# Java Course Duration Estimate

Документ актуализирован 9 июня 2026 года и фиксирует ориентировочную длительность полного пути студента с нуля:

```text
Java Base
↓
Java Core
↓
Java Pre-project
↓
Java Advanced
```

Оценка рассчитана для студента, который учится стабильно: 4-5 дней в неделю по 2-3 часа, выполняет обязательную практику, проходит ревью и доводит проекты до рабочего состояния. Это примерно 8-15 учебных часов в неделю.

Основание оценки:

- готовый Java Base содержит 13 модулей, 33 теоретических материала и 85 практических задач;
- целевая программа Java Core содержит 11 модулей и 72 содержательные лекции, модульную практику, 9 мини-проектов и итоговый проект;
- в Core пока полностью наполнен только модуль `1.0`: 4 лекции, устное ревью и 12 задач; оценка Core относится к полной целевой программе, а не к текущему объему готовых файлов;
- Pre-project и Advanced оцениваются по соответствующим curriculum proposal и требуют существенной проектной практики.

Диапазоны ниже учитывают не только чтение материалов, но и решение задач, исправление замечаний, повтор сложных тем и проектные доработки.

## Сводная оценка

| Курс | Реалистичное время | Интенсивно | Комфортно |
|---|---:|---:|---:|
| Java Base | 8-10 недель | 6-7 недель | 12-14 недель |
| Java Core | 22-28 недель | 16-20 недель | 30-38 недель |
| Java Pre-project | 12-16 недель | 9-12 недель | 18-22 недели |
| Java Advanced | 16-22 недели | 12-16 недель | 24-32 недели |
| Весь путь до завершения Pre-project | 42-54 недели, примерно 10-13 месяцев | 31-39 недель, примерно 7-9 месяцев | 60-74 недели, примерно 14-17 месяцев |
| Весь путь с Advanced | 58-76 недель, примерно 14-18 месяцев | 43-55 недель, примерно 10-13 месяцев | 84-106 недель, примерно 19-24 месяца |

Базовая плановая оценка:

```text
Base: 9 недель
Core: 25 недель
Pre-project: 14 недель
Advanced: 19 недель
Итого до завершения Pre-project: 48 недель, примерно 11 месяцев
Итого с Advanced: 67 недель, примерно 15-16 месяцев
```

С учетом каникул, болезней, пересдач и пауз путь до уверенного завершения Pre-project разумно планировать как 12-15 календарных месяцев. Полный путь с дополнительным Advanced — как 16-20 календарных месяцев.

## Java Base

Ориентир: **8-10 недель**.

Интенсивно: **6-7 недель**.

Комфортно: **12-14 недель**.

Base отвечает за вход в программирование и базовый Java-синтаксис:

- `main`, вывод, переменные;
- базовые типы, изученные в Base: `int`, `double`, `boolean`, `char`, `String`;
- простые преобразования `int` -> `double` и `(int) double`;
- методы;
- условия;
- циклы, включая enhanced `for`;
- массивы;
- строки;
- базовый ввод;
- базовые исключения;
- простые `ArrayList` и `HashMap`;
- простые классы и объекты;
- базовый Debugger и stack trace.

При 33 теоретических материалах и 85 задачах прежняя оценка 6-8 недель возможна только при высокой недельной нагрузке и небольшом числе доработок. Для студента с нуля основной риск растягивания — методы, циклы, массивы, строки и декомпозиция.

## Java Core

Ориентир: **22-28 недель**.

Интенсивно: **16-20 недель**.

Комфортно: **30-38 недель**.

Core длиннее Base, потому что это переход от синтаксиса к нормальному Java-приложению без фреймворков.

Целевая программа Core включает 72 лекции. Почти каждый содержательный модуль заканчивается практикой и мини-проектом, затем выполняется итоговый проект. При реалистичном темпе это в среднем 2-2.5 недели на обычный модуль, 3-4 недели на SQL/JDBC и 3-4 недели на итоговую сборку.

В Core входят:

- полная система примитивных типов;
- packages и структура проекта;
- инкапсуляция, конструкторы, объектные инварианты;
- `toString`, `equals/hashCode`, `enum`;
- наследование, полиморфизм, интерфейсы;
- Maven/Gradle basics;
- JUnit и debugging тестов;
- Collections Framework и generics;
- exceptions глубже;
- Files API и текстовые форматы;
- базовая архитектура `model/service/repository/app`;
- SQL, JDBC и Repository pattern;
- lambda и Stream API;
- базовая многопоточность;
- итоговый Core-проект.

Главный риск Core: это самый большой концептуальный скачок. Студент должен не просто узнать конструкции, а начать собирать приложение слоями, писать тесты, работать с БД и понимать границы ответственности классов. Попытка пройти полную программу за 10-12 недель оставляет около одной недели на модуль вместе с проектами и поэтому подходит только как режим повторения, а не обучения с уровня Base.

## Java Pre-project

Ориентир: **12-16 недель**.

Интенсивно: **9-12 недель**.

Комфортно: **18-22 недели**.

Pre-project отвечает за переход от обычного Java-приложения без фреймворков к Spring web/API-приложению.

В Pre-project входят:

- проектный Git workflow;
- JDBC CRUD и Repository;
- Hibernate/JPA;
- Spring MVC + CRUD-приложение;
- Spring Boot;
- Spring Security как доработка CRUD-приложения;
- REST CRUD Spring Boot приложение;
- элементы JavaScript для асинхронных запросов;
- Bootstrap для оформления личного кабинета;
- базовое тестирование web/service слоев;
- итоговая pre-project сборка.

Главный риск Pre-project: остаться набором CRUD-задач без явной архитектуры, Git workflow, тестирования и связи между MVC, REST и frontend.

## Java Advanced

Ориентир: **16-22 недели**.

Интенсивно: **12-16 недель**.

Комфортно: **24-32 недели**.

Advanced является дополнительным курсом после Pre-project и отвечает за инфраструктуру, микросервисы и distributed backend.

В Advanced входят:

- Docker и Docker Compose;
- микросервисы vs монолит;
- Spring Cloud;
- Eureka Discovery Server;
- Config Server;
- Feign Client, RestTemplate, WebClient;
- MongoDB image basics;
- API Gateway;
- LoadBalancer и Circuit Breaker;
- Zipkin/Sleuth;
- ELK;
- Spring Cloud Stream;
- Kafka/RabbitMQ;
- Spring Cloud Bus;
- Reactor/WebFlux;
- Kubernetes и Spring Cloud Kubernetes.

Главный риск Advanced: перегруз распределенной инфраструктурой. Его нельзя начинать, пока студент не собрал монолитное Spring Boot web/API-приложение и не понимает Spring на уровне Pre-project.

## Итоговая рекомендация

Минимально возможный темп:

```text
Base: 1.5-1.75 месяца
Core: 4-5 месяцев
Pre-project: 2-3 месяца
Advanced: 3-4 месяца
Итого до завершения Pre-project: 7-9 месяцев
Итого с Advanced: 10-13 месяцев
```

Реалистичный учебный темп:

```text
Base: 2-2.5 месяца
Core: 5.5-7 месяцев
Pre-project: 3-4 месяца
Advanced: 4-5.5 месяца
Итого до завершения Pre-project: 10-13 месяцев чистого обучения
Итого с Advanced: 14-18 месяцев чистого обучения
```

Комфортный темп для студента с нуля:

```text
Base: 3-3.5 месяца
Core: 7.5-9.5 месяца
Pre-project: 4.5-5.5 месяца
Advanced: 6-8 месяцев
Итого до завершения Pre-project: 14-17 месяцев
Итого с Advanced: 19-24 месяца
```

Для коммерчески полезного результата лучше планировать **12-15 календарных месяцев** на путь от нуля до уверенного завершения Pre-project с учетом пауз и доработок. Advanced лучше считать отдельным усилением еще на **4-6 месяцев** при стабильном темпе.

Сокращать срок следует уменьшением необязательного объема и числа параллельных технологий, а не пропуском практики, тестов и итоговых проектов. Именно эти части превращают знакомство с темой в рабочий навык.
