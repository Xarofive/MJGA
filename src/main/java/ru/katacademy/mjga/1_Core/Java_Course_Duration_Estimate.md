# Java Course Duration Estimate

Документ фиксирует ориентировочную длительность полного пути студента с нуля:

```text
Java Base
↓
Java Core
↓
Java Pre-project
↓
Java Advanced
```

Оценка рассчитана для студента, который учится стабильно: 4-5 дней в неделю, по 2-3 часа в день, выполняет практику, проходит ревью и доводит мини-проекты до рабочего состояния.

## Сводная оценка

| Курс | Реалистичное время | Интенсивно | Комфортно |
|---|---:|---:|---:|
| Java Base | 6-8 недель | 4-5 недель | 8-10 недель |
| Java Core | 14-18 недель | 10-12 недель | 20-24 недели |
| Java Pre-project | 10-14 недель | 8-10 недель | 14-18 недель |
| Java Advanced | 12-18 недель | 10-12 недель | 18-24 недели |
| Весь путь до завершения Pre-project | 7-9 месяцев | 5-6 месяцев | 9-12 месяцев |
| Весь путь с Advanced | 10-14 месяцев | 8-9 месяцев | 14-18 месяцев |

Базовая плановая оценка:

```text
Base: 8 недель
Core: 16 недель
Pre-project: 12 недель
Advanced: 16 недель
Итого до завершения Pre-project: 36 недель, примерно 8-9 месяцев
Итого с Advanced: 52 недели, примерно 12-13 месяцев
```

С учетом каникул, болезней, пересдач, ревью, слабых мест и проектных доработок путь до уверенного завершения Pre-project реалистично планировать как 9-12 месяцев. Полный путь с дополнительным Advanced реалистично планировать как 12-15 месяцев.

## Java Base

Ориентир: **6-8 недель**.

Интенсивно: **4-5 недель**.

Комфортно: **8-10 недель**.

Base отвечает за вход в программирование и базовый Java-синтаксис:

- `main`, вывод, переменные;
- базовые типы, изученные в Base: `int`, `double`, `boolean`, `char`, `String`;
- методы;
- условия;
- циклы;
- массивы;
- строки;
- базовый ввод;
- базовые исключения;
- простые `ArrayList` и `HashMap`;
- простые классы и объекты;
- базовый Debugger и stack trace.

Риск растягивания Base: студент с нуля может задержаться на циклах, методах, массивах, строках и декомпозиции задач.

## Java Core

Ориентир: **14-18 недель**.

Интенсивно: **10-12 недель**.

Комфортно: **20-24 недели**.

Core длиннее Base, потому что это переход от синтаксиса к нормальному Java-приложению без фреймворков.

В Core входят:

- полная система примитивных типов;
- packages и структура проекта;
- инкапсуляция, конструкторы, объектные инварианты;
- `toString`, `equals/hashCode`, `enum`;
- наследование, полиморфизм, интерфейсы;
- Maven/Gradle basics;
- JUnit и debugging тестов;
- logging basics;
- Collections Framework и generics;
- exceptions глубже;
- Files API и конфигурация;
- базовая архитектура `model/service/repository/app`;
- SQL, PostgreSQL, JDBC и Repository pattern;
- HTTP basics и Java `HttpClient`;
- lambda и Stream API;
- базовая многопоточность;
- итоговый Core-проект.

Главный риск Core: это самый большой концептуальный скачок. Студент должен не просто узнать конструкции, а начать собирать приложение слоями, писать тесты, работать с БД, логировать ошибки и понимать границы ответственности классов.

## Java Pre-project

Ориентир: **10-14 недель**.

Интенсивно: **8-10 недель**.

Комфортно: **14-18 недель**.

Pre-project отвечает за переход от обычного Java-приложения без фреймворков к Spring web/API-приложению.

В Pre-project входят:

- проектный Git workflow;
- JDBC CRUD и Repository;
- Hibernate/JPA;
- Spring MVC + CRUD-приложение;
- Spring Boot;
- Spring Security как доработка CRUD-приложения;
- REST CRUD Spring Boot приложение;
- элементы JavaScript для асинхронных запросов;
- Bootstrap для оформления личного кабинета;
- базовое тестирование web/service слоев;
- итоговая pre-project сборка.

Главный риск Pre-project: остаться набором CRUD-задач без явной архитектуры, Git workflow, тестирования и связи между MVC, REST и frontend.

## Java Advanced

Ориентир: **12-18 недель**.

Интенсивно: **10-12 недель**.

Комфортно: **18-24 недели**.

Advanced является дополнительным курсом после Pre-project и отвечает за инфраструктуру, микросервисы и distributed backend.

В Advanced входят:

- Docker и Docker Compose;
- микросервисы vs монолит;
- Spring Cloud;
- Eureka Discovery Server;
- Config Server;
- Feign Client, RestTemplate, WebClient;
- MongoDB image basics;
- API Gateway;
- LoadBalancer и Circuit Breaker;
- Zipkin/Sleuth;
- ELK;
- Spring Cloud Stream;
- Kafka/RabbitMQ;
- Spring Cloud Bus;
- Reactor/WebFlux;
- Kubernetes и Spring Cloud Kubernetes.

Главный риск Advanced: перегруз распределенной инфраструктурой. Его нельзя начинать, пока студент не собрал монолитное Spring Boot web/API-приложение и не понимает Spring на уровне Pre-project.

## Итоговая рекомендация

Минимально возможный темп:

```text
Base: 1-1.5 месяца
Core: 2.5-3 месяца
Pre-project: 2-2.5 месяца
Advanced: 2.5-3 месяца
Итого до завершения Pre-project: 5-6 месяцев
Итого с Advanced: 8-9 месяцев
```

Реалистичный учебный темп:

```text
Base: 2 месяца
Core: 4 месяца
Pre-project: 3 месяца
Advanced: 4 месяца
Итого до завершения Pre-project: 8-9 месяцев
Итого с Advanced: 12-13 месяцев
```

Комфортный темп для студента с нуля:

```text
Base: 2-2.5 месяца
Core: 5-6 месяцев
Pre-project: 3.5-4.5 месяца
Advanced: 4.5-6 месяцев
Итого до завершения Pre-project: 11-13 месяцев
Итого с Advanced: 15-18 месяцев
```

Для коммерчески полезного результата лучше планировать не меньше **9-12 месяцев** на путь от нуля до уровня, где студент способен уверенно пройти Pre-project и выйти на junior-level backend-практику. Advanced лучше считать отдельным усилением после этого этапа.
