# Java Course Duration Estimate

Документ фиксирует ориентировочную длительность полного пути студента с нуля:

```text
Java Base
↓
Java Core
↓
Java Advanced
```

Оценка рассчитана для студента, который учится стабильно: 4-5 дней в неделю, по 2-3 часа в день, выполняет практику, проходит ревью и доводит мини-проекты до рабочего состояния.

## Сводная оценка

| Курс | Реалистичное время | Интенсивно | Комфортно |
|---|---:|---:|---:|
| Java Base | 6-8 недель | 4-5 недель | 8-10 недель |
| Java Core | 14-18 недель | 10-12 недель | 20-24 недели |
| Java Advanced | 16-22 недели | 12-14 недель | 24-30 недель |
| Весь путь | 9-12 месяцев | 6-7 месяцев | 12-15 месяцев |

Базовая плановая оценка:

```text
Base: 8 недель
Core: 16 недель
Advanced: 20 недель
Итого: 44 недели, примерно 10-11 месяцев
```

С учетом каникул, болезней, пересдач, ревью, слабых мест и проектных доработок полный путь реалистично планировать как 12 месяцев.

## Java Base

Ориентир: **6-8 недель**.

Интенсивно: **4-5 недель**.

Комфортно: **8-10 недель**.

Base отвечает за вход в программирование и базовый Java-синтаксис:

- `main`, вывод, переменные;
- базовые типы, изученные в Base: `int`, `double`, `boolean`, `char`, `String`;
- методы;
- условия;
- циклы;
- массивы;
- строки;
- базовый ввод;
- базовые исключения;
- простые `ArrayList` и `HashMap`;
- простые классы и объекты;
- базовый Debugger и stack trace.

Риск растягивания Base: студент с нуля может задержаться на циклах, методах, массивах, строках и декомпозиции задач.

## Java Core

Ориентир: **14-18 недель**.

Интенсивно: **10-12 недель**.

Комфортно: **20-24 недели**.

Core длиннее Base, потому что это переход от синтаксиса к нормальному Java-приложению без фреймворков.

В Core входят:

- полная система примитивных типов;
- Git basics;
- packages и структура проекта;
- инкапсуляция, конструкторы, объектные инварианты;
- `toString`, `equals/hashCode`, `enum`;
- наследование, полиморфизм, интерфейсы;
- Maven/Gradle basics;
- JUnit и debugging тестов;
- logging basics;
- Collections Framework и generics;
- exceptions глубже;
- Files API и конфигурация;
- базовая архитектура `model/service/repository/app`;
- SQL, PostgreSQL, JDBC и Repository pattern;
- HTTP basics и Java `HttpClient`;
- lambda и Stream API;
- базовая многопоточность;
- итоговый Core-проект.

Главный риск Core: это самый большой концептуальный скачок. Студент должен не просто узнать конструкции, а начать собирать приложение слоями, писать тесты, работать с БД, логировать ошибки и понимать границы ответственности классов.

## Java Advanced

Ориентир: **16-22 недели**.

Интенсивно: **12-14 недель**.

Комфортно: **24-30 недель**.

Advanced отвечает за переход от обычного Java-приложения к backend-сервису.

В Advanced входят:

- Spring Core и Dependency Injection;
- Spring Boot;
- REST API, JSON, validation;
- Spring Data JPA / Hibernate;
- миграции БД: Flyway/Liquibase;
- security basics: bcrypt, JWT, роли;
- integration testing: Testcontainers, MockMvc;
- observability: structured logs, metrics, Prometheus, health checks;
- resilience: timeout, retry, backoff, circuit breaker;
- Docker и Docker Compose;
- Redis/cache basics;
- advanced concurrency: `CompletableFuture`, locks, atomics, Java Memory Model overview;
- CI/CD basics;
- предпроектная работа / backend service.

Главный риск Advanced: перегруз инфраструктурой. Его нельзя начинать, пока студент не умеет писать, тестировать и запускать обычное Java-приложение без Spring.

## Итоговая рекомендация

Минимально возможный темп:

```text
Base: 1-1.5 месяца
Core: 2.5-3 месяца
Advanced: 3-3.5 месяца
Итого: 6-7 месяцев
```

Реалистичный учебный темп:

```text
Base: 2 месяца
Core: 4 месяца
Advanced: 5 месяцев
Итого: 10-11 месяцев
```

Комфортный темп для студента с нуля:

```text
Base: 2-2.5 месяца
Core: 5-6 месяцев
Advanced: 6-7 месяцев
Итого: 12-15 месяцев
```

Для коммерчески полезного результата лучше планировать не меньше **10-12 месяцев** на полный путь от нуля до уровня, где студент способен уверенно войти в pre-project или junior-level backend-практику.
