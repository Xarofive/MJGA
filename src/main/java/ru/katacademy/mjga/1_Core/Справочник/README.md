# Справочник Java Core

## Назначение

Справочник содержит короткие карты и таблицы для быстрого поиска и повторения.
Это не отдельный учебный модуль и не обязательный маршрут курса.

Новые темы сначала объясняются в лекциях на примерах и закрепляются практикой.
После этого справочник помогает:

- быстро восстановить определение;
- сравнить похожие конструкции;
- проверить, к чему применяется ключевое слово;
- подготовиться к ревью;
- найти лекцию с подробным объяснением.

Не нужно читать все материалы справочника заранее или заучивать таблицы
целиком.

## Доступные материалы

### [Ключевые слова Java](Ключевые_слова_Java.md)

Содержит:

- карту основных ключевых слов по назначению;
- области применения ключевых слов;
- ссылки на модули с подробным объяснением;
- расширенный разбор `static` и `final`;
- примеры литералов и отличие литералов от ключевых слов.

## План развития

Новые страницы добавляются после появления соответствующей темы в основном
курсе.

Планируемые материалы:

- типы Java и wrapper-классы;
- модификаторы доступа;
- объектные контракты `equals`, `hashCode`, `toString`;
- выбор коллекции;
- generics;
- исключения;
- Files API;
- Stream API;
- Maven и Gradle;
- JUnit 5;
- SQL и JDBC;
- базовая многопоточность.

Пустые страницы заранее не создаются: справочник должен отражать уже
объясненный материал, а не опережать учебный маршрут.

## Правило для лекций

При первом появлении термин:

1. Объясняется в самой лекции.
2. Показывается на небольшом примере.
3. Повторяется в словаре лекции.
4. При необходимости получает ссылку на соответствующую страницу справочника.

Переход в справочник не должен быть обязательным для понимания лекции.
