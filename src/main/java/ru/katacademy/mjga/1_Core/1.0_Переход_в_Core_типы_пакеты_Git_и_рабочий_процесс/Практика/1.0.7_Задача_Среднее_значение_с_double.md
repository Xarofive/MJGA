# Задача 1.0.7: Среднее значение с `double`

## Цель задачи

Закрепить целочисленное деление и явное преобразование к `double`.

## Сценарий

Есть сумма баллов и количество элементов. Нужно вернуть среднее значение.

## Какие навыки пригодятся

- Деление целых чисел.
- Явное преобразование к `double`.
- Проверка деления на ноль.
- Возврат дробного результата из метода.

## Техническое задание

Напишите метод:

```java
static double average(int sum, int count)
```

Метод должен:

- вернуть среднее значение как `double`;
- вернуть `0.0`, если `count` равен `0`.

## Примеры

```java
average(5, 2)  // 2.5
average(10, 4) // 2.5
average(7, 0)  // 0.0
```

## Ограничения

- Не используйте `System.out.println`.
- Cast должен стоять до деления, а не после уже выполненного `int / int`.

## Где это пригодится

Сегодня: вы учитесь получать дробный результат там, где обычное `int / int` его потеряет.

В задачах курса: среднее значение часто используется для оценок, рейтингов и статистики.

В проектах: такие расчеты нужны в отчетах, аналитике и проверке качества данных.

## Критерии приемки

Решение считается верным, если метод возвращает дробный результат там, где он нужен.
