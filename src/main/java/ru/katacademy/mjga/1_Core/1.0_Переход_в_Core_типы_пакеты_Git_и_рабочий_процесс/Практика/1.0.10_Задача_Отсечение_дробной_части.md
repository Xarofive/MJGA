# Задача 1.0.10: Отсечение дробной части

## Цель задачи

Закрепить преобразование `double` в `int` и отличие отсечения от округления.

## Сценарий

Есть дробное значение. Нужно получить целую часть без округления.

## Какие навыки пригодятся

- Тип `double`.
- Явное преобразование к `int`.
- Отличие отсечения дробной части от округления.
- Проверка поведения на отрицательных числах.

## Техническое задание

Напишите метод:

```java
static int truncate(double value)
```

Метод должен вернуть целую часть числа через явное преобразование к `int`.

## Примеры

```java
truncate(99.99)  // 99
truncate(12.01)  // 12
truncate(-3.9)   // -3
```

## Ограничения

- Не используйте `Math.round`, `Math.floor`, `Math.ceil`.
- Не печатайте результат.

## Где это пригодится

Сегодня: вы видите, что cast из `double` в `int` не округляет число.

В задачах курса: это поможет понимать результаты преобразований и не путать их с математическим округлением.

В проектах: отсечение иногда используют для индексов, целой части измерений или грубого ограничения значения.

## Критерии приемки

Решение считается верным, если метод именно отсекает дробную часть.
