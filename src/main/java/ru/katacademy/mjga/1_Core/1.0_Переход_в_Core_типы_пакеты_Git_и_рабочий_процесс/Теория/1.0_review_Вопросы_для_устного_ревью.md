# Вопросы для устного ревью

## Как проходит ревью

Ревью по модулю сдается устно.

Цель ревью - проверить не заучивание формулировок, а понимание. Отвечайте своими словами, приводите короткие примеры кода и объясняйте, почему выбираете именно такое решение.

На ревью могут попросить:

- объяснить термин простыми словами;
- найти ошибку в небольшом фрагменте кода;
- выбрать подходящий тип данных;
- предложить улучшение кода;
- объяснить, как разложить классы по пакетам.

## Мини-задачи на ревью

### Задача 1. Найти проблему в `main`

```java
public static void main(String[] args) {
    int price = 1000;
    int discount = 15;
    int result = price - price * discount / 100;
    System.out.println(result);
}
```

Вопросы:

- Что можно улучшить в этом коде?
- Какой метод можно выделить?
- Почему это будет рефакторинг?
