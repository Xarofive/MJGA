# Задача 1.0.2: Подсчет слов длиннее порога

## Уровень сложности

**2 из 5 — базовый.** Нужно совместить цикл, условие, работу со строками и защиту от `null`.

## Цель задачи

Повторить массивы, строки, условия и методы, а также явно обработать неудобные входные данные.

## Сценарий

Есть массив слов. Нужно посчитать, сколько строк длиннее заданного минимального значения.

## Какие навыки пригодятся

- Работа с массивом через индексный `for`.
- Проверка строки на `null` перед вызовом методов.
- Метод `length()` у строки.
- Условие `if` и счетчик.

## Техническое задание

Напишите метод:

```java
static int countWordsLongerThan(String[] words, int minLength)
```

Метод должен:

- вернуть количество строк, длина которых больше `minLength`;
- вернуть `0`, если массив пустой;
- игнорировать элементы `null` внутри массива.

## Примеры

```java
countWordsLongerThan(new String[]{"Java", "Core", "if"}, 3) // 2
countWordsLongerThan(new String[]{"", null, "tool"}, 2)     // 1
countWordsLongerThan(new String[]{}, 5)                     // 0
```

## Ограничения

- Используйте обычный цикл `for` с индексом.
- Не используйте `for-each`.
- Не используйте `System.out.println`.

## Где это пригодится

Сегодня: вы повторяете строки, массивы и защиту от ошибки при `null`.

В задачах курса: похожий подход понадобится при фильтрации массивов и списков.

В проектах: так проверяют и считают подходящие значения, например длинные названия, валидные строки или заполненные поля.

## Критерии приемки

Решение считается верным, если метод корректно считает строки и не падает на `null`.
