# Задача 1.0.6: Подсчет активных флагов

## Уровень сложности

**1 из 5 — начальный.** Это прямой подсчет подходящих элементов одним циклом.

## Цель задачи

Закрепить массив `boolean[]`, цикл с индексом и подсчет значений `true`.

## Сценарий

Есть массив флагов. `true` означает активное состояние, `false` - неактивное.

## Какие навыки пригодятся

- Массив `boolean[]`.
- Индексный цикл `for`.
- Проверка значения `true`.
- Счетчик подходящих элементов.

## Техническое задание

Напишите метод:

```java
static int countActive(boolean[] flags)
```

Метод должен вернуть количество элементов со значением `true`.

## Примеры

```java
countActive(new boolean[]{true, false, true}) // 2
countActive(new boolean[]{false, false})      // 0
countActive(new boolean[]{})                  // 0
```

## Ограничения

- Используйте обычный цикл `for` с индексом.
- Не используйте `for-each`.
- Не печатайте результат.

## Где это пригодится

Сегодня: вы закрепляете работу с массивом логических значений.

В задачах курса: такой подсчет пригодится при анализе массивов и коллекций.

В проектах: флаги часто используются для включенных настроек, активных пользователей, выбранных пунктов и доступных действий.

## Критерии приемки

Решение считается верным, если метод корректно считает `true` в массиве любой длины.
