# Задача 1.0.5: Следующая латинская буква

## Уровень сложности

**2 из 5 — базовый.** Помимо условий нужно учесть границы диапазона и явное преобразование арифметического результата обратно в `char`.

## Цель задачи

Закрепить, что `char` связан с числовым кодом символа и над ним возможна арифметика.

## Сценарий

Нужно получить следующую заглавную латинскую букву.

## Какие навыки пригодятся

- Тип `char`.
- Сравнение символов.
- Арифметика с `char`.
- Явное преобразование результата обратно к `char`.

## Техническое задание

Напишите метод:

```java
static char nextLatinLetter(char letter)
```

Метод должен:

- вернуть следующую букву для `'A'` ... `'Y'`;
- вернуть `'A'`, если пришла `'Z'`;
- вернуть исходный символ, если это не заглавная латинская буква.

## Примеры

```java
nextLatinLetter('A') // 'B'
nextLatinLetter('Y') // 'Z'
nextLatinLetter('Z') // 'A'
nextLatinLetter('7') // '7'
```

## Ограничения

- Не используйте строки.
- Не печатайте результат.

## Где это пригодится

Сегодня: вы видите, что за символом в Java стоит числовой код.

В задачах курса: это пригодится при простых проверках символов, кодов и диапазонов.

В проектах: похожие проверки встречаются в парсерах, валидаторах и обработке коротких кодов.

## Критерии приемки

Решение считается верным, если метод корректно работает с буквами и не меняет остальные символы.
