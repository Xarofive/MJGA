# Аудит покрытия лекций практическими задачами

## Область проверки

Аудит относится к фактически подготовленному модулю `1.0`: четырем лекциям и задачам `1.0.1`-`1.0.12`.

Шкала сложности задач:

- **1 из 5 — начальный:** одно действие по явно заданному образцу;
- **2 из 5 — базовый:** сочетание нескольких знакомых конструкций или обработка граничного случая;
- **3 из 5 — средний:** несколько правил, методов или классов, где важен выбор типа и структуры;
- **4 из 5 — повышенный:** самостоятельная декомпозиция и несколько взаимодействующих частей;
- **5 из 5 — комплексный:** мини-проект с несколькими темами модуля и рабочим процессом.

В текущем наборе нет задач уровней 4 и 5. Это нормально для отдельных упражнений, но модулю не хватает комплексной работы, которая соединяет темы.

## Итог

Нет, задачи не покрывают весь материал лекций.

Сильнее всего покрыты примитивные типы, numeric promotion, cast, целочисленное деление и переполнение. Частично покрыты повтор Base, рефакторинг, граничные случаи и разделение ответственности. Практически не покрыты пакеты и импорты.

## Что добавить для полного покрытия

1. **Диагностика Base, 2/5:** исправить готовый метод с массивом, `null`, неверной границей цикла и плохими именами; описать найденные крайние случаи.
2. **Коллекции и Debugger, 3/5:** найти ошибку в обработке `ArrayList`/`HashMap`, пройти заданные breakpoint и зафиксировать значения переменных.
3. **Диапазоны и точность типов, 2/5:** подобрать типы, проверить `MIN_VALUE`/`MAX_VALUE`, воспроизвести неточность `0.1 + 0.2` и объяснить, почему деньги хранятся в целых единицах.
4. **Преобразования, 2/5:** отдельно показать widening conversion и потерю данных при `long -> int` и `double -> int`.
5. **Пакеты и импорты, 4/5:** разнести `Main`, `Product`, `DeliveryCalculator` и `MoneyFormatter` по `app`, `model`, `service`, `util`; использовать явные импорты и исправить один неверный `package`.
6. **Итоговая задача модуля, 5/5:** собрать маленький расчет доставки с пакетами, несколькими классами, корректными типами и README с инструкцией запуска.
