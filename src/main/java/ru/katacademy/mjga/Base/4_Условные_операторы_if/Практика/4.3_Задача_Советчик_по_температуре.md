<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">Задача 4.3: Советчик по температуре</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться классифицировать число по диапазонам и аккуратно работать с граничными значениями.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">🎯 Сценарий</h3>
<p style="margin-bottom: 10px;">Приложение погоды должно превратить температуру в понятную категорию: морозно, прохладно, тепло или жарко.</p>
<p style="margin: 0;"><strong>Ваша задача:</strong> реализовать метод <code>temperatureAdvice</code>, который возвращает категорию по температуре.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Диапазоны без дырок</h4>
<p>Если проверки идут от меньших значений к большим, можно проверять верхние границы.</p>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>if (temp &lt; 0) {
    return "freezing";
} else if (temp &lt;= 17) {
    return "chilly";
}</code></pre>
<p style="margin-bottom: 0;">Важно, чтобы каждая температура попадала ровно в один диапазон.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<p><strong>Создайте метод с точной сигнатурой:</strong></p>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 16px; border-radius: 8px; overflow-x: auto;"><code>static String temperatureAdvice(int temp)</code></pre>
<ul>
	<li><code>"freezing"</code> — температура ниже 0°C;</li>
	<li><code>"chilly"</code> — от 0°C до 17°C включительно;</li>
	<li><code>"warm"</code> — от 18°C до 27°C включительно;</li>
	<li><code>"hot"</code> — выше 27°C.</li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Результат программы</h2>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 16px; border-radius: 8px; overflow-x: auto;"><code>temperatureAdvice(-10) -> "freezing"
temperatureAdvice(0)   -> "chilly"
temperatureAdvice(25)  -> "warm"
temperatureAdvice(18)  -> "warm"
temperatureAdvice(40)  -> "hot"</code></pre>

<div style="background: #fff3e0; border: 1px solid #ff9800; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #e65100; margin-top: 0;">🤔 Момент для размышления</h4>
<p style="margin-bottom: 0;">Почему <code>0</code> относится к <code>"chilly"</code>, а <code>18</code> уже к <code>"warm"</code>? Какие ошибки чаще всего возникают на границах?</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📝 Структура метода</h2>
<pre style="background: #263238; color: #aed581; padding: 20px; border-radius: 8px; overflow-x: auto;"><code>static String temperatureAdvice(int temp) {
    // freezing
    // chilly
    // warm
    // hot
}</code></pre>

<h2 style="color: #34495e; margin-top: 28px;">✅ Чек-лист проверки</h2>
<div style="background: #f8f9fa; border-left: 4px solid #28a745; padding: 16px; margin: 16px 0;">
<ul style="margin-bottom: 0;">
	<li>☐ Все четыре диапазона описаны</li>
	<li>☐ <code>0</code> возвращает <code>"chilly"</code></li>
	<li>☐ <code>17</code> возвращает <code>"chilly"</code></li>
	<li>☐ <code>18</code> и <code>27</code> возвращают <code>"warm"</code></li>
	<li>☐ Метод ничего не печатает</li>
</ul>
</div>

<div style="background: #fff8e1; border-left: 4px solid #ffc107; padding: 16px; margin: 16px 0;">
<h4 style="color: #f57c00; margin-top: 0;">⚠️ Технические ограничения</h4>
<ul style="margin-bottom: 0;">
	<li>Не используйте циклы и <code>switch</code></li>
	<li>Не используйте <code>System.out.println</code></li>
	<li>Не оставляйте промежутков между диапазонами</li>
	<li>Откажитесь от проверки только части условий: автопроверка проверит границы</li>
</ul>
</div>
