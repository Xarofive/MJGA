<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🔐 Валидатор паролей</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться использовать длину строки в условных операторах и выполнять проверки в заданном порядке.</p>
</div>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #e65100; margin-top: 0;">🔧 Метод passwordStatus</h4>
<p><strong>Сигнатура:</strong></p>
<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>static String passwordStatus(String password)</code></div>
<p><strong>Правила валидации:</strong></p>
<ol style="margin: 10px 0;">
	<li><code>"too short"</code> &mdash; пароль короче 8 символов;</li>
	<li><code>"too long"</code> &mdash; пароль длиннее 16 символов;</li>
	<li><code>"ok"</code> &mdash; пароль прошёл все проверки.</li>
</ol>
<p><strong>Требования:</strong> вычислите длину через <code>password.length()</code>, используйте <code>if</code>, не используйте циклы и <code>switch</code>.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Примеры работы</h2>
<div style="background: #e3f2fd; border: 1px solid #64b5f6; border-radius: 10px; padding: 16px; margin: 20px 0;">
<pre style="margin: 0;"><code>passwordStatus("short")             → "too short"
passwordStatus("12345678")          → "ok"
passwordStatus("SeventeenCharacters") → "too long"
passwordStatus("SixteenCharsHere")  → "ok"
passwordStatus("")                  → "too short"</code></pre>
</div>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #c62828; margin-top: 0;">⚠️ Частые ошибки</h4>
<p><strong>Границы:</strong> длина 8 и 16 допустимы.</p>
<p><strong>Неверная проверка строки:</strong> длина строки в Java получается через <code>password.length()</code>, а не через поле <code>length</code>.</p>
<p style="margin-bottom: 0;"><strong>Лишний вывод:</strong> метод должен вернуть строку.</p>
</div>
