<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">Задача 4.7: Валидатор паролей</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться использовать числовые границы в условных операторах и выполнять проверки в заданном порядке.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">🎯 Сценарий</h3>
<p style="margin-bottom: 10px;">Сервис регистрации проверяет длину пароля до сохранения аккаунта.</p>
<p style="margin: 0;"><strong>Ваша задача:</strong> вернуть статус пароля по его длине.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Минимум и максимум</h4>
<p>Сначала удобно проверить значения вне допустимого диапазона, а все остальные считать корректными.</p>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>if (length &lt; 8) {
    return "too short";
}
if (length &gt; 16) {
    return "too long";
}
return "ok";</code></pre>
<p style="margin-bottom: 0;">Границы <code>8</code> и <code>16</code> допустимы.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<p><strong>Создайте метод с точной сигнатурой:</strong></p>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 16px; border-radius: 8px; overflow-x: auto;"><code>static String passwordStatus(int length)</code></pre>
<ol>
	<li><code>"too short"</code> — пароль короче 8 символов;</li>
	<li><code>"too long"</code> — пароль длиннее 16 символов;</li>
	<li><code>"ok"</code> — пароль прошёл все проверки.</li>
</ol>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Результат программы</h2>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 16px; border-radius: 8px; overflow-x: auto;"><code>passwordStatus(5)  -> "too short"
passwordStatus(8)  -> "ok"
passwordStatus(17) -> "too long"
passwordStatus(16) -> "ok"
passwordStatus(0)  -> "too short"</code></pre>

<div style="background: #fff3e0; border: 1px solid #ff9800; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #e65100; margin-top: 0;">🤔 Момент для размышления</h4>
<p style="margin-bottom: 0;">Почему проверка <code>length == 8</code> не нужна отдельно, если границы уже заданы через <code>&lt; 8</code> и <code>&gt; 16</code>?</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📝 Структура метода</h2>
<pre style="background: #263238; color: #aed581; padding: 20px; border-radius: 8px; overflow-x: auto;"><code>static String passwordStatus(int length) {
    // too short
    // too long
    // ok
}</code></pre>
