<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">Задача 4.10: Система оценивания результатов</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться использовать ранний <code>return</code>, чтобы избежать глубокой вложенности условий.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">🎯 Сценарий</h3>
<p style="margin-bottom: 10px;">Образовательная платформа переводит числовой результат теста в понятную метку: не сдал, сдал или отлично.</p>
<p style="margin: 0;"><strong>Ваша задача:</strong> вернуть метку результата по количеству баллов.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Ранний return</h4>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>static String checkValue(int x) {
    if (x &lt; 0) {
        return "invalid";
    }
    return "valid";
}</code></pre>
<p style="margin-bottom: 0;">После <code>return</code> метод завершается. Поэтому можно проверять особые случаи сверху и не вкладывать все условия друг в друга.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<p><strong>Создайте метод с точной сигнатурой:</strong></p>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 16px; border-radius: 8px; overflow-x: auto;"><code>static String scoreLabel(int score)</code></pre>
<ul>
	<li><code>"invalid"</code> — баллы меньше 0 или больше 100;</li>
	<li><code>"failed"</code> — от 0 до 69 включительно;</li>
	<li><code>"passed"</code> — от 70 до 89 включительно;</li>
	<li><code>"excellent"</code> — от 90 до 100 включительно.</li>
</ul>
<p><strong>Дополнительно:</strong> используйте минимум два оператора <code>return</code>, сначала проверяйте невалидные значения, избегайте глубокой вложенности <code>if-else</code>.</p>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Результат программы</h2>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 16px; border-radius: 8px; overflow-x: auto;"><code>scoreLabel(95)  -> "excellent"
scoreLabel(70)  -> "passed"
scoreLabel(69)  -> "failed"
scoreLabel(101) -> "invalid"
scoreLabel(-1)  -> "invalid"
scoreLabel(100) -> "excellent"</code></pre>

<div style="background: #fff3e0; border: 1px solid #ff9800; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #e65100; margin-top: 0;">🤔 Момент для размышления</h4>
<p style="margin-bottom: 0;">Почему проверку <code>score &lt; 0 || score &gt; 100</code> стоит поставить первой?</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📝 Структура метода</h2>
<pre style="background: #263238; color: #aed581; padding: 20px; border-radius: 8px; overflow-x: auto;"><code>static String scoreLabel(int score) {
    if (score &lt; 0 || score &gt; 100) {
        return "invalid";
    }
    // failed / passed / excellent
}</code></pre>

<h2 style="color: #34495e; margin-top: 28px;">✅ Чек-лист проверки</h2>
<div style="background: #f8f9fa; border-left: 4px solid #28a745; padding: 16px; margin: 16px 0;">
<ul style="margin-bottom: 0;">
	<li>☐ Невалидные значения проверяются первыми</li>
	<li>☐ Значения <code>0</code>, <code>69</code>, <code>70</code>, <code>89</code>, <code>90</code>, <code>100</code> обработаны верно</li>
	<li>☐ В коде минимум два <code>return</code></li>
	<li>☐ Нет глубокой вложенности условий</li>
	<li>☐ Нет вывода в консоль, циклов и других операторов выбора</li>
</ul>
</div>

<div style="background: #fff8e1; border-left: 4px solid #ffc107; padding: 16px; margin: 16px 0;">
<h4 style="color: #f57c00; margin-top: 0;">⚠️ Технические ограничения</h4>
<ul style="margin-bottom: 0;">
	<li>Не используйте циклы и другие операторы выбора</li>
	<li>Не используйте <code>System.out.println</code></li>
	<li>Не забывайте верхнюю границу <code>score &gt; 100</code></li>
	<li>Откажитесь от большой вложенной пирамиды <code>if</code> внутри <code>if</code></li>
</ul>
</div>
