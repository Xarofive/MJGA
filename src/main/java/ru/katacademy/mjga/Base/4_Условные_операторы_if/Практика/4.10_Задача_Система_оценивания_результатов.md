<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">📊 Система оценивания результатов</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться использовать ранний <code>return</code>, чтобы избежать глубокой вложенности условий.</p>
</div>

<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Ранний return</h4>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>static String checkValue(int x) {
    if (x &lt; 0) {
        return "invalid";
    }

    return "valid";
}</code></pre>
<p style="margin-bottom: 0;">После <code>return</code> метод завершается, поэтому код ниже не выполняется.</p>
</div>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #e65100; margin-top: 0;">🔧 Метод scoreLabel</h4>
<p><strong>Сигнатура:</strong></p>
<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>static String scoreLabel(int score)</code></div>
<p><strong>Система оценивания:</strong></p>
<ul style="margin: 10px 0;">
	<li><code>"invalid"</code> &mdash; баллы меньше 0 или больше 100;</li>
	<li><code>"failed"</code> &mdash; от 0 до 69 включительно;</li>
	<li><code>"passed"</code> &mdash; от 70 до 89 включительно;</li>
	<li><code>"excellent"</code> &mdash; от 90 до 100 включительно.</li>
</ul>
<p><strong>Требования:</strong></p>
<ul style="margin: 10px 0;">
	<li>используйте минимум два оператора <code>return</code>;</li>
	<li>сначала проверяйте невалидные значения;</li>
	<li>избегайте глубокой вложенности <code>if-else</code>;</li>
	<li>не используйте циклы, <code>switch</code> и вывод в консоль.</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Примеры работы</h2>
<div style="background: #e3f2fd; border: 1px solid #64b5f6; border-radius: 10px; padding: 16px; margin: 20px 0;">
<pre style="margin: 0;"><code>scoreLabel(95)  → "excellent"
scoreLabel(70)  → "passed"
scoreLabel(69)  → "failed"
scoreLabel(101) → "invalid"
scoreLabel(-1)  → "invalid"
scoreLabel(100) → "excellent"</code></pre>
</div>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #c62828; margin-top: 0;">⚠️ Частые ошибки</h4>
<p><strong>Не проверили верхнюю границу:</strong> 101 должен вернуть <code>"invalid"</code>.</p>
<p><strong>Перепутали границы:</strong> 69 &mdash; <code>"failed"</code>, 70 &mdash; <code>"passed"</code>, 90 &mdash; <code>"excellent"</code>.</p>
<p style="margin-bottom: 0;"><strong>Слишком глубокая вложенность:</strong> ранний <code>return</code> делает код короче и понятнее.</p>
</div>

<div style="background: #e8f5e9; border: 1px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">

</ul>
