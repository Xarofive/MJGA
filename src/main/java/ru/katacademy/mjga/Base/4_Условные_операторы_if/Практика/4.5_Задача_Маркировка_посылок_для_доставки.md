<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">Задача 4.5: Маркировка посылок для доставки</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться комбинировать числовые и булевы условия при выборе категории.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">🎯 Сценарий</h3>
<p style="margin-bottom: 10px;">Склад маркирует посылки по весу и признаку хрупкости, чтобы курьеры понимали, как с ними обращаться.</p>
<p style="margin: 0;"><strong>Ваша задача:</strong> вернуть метку доставки по параметрам <code>weight</code> и <code>fragile</code>.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Число + boolean</h4>
<p>Булев параметр уже является условием. Его можно проверять напрямую.</p>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>if (fragile) {
    // правила для хрупкого груза
} else {
    // правила для обычного груза
}</code></pre>
<p style="margin-bottom: 0;">Сначала проверяйте валидность веса, чтобы не классифицировать невозможные посылки.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<p><strong>Создайте метод с точной сигнатурой:</strong></p>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 16px; border-radius: 8px; overflow-x: auto;"><code>static String deliveryLabel(int weight, boolean fragile)</code></pre>
<ul>
	<li><code>"invalid"</code> — вес <code>&lt;= 0</code>;</li>
	<li><code>"handle with care"</code> — хрупкий груз весом до 10 кг включительно;</li>
	<li><code>"special handling"</code> — хрупкий груз весом более 10 кг;</li>
	<li><code>"standard"</code> — обычный груз весом до 10 кг включительно;</li>
	<li><code>"bulk"</code> — обычный груз весом более 10 кг.</li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Результат программы</h2>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 16px; border-radius: 8px; overflow-x: auto;"><code>deliveryLabel(5, true)   -> "handle with care"
deliveryLabel(11, true)  -> "special handling"
deliveryLabel(6, false)  -> "standard"
deliveryLabel(15, false) -> "bulk"
deliveryLabel(10, true)  -> "handle with care"
deliveryLabel(0, false)  -> "invalid"</code></pre>

<div style="background: #fff3e0; border: 1px solid #ff9800; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #e65100; margin-top: 0;">🤔 Момент для размышления</h4>
<p style="margin-bottom: 0;">Почему вес <code>10</code> попадает в категории «до 10 кг включительно», а вес <code>11</code> уже меняет метку?</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📝 Структура метода</h2>
<pre style="background: #263238; color: #aed581; padding: 20px; border-radius: 8px; overflow-x: auto;"><code>static String deliveryLabel(int weight, boolean fragile) {
    // 1. invalid
    // 2. fragile + weight
    // 3. not fragile + weight
}</code></pre>

<h2 style="color: #34495e; margin-top: 28px;">✅ Чек-лист проверки</h2>
<div style="background: #f8f9fa; border-left: 4px solid #28a745; padding: 16px; margin: 16px 0;">
<ul style="margin-bottom: 0;">
	<li>☐ Вес <code>&lt;= 0</code> обрабатывается первым</li>
	<li>☐ Хрупкий и обычный груз различаются</li>
	<li>☐ Граница <code>10</code> обработана включительно</li>
	<li>☐ Все пять строк результата используются точно</li>
	<li>☐ Нет вывода в консоль</li>
</ul>
</div>

<div style="background: #fff8e1; border-left: 4px solid #ffc107; padding: 16px; margin: 16px 0;">
<h4 style="color: #f57c00; margin-top: 0;">⚠️ Технические ограничения</h4>
<ul style="margin-bottom: 0;">
	<li>Не используйте циклы и другие операторы выбора</li>
	<li>Не игнорируйте параметр <code>fragile</code></li>
	<li>Не меняйте границу <code>10</code> на <code>9</code> или <code>11</code></li>
	<li>Откажитесь от вывода: метод должен возвращать строку</li>
</ul>
</div>
