<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🔍 Определение знака числа</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться использовать условные операторы для классификации чисел и возвращать результат из метода в виде строки.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">💼 Контекст и мотивация</h3>
<p style="margin-bottom: 10px;">Определение знака числа встречается в финансовых расчётах, аналитике, игровой логике и обработке датчиков. Важно отдельно обработать ноль: он не является ни положительным, ни отрицательным числом.</p>
<p style="margin: 0;"><strong>Задача:</strong> создать метод для текстового описания знака числа.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Условный оператор if</h4>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>if (условие) {
    // первый вариант
} else if (другоеУсловие) {
    // второй вариант
} else {
    // если ни одно условие не выполнено
}</code></pre>
<p style="margin-bottom: 0;">Условия проверяются сверху вниз. Выполняется первый подходящий блок.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #e65100; margin-top: 0;">🔧 Метод describeSign</h4>
<p><strong>Сигнатура:</strong></p>
<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>static String describeSign(int n)</code></div>
<p><strong>Требования:</strong></p>
<ul style="margin: 10px 0;">
	<li>принимает один параметр типа <code>int</code>;</li>
	<li>возвращает строку (<code>String</code>);</li>
	<li>для положительных чисел (<code>n &gt; 0</code>) возвращает <code>"positive"</code>;</li>
	<li>для нуля (<code>n == 0</code>) возвращает <code>"zero"</code>;</li>
	<li>для отрицательных чисел (<code>n &lt; 0</code>) возвращает <code>"negative"</code>;</li>
	<li>обязательно использовать цепочку <code>if</code> / <code>else if</code> / <code>else</code>;</li>
	<li>запрещено использовать циклы и оператор <code>switch</code>.</li>
</ul>
<p><strong>Формат решения:</strong> отправьте только метод <code>static String describeSign(int n) { ... }</code>, без класса <code>Main</code> и без <code>import</code>.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Примеры работы</h2>
<div style="background: #e3f2fd; border: 1px solid #64b5f6; border-radius: 10px; padding: 16px; margin: 20px 0;">
<pre style="margin: 0;"><code>describeSign(9)  → "positive"
describeSign(-5) → "negative"
describeSign(0)  → "zero"
describeSign(1)  → "positive"</code></pre>
</div>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #c62828; margin-top: 0;">⚠️ Частые ошибки</h4>
<p><strong>Ноль обработан неверно:</strong> ноль должен возвращать <code>"zero"</code>.</p>
<p><strong>Опечатки в строках:</strong> значения должны быть точно <code>"positive"</code>, <code>"negative"</code>, <code>"zero"</code>.</p>
<p style="margin-bottom: 0;"><strong>Неправильная сигнатура:</strong> нужен тип возврата <code>String</code>.</p>
</div>
