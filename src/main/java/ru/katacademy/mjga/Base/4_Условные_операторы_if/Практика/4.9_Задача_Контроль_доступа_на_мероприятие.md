<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">Задача 4.9: Контроль доступа на мероприятие</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться объединять условия через <code>&amp;&amp;</code> и <code>||</code>.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">🎯 Сценарий</h3>
<p style="margin-bottom: 10px;">На мероприятии обычный посетитель проходит только при возрасте 18+ и билете, а VIP проходит независимо от остальных параметров.</p>
<p style="margin: 0;"><strong>Ваша задача:</strong> вернуть статус доступа по возрасту, билету и VIP-статусу.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">&& и || вместе</h4>
<p><code>&amp;&amp;</code> означает «и», <code>||</code> означает «или». Скобки помогают явно показать смысл правила.</p>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>if ((age &gt;= 18 &amp;&amp; hasTicket) || vip) {
    return "allowed";
}</code></pre>
<p style="margin-bottom: 0;">Так читается правило: либо посетитель 18+ с билетом, либо VIP.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<p><strong>Создайте метод с точной сигнатурой:</strong></p>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 16px; border-radius: 8px; overflow-x: auto;"><code>static String accessStatus(int age, boolean hasTicket, boolean vip)</code></pre>
<ul>
	<li><code>"allowed"</code> — если возраст не меньше 18 и есть билет;</li>
	<li><code>"allowed"</code> — если есть VIP-статус, независимо от возраста и билета;</li>
	<li><code>"denied"</code> — во всех остальных случаях.</li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Результат программы</h2>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 16px; border-radius: 8px; overflow-x: auto;"><code>accessStatus(20, true, false)  -> "allowed"
accessStatus(16, false, true)  -> "allowed"
accessStatus(17, true, false)  -> "denied"
accessStatus(20, false, false) -> "denied"
accessStatus(18, true, false)  -> "allowed"
accessStatus(20, true, true)   -> "allowed"</code></pre>

<div style="background: #fff3e0; border: 1px solid #ff9800; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #e65100; margin-top: 0;">🤔 Момент для размышления</h4>
<p style="margin-bottom: 0;">Почему VIP-статус вынесен через <code>||</code>, а билет связан с возрастом через <code>&amp;&amp;</code>?</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📝 Структура метода</h2>
<pre style="background: #263238; color: #aed581; padding: 20px; border-radius: 8px; overflow-x: auto;"><code>static String accessStatus(int age, boolean hasTicket, boolean vip) {
    if ((age &gt;= 18 &amp;&amp; hasTicket) || vip) {
        return "allowed";
    }
    return "denied";
}</code></pre>

<div style="background: #fff8e1; border-left: 4px solid #ffc107; padding: 16px; margin: 16px 0;">

<ul style="margin-bottom: 0;">

	<li>Не требуйте билет для VIP</li>
