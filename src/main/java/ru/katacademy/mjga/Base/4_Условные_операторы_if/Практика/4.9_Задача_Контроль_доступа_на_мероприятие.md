<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🎭 Контроль доступа на мероприятие</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться объединять условия через <code>&amp;&amp;</code> и <code>||</code>.</p>
</div>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #e65100; margin-top: 0;">🔧 Метод accessStatus</h4>
<p><strong>Сигнатура:</strong></p>
<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>static String accessStatus(int age, boolean hasTicket, boolean vip)</code></div>
<p><strong>Правила доступа:</strong></p>
<ul style="margin: 10px 0;">
	<li><code>"allowed"</code> &mdash; если возраст не меньше 18 и есть билет;</li>
	<li><code>"allowed"</code> &mdash; если есть VIP-статус, независимо от возраста и билета;</li>
	<li><code>"denied"</code> &mdash; во всех остальных случаях.</li>
</ul>
<p><strong>Требования:</strong> обязательно используйте операторы <code>&amp;&amp;</code> и <code>||</code>, не используйте циклы, <code>switch</code> и вывод в консоль.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Примеры работы</h2>
<div style="background: #e3f2fd; border: 1px solid #64b5f6; border-radius: 10px; padding: 16px; margin: 20px 0;">
<pre style="margin: 0;"><code>accessStatus(20, true, false)  → "allowed"
accessStatus(16, false, true)  → "allowed"
accessStatus(17, true, false)  → "denied"
accessStatus(20, false, false) → "denied"
accessStatus(18, true, false)  → "allowed"
accessStatus(20, true, true)   → "allowed"</code></pre>
</div>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #c62828; margin-top: 0;">⚠️ Частые ошибки</h4>
<p><strong>Смешали И и ИЛИ:</strong> билет нужен только для обычного доступа, VIP проходит отдельно.</p>
<p><strong>Забыли скобки:</strong> условие <code>(age &gt;= 18 &amp;&amp; hasTicket) || vip</code> читается однозначно.</p>
<p style="margin-bottom: 0;"><strong>Граница возраста:</strong> 18 лет уже допускается при наличии билета.</p>
</div>
