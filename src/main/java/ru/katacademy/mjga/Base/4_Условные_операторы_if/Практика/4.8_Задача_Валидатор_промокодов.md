<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🎟️ Валидатор промокодов</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Потренироваться проверять строки, длину строки и булев флаг в правильном порядке.</p>
</div>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #e65100; margin-top: 0;">🔧 Метод promoStatus</h4>
<p><strong>Сигнатура:</strong></p>
<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>static String promoStatus(String code, boolean isMember)</code></div>
<p><strong>Правила валидации:</strong></p>
<ol style="margin: 10px 0;">
	<li><code>"missing"</code> &mdash; код пустой;</li>
	<li><code>"invalid length"</code> &mdash; длина кода не равна 6 символам;</li>
	<li><code>"members only"</code> &mdash; код <code>"GOSALE"</code>, но пользователь не участник программы;</li>
	<li><code>"accepted"</code> &mdash; код прошёл все проверки.</li>
</ol>
<p><strong>Требования:</strong> промокоды уже передаются в верхнем регистре без пробелов. Для сравнения строк используйте <code>code.equals("GOSALE")</code>. Не используйте циклы и <code>switch</code>.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Примеры работы</h2>
<div style="background: #e3f2fd; border: 1px solid #64b5f6; border-radius: 10px; padding: 16px; margin: 20px 0;">
<pre style="margin: 0;"><code>promoStatus("GOSALE", true)  → "accepted"
promoStatus("GOSALE", false) → "members only"
promoStatus("", true)        → "missing"
promoStatus("ABC", true)     → "invalid length"
promoStatus("WINTER", false) → "accepted"</code></pre>
</div>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #c62828; margin-top: 0;">⚠️ Частые ошибки</h4>
<p><strong>Сравнение строк через <code>==</code>:</strong> в Java для содержимого строки используйте <code>equals</code>.</p>
<p><strong>Неверный порядок:</strong> пустой код должен вернуть <code>"missing"</code>, а не <code>"invalid length"</code>.</p>
<p style="margin-bottom: 0;"><strong>Членство:</strong> только <code>"GOSALE"</code> требует <code>isMember == true</code>.</p>
</div>
