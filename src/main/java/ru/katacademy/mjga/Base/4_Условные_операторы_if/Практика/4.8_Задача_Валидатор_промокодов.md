<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🎟️ Валидатор промокодов</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Потренироваться проверять числовые параметры и булевы флаги в правильном порядке.</p>
</div>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #e65100; margin-top: 0;">🔧 Метод promoStatus</h4>
<p><strong>Сигнатура:</strong></p>
<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>static String promoStatus(int codeLength, boolean isGoSale, boolean isMember)</code></div>
<p><strong>Правила валидации:</strong></p>
<ol style="margin: 10px 0;">
	<li><code>"missing"</code> &mdash; длина кода равна 0;</li>
	<li><code>"invalid length"</code> &mdash; длина кода не равна 6 символам;</li>
	<li><code>"members only"</code> &mdash; это специальный код <code>GOSALE</code>, но пользователь не участник программы;</li>
	<li><code>"accepted"</code> &mdash; код прошёл все проверки.</li>
</ol>
<p><strong>Требования:</strong> сам текст промокода проверять не нужно. Если промокод равен <code>GOSALE</code>, параметр <code>isGoSale</code> уже будет равен <code>true</code>. Не используйте циклы и <code>switch</code>.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Примеры работы</h2>
<div style="background: #e3f2fd; border: 1px solid #64b5f6; border-radius: 10px; padding: 16px; margin: 20px 0;">
<pre style="margin: 0;"><code>promoStatus(6, true, true)   → "accepted"
promoStatus(6, true, false)  → "members only"
promoStatus(0, false, true)  → "missing"
promoStatus(3, false, true)  → "invalid length"
promoStatus(6, false, false) → "accepted"</code></pre>
</div>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #c62828; margin-top: 0;">⚠️ Частые ошибки</h4>
<p><strong>Неверный порядок:</strong> длина 0 должна вернуть <code>"missing"</code>, а не <code>"invalid length"</code>.</p>
<p><strong>Проверка длины:</strong> допустима только длина 6.</p>
<p style="margin-bottom: 0;"><strong>Членство:</strong> только специальный код <code>isGoSale == true</code> требует <code>isMember == true</code>.</p>
</div>

<div style="background: #e8f5e9; border: 1px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #2e7d32; margin-top: 0;">✅ Чек-лист самопроверки</h4>

	<li><label><input type="checkbox"> <code>codeLength == 0</code> проверяется первым.</label></li>
	<li><label><input type="checkbox"> Длина проверяется через параметр <code>codeLength</code>.</label></li>
	<li><label><input type="checkbox"> Для специального кода используется параметр <code>isGoSale</code>.</label></li>
	<li><label><input type="checkbox"> Обычный код длиной 6 принимается даже без членства.</label></li>
	<li><label><input type="checkbox"> Метод ничего не печатает.</label></li>
</ul>
