<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">Задача 4.8: Валидатор промокодов</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Потренироваться проверять числовые параметры и булевы флаги в правильном порядке.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">🎯 Сценарий</h3>
<p style="margin-bottom: 10px;">Интернет-магазин проверяет промокод: он может отсутствовать, иметь неправильную длину или быть специальным кодом только для участников.</p>
<p style="margin: 0;"><strong>Ваша задача:</strong> вернуть статус промокода по длине, признаку <code>JAVASALE</code> и членству пользователя.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Валидация по шагам</h4>
<p>Проверки лучше выполнять от базовых к более специфичным: сначала отсутствие значения, затем длина, затем специальные правила.</p>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>if (codeLength == 0) {
    return "missing";
}
if (codeLength != 6) {
    return "invalid length";
}</code></pre>
<p style="margin-bottom: 0;">Если перепутать порядок, пустой код может ошибочно стать <code>"invalid length"</code>.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<p><strong>Создайте метод с точной сигнатурой:</strong></p>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 16px; border-radius: 8px; overflow-x: auto;"><code>static String promoStatus(int codeLength, boolean isJavaSale, boolean isMember)</code></pre>
<ol>
	<li><code>"missing"</code> — длина кода равна 0;</li>
	<li><code>"invalid length"</code> — длина кода не равна 6 символам;</li>
	<li><code>"members only"</code> — это специальный код <code>JAVASALE</code>, но пользователь не участник программы;</li>
	<li><code>"accepted"</code> — код прошёл все проверки.</li>
</ol>
<p>Сам текст промокода проверять не нужно: если промокод равен <code>JAVASALE</code>, параметр <code>isJavaSale</code> уже будет <code>true</code>.</p>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Результат программы</h2>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 16px; border-radius: 8px; overflow-x: auto;"><code>promoStatus(6, true, true)   -> "accepted"
promoStatus(6, true, false)  -> "members only"
promoStatus(0, false, true)  -> "missing"
promoStatus(3, false, true)  -> "invalid length"
promoStatus(6, false, false) -> "accepted"</code></pre>

<div style="background: #fff3e0; border: 1px solid #ff9800; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #e65100; margin-top: 0;">🤔 Момент для размышления</h4>
<p style="margin-bottom: 0;">Почему обычный код длиной 6 принимается даже без членства, а специальный <code>JAVASALE</code> требует участника программы?</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📝 Структура метода</h2>
<pre style="background: #263238; color: #aed581; padding: 20px; border-radius: 8px; overflow-x: auto;"><code>static String promoStatus(int codeLength, boolean isJavaSale, boolean isMember) {
    // 1. missing
    // 2. invalid length
    // 3. members only
    // 4. accepted
}</code></pre>

<h2 style="color: #34495e; margin-top: 28px;">✅ Чек-лист проверки</h2>
<div style="background: #f8f9fa; border-left: 4px solid #28a745; padding: 16px; margin: 16px 0;">
<ul style="margin-bottom: 0;">
	<li>☐ <code>codeLength == 0</code> проверяется первым</li>
	<li>☐ Длина проверяется через <code>codeLength != 6</code></li>
	<li>☐ Для специального кода используется <code>isJavaSale</code></li>
	<li>☐ Обычный код длиной 6 принимается без членства</li>
	<li>☐ Метод ничего не печатает</li>
</ul>
</div>

<div style="background: #fff8e1; border-left: 4px solid #ffc107; padding: 16px; margin: 16px 0;">
<h4 style="color: #f57c00; margin-top: 0;">⚠️ Технические ограничения</h4>
<ul style="margin-bottom: 0;">
	<li>Не используйте циклы и другие операторы выбора</li>
	<li>Не проверяйте сам текст промокода</li>
	<li>Не используйте вывод в консоль</li>
	<li>Откажитесь от проверки членства до проверки длины</li>
</ul>
</div>
