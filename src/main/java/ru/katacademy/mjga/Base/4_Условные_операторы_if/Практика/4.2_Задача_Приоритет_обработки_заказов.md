<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">📦 Приоритет обработки заказов</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться строить несколько проверок подряд и правильно расставлять приоритет условий.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">💼 Контекст и мотивация</h3>
<p style="margin: 0;">Сервис обработки заказов должен определить, какой заказ нужно обработать быстрее: экспресс-заказ, крупный заказ или обычный заказ.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #e65100; margin-top: 0;">🔧 Метод orderPriority</h4>
<p><strong>Сигнатура:</strong></p>
<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>static String orderPriority(int total, boolean express)</code></div>
<p><strong>Бизнес-логика:</strong></p>
<ul style="margin: 10px 0;">
	<li><code>"invalid"</code> &mdash; сумма заказа <code>&lt;= 0</code>;</li>
	<li><code>"rush"</code> &mdash; экспресс-доставка при любой допустимой сумме;</li>
	<li><code>"priority"</code> &mdash; обычная доставка при сумме <code>&gt; 100</code>;</li>
	<li><code>"standard"</code> &mdash; обычная доставка при сумме от 1 до 100 включительно.</li>
</ul>
<p><strong>Требования:</strong></p>
<ul style="margin: 10px 0;">
	<li>сначала проверяйте недопустимую сумму;</li>
	<li>используйте условные операторы <code>if</code>;</li>
	<li>не используйте циклы, <code>switch</code> и вывод в консоль;</li>
	<li>отправьте только метод, без класса и импортов.</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Примеры работы</h2>
<div style="background: #e3f2fd; border: 1px solid #64b5f6; border-radius: 10px; padding: 16px; margin: 20px 0;">
<pre style="margin: 0;"><code>orderPriority(50, true)   → "rush"
orderPriority(150, false) → "priority"
orderPriority(75, false)  → "standard"
orderPriority(0, true)    → "invalid"
orderPriority(200, true)  → "rush"</code></pre>
</div>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #c62828; margin-top: 0;">⚠️ Частые ошибки</h4>
<p><strong>Неверный порядок:</strong> если сначала проверить <code>express</code>, то <code>orderPriority(0, true)</code> ошибочно вернёт <code>"rush"</code>.</p>
<p style="margin-bottom: 0;"><strong>Лишний вывод:</strong> метод должен возвращать строку, а не печатать её.</p>
</div>
