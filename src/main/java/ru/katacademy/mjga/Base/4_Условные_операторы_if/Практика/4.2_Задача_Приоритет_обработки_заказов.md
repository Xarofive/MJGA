<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">Задача 4.2: Приоритет обработки заказов</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться строить несколько проверок подряд и правильно расставлять приоритет условий.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">🎯 Сценарий</h3>
<p style="margin-bottom: 10px;">Сервис обработки заказов должен понять, что делать быстрее: экспресс-заказ, крупный заказ или обычный заказ.</p>
<p style="margin: 0;"><strong>Ваша задача:</strong> вернуть приоритет заказа по сумме и флагу экспресс-доставки.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Порядок проверок</h4>
<p>Если одно условие важнее остальных, его нужно проверить раньше. Здесь невалидная сумма важнее экспресс-доставки.</p>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>if (total &lt;= 0) {
    return "invalid";
}
if (express) {
    return "rush";
}</code></pre>
<p style="margin-bottom: 0;">Ранний <code>return</code> завершает метод, поэтому следующие проверки уже не выполняются.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<p><strong>Создайте метод с точной сигнатурой:</strong></p>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 16px; border-radius: 8px; overflow-x: auto;"><code>static String orderPriority(int total, boolean express)</code></pre>
<ul>
	<li><code>"invalid"</code> — сумма заказа <code>&lt;= 0</code>;</li>
	<li><code>"rush"</code> — экспресс-доставка при любой допустимой сумме;</li>
	<li><code>"priority"</code> — обычная доставка при сумме <code>&gt; 100</code>;</li>
	<li><code>"standard"</code> — обычная доставка при сумме от 1 до 100 включительно.</li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Результат программы</h2>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 16px; border-radius: 8px; overflow-x: auto;"><code>orderPriority(50, true)   -> "rush"
orderPriority(150, false) -> "priority"
orderPriority(75, false)  -> "standard"
orderPriority(0, true)    -> "invalid"
orderPriority(200, true)  -> "rush"</code></pre>
