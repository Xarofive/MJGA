<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🌍 Извлечение подстроки с учётом Unicode</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться извлекать подстроку по Unicode code points, не разрывая эмодзи и другие символы.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p><strong>Реализуйте метод:</strong></p>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px;"><code class="language-java">static String substringCodePoints(String s, int start, int length)</code></pre>
<ul>
	<li><code>start</code> и <code>length</code> заданы в code points, не в UTF-16 индексах</li>
	<li>Если <code>start + length</code> выходит за конец строки, верните до конца строки</li>
	<li>Если <code>length == 0</code>, верните пустую строку</li>
	<li>Используйте <code>codePointCount</code> и <code>offsetByCodePoints</code></li>
	<li>Решение должно содержать только метод без вывода в консоль</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Примеры работы</h2>
<ul>
	<li><code>substringCodePoints("Hello, World!", 7, 5)</code> → <code>"World"</code></li>
	<li><code>substringCodePoints("Hello😊世界", 5, 3)</code> → <code>"😊世界"</code></li>
	<li><code>substringCodePoints("Привет", 3, 10)</code> → <code>"вет"</code></li>
	<li><code>substringCodePoints("Test", 2, 0)</code> → <code>""</code></li>
</ul>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #c62828; margin-top: 0;">⚠️ Частая ошибка</h4>
<p style="margin-bottom: 0;">Не используйте напрямую <code>s.substring(start, start + length)</code>: эти индексы считаются в UTF-16, а задача требует code points.</p>
</div>
