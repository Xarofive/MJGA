<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🔤 Анализ размера строки: байты и символы</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться различать размер строки в UTF-8 байтах и количество Unicode code points в Java.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p><strong>Реализуйте метод:</strong></p>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px;"><code class="language-java">static int[] byteCodePointCount(String s)</code></pre>
<ul>
	<li>Возвращает массив из двух чисел</li>
	<li><code>result[0]</code> &mdash; количество байт в UTF-8</li>
	<li><code>result[1]</code> &mdash; количество Unicode code points</li>
	<li>Для байт используйте <code>s.getBytes(StandardCharsets.UTF_8).length</code></li>
	<li>Для code points используйте <code>s.codePointCount(0, s.length())</code></li>
	<li>Решение должно содержать только метод без вывода в консоль</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Примеры работы</h2>
<ul>
	<li><code>byteCodePointCount("")</code> → <code>[0, 0]</code></li>
	<li><code>byteCodePointCount("Hello")</code> → <code>[5, 5]</code></li>
	<li><code>byteCodePointCount("Привет")</code> → <code>[12, 6]</code></li>
	<li><code>byteCodePointCount("Go🚀")</code> → <code>[6, 3]</code></li>
</ul>
