<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🔁 Разворот строки методом двух указателей</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Развернуть строку по Unicode code points, сохраняя целостность многобайтовых символов.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Сценарий</h2>
<p>Нужно развернуть строку, в которой могут быть русские буквы и эмодзи. Обычный разворот по <code>char</code> может испортить символы, поэтому работа идёт через code points.</p>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<ul>
	<li><code>s.codePoints().toArray()</code> превращает строку в массив Unicode code points.</li>
	<li>Метод двух указателей меняет местами элементы с начала и конца массива.</li>
	<li><code>StringBuilder.appendCodePoint</code> собирает строку из code points.</li>
	<li>Такой подход безопаснее разворота по отдельным <code>char</code>.</li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p><strong>Реализуйте метод:</strong></p>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px;"><code class="language-java">static String reverseCodePoints(String s)</code></pre>
<ul>
	<li>Преобразуйте строку в массив code points: <code>s.codePoints().toArray()</code>.</li>
	<li>Разверните массив методом двух указателей.</li>
	<li>Соберите строку через <code>StringBuilder.appendCodePoint</code>.</li>
	<li>Решение должно содержать только метод без вывода в консоль.</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Результат программы</h2>
<ul>
	<li><code>reverseCodePoints("abc")</code> → <code>"cba"</code></li>
	<li><code>reverseCodePoints("Привет")</code> → <code>"тевирП"</code></li>
	<li><code>reverseCodePoints("Go🚀")</code> → <code>"🚀oG"</code></li>
	<li><code>reverseCodePoints("")</code> → <code>""</code></li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">🤔 Момент для размышления</h2>
<p>Почему <code>new StringBuilder(s).reverse()</code> не раскрывает механику этой задачи, даже если для части строк может дать похожий результат?</p>

<h2 style="color: #34495e; margin-top: 28px;">📝 Структура программы</h2>
<ol>
	<li>Получите массив code points.</li>
	<li>Создайте два индекса: левый и правый.</li>
	<li>Меняйте элементы местами, пока индексы не встретятся.</li>
	<li>Соберите новую строку через <code>StringBuilder</code>.</li>
	<li>Верните результат.</li>
</ol>

<h2 style="color: #34495e; margin-top: 28px;">✅ Чек-лист проверки</h2>
<div style="background: #e8f5e9; border: 1px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<ul style="margin-bottom: 0;">
	<li><label><input type="checkbox"> Используется массив code points.</label></li>
	<li><label><input type="checkbox"> Разворот сделан двумя указателями.</label></li>
	<li><label><input type="checkbox"> Сборка идёт через <code>appendCodePoint</code>.</label></li>
	<li><label><input type="checkbox"> Эмодзи остаются целыми.</label></li>
	<li><label><input type="checkbox"> Метод не печатает результат.</label></li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">⚠️ Технические ограничения</h2>
<ul>
	<li>Не разворачивайте строку обычным проходом по <code>char</code>.</li>
	<li>Не используйте готовый <code>StringBuilder.reverse()</code> как основное решение.</li>
	<li>Не выводите результат в консоль.</li>
	<li>Откажитесь от конкатенации строк через <code>+=</code> в цикле.</li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">🐛 Отладка: если что-то не работает</h2>
<ul>
	<li>Если эмодзи ломается, проверьте, что сборка идёт через <code>appendCodePoint</code>.</li>
	<li>Если строка не разворачивается, проверьте обмен элементов и движение указателей.</li>
	<li>Если пустая строка падает, убедитесь, что цикл корректно работает с массивом длины 0.</li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">💪 Дополнительный вызов (после сдачи задачи)</h2>
<p>Проверьте метод на строке из нескольких эмодзи подряд и объясните, почему результат остаётся читаемым.</p>

<h2 style="color: #34495e; margin-top: 28px;">🎨 Где это пригодится</h2>
<p>Unicode-безопасная работа со строками нужна в текстовых редакторах, чатах, локализованных интерфейсах, проверках палиндромов и любых приложениях, где пользовательский текст не ограничен английскими буквами.</p>
