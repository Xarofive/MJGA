<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🧹 Нормализация пробелов в тексте</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Удалить лишние пробелы по краям и заменить любые группы пробельных символов одним обычным пробелом.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px;"><code class="language-java">static String normalizeSpaces(String s)</code></pre>
<ul>
	<li>Пробелы в начале и конце нужно убрать</li>
	<li>Любую группу whitespace внутри строки заменить одним пробелом</li>
	<li>Для строки только из пробелов вернуть пустую строку</li>
	<li>Используйте <code>trim()</code>, <code>split("\\s+")</code> и <code>String.join(" ", words)</code></li>
	<li>Решение должно содержать только метод без вывода в консоль</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Примеры работы</h2>
<ul>
	<li><code>normalizeSpaces("  hello   world  ")</code> → <code>"hello world"</code></li>
	<li><code>normalizeSpaces("a\tb\nc")</code> → <code>"a b c"</code></li>
	<li><code>normalizeSpaces("     ")</code> → <code>""</code></li>
</ul>
