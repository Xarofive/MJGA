<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🔍 Поиск подстроки без учёта регистра</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Реализовать поиск подстроки, при котором <code>"Hello"</code>, <code>"HELLO"</code> и <code>"hello"</code> считаются совпадающими по регистру.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Сценарий</h2>
<p>Пользователь ищет слово в тексте, но может ввести его в любом регистре. Метод должен найти совпадение независимо от того, какие буквы были заглавными.</p>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<ul>
	<li><code>contains</code> ищет точное вхождение строки.</li>
	<li>Чтобы игнорировать регистр, обе строки нужно привести к одному регистру.</li>
	<li>Пустая подстрока считается найденной.</li>
	<li>Метод должен вернуть <code>boolean</code>, а не печатать результат.</li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p><strong>Реализуйте метод:</strong></p>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px;"><code class="language-java">static boolean containsIgnoreCase(String s, String sub)</code></pre>
<ul>
	<li>Верните <code>true</code>, если <code>sub</code> содержится в <code>s</code> без учёта регистра.</li>
	<li>Если <code>sub</code> пустая строка, верните <code>true</code>.</li>
	<li>Используйте <code>toLowerCase()</code> или <code>toUpperCase()</code> для обеих строк.</li>
	<li>После нормализации регистра используйте <code>contains</code>.</li>
	<li>Решение должно содержать только метод без вывода в консоль.</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Результат программы</h2>
<ul>
	<li><code>containsIgnoreCase("Hello World", "HELLO")</code> → <code>true</code></li>
	<li><code>containsIgnoreCase("Programming", "GRAM")</code> → <code>true</code></li>
	<li><code>containsIgnoreCase("Java", "python")</code> → <code>false</code></li>
	<li><code>containsIgnoreCase("any string", "")</code> → <code>true</code></li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">🤔 Момент для размышления</h2>
<p>Почему нужно приводить к одному регистру обе строки, а не только подстроку <code>sub</code>?</p>

<h2 style="color: #34495e; margin-top: 28px;">📝 Структура программы</h2>
<ol>
	<li>Проверьте пустую подстроку.</li>
	<li>Приведите исходную строку к одному регистру.</li>
	<li>Приведите подстроку к тому же регистру.</li>
	<li>Вызовите <code>contains</code>.</li>
	<li>Верните результат.</li>
</ol>

<h2 style="color: #34495e; margin-top: 28px;">✅ Чек-лист проверки</h2>
<div style="background: #e8f5e9; border: 1px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<ul style="margin-bottom: 0;">
	<li><label><input type="checkbox"> Метод называется <code>containsIgnoreCase</code>.</label></li>
	<li><label><input type="checkbox"> Пустая подстрока возвращает <code>true</code>.</label></li>
	<li><label><input type="checkbox"> Обе строки приводятся к одному регистру.</label></li>
	<li><label><input type="checkbox"> Метод возвращает <code>boolean</code>, а не печатает результат.</label></li>
	<li><label><input type="checkbox"> После нормализации используется <code>contains</code>.</label></li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">⚠️ Технические ограничения</h2>
<ul>
	<li>Не используйте вывод в консоль.</li>
	<li>Не сравнивайте строки через <code>==</code>.</li>
	<li>Не приводите к регистру только одну из двух строк.</li>
	<li>Откажитесь от ручного перебора символов: в этой задаче нужен <code>contains</code> после нормализации регистра.</li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">🐛 Отладка: если что-то не работает</h2>
<ul>
	<li>Если <code>"HELLO"</code> не находится в <code>"Hello World"</code>, проверьте приведение обеих строк к одному регистру.</li>
	<li>Если пустая подстрока даёт <code>false</code>, добавьте отдельную проверку или вспомните поведение <code>contains("")</code>.</li>
	<li>Если метод печатает <code>true</code>, но не проходит тест, возвращайте значение через <code>return</code>.</li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">🎨 Где это пригодится</h2>
<p>Поиск без учёта регистра нужен в поисковых строках, фильтрах, проверке команд, обработке пользовательского ввода, модерации текста и любых интерфейсах, где регистр не должен мешать совпадению.</p>
