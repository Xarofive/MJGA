<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🔍 Поиск подстроки без учёта регистра</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Реализовать поиск подстроки, при котором <code>"Hello"</code>, <code>"HELLO"</code> и <code>"hello"</code> считаются совпадающими по регистру.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px;"><code class="language-java">static boolean containsIgnoreCase(String s, String sub)</code></pre>
<ul>
	<li>Верните <code>true</code>, если <code>sub</code> содержится в <code>s</code> без учёта регистра</li>
	<li>Если <code>sub</code> пустая строка, верните <code>true</code></li>
	<li>Используйте <code>toLowerCase()</code> или <code>toUpperCase()</code> для обеих строк</li>
	<li>После нормализации регистра используйте <code>contains</code></li>
	<li>Решение должно содержать только метод без вывода в консоль</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Примеры работы</h2>
<ul>
	<li><code>containsIgnoreCase("Hello World", "HELLO")</code> → <code>true</code></li>
	<li><code>containsIgnoreCase("Programming", "GRAM")</code> → <code>true</code></li>
	<li><code>containsIgnoreCase("Java", "python")</code> → <code>false</code></li>
	<li><code>containsIgnoreCase("any string", "")</code> → <code>true</code></li>
</ul>
