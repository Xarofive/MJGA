<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🔗 Эффективная конкатенация чисел в строку</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться собирать строку из чисел через <code>StringBuilder</code>, не создавая лишние промежуточные строки в цикле.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px;"><code class="language-java">static String joinNumbers(int[] nums)</code></pre>
<ul>
	<li>Верните строку, где числа разделены запятыми без пробелов</li>
	<li>Для пустого массива верните пустую строку</li>
	<li>После последнего числа запятая не ставится</li>
	<li>Используйте <code>StringBuilder</code></li>
	<li>Не используйте конкатенацию строк через <code>+</code> или <code>+=</code> внутри цикла</li>
	<li>Решение должно содержать только метод без вывода в консоль</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Примеры работы</h2>
<ul>
	<li><code>joinNumbers(new int[]{1, 2, 3})</code> → <code>"1,2,3"</code></li>
	<li><code>joinNumbers(new int[]{})</code> → <code>""</code></li>
	<li><code>joinNumbers(new int[]{42})</code> → <code>"42"</code></li>
	<li><code>joinNumbers(new int[]{-5, 0, 10})</code> → <code>"-5,0,10"</code></li>
</ul>
