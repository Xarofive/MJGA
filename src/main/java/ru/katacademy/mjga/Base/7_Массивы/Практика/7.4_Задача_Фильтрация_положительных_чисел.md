<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🔍 Фильтрация положительных чисел</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться создавать новый массив на основе исходного, не изменяя входные данные.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Сценарий</h2>
<p>Есть массив с разными числами: отрицательными, нулём и положительными. Нужно оставить только полезные значения, сохранив их порядок, и вернуть новый массив.</p>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<ul>
	<li>Положительное число строго больше <code>0</code>.</li>
	<li>Ноль не считается положительным.</li>
	<li>Чтобы создать массив точной длины, сначала удобно посчитать подходящие элементы.</li>
	<li>Если исходный массив нельзя менять, результат создаётся отдельно.</li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p><strong>Реализуйте метод:</strong></p>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px;"><code class="language-java">static int[] filterPositive(int[] src)</code></pre>
<ul>
	<li>Метод принимает массив целых чисел.</li>
	<li>Возвращает новый массив только с положительными числами, то есть строго больше <code>0</code>.</li>
	<li>Порядок элементов должен сохраниться.</li>
	<li>Исходный массив изменять нельзя.</li>
	<li>Если положительных чисел нет, верните пустой массив <code>new int[0]</code>.</li>
	<li>Решение должно содержать только метод без вывода в консоль.</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Результат программы</h2>
<ul>
	<li><code>filterPositive(new int[]{-2, -1, 0, 3, 5, -4})</code> → <code>[3, 5]</code></li>
	<li><code>filterPositive(new int[]{0, -7})</code> → <code>[]</code></li>
	<li><code>filterPositive(new int[]{1, 2, 3})</code> → <code>[1, 2, 3]</code></li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">🤔 Момент для размышления</h2>
<p>Почему нельзя создать результат той же длины, что и исходный массив, и просто оставить лишние ячейки нулями?</p>

<h2 style="color: #34495e; margin-top: 28px;">📝 Структура программы</h2>
<ol>
	<li>Посчитайте количество положительных элементов.</li>
	<li>Создайте массив результата нужной длины.</li>
	<li>Повторно пройдите по исходному массиву.</li>
	<li>Скопируйте только значения <code>&gt; 0</code>.</li>
	<li>Верните новый массив.</li>
</ol>

<h2 style="color: #34495e; margin-top: 28px;">✅ Чек-лист проверки</h2>
<div style="background: #e8f5e9; border: 1px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<ul style="margin-bottom: 0;">
	<li><label><input type="checkbox"> Метод называется <code>filterPositive</code>.</label></li>
	<li><label><input type="checkbox"> Возвращается <code>int[]</code>.</label></li>
	<li><label><input type="checkbox"> Используется проверка <code>value &gt; 0</code>.</label></li>
	<li><label><input type="checkbox"> Исходный массив не изменяется.</label></li>
	<li><label><input type="checkbox"> В результате нет лишних нулей.</label></li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">⚠️ Технические ограничения</h2>
<ul>
	<li>Не считайте <code>0</code> положительным числом.</li>
	<li>Не изменяйте элементы <code>src</code>.</li>
	<li>Не печатайте результат в консоль.</li>
	<li>Откажитесь от массива результата с лишними значениями.</li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">🐛 Отладка: если что-то не работает</h2>
<ul>
	<li>Если в результате появился ноль, проверьте условие <code>&gt; 0</code> и длину результата.</li>
	<li>Если порядок изменился, копируйте элементы при обычном проходе слева направо.</li>
	<li>Если исходный массив изменился, уберите любые присваивания в <code>src[index]</code>.</li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">💪 Дополнительный вызов (после сдачи задачи)</h2>
<p>Сделайте похожий метод, который возвращает только отрицательные числа.</p>

<h2 style="color: #34495e; margin-top: 28px;">🎨 Где это пригодится</h2>
<p>Фильтрация массива нужна при отборе валидных значений, заказов с положительной суммой, успешных результатов, активных идентификаторов и любых данных, где нужно оставить только элементы с нужным признаком.</p>
