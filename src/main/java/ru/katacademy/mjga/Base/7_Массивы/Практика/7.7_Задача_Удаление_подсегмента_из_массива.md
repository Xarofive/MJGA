<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">✂️ Удаление диапазона из массива</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться удалять диапазон элементов, создавая новый массив без выбранного участка.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code class="language-java">static int[] removeRange(int[] array, int from, int to)</code></pre>

<ul style="margin: 10px 0;">
	<li><code>from</code> &mdash; начальный индекс удаляемого диапазона, включительно</li>
	<li><code>to</code> &mdash; конечный индекс, не включительно</li>
	<li>Если <code>from &gt;= to</code>, <code>from &lt; 0</code> или <code>to &gt; array.length</code>, верните копию исходного массива</li>
	<li>Иначе верните новый массив без элементов с индексами от <code>from</code> до <code>to - 1</code></li>
	<li>Порядок оставшихся элементов должен сохраниться</li>
	<li>Решение должно содержать только метод без вывода в консоль</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Примеры работы</h2>

<ul>
	<li><code>removeRange(new int[]{0, 1, 2, 3, 4, 5, 6}, 2, 5)</code> → <code>[0, 1, 5, 6]</code></li>
	<li><code>removeRange(new int[]{1, 2, 3}, 0, 2)</code> → <code>[3]</code></li>
	<li><code>removeRange(new int[]{1, 2, 3}, 1, 1)</code> → <code>[1, 2, 3]</code></li>
</ul>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #c62828; margin-top: 0;">⚠️ Частые ошибки</h4>
<p><strong>Ошибка 1:</strong> считать <code>to</code> включительным индексом. В задаче <code>to</code> не входит в удаляемый диапазон.</p>
<p><strong>Ошибка 2:</strong> не проверить границы перед копированием.</p>
<p style="margin-bottom: 0;"><strong>Ошибка 3:</strong> изменить исходный массив вместо создания нового.</p>
</div>
