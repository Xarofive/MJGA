<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">⏱️ Обратный отсчёт в массиве</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Освоить классический цикл с декрементом и создание массива результата заранее известной длины.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Сценарий</h2>
<p>Нужно подготовить массив для обратного отсчёта: от заданного числа до нуля. Такой массив можно вернуть из метода и дальше использовать в тестах, таймерах или учебной визуализации.</p>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<ul>
	<li>Длина массива задаётся при создании и не меняется.</li>
	<li>Для значения <code>start</code> результат содержит <code>start + 1</code> элементов, потому что ноль тоже входит.</li>
	<li>Классический <code>for</code> удобен, когда нужно управлять индексом и значением отдельно.</li>
	<li>Отрицательное значение не даёт корректного отсчёта, поэтому возвращается пустой массив.</li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p><strong>Реализуйте метод:</strong></p>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px;"><code class="language-java">static int[] buildCountdown(int start)</code></pre>
<ul>
	<li>Метод принимает начальное значение <code>start</code>.</li>
	<li>Возвращает массив чисел от <code>start</code> до <code>0</code> включительно.</li>
	<li>Если <code>start &lt; 0</code>, возвращает пустой массив <code>new int[0]</code>.</li>
	<li>Для неотрицательного <code>start</code> длина результата равна <code>start + 1</code>.</li>
	<li>Используйте классический цикл с декрементом.</li>
	<li>Решение должно содержать только метод без вывода в консоль.</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Результат программы</h2>
<ul>
	<li><code>buildCountdown(5)</code> → <code>[5, 4, 3, 2, 1, 0]</code></li>
	<li><code>buildCountdown(0)</code> → <code>[0]</code></li>
	<li><code>buildCountdown(-5)</code> → <code>[]</code></li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">🤔 Момент для размышления</h2>
<p>Почему для <code>start = 5</code> длина массива равна 6, а не 5?</p>

<h2 style="color: #34495e; margin-top: 28px;">📝 Структура программы</h2>
<ol>
	<li>Проверьте отрицательное значение и верните пустой массив.</li>
	<li>Создайте массив длиной <code>start + 1</code>.</li>
	<li>Заполните его значениями от <code>start</code> до <code>0</code>.</li>
	<li>Верните заполненный массив.</li>
</ol>

	<li><label><input type="checkbox"> Для отрицательного входа возвращается пустой массив.</label></li>

<h2 style="color: #34495e; margin-top: 28px;">⚠️ Технические ограничения</h2>
<ul>
	<li>Не печатайте результат: автопроверка ожидает возвращаемый массив.</li>
	<li>Не используйте коллекции вместо массива.</li>
	<li>Не пропускайте обработку <code>start &lt; 0</code>.</li>
	<li>Откажитесь от готового ручного массива для конкретного числа.</li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">🐛 Отладка: если что-то не работает</h2>
<ul>
	<li>Если ноль пропал, проверьте условие цикла: оно должно включать <code>0</code>.</li>
	<li>Если цикл не заканчивается, проверьте декремент <code>i--</code>.</li>
	<li>Если возникает ошибка размера массива, сначала обрабатывайте отрицательный <code>start</code>.</li>
</ul>
