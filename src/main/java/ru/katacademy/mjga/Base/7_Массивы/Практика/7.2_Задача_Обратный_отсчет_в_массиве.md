<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">⏱️ Обратный отсчёт в массиве</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Освоить классический цикл с декрементом и создание массива результата заранее известной длины.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;"><strong>Реализуйте метод <code>buildCountdown</code>:</strong></p>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code class="language-java">static int[] buildCountdown(int start)</code></pre>

<ul style="margin: 10px 0;">
	<li>Принимает начальное значение <code>start</code></li>
	<li>Возвращает массив чисел от <code>start</code> до <code>0</code> включительно</li>
	<li>Если <code>start &lt; 0</code>, возвращает пустой массив <code>new int[0]</code></li>
	<li>Для неотрицательного <code>start</code> длина результата равна <code>start + 1</code></li>
	<li>Используйте классический цикл с декрементом</li>
	<li>Решение должно содержать только метод без вывода в консоль</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Примеры работы</h2>

<ul>
	<li><code>buildCountdown(5)</code> → <code>[5, 4, 3, 2, 1, 0]</code></li>
	<li><code>buildCountdown(0)</code> → <code>[0]</code></li>
	<li><code>buildCountdown(-5)</code> → <code>[]</code></li>
</ul>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #c62828; margin-top: 0;">⚠️ Частые ошибки</h4>
<p><strong>Ошибка 1:</strong> условие <code>i &gt; 0</code> пропускает ноль.</p>
<p><strong>Ошибка 2:</strong> инкремент <code>i++</code> вместо декремента <code>i--</code>.</p>
<p style="margin-bottom: 0;"><strong>Ошибка 3:</strong> забыть обработать отрицательный <code>start</code>.</p>
</div>
