<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">📋 Независимая копия массива</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Написать метод <code>cloneArray</code>, который возвращает полностью независимую копию массива целых чисел.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Сценарий</h2>
<p>Метод получает массив, который может использоваться где-то ещё. Нужно вернуть копию, чтобы изменение результата не меняло исходные данные через ту же ссылку.</p>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<ul>
	<li>В Java переменная массива хранит ссылку на объект массива.</li>
	<li><code>int[] copy = src;</code> не копирует элементы, а создаёт вторую ссылку на тот же массив.</li>
	<li>Независимая копия требует создания нового массива.</li>
	<li><code>null</code> нужно обработать до обращения к <code>src.length</code>.</li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p><strong>Реализуйте метод:</strong></p>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px;"><code class="language-java">static int[] cloneArray(int[] src)</code></pre>
<ul>
	<li>Если <code>src == null</code>, верните <code>null</code>.</li>
	<li>Для непустого и пустого массива верните новый массив той же длины.</li>
	<li>Скопируйте все элементы из <code>src</code> в новый массив.</li>
	<li>Изменение копии не должно менять исходный массив.</li>
	<li>Решение должно содержать только метод без вывода в консоль.</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Результат программы</h2>
<ul>
	<li><code>cloneArray(new int[]{1, 2, 3})</code> → <code>[1, 2, 3]</code></li>
	<li><code>cloneArray(new int[]{})</code> → <code>[]</code></li>
	<li><code>cloneArray(null)</code> → <code>null</code></li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">🤔 Момент для размышления</h2>
<p>Почему после <code>int[] copy = src;</code> изменение <code>copy[0]</code> меняет и <code>src[0]</code>?</p>

<h2 style="color: #34495e; margin-top: 28px;">📝 Структура программы</h2>
<ol>
	<li>Проверьте <code>src == null</code>.</li>
	<li>Создайте новый массив длиной <code>src.length</code>.</li>
	<li>Скопируйте элементы циклом по индексам.</li>
	<li>Верните новый массив.</li>
</ol>

<h2 style="color: #34495e; margin-top: 28px;">✅ Чек-лист проверки</h2>
<div style="background: #e8f5e9; border: 1px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">

	<li><label><input type="checkbox"> Метод называется <code>cloneArray</code>.</label></li>
	<li><label><input type="checkbox"> Параметр имеет тип <code>int[]</code>.</label></li>
	<li><label><input type="checkbox"> Обработан <code>null</code>.</label></li>
	<li><label><input type="checkbox"> Возвращается новый массив.</label></li>
	<li><label><input type="checkbox"> Изменение копии не влияет на оригинал.</label></li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">⚠️ Технические ограничения</h2>
<ul>
	<li>Не возвращайте <code>src</code> для непустого или пустого массива.</li>
	<li>Не используйте вывод в консоль.</li>
	<li>Не меняйте элементы исходного массива.</li>
	<li>Откажитесь от присваивания ссылки вместо копирования данных.</li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">🐛 Отладка: если что-то не работает</h2>
<ul>
	<li>Если падает <code>NullPointerException</code>, проверьте обработку <code>null</code> до <code>src.length</code>.</li>
	<li>Если изменение копии меняет оригинал, вы вернули ту же ссылку.</li>
	<li>Если в копии нули, проверьте цикл копирования элементов.</li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">💪 Дополнительный вызов (после сдачи задачи)</h2>
<p>Реализуйте копирование через <code>System.arraycopy</code> и сравните решение с циклом.</p>
