<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">Независимая копия массива</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">Цель задачи</h3>
<p style="margin: 0;">Написать метод <code>cloneArray</code>, который возвращает полностью независимую копию массива целых чисел.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">Контекст и мотивация</h3>
<p style="margin-bottom: 10px;">При присваивании массива копируется ссылка, а не данные. Если изменить массив через одну переменную, изменение будет видно через другую. Независимая копия защищает исходные данные от случайных изменений.</p>
<p style="margin: 0;"><strong>Задача:</strong> реализовать метод <code>cloneArray(src)</code>, возвращающий новый массив с теми же элементами.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code class="language-java">static int[] cloneArray(int[] src)</code></pre>

<ul style="margin: 10px 0;">
	<li>Если <code>src == null</code>, верните <code>null</code></li>
	<li>Для непустого и пустого массива верните новый массив той же длины</li>
	<li>Скопируйте все элементы из <code>src</code> в новый массив</li>
	<li>Изменение копии не должно менять исходный массив</li>
	<li>Решение должно содержать только метод без вывода в консоль</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Примеры работы</h2>

<ul>
	<li><code>cloneArray(new int[]{1, 2, 3})</code> → <code>[1, 2, 3]</code></li>
	<li><code>cloneArray(new int[]{})</code> → <code>[]</code></li>
	<li><code>cloneArray(null)</code> → <code>null</code></li>
</ul>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #c62828; margin-top: 0;">Частые ошибки</h4>
<p><strong>Ошибка 1:</strong> <code>int[] clone = src;</code> копирует ссылку, а не массив.</p>
<p><strong>Ошибка 2:</strong> забыть обработать <code>null</code>.</p>
<p style="margin-bottom: 0;"><strong>Ошибка 3:</strong> создать новый массив, но не скопировать элементы.</p>
</div>
