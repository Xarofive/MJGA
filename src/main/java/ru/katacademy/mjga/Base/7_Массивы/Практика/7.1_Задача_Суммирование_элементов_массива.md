<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🔢 Суммирование элементов массива</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Освоить обход массива через enhanced for и накопление суммы с помощью оператора <code>+=</code>.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">💼 Контекст и мотивация</h3>
<p style="margin-bottom: 10px;">Обработка коллекций данных встречается в корзинах интернет-магазинов, финансовых отчётах, аналитике и мониторинге. Частый базовый шаг &mdash; пройти по набору чисел и получить итоговую сумму.</p>
<p style="margin: 0;"><strong>Задача:</strong> вычислить сумму всех элементов массива.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;"><strong>Создайте программу для вычисления суммы элементов массива:</strong></p>

<ul style="margin: 10px 0;">
	<li>Создайте массив <code>numbers</code> с точными значениями: <code>{10, 20, 30, 40, 50}</code></li>
	<li>Объявите переменную <code>sum</code> до цикла и инициализируйте её нулём</li>
	<li>Используйте enhanced for: <code>for (int value : numbers)</code></li>
	<li>Внутри цикла добавляйте текущий элемент к сумме через <code>sum += value</code></li>
	<li>После цикла выведите сумму через <code>System.out.println(sum)</code></li>
	<li>В программе должен быть один цикл и один вывод после цикла</li>
</ul>
</div>

<h4 style="color: #c62828; margin-top: 0;">⚠️ Частые ошибки</h4>
<p><strong>Ошибка 1:</strong> выводить промежуточные значения внутри цикла. Нужен только итог.</p>
<p><strong>Ошибка 2:</strong> использовать <code>sum++</code>. Это прибавляет единицу, а не значение элемента.</p>
<p style="margin-bottom: 0;"><strong>Ошибка 3:</strong> менять значения массива. В этой задаче массив только читается.</p>
