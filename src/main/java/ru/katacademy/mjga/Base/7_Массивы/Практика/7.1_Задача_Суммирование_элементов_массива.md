<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🔢 Суммирование элементов массива</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Освоить обход массива через enhanced for и накопление суммы с помощью оператора <code>+=</code>.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Сценарий</h2>
<p>У вас есть набор чисел, например значения в отчёте или стоимости товаров. Нужно пройти по всем элементам массива и получить один итог: сумму.</p>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<ul>
	<li>Массив хранит несколько значений одного типа.</li>
	<li>Enhanced for удобен, когда нужен каждый элемент без работы с индексами.</li>
	<li><code>sum += value</code> означает: взять старую сумму и добавить текущий элемент.</li>
	<li>Итоговый вывод нужно делать после цикла, когда сумма уже собрана полностью.</li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<ul style="margin: 10px 0;">
	<li>Создайте полную Java-программу с классом <code>Main</code>.</li>
	<li>Создайте массив <code>numbers</code> со значениями <code>{10, 20, 30, 40, 50}</code>.</li>
	<li>Объявите переменную <code>sum</code> до цикла и инициализируйте её нулём.</li>
	<li>Используйте enhanced for: <code>for (int value : numbers)</code>.</li>
	<li>Внутри цикла добавляйте текущий элемент к сумме через <code>sum += value</code>.</li>
	<li>После цикла выведите сумму через <code>System.out.println(sum)</code>.</li>
	<li>В программе должен быть один цикл и один вывод после цикла.</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">⚠️ Технические ограничения</h2>
<ul>
	<li>Не выводите промежуточные суммы внутри цикла.</li>
	<li>Не меняйте значения исходного массива.</li>
	<li>Не используйте готовый ответ <code>150</code> напрямую в выводе.</li>
	<li>Откажитесь от дополнительных методов: здесь нужна простая программа в <code>main</code>.</li>

<h2 style="color: #34495e; margin-top: 28px;">🐛 Отладка: если что-то не работает</h2>
<ul>
	<li>Если вывод не 150, проверьте значения массива и начальное значение <code>sum</code>.</li>
	<li>Если выводится несколько строк, перенесите <code>println</code> за пределы цикла.</li>
	<li>Если сумма слишком маленькая, убедитесь, что добавляется именно <code>value</code>, а не единица.</li>
</ul>
