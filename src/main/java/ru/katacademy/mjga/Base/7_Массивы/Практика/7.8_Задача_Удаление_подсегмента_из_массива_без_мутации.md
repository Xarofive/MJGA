<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">✂️ Удаление диапазона без изменения исходного массива</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Закрепить удаление диапазона и отдельно проверить, что результат не разделяет изменяемое состояние с исходным массивом.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">💼 Контекст и мотивация</h3>
<p style="margin-bottom: 10px;">В Java массивы передаются в методы по ссылке на объект. Поэтому метод, который возвращает результат обработки, должен явно создавать новый массив, если исходные данные нельзя менять.</p>
<p style="margin: 0;"><strong>Задача:</strong> реализовать безопасное удаление диапазона с гарантией независимого результата.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code class="language-java">static int[] removeRangeSafe(int[] array, int from, int to)</code></pre>

<ul style="margin: 10px 0;">
	<li>Метод удаляет диапазон <code>[from, to)</code></li>
	<li>При некорректных границах возвращает независимую копию исходного массива</li>
	<li>При корректных границах возвращает новый массив без удаляемого диапазона</li>
	<li>Возвращённый массив не должен быть тем же объектом, что и <code>array</code></li>
	<li>Если <code>array == null</code>, верните <code>null</code></li>
	<li>Решение должно содержать только метод без вывода в консоль</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Примеры работы</h2>

<ul>
	<li><code>removeRangeSafe(new int[]{0, 1, 2, 3, 4}, 1, 4)</code> → <code>[0, 4]</code></li>
	<li><code>removeRangeSafe(new int[]{1, 2, 3}, -1, 2)</code> → <code>[1, 2, 3]</code>, но это новый массив</li>
	<li><code>removeRangeSafe(null, 0, 1)</code> → <code>null</code></li>
</ul>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #c62828; margin-top: 0;">⚠️ Частые ошибки</h4>
<p><strong>Ошибка 1:</strong> вернуть <code>array</code> при некорректных границах. Нужна копия.</p>
<p><strong>Ошибка 2:</strong> забыть обработать <code>null</code>.</p>
<p style="margin-bottom: 0;"><strong>Ошибка 3:</strong> удалить неправильное количество элементов из-за включительного понимания <code>to</code>.</p>
</div>
