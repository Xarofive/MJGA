<p style="margin: 0;">Научиться обходить массив, определять чётность через оператор <code>%</code> и вести два независимых счётчика.</p>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;"><strong>Создайте программу для подсчёта чётных и нечётных чисел:</strong></p>

<ul style="margin: 10px 0;">
	<li>Создайте массив <code>nums</code> со значениями <code>{1, 2, 3, 4, 5, 6}</code></li>
	<li>Объявите счётчики <code>evenCount</code> и <code>oddCount</code> до цикла</li>
	<li>Используйте один enhanced for для обхода массива</li>
	<li>Число чётное, если <code>value % 2 == 0</code></li>
	<li>После цикла выведите два числа одной строкой: сначала чётные, затем нечётные</li>
	<li>Не используйте <code>break</code>, <code>continue</code> и дополнительные методы</li>
