<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🔗 Объединение двух массивов</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Написать метод <code>mergeArrays</code>, который создаёт новый массив из элементов двух исходных массивов.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code class="language-java">static int[] mergeArrays(int[] a, int[] b)</code></pre>

<ul style="margin: 10px 0;">
	<li>Метод принимает два массива <code>a</code> и <code>b</code></li>
	<li>Возвращает новый массив: сначала все элементы <code>a</code>, затем все элементы <code>b</code></li>
	<li>Исходные массивы изменять нельзя</li>
	<li>Пустые массивы должны обрабатываться корректно</li>
	<li>Можно использовать циклы или <code>System.arraycopy</code></li>
	<li>Решение должно содержать только метод без вывода в консоль</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Примеры работы</h2>

<ul>
	<li><code>mergeArrays(new int[]{1, 2}, new int[]{3, 4, 5})</code> → <code>[1, 2, 3, 4, 5]</code></li>
	<li><code>mergeArrays(new int[]{}, new int[]{1})</code> → <code>[1]</code></li>
	<li><code>mergeArrays(new int[]{10}, new int[]{})</code> → <code>[10]</code></li>
</ul>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #c62828; margin-top: 0;">⚠️ Частые ошибки</h4>
<p><strong>Ошибка 1:</strong> создать массив длиной только <code>a.length</code> или только <code>b.length</code>.</p>
<p><strong>Ошибка 2:</strong> перезаписать элементы первого массива при копировании второго.</p>
<p style="margin-bottom: 0;"><strong>Ошибка 3:</strong> вернуть один из исходных массивов вместо нового.</p>
</div>
