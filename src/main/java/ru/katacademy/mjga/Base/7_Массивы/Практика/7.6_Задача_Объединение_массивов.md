<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🔗 Объединение двух массивов</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Написать метод <code>mergeArrays</code>, который создаёт новый массив из элементов двух исходных массивов.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Сценарий</h2>
<p>Есть две части данных: первая и вторая. Нужно собрать их в один массив, сохранив порядок: сначала все элементы первого массива, затем все элементы второго.</p>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<ul>
	<li>Длина результата равна сумме длин двух массивов.</li>
	<li>Второй массив нужно копировать со смещением <code>a.length</code>.</li>
	<li>Исходные массивы не должны изменяться.</li>
	<li>Пустой массив тоже имеет длину и корректно участвует в объединении.</li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p><strong>Реализуйте метод:</strong></p>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px;"><code class="language-java">static int[] mergeArrays(int[] a, int[] b)</code></pre>
<ul>
	<li>Метод принимает два массива <code>a</code> и <code>b</code>.</li>
	<li>Возвращает новый массив: сначала все элементы <code>a</code>, затем все элементы <code>b</code>.</li>
	<li>Исходные массивы изменять нельзя.</li>
	<li>Пустые массивы должны обрабатываться корректно.</li>
	<li>Можно использовать циклы или <code>System.arraycopy</code>.</li>
	<li>Решение должно содержать только метод без вывода в консоль.</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Результат программы</h2>
<ul>
	<li><code>mergeArrays(new int[]{1, 2}, new int[]{3, 4, 5})</code> → <code>[1, 2, 3, 4, 5]</code></li>
	<li><code>mergeArrays(new int[]{}, new int[]{1})</code> → <code>[1]</code></li>
	<li><code>mergeArrays(new int[]{10}, new int[]{})</code> → <code>[10]</code></li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">🤔 Момент для размышления</h2>
<p>Почему при копировании второго массива индекс результата должен начинаться не с нуля, а с <code>a.length</code>?</p>

<h2 style="color: #34495e; margin-top: 28px;">📝 Структура программы</h2>
<ol>
	<li>Создайте массив длиной <code>a.length + b.length</code>.</li>
	<li>Скопируйте элементы <code>a</code> в начало результата.</li>
	<li>Скопируйте элементы <code>b</code> после элементов <code>a</code>.</li>
	<li>Верните новый массив.</li>
</ol>

<h2 style="color: #34495e; margin-top: 28px;">✅ Чек-лист проверки</h2>
<div style="background: #e8f5e9; border: 1px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<ul style="margin-bottom: 0;">
	<li><label><input type="checkbox"> Метод называется <code>mergeArrays</code>.</label></li>
	<li><label><input type="checkbox"> Длина результата равна <code>a.length + b.length</code>.</label></li>
	<li><label><input type="checkbox"> Порядок элементов сохранён.</label></li>
	<li><label><input type="checkbox"> Пустые массивы работают корректно.</label></li>
	<li><label><input type="checkbox"> В методе нет вывода в консоль.</label></li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">⚠️ Технические ограничения</h2>
<ul>
	<li>Не возвращайте один из исходных массивов.</li>
	<li>Не меняйте элементы <code>a</code> и <code>b</code>.</li>
	<li>Не создавайте результат длиной только одного массива.</li>
	<li>Откажитесь от вывода в консоль: метод должен возвращать массив.</li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">🐛 Отладка: если что-то не работает</h2>
<ul>
	<li>Если часть элементов потерялась, проверьте длину результата.</li>
	<li>Если второй массив перезаписал первый, проверьте смещение при копировании.</li>
	<li>Если пустые массивы ломают метод, проверьте границы циклов.</li>
</ul>
