<h1 style="color: #2c3e50; border-bottom: 3px solid #3498db; padding-bottom: 10px;">Задача 3.4: Персонализированное приветствие</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться разделять формирование строки и её вывод.</p>
</div>

<h2 style="color: #34495e;">Условие</h2>
<p>Создайте программу с методом <code>buildGreeting</code>, который принимает имя и возвращает строку-приветствие. Метод <code>main</code> должен вызвать <code>buildGreeting</code> и вывести результат.</p>

<h2 style="color: #34495e;">Структура программы</h2>
<ul>
	<li>Класс <code>Main</code>.</li>
	<li>Ровно два метода: <code>main</code> и <code>buildGreeting</code>.</li>
</ul>

<h2 style="color: #34495e;">Сигнатура метода</h2>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 15px; border-radius: 8px; overflow-x: auto;"><code>static String buildGreeting(String name)</code></pre>

<h2 style="color: #34495e;">Требования</h2>
<ul>
	<li><code>buildGreeting</code> принимает один параметр <code>name</code> типа <code>String</code>.</li>
	<li><code>buildGreeting</code> возвращает строку, в которой используется переданное имя.</li>
	<li><code>buildGreeting</code> не должен ничего выводить на экран.</li>
	<li><code>main</code> вызывает <code>buildGreeting</code> с любым именем.</li>
	<li><code>main</code> выводит результат через <code>System.out.println</code>.</li>
</ul>

<h2 style="color: #34495e;">Примеры допустимого результата</h2>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 15px; border-radius: 8px; overflow-x: auto;"><code>buildGreeting("Анна") -> "Привет, Анна!"
buildGreeting("Мир")  -> "Добро пожаловать, Мир!"</code></pre>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #c62828; margin-top: 0;">Частые ошибки</h3>
<ul>
	<li><code>buildGreeting</code> печатает строку вместо возврата.</li>
	<li>В возвращаемой строке не используется параметр <code>name</code>.</li>
	<li>Метод объявлен как <code>void</code>, хотя должен вернуть <code>String</code>.</li>
	<li><code>main</code> выводит строку напрямую и не вызывает <code>buildGreeting</code>.</li>
</ul>
</div>
