<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">Задача 3.4: Персонализированное приветствие</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться разделять формирование строки и её вывод, создавая переиспользуемый метод для текстового сообщения.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">🎯 Сценарий</h3>
<p style="margin-bottom: 10px;">В приложении нужно сформировать приветствие для пользователя. Само приветствие может понадобиться в консоли, письме, чат-боте или веб-интерфейсе.</p>
<p style="margin: 0;"><strong>Ваша задача:</strong> создать метод <code>buildGreeting</code>, который возвращает готовую строку, а <code>main</code> выводит результат.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Разделение ответственности</h4>
<p>Метод, который формирует строку, не должен сам решать, где её показывать. Он возвращает значение, а вызывающий код использует его дальше.</p>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>static String buildGreeting(String name) {
    return "Привет, " + name + "!";
}</code></pre>
<p style="margin-bottom: 0;">Такой метод легче тестировать: можно проверить возвращаемую строку без анализа консольного вывода.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<p><strong>Создайте программу с двумя методами:</strong></p>
<ul>
	<li>Класс <code>Main</code>.</li>
	<li>Ровно два метода: <code>main</code> и <code>buildGreeting</code>.</li>
	<li><code>buildGreeting</code> принимает один параметр <code>name</code> типа <code>String</code>.</li>
	<li><code>buildGreeting</code> возвращает строку, в которой используется переданное имя.</li>
	<li><code>buildGreeting</code> не должен ничего выводить на экран.</li>
	<li><code>main</code> вызывает <code>buildGreeting</code> с любым именем и выводит результат.</li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Результат программы</h2>
<p>Допустимы разные тексты приветствия, если используется переданное имя:</p>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 16px; border-radius: 8px; overflow-x: auto;"><code>buildGreeting("Анна") -> "Привет, Анна!"
buildGreeting("Мир")  -> "Добро пожаловать, Мир!"</code></pre>

<div style="background: #fff3e0; border: 1px solid #ff9800; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #e65100; margin-top: 0;">🤔 Момент для размышления</h4>
<p style="margin-bottom: 0;">Почему <code>buildGreeting</code> должен возвращать строку, а не вызывать <code>System.out.println</code> внутри себя?</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📝 Структура программы</h2>
<pre style="background: #263238; color: #aed581; padding: 20px; border-radius: 8px; overflow-x: auto;"><code>public class Main {
    public static void main(String[] args) {
        String greeting = buildGreeting("Анна");
        System.out.println(greeting);
    }

    static String buildGreeting(String name) {
        return "Привет, " + name + "!";
    }
}</code></pre>

<h2 style="color: #34495e; margin-top: 28px;">✅ Чек-лист проверки</h2>
<div style="background: #f8f9fa; border-left: 4px solid #28a745; padding: 16px; margin: 16px 0;">
<ul style="margin-bottom: 0;">
	<li>☐ Есть ровно два метода: <code>main</code> и <code>buildGreeting</code></li>
	<li>☐ Сигнатура: <code>static String buildGreeting(String name)</code></li>
	<li>☐ В возвращаемой строке используется <code>name</code></li>
	<li>☐ Внутри <code>buildGreeting</code> есть <code>return</code></li>
	<li>☐ Вывод находится в <code>main</code>, а не в <code>buildGreeting</code></li>
</ul>
</div>

<div style="background: #fff8e1; border-left: 4px solid #ffc107; padding: 16px; margin: 16px 0;">
<h4 style="color: #f57c00; margin-top: 0;">⚠️ Технические ограничения</h4>
<ul style="margin-bottom: 0;">
	<li>Не объявляйте <code>buildGreeting</code> как <code>void</code></li>
	<li>Не добавляйте второй параметр</li>
	<li>Не создавайте дополнительные методы</li>
	<li>Откажитесь от прямого вывода строки в <code>main</code> без вызова <code>buildGreeting</code></li>
</ul>
</div>
