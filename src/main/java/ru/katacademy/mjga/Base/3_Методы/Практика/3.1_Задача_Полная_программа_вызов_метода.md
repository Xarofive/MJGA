<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">Задача 3.1: Полная программа — вызов метода</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Освоить создание полноценной Java-программы с методом <code>main</code> и вспомогательным методом, который вызывается из <code>main</code>.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">🎯 Сценарий</h3>
<p style="margin-bottom: 10px;">Вы выносите вывод приветственного баннера в отдельный метод, чтобы <code>main</code> не занимался деталями оформления.</p>
<p style="margin: 0;"><strong>Ваша задача:</strong> создать метод <code>showBanner</code> и вызвать его из <code>main</code>.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Вспомогательные методы</h4>
<p>Метод — это именованный блок кода. Его можно вызвать по имени с круглыми скобками.</p>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>static void showBanner() {
    System.out.println("Welcome!");
}</code></pre>
<p style="margin-bottom: 0;">В учебных задачах методы рядом с <code>main</code> объявляются как <code>static</code>, чтобы их можно было вызвать напрямую из <code>main</code>.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<p><strong>Создайте программу с методом вывода баннера:</strong></p>
<ul>
	<li>Класс должен называться <code>Main</code>.</li>
	<li>В классе должны быть ровно два метода: <code>main</code> и <code>showBanner</code>.</li>
	<li><code>main</code> должен вызвать <code>showBanner()</code>.</li>
	<li><code>showBanner</code> должен быть без параметров и без возвращаемого значения.</li>
	<li><code>showBanner</code> выводит любой текст или число через <code>System.out.println</code>.</li>
	<li><code>main</code> не должен выводить текст напрямую.</li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Результат программы</h2>
<p>Можно вывести любой баннер. Например:</p>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 16px; border-radius: 8px; overflow-x: auto;"><code>Welcome!</code></pre>

<div style="background: #fff3e0; border: 1px solid #ff9800; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #e65100; margin-top: 0;">🤔 Момент для размышления</h4>
<p style="margin-bottom: 0;">Почему лучше вынести баннер в отдельный метод, даже если сейчас он выводит всего одну строку?</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📝 Структура программы</h2>
<pre style="background: #263238; color: #aed581; padding: 20px; border-radius: 8px; overflow-x: auto;"><code>public class Main {
    public static void main(String[] args) {
        showBanner();
    }

    static void showBanner() {
        // вывод баннера
    }
}</code></pre>

<h2 style="color: #34495e; margin-top: 28px;">✅ Чек-лист проверки</h2>
<div style="background: #f8f9fa; border-left: 4px solid #28a745; padding: 16px; margin: 16px 0;">
<ul style="margin-bottom: 0;">
	<li>☐ Есть класс <code>Main</code></li>
	<li>☐ В программе ровно два метода: <code>main</code> и <code>showBanner</code></li>
	<li>☐ <code>main</code> вызывает <code>showBanner()</code></li>
	<li>☐ <code>showBanner</code> объявлен как <code>static void showBanner()</code></li>
	<li>☐ Вывод находится внутри <code>showBanner</code>, а не внутри <code>main</code></li>
</ul>
</div>

<div style="background: #fff8e1; border-left: 4px solid #ffc107; padding: 16px; margin: 16px 0;">
<h4 style="color: #f57c00; margin-top: 0;">⚠️ Технические ограничения</h4>
<ul style="margin-bottom: 0;">
	<li>Не создавайте дополнительные методы</li>
	<li>Не добавляйте параметры в <code>showBanner</code></li>
	<li>Не возвращайте значение из <code>showBanner</code></li>
	<li>Откажитесь от прямого <code>System.out.println</code> в <code>main</code></li>
</ul>
</div>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
