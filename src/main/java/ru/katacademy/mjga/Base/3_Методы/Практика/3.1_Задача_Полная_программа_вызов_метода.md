<h1 style="color: #2c3e50; border-bottom: 3px solid #3498db; padding-bottom: 10px;">Задача 3.1: Полная программа &mdash; вызов метода</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Освоить создание полноценной Java-программы с методом <code>main</code> и вспомогательным методом.</p>
</div>

<h2 style="color: #34495e;">Условие</h2>
<p>Создайте программу с методом <code>showBanner</code>, который выводит приветственный баннер.</p>

<h2 style="color: #34495e;">Требования</h2>
<ul>
	<li>Программа должна содержать класс <code>Main</code>.</li>
	<li>В классе должны быть ровно два метода: <code>main</code> и <code>showBanner</code>.</li>
	<li>Метод <code>main</code> должен вызвать <code>showBanner()</code>.</li>
	<li>Метод <code>showBanner</code> должен быть без параметров и без возвращаемого значения.</li>
	<li>Для вывода используйте <code>System.out.println</code>.</li>
	<li>Метод <code>main</code> не должен выводить текст напрямую.</li>
</ul>

<h2 style="color: #34495e;">Пример структуры</h2>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 15px; border-radius: 8px; overflow-x: auto;"><code>public class Main {
    public static void main(String[] args) {
        showBanner();
    }

    static void showBanner() {
        System.out.println("Welcome!");
    }
}</code></pre>

<h2 style="color: #34495e;">Что вывести</h2>
<p>Можно вывести любой текст или число. Например:</p>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 15px; border-radius: 8px; overflow-x: auto;"><code>Welcome!</code></pre>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #c62828; margin-top: 0;">Частые ошибки</h3>
<ul>
	<li>Забыли класс <code>Main</code>.</li>
	<li>Создали лишние методы.</li>
	<li>Вызвали <code>System.out.println</code> внутри <code>main</code>, а не внутри <code>showBanner</code>.</li>
	<li>Забыли вызвать <code>showBanner()</code>.</li>
	<li>Объявили <code>showBanner</code> без <code>static</code>.</li>
</ul>
</div>

<div style="background: #fff3cd; border-left: 4px solid #ffc107; padding: 15px; margin: 20px 0;">
<h3 style="color: #856404; margin-top: 0;">Подсказка</h3>
<p style="margin: 0;">Метод без параметров и результата объявляется так: <code>static void showBanner() { ... }</code>.</p>
