<h1 style="color: #2c3e50; border-bottom: 3px solid #3498db; padding-bottom: 10px;">Задача 3.7: Вычисление среднего арифметического</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться разбивать вычисление на промежуточные шаги с использованием переменных.</p>
</div>

<h2 style="color: #34495e;">Условие</h2>
<p>Создайте метод для вычисления среднего арифметического двух дробных чисел.</p>

<h2 style="color: #34495e;">Сигнатура метода</h2>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 15px; border-radius: 8px; overflow-x: auto;"><code>static double averagePair(double first, double second)</code></pre>

<h2 style="color: #34495e;">Требования</h2>
<ul>
	<li>Метод должен называться <code>averagePair</code>.</li>
	<li>Метод принимает два параметра типа <code>double</code>.</li>
	<li>Создайте переменную <code>total</code> типа <code>double</code> для суммы.</li>
	<li>Создайте переменную <code>average</code> типа <code>double</code> для результата.</li>
	<li>Верните <code>average</code> через <code>return</code>.</li>
	<li>Не выводите данные через <code>System.out.print*</code>.</li>
</ul>

<h2 style="color: #34495e;">Алгоритм</h2>
<ol>
	<li>Вычислите сумму <code>first + second</code> и сохраните её в <code>total</code>.</li>
	<li>Разделите <code>total</code> на <code>2.0</code> и сохраните результат в <code>average</code>.</li>
	<li>Верните <code>average</code>.</li>
</ol>

<h2 style="color: #34495e;">Примеры работы</h2>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 15px; border-radius: 8px; overflow-x: auto;"><code>averagePair(10.0, 20.0) -> 15.0
averagePair(3.5, 7.5)   -> 5.5
averagePair(-4.0, 12.0) -> 4.0
averagePair(7.0, 7.0)   -> 7.0</code></pre>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #c62828; margin-top: 0;">Частые ошибки</h3>
<ul>
	<li>Возвращается сумма, а не среднее значение.</li>
	<li>Используется тип <code>int</code> вместо <code>double</code>.</li>
	<li>Метод печатает результат вместо возврата.</li>
	<li>Переменные названы не по смыслу или не используются.</li>
</ul>
</div>

<div style="background: #fff3cd; border-left: 4px solid #ffc107; padding: 15px; margin: 20px 0;">
<h3 style="color: #856404; margin-top: 0;">Подсказка</h3>
<pre style="background: #1e293b; color: #e2e8f0; padding: 12px; border-radius: 5px;"><code>double total = first + second;
double average = total / 2.0;
return average;</code></pre>
