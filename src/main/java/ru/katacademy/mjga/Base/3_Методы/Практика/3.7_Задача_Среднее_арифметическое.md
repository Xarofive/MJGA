<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">Задача 3.7: Вычисление среднего арифметического</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться разбивать вычисление на промежуточные шаги с использованием переменных <code>total</code> и <code>average</code>.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">🎯 Сценарий</h3>
<p style="margin-bottom: 10px;">В аналитике часто нужно посчитать среднее значение и при этом сохранить промежуточную сумму для проверки.</p>
<p style="margin: 0;"><strong>Ваша задача:</strong> создать метод <code>averagePair</code>, который считает среднее арифметическое двух дробных чисел.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Среднее арифметическое</h4>
<p>Среднее двух чисел — это их сумма, разделённая на количество чисел.</p>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>double total = first + second;
double average = total / 2.0;
return average;</code></pre>
<p style="margin-bottom: 0;">Тип <code>double</code> нужен, чтобы корректно работать с дробными значениями.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<p><strong>Создайте метод с точной сигнатурой:</strong></p>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 16px; border-radius: 8px; overflow-x: auto;"><code>static double averagePair(double first, double second)</code></pre>
<ol>
	<li>Вычислите сумму <code>first + second</code> и сохраните её в <code>double total</code>.</li>
	<li>Разделите <code>total</code> на <code>2.0</code> и сохраните результат в <code>double average</code>.</li>
	<li>Верните <code>average</code> через <code>return</code>.</li>
</ol>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Результат программы</h2>
<p>Автопроверка будет вызывать метод с разными аргументами:</p>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 16px; border-radius: 8px; overflow-x: auto;"><code>averagePair(10.0, 20.0) -> 15.0
averagePair(3.5, 7.5)   -> 5.5
averagePair(-4.0, 12.0) -> 4.0
averagePair(7.0, 7.0)   -> 7.0</code></pre>

<div style="background: #fff3e0; border: 1px solid #ff9800; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #e65100; margin-top: 0;">🤔 Момент для размышления</h4>
<p style="margin-bottom: 0;">Почему в этой задаче делим на <code>2.0</code>, а не на <code>2</code>? Как это подчёркивает работу с дробными числами?</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📝 Структура метода</h2>
<pre style="background: #263238; color: #aed581; padding: 20px; border-radius: 8px; overflow-x: auto;"><code>static double averagePair(double first, double second) {
    double total = first + second;
    double average = total / 2.0;
    return average;
}</code></pre>

<h2 style="color: #34495e; margin-top: 28px;">✅ Чек-лист проверки</h2>
<div style="background: #f8f9fa; border-left: 4px solid #28a745; padding: 16px; margin: 16px 0;">
<ul style="margin-bottom: 0;">
	<li>☐ Метод называется <code>averagePair</code></li>
	<li>☐ Сигнатура: <code>static double averagePair(double first, double second)</code></li>
	<li>☐ Есть переменная <code>double total</code></li>
	<li>☐ Есть переменная <code>double average</code></li>
	<li>☐ Возвращается среднее значение через <code>return average</code></li>
	<li>☐ Нет вывода через <code>System.out.print*</code></li>
</ul>
</div>

<div style="background: #fff8e1; border-left: 4px solid #ffc107; padding: 16px; margin: 16px 0;">
<h4 style="color: #f57c00; margin-top: 0;">⚠️ Технические ограничения</h4>
<ul style="margin-bottom: 0;">
	<li>Не используйте <code>int</code> для параметров, <code>total</code> или <code>average</code></li>
	<li>Не выводите результат внутри метода</li>
	<li>Не создавайте дополнительные методы</li>
	<li>Откажитесь от возврата суммы: нужно вернуть именно среднее значение</li>
</ul>
</div>
