<h1 style="color: #2c3e50; border-bottom: 3px solid #3498db; padding-bottom: 10px;">Задача 3.6: Калькулятор скидок</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться создавать вычислительный метод, который принимает параметры и возвращает результат.</p>
</div>

<h2 style="color: #34495e;">Условие</h2>
<p>Создайте программу с методом <code>discount</code>, который рассчитывает финальную цену товара после скидки.</p>

<h2 style="color: #34495e;">Сигнатура метода</h2>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 15px; border-radius: 8px; overflow-x: auto;"><code>static int discount(int price, int percent)</code></pre>

<h2 style="color: #34495e;">Требования</h2>
<ul>
	<li>Класс должен называться <code>Main</code>.</li>
	<li>В программе должны быть методы <code>main</code> и <code>discount</code>.</li>
	<li><code>discount</code> принимает исходную цену <code>price</code> и процент скидки <code>percent</code>.</li>
	<li><code>discount</code> возвращает финальную цену после применения скидки.</li>
	<li><code>discount</code> не должен ничего выводить на экран.</li>
	<li><code>main</code> должен вызвать <code>discount</code> и вывести результат через <code>System.out.println</code>.</li>
</ul>

<h2 style="color: #34495e;">Формула</h2>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 15px; border-radius: 8px; overflow-x: auto;"><code>размерСкидки = price * percent / 100
финальнаяЦена = price - размерСкидки</code></pre>

<h2 style="color: #34495e;">Примеры работы</h2>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 15px; border-radius: 8px; overflow-x: auto;"><code>discount(1000, 20) -> 800
discount(500, 50)  -> 250
discount(750, 0)   -> 750
discount(200, 100) -> 0
discount(999, 1)   -> 990</code></pre>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #c62828; margin-top: 0;">Частые ошибки</h3>
<ul>
	<li><code>discount</code> печатает результат вместо возврата.</li>
	<li>Не используется один из параметров.</li>
	<li>Неверный порядок операций при расчёте процента.</li>
	<li>Метод объявлен как <code>void</code>.</li>
</ul>
</div>

<div style="background: #fff3cd; border-left: 4px solid #ffc107; padding: 15px; margin: 20px 0;">
<h3 style="color: #856404; margin-top: 0;">Подсказка</h3>
<pre style="background: #1e293b; color: #e2e8f0; padding: 12px; border-radius: 5px;"><code>int discountAmount = price * percent / 100;
return price - discountAmount;</code></pre>
