<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">Задача 3.6: Калькулятор скидок</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться создавать вычислительный метод, который принимает параметры, выполняет расчёт и возвращает результат.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">🎯 Сценарий</h3>
<p style="margin-bottom: 10px;">В интернет-магазине нужно посчитать финальную цену товара после применения скидки.</p>
<p style="margin: 0;"><strong>Ваша задача:</strong> реализовать метод <code>discount</code>, который получает цену и процент скидки, а возвращает цену после скидки.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Вычислительный метод</h4>
<p>Метод с бизнес-логикой должен считать и возвращать значение. Вывод результата лучше оставить вызывающему коду.</p>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>int discountAmount = price * percent / 100;
return price - discountAmount;</code></pre>
<p style="margin-bottom: 0;">Такой метод легче тестировать: для одинаковых аргументов он всегда возвращает одинаковый результат.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<p><strong>Создайте программу с методом расчёта скидки:</strong></p>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 16px; border-radius: 8px; overflow-x: auto;"><code>static int discount(int price, int percent)</code></pre>
<ul>
	<li>Класс должен называться <code>Main</code>.</li>
	<li>В программе должны быть методы <code>main</code> и <code>discount</code>.</li>
	<li><code>discount</code> принимает исходную цену <code>price</code> и процент скидки <code>percent</code>.</li>
	<li><code>discount</code> возвращает финальную цену после применения скидки.</li>
	<li><code>discount</code> не должен ничего выводить на экран.</li>
	<li><code>main</code> должен вызвать <code>discount</code> и вывести результат через <code>System.out.println</code>.</li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Результат программы</h2>
<p>Метод должен проходить такие проверки:</p>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 16px; border-radius: 8px; overflow-x: auto;"><code>discount(1000, 20) -> 800
discount(500, 50)  -> 250
discount(750, 0)   -> 750
discount(200, 100) -> 0
discount(999, 1)   -> 990</code></pre>

<div style="background: #fff3e0; border: 1px solid #ff9800; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #e65100; margin-top: 0;">🤔 Момент для размышления</h4>
<p style="margin-bottom: 0;">Почему <code>discount(999, 1)</code> возвращает <code>990</code>, а не <code>989.01</code>? Подумайте о целочисленном делении.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📝 Структура программы</h2>
<pre style="background: #263238; color: #aed581; padding: 20px; border-radius: 8px; overflow-x: auto;"><code>public class Main {
    public static void main(String[] args) {
        int result = discount(1000, 20);
        System.out.println(result);
    }

    static int discount(int price, int percent) {
        int discountAmount = price * percent / 100;
        return price - discountAmount;
    }
}</code></pre>

<h2 style="color: #34495e; margin-top: 28px;">✅ Чек-лист проверки</h2>
<div style="background: #f8f9fa; border-left: 4px solid #28a745; padding: 16px; margin: 16px 0;">
<ul style="margin-bottom: 0;">
	<li>☐ Есть методы <code>main</code> и <code>discount</code></li>
	<li>☐ Сигнатура: <code>static int discount(int price, int percent)</code></li>
	<li>☐ В расчёте используются оба параметра</li>
	<li>☐ <code>discount</code> возвращает результат через <code>return</code></li>
	<li>☐ Вывод результата находится в <code>main</code>, а не в <code>discount</code></li>
</ul>
</div>

<div style="background: #fff8e1; border-left: 4px solid #ffc107; padding: 16px; margin: 16px 0;">
<h4 style="color: #f57c00; margin-top: 0;">⚠️ Технические ограничения</h4>
<ul style="margin-bottom: 0;">
	<li>Не объявляйте <code>discount</code> как <code>void</code></li>
	<li>Не используйте <code>double</code>: задача тренирует целочисленный расчёт</li>
	<li>Не выводите данные внутри <code>discount</code></li>
	<li>Откажитесь от готовых ответов для конкретных примеров: метод должен считать по формуле</li>
</ul>
</div>
