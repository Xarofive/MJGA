<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">Задача 3.2: Первый метод с параметрами — сумма двух чисел</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться создавать метод с параметрами и возвращаемым значением.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">🎯 Сценарий</h3>
<p style="margin-bottom: 10px;">Вы пишете первый универсальный вычислительный метод: он получает разные числа при разных вызовах и возвращает результат.</p>
<p style="margin: 0;"><strong>Ваша задача:</strong> создать метод <code>sum</code>, который принимает два целых числа и возвращает их сумму.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Параметры и return</h4>
<p>Параметры — это входные данные метода. Возвращаемый тип показывает, какой результат метод отдаёт вызывающему коду.</p>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>static int sum(int a, int b) {
    return a + b;
}</code></pre>
<p style="margin-bottom: 0;">Метод с типом возврата <code>int</code> обязан вернуть целое число через <code>return</code>.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<p><strong>Создайте метод с точной сигнатурой:</strong></p>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 16px; border-radius: 8px; overflow-x: auto;"><code>static int sum(int a, int b)</code></pre>
<ul>
	<li>Метод должен называться <code>sum</code>.</li>
	<li>Метод принимает два параметра типа <code>int</code>.</li>
	<li>Метод возвращает сумму двух параметров.</li>
	<li>Используйте <code>return</code>.</li>
	<li>Не выводите результат на экран внутри метода.</li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Результат программы</h2>
<p>Автопроверка будет вызывать метод с разными аргументами:</p>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 16px; border-radius: 8px; overflow-x: auto;"><code>sum(2, 3)     -> 5
sum(0, 0)     -> 0
sum(-1, 5)    -> 4
sum(100, 200) -> 300</code></pre>

<div style="background: #fff3e0; border: 1px solid #ff9800; border-radius: 8px; padding: 14px; margin: 16px 0;">

<p style="margin-bottom: 0;">Почему метод должен возвращать значение, а не печатать его? Как автопроверка сможет сравнить результат, если метод только выводит текст?</p>
</div>

}</code></pre>
