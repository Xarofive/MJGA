<h1 style="color: #2c3e50; border-bottom: 3px solid #3498db; padding-bottom: 10px;">Задача 3.2: Первый метод с параметрами &mdash; сумма двух чисел</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться создавать метод с параметрами и возвращаемым значением.</p>
</div>

<h2 style="color: #34495e;">Условие</h2>
<p>Создайте метод, который принимает два целых числа и возвращает их сумму.</p>

<h2 style="color: #34495e;">Сигнатура метода</h2>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 15px; border-radius: 8px; overflow-x: auto;"><code>static int sum(int a, int b)</code></pre>

<h2 style="color: #34495e;">Требования</h2>
<ul>
	<li>Метод должен называться <code>sum</code>.</li>
	<li>Метод принимает два параметра типа <code>int</code>.</li>
	<li>Метод возвращает сумму двух параметров.</li>
	<li>Не выводите результат на экран внутри метода.</li>
	<li>Используйте <code>return</code>.</li>
</ul>

<h2 style="color: #34495e;">Примеры работы</h2>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 15px; border-radius: 8px; overflow-x: auto;"><code>sum(2, 3)     -> 5
sum(0, 0)     -> 0
sum(-1, 5)    -> 4
sum(100, 200) -> 300</code></pre>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #c62828; margin-top: 0;">Частые ошибки</h3>
<ul>
	<li><code>static void sum(...)</code> &mdash; неверно, метод должен возвращать <code>int</code>.</li>
	<li><code>System.out.println(a + b)</code> внутри метода &mdash; неверно, нужно вернуть значение.</li>
	<li>Забыли <code>return</code>.</li>
	<li>Перепутали сложение с другой операцией.</li>
</ul>
</div>

<div style="background: #fff3cd; border-left: 4px solid #ffc107; padding: 15px; margin: 20px 0;">
<h3 style="color: #856404; margin-top: 0;">Подсказка</h3>
<pre style="background: #1e293b; color: #e2e8f0; padding: 12px; border-radius: 5px;"><code>return a + b;</code></pre>
