<h1 style="color: #2c3e50; border-bottom: 3px solid #3498db; padding-bottom: 10px;">Задача 3.5: Персональное приветствие с возрастом</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться создавать метод с несколькими параметрами разных типов.</p>
</div>

<h2 style="color: #34495e;">Условие</h2>
<p>Создайте программу с методом <code>personalGreeting</code>, который принимает имя и возраст, а затем выводит персональное приветствие.</p>

<h2 style="color: #34495e;">Структура программы</h2>
<ul>
	<li>Класс <code>Main</code>.</li>
	<li>Ровно два метода: <code>main</code> и <code>personalGreeting</code>.</li>
</ul>

<h2 style="color: #34495e;">Сигнатура метода</h2>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 15px; border-radius: 8px; overflow-x: auto;"><code>static void personalGreeting(String name, int age)</code></pre>

<h2 style="color: #34495e;">Требования</h2>
<ul>
	<li>Метод принимает два параметра: <code>name</code> типа <code>String</code> и <code>age</code> типа <code>int</code>.</li>
	<li>Метод ничего не возвращает.</li>
	<li>Метод использует <code>System.out.println</code> для вывода приветствия.</li>
	<li>В выводе должны использоваться оба параметра.</li>
	<li><code>main</code> вызывает <code>personalGreeting</code> с любыми корректными аргументами.</li>
	<li><code>main</code> не должен выводить текст напрямую.</li>
</ul>

<h2 style="color: #34495e;">Пример вывода</h2>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 15px; border-radius: 8px; overflow-x: auto;"><code>Привет, Анна! Тебе 25 лет.</code></pre>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #c62828; margin-top: 0;">Частые ошибки</h3>
<ul>
	<li>Не используется один из параметров.</li>
	<li>Метод возвращает строку, хотя должен быть <code>void</code>.</li>
	<li>Вызов метода содержит только один аргумент.</li>
	<li>Вывод находится в <code>main</code>, а не в <code>personalGreeting</code>.</li>
</ul>
</div>
