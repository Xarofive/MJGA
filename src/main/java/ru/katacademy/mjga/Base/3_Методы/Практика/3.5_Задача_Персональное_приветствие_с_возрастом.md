<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">Задача 3.5: Персональное приветствие с возрастом</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться создавать метод с несколькими параметрами разных типов и использовать их в одном выводе.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">🎯 Сценарий</h3>
<p style="margin-bottom: 10px;">Система регистрации выводит приветствие участнику, используя текстовые данные и число: имя и возраст.</p>
<p style="margin: 0;"><strong>Ваша задача:</strong> создать метод <code>personalGreeting</code>, который принимает <code>String</code> и <code>int</code>, а затем выводит сообщение.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Несколько параметров</h4>
<p>Метод может принимать несколько параметров разных типов. Порядок аргументов при вызове должен совпадать с порядком параметров.</p>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>static void personalGreeting(String name, int age) {
    System.out.println("Привет, " + name + "! Тебе " + age + " лет.");
}</code></pre>
<p style="margin-bottom: 0;">Так как метод сам выводит сообщение, его тип возврата — <code>void</code>.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<p><strong>Создайте программу с двумя методами:</strong></p>
<ul>
	<li>Класс <code>Main</code>.</li>
	<li>Ровно два метода: <code>main</code> и <code>personalGreeting</code>.</li>
	<li><code>personalGreeting</code> принимает <code>name</code> типа <code>String</code> и <code>age</code> типа <code>int</code>.</li>
	<li><code>personalGreeting</code> ничего не возвращает.</li>
	<li><code>personalGreeting</code> выводит приветствие через <code>System.out.println</code>.</li>
	<li><code>main</code> вызывает <code>personalGreeting</code> с любыми корректными аргументами.</li>
	<li><code>main</code> не должен выводить текст напрямую.</li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Результат программы</h2>
<p>Пример допустимого вывода:</p>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 16px; border-radius: 8px; overflow-x: auto;"><code>Привет, Анна! Тебе 25 лет.</code></pre>

<div style="background: #fff3e0; border: 1px solid #ff9800; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #e65100; margin-top: 0;">🤔 Момент для размышления</h4>
<p style="margin-bottom: 0;">Почему в этой задаче метод <code>personalGreeting</code> выводит сам, а в предыдущей <code>buildGreeting</code> возвращал строку?</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📝 Структура программы</h2>
<pre style="background: #263238; color: #aed581; padding: 20px; border-radius: 8px; overflow-x: auto;"><code>public class Main {
    public static void main(String[] args) {
        personalGreeting("Анна", 25);
    }

    static void personalGreeting(String name, int age) {
        // выведите сообщение с name и age
    }
}</code></pre>

<h2 style="color: #34495e; margin-top: 28px;">✅ Чек-лист проверки</h2>
<div style="background: #f8f9fa; border-left: 4px solid #28a745; padding: 16px; margin: 16px 0;">
<ul style="margin-bottom: 0;">
	<li>☐ Есть ровно два метода: <code>main</code> и <code>personalGreeting</code></li>
	<li>☐ Сигнатура: <code>static void personalGreeting(String name, int age)</code></li>
	<li>☐ В выводе используются оба параметра</li>
	<li>☐ <code>main</code> вызывает метод с двумя аргументами</li>
	<li>☐ В <code>main</code> нет прямого вывода</li>
</ul>
</div>

<div style="background: #fff8e1; border-left: 4px solid #ffc107; padding: 16px; margin: 16px 0;">
<h4 style="color: #f57c00; margin-top: 0;">⚠️ Технические ограничения</h4>
<ul style="margin-bottom: 0;">
	<li>Не возвращайте строку из <code>personalGreeting</code></li>
	<li>Не убирайте один из параметров</li>
	<li>Не создавайте дополнительные методы</li>
	<li>Откажитесь от вывода в <code>main</code>: весь вывод должен быть внутри <code>personalGreeting</code></li>
</ul>
</div>
