<h1 style="color: #2c3e50; border-bottom: 3px solid #3498db; padding-bottom: 10px;">Задача 3.3: Умножение чисел</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться сохранять промежуточный результат вычисления в переменную и возвращать его из метода.</p>
</div>

<h2 style="color: #34495e;">Условие</h2>
<p>Создайте метод для умножения двух целых чисел.</p>

<h2 style="color: #34495e;">Сигнатура метода</h2>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 15px; border-radius: 8px; overflow-x: auto;"><code>static int multiply(int a, int b)</code></pre>

<h2 style="color: #34495e;">Требования</h2>
<ul>
	<li>Метод должен называться <code>multiply</code>.</li>
	<li>Метод принимает два параметра типа <code>int</code>.</li>
	<li>Создайте переменную <code>result</code> типа <code>int</code>.</li>
	<li>Сохраните в <code>result</code> произведение <code>a * b</code>.</li>
	<li>Верните <code>result</code> через <code>return</code>.</li>
	<li>Не выводите данные через <code>System.out.print*</code>.</li>
</ul>

<h2 style="color: #34495e;">Примеры работы</h2>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 15px; border-radius: 8px; overflow-x: auto;"><code>multiply(5, 3)   -> 15
multiply(7, 0)   -> 0
multiply(-4, 6)  -> -24
multiply(-3, -8) -> 24</code></pre>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #c62828; margin-top: 0;">Частые ошибки</h3>
<ul>
	<li>Возвращается сумма вместо произведения.</li>
	<li>Метод печатает результат вместо возврата.</li>
	<li>Переменная <code>result</code> объявлена как <code>double</code>, хотя нужен <code>int</code>.</li>
	<li>Забыли вернуть результат.</li>
</ul>
</div>

<pre style="background: #1e293b; color: #e2e8f0; padding: 12px; border-radius: 5px;"><code>int result = a * b;
return result;</code></pre>
