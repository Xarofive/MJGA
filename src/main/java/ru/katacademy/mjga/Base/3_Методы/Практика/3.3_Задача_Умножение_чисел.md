<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">Задача 3.3: Умножение чисел</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться сохранять промежуточный результат вычисления в переменную и возвращать его из метода.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">🎯 Сценарий</h3>
<p style="margin-bottom: 10px;">Вы пишете вычислительный метод, где результат нужно не только получить, но и сохранить в понятную переменную.</p>
<p style="margin: 0;"><strong>Ваша задача:</strong> реализовать умножение двух чисел через переменную <code>result</code>.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Промежуточные переменные</h4>
<p>Именованная переменная делает вычисление понятнее и помогает проверять результат в отладчике.</p>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>int result = a * b;
return result;</code></pre>
<p style="margin-bottom: 0;">Здесь <code>result</code> должен быть типа <code>int</code>, потому что оба параметра и результат — целые числа.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<p><strong>Создайте метод с точной сигнатурой:</strong></p>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 16px; border-radius: 8px; overflow-x: auto;"><code>static int multiply(int a, int b)</code></pre>
<ul>
	<li>Метод должен называться <code>multiply</code>.</li>
	<li>Метод принимает два параметра типа <code>int</code>.</li>
	<li>Создайте переменную <code>result</code> типа <code>int</code>.</li>
	<li>Сохраните в <code>result</code> произведение <code>a * b</code>.</li>
	<li>Верните <code>result</code> через <code>return</code>.</li>
	<li>Не выводите данные через <code>System.out.print*</code>.</li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Результат программы</h2>
<p>Автопроверка будет вызывать метод с разными аргументами:</p>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 16px; border-radius: 8px; overflow-x: auto;"><code>multiply(5, 3)   -> 15
multiply(7, 0)   -> 0
multiply(-4, 6)  -> -24
multiply(-3, -8) -> 24</code></pre>

<div style="background: #fff3e0; border: 1px solid #ff9800; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #e65100; margin-top: 0;">🤔 Момент для размышления</h4>
<p style="margin-bottom: 0;">Чем вариант с переменной <code>result</code> удобнее при отладке, чем короткий вариант <code>return a * b;</code>?</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📝 Структура метода</h2>
<pre style="background: #263238; color: #aed581; padding: 20px; border-radius: 8px; overflow-x: auto;"><code>static int multiply(int a, int b) {
    int result = a * b;
    return result;
}</code></pre>

<h2 style="color: #34495e; margin-top: 28px;">✅ Чек-лист проверки</h2>
<div style="background: #f8f9fa; border-left: 4px solid #28a745; padding: 16px; margin: 16px 0;">
<ul style="margin-bottom: 0;">
	<li>☐ Метод называется <code>multiply</code></li>
	<li>☐ Сигнатура: <code>static int multiply(int a, int b)</code></li>
	<li>☐ Есть переменная <code>int result</code></li>
	<li>☐ В <code>result</code> сохраняется <code>a * b</code></li>
	<li>☐ Возвращается <code>result</code>, а не выводится на экран</li>
</ul>
</div>

<div style="background: #fff8e1; border-left: 4px solid #ffc107; padding: 16px; margin: 16px 0;">
<h4 style="color: #f57c00; margin-top: 0;">⚠️ Технические ограничения</h4>
<ul style="margin-bottom: 0;">
	<li>Не используйте <code>double</code> для <code>result</code></li>
	<li>Не используйте <code>System.out.println</code> внутри метода</li>
	<li>Не создавайте дополнительные методы</li>
	<li>Откажитесь от сложения: нужна операция умножения <code>*</code></li>
</ul>
</div>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
