<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">📊 Парсер CSV-строки с числами</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться разбирать строку с числами, проверять каждый элемент и возвращать массив целых чисел.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">💼 Контекст и мотивация</h3>
<p style="margin-bottom: 10px;">CSV-строки часто встречаются в настройках, логах и импорте данных. Даже простой формат <code>"1, 2, 3"</code> требует аккуратной обработки: пустые элементы, лишние пробелы и нечисловые значения должны давать понятную ошибку.</p>
<p style="margin: 0;"><strong>Задача:</strong> реализовать метод для преобразования CSV-строки в массив <code>int[]</code>.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>

<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p><code>split(",")</code> делит строку по запятым, <code>trim()</code> убирает пробелы по краям, <code>Integer.parseInt</code> преобразует элемент в число.</p>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code class="language-java">String[] parts = input.split(",");
int[] numbers = new int[parts.length];</code></pre>
<p style="margin-bottom: 0;">Индексы в массиве начинаются с нуля, поэтому второй элемент имеет индекс <code>1</code>.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;"><strong>Реализуйте метод <code>parseCsvInts</code>:</strong></p>
<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>static int[] parseCsvInts(String input)</code></div>

<h4 style="color: #e65100; margin-top: 16px;">📝 Требования</h4>
<ul style="margin: 10px 0;">
	<li>Если <code>input == null</code>, выбросьте <code>IllegalArgumentException("input is null")</code></li>
	<li>Если строка пустая или состоит только из пробелов, выбросьте <code>IllegalArgumentException("empty input")</code></li>
	<li>Разделитель — запятая</li>
	<li>Пробелы вокруг каждого элемента нужно удалить</li>
	<li>Если элемент пустой после <code>trim()</code>, выбросьте <code>IllegalArgumentException</code>, сообщение содержит индекс элемента</li>
	<li>Если элемент не является числом, выбросьте <code>IllegalArgumentException</code>, сообщение содержит индекс и проблемное значение</li>
	<li>При успехе верните массив <code>int[]</code></li>
</ul>

<h4 style="color: #e65100; margin-top: 16px;">🚫 Ограничения</h4>
<ul style="margin: 10px 0;">
	<li>Не используйте <code>ArrayList</code> и другие коллекции</li>
	<li>Не используйте вывод на экран</li>
	<li>Не создавайте дополнительные методы</li>
	<li>Не указывайте <code>package</code> внутри решения</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Примеры работы</h2>

<div style="background: linear-gradient(135deg, #e3f2fd 0%, #bbdefb 100%); border-radius: 10px; padding: 20px; margin: 20px 0; border: 1px solid #64b5f6;">
<p><code>parseCsvInts("1,2,3")</code> → <code>new int[]{1, 2, 3}</code></p>
<p><code>parseCsvInts(" 10 , -5 , 0 ")</code> → <code>new int[]{10, -5, 0}</code></p>
<p><code>parseCsvInts("")</code> → <code>IllegalArgumentException("empty input")</code></p>
<p><code>parseCsvInts("5,,7")</code> → <code>IllegalArgumentException</code>, сообщение содержит индекс <code>1</code></p>
<p style="margin-bottom: 0;"><code>parseCsvInts("1,abc,3")</code> → <code>IllegalArgumentException</code>, сообщение содержит <code>1</code> и <code>abc</code></p>
</div>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #c62828; margin-top: 0;">⚠️ Частые ошибки</h4>
<p><strong>Ошибка 1:</strong> не проверять пустую строку до <code>split</code>.</p>
<p><strong>Ошибка 2:</strong> забыть <code>trim()</code> для каждого элемента.</p>
<p><strong>Ошибка 3:</strong> возвращать частично заполненный массив после ошибки.</p>
<p style="margin-bottom: 0;"><strong>Ошибка 4:</strong> не указывать индекс проблемного элемента в сообщении.</p>
</div>
