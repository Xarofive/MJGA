<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🧮 Калькулятор арифметических операций</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться выбрасывать разные исключения для разных ошибочных ситуаций и защищать код от деления на ноль.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">💼 Контекст и мотивация</h3>
<p style="margin-bottom: 10px;">Даже простой калькулятор должен корректно реагировать на невозможные операции: неизвестный оператор и деление на ноль. В Java такие ситуации удобно выражать через <code>IllegalArgumentException</code> и <code>ArithmeticException</code>.</p>
<p style="margin: 0;"><strong>Задача:</strong> реализовать метод-калькулятор для четырёх базовых операций.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>

<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p><code>ArithmeticException</code> подходит для математических ошибок, например деления на ноль. <code>IllegalArgumentException</code> подходит для недопустимого аргумента, например неизвестного оператора.</p>
<p style="margin-bottom: 0;">Оператор <code>switch</code> будет отдельной темой в следующем модуле, поэтому здесь используйте цепочку <code>if / else if / else</code>.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;"><strong>Реализуйте метод <code>calcOp</code>:</strong></p>
<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>static int calcOp(int a, int b, String op)</code></div>

<h4 style="color: #e65100; margin-top: 16px;">📝 Требования</h4>
<ul style="margin: 10px 0;">
	<li><code>"+"</code> — вернуть <code>a + b</code></li>
	<li><code>"-"</code> — вернуть <code>a - b</code></li>
	<li><code>"*"</code> — вернуть <code>a * b</code></li>
	<li><code>"/"</code> — вернуть целочисленное деление <code>a / b</code></li>
	<li>Если <code>op == null</code> или операция неизвестна, выбросьте <code>IllegalArgumentException</code> с текстом, содержащим операцию</li>
	<li>Если операция <code>"/"</code> и <code>b == 0</code>, выбросьте <code>ArithmeticException("divide by zero")</code></li>
</ul>

<h4 style="color: #e65100; margin-top: 16px;">🚫 Ограничения</h4>
<ul style="margin: 10px 0;">
	<li>Не используйте <code>switch</code></li>
	<li>Не используйте вывод на экран</li>
	<li>Не создавайте дополнительные методы</li>
	<li>Не указывайте <code>package</code> внутри решения</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Примеры работы</h2>

<div style="background: linear-gradient(135deg, #e3f2fd 0%, #bbdefb 100%); border-radius: 10px; padding: 20px; margin: 20px 0; border: 1px solid #64b5f6;">
<p><code>calcOp(7, 5, "+")</code> → <code>12</code></p>
<p><code>calcOp(7, 5, "-")</code> → <code>2</code></p>
<p><code>calcOp(7, 5, "*")</code> → <code>35</code></p>
<p><code>calcOp(7, 5, "/")</code> → <code>1</code></p>
<p><code>calcOp(7, 0, "/")</code> → <code>ArithmeticException("divide by zero")</code></p>
<p style="margin-bottom: 0;"><code>calcOp(7, 5, "^")</code> → <code>IllegalArgumentException</code></p>
</div>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #c62828; margin-top: 0;">⚠️ Частые ошибки</h4>
<p><strong>Ошибка 1:</strong> выполнить деление до проверки <code>b == 0</code>.</p>
<p><strong>Ошибка 2:</strong> вернуть <code>0</code> для неизвестной операции вместо исключения.</p>
<p style="margin-bottom: 0;"><strong>Ошибка 3:</strong> забыть обработать <code>op == null</code>.</p>
</div>
