<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🔢 Безопасное преобразование строки в число</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться преобразовывать строку в целое число и выбрасывать понятное исключение для некорректного ввода.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">💼 Контекст и мотивация</h3>
<p style="margin-bottom: 10px;">Пользовательский ввод почти всегда приходит как строка. Даже если поле называется "возраст" или "количество", сначала программа получает текст и только потом превращает его в число.</p>
<p style="margin: 0;"><strong>Задача:</strong> реализовать метод, который очищает строку от пробелов, преобразует её в <code>int</code> и сообщает об ошибке через исключение.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>

<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Integer.parseInt и NumberFormatException</h4>
<p><code>Integer.parseInt(text)</code> преобразует строку в <code>int</code>. Если строка не является корректным целым числом, метод выбрасывает <code>NumberFormatException</code>.</p>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code class="language-java">int value = Integer.parseInt("42"); // 42</code></pre>
<p style="margin-bottom: 0;">Перед преобразованием используйте <code>trim()</code>, чтобы убрать пробелы по краям.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;"><strong>Реализуйте метод <code>safeParseInt</code>:</strong></p>
<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>static int safeParseInt(String text)</code></div>

<h4 style="color: #e65100; margin-top: 16px;">📝 Требования</h4>
<ul style="margin: 10px 0;">
	<li>Если <code>text == null</code>, выбросьте <code>IllegalArgumentException</code> с текстом <code>"input is null"</code></li>
	<li>Удалите пробелы по краям через <code>trim()</code></li>
	<li>Если после очистки строка пустая, выбросьте <code>IllegalArgumentException</code> с текстом <code>"empty input"</code></li>
	<li>Если строка корректная, верните число</li>
	<li>Если <code>Integer.parseInt</code> выбросил <code>NumberFormatException</code>, выбросьте <code>IllegalArgumentException</code> с текстом, который содержит исходную строку</li>
</ul>

<h4 style="color: #e65100; margin-top: 16px;">🚫 Ограничения</h4>
<ul style="margin: 10px 0;">
	<li>Не используйте вывод на экран</li>
	<li>Не создавайте дополнительные методы</li>
	<li>Не указывайте <code>package</code> внутри решения</li>
	<li>Не заменяйте ошибку специальным числом вроде <code>-1</code> или <code>0</code></li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Примеры работы</h2>

<div style="background: linear-gradient(135deg, #e3f2fd 0%, #bbdefb 100%); border-radius: 10px; padding: 20px; margin: 20px 0; border: 1px solid #64b5f6;">
<p><code>safeParseInt("42")</code> → <code>42</code></p>
<p><code>safeParseInt(" -7 ")</code> → <code>-7</code></p>
<p><code>safeParseInt("")</code> → <code>IllegalArgumentException("empty input")</code></p>
<p><code>safeParseInt(null)</code> → <code>IllegalArgumentException("input is null")</code></p>
<p style="margin-bottom: 0;"><code>safeParseInt("12px")</code> → <code>IllegalArgumentException</code>, сообщение содержит <code>12px</code></p>
</div>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #c62828; margin-top: 0;">⚠️ Частые ошибки</h4>
<p><strong>Ошибка 1:</strong> забыть обработать <code>null</code> до вызова <code>trim()</code>.</p>
<p><strong>Ошибка 2:</strong> вернуть <code>0</code> при ошибке и потерять информацию о проблеме.</p>
<p style="margin-bottom: 0;"><strong>Ошибка 3:</strong> выбросить исключение без исходного значения, из-за чего непонятно, что именно не распарсилось.</p>
</div>
