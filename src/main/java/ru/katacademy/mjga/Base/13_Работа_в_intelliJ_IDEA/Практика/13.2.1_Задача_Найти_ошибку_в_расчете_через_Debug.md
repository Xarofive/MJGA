<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🧮 Найти ошибку в расчёте через Debug</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>

<p style="margin: 0;">Научиться использовать Debug для поиска первой строки, после которой результат вычисления становится неправильным.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">💼 Контекст и мотивация</h3>

<p style="margin-bottom: 10px;">В учебных задачах ошибка часто находится не там, где программа выводит результат, а на несколько строк раньше. Debug помогает пройти расчёт по шагам и увидеть момент, когда значение стало неверным.</p>

<p style="margin: 0;"><strong>Задача:</strong> исправить расчёт итоговой цены заказа после скидки.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>

<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Проверяйте промежуточные значения</h4>

<p>Поставьте breakpoint перед подозрительным расчётом и выполните код через <strong>Step Over</strong>:</p>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code class="language-java">int total = price * quantity;
int discountAmount = total * discountPercent / 100;
int finalPrice = total - discountAmount;</code></pre>

<p style="margin-bottom: 0;">После каждой строки смотрите значения <code>total</code>, <code>discountAmount</code> и <code>finalPrice</code> в окне Debug.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;"><strong>Создайте класс:</strong></p>

<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>class OrderCalculator</code></div>

<h4 style="color: #e65100; margin-top: 16px;">⚡ Метод calculateFinalPrice</h4>

<p><strong>Сигнатура:</strong></p>

<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>static int calculateFinalPrice(int price, int quantity, int discountPercent)</code></div>

<p><strong>Требования:</strong></p>

<ul style="margin: 10px 0;">
	<li>Метод принимает цену одной единицы товара, количество и процент скидки</li>
	<li>Сначала считает полную стоимость: <code>price * quantity</code></li>
	<li>Затем считает размер скидки: <code>total * discountPercent / 100</code></li>
	<li>Возвращает итоговую цену: <code>total - discountAmount</code></li>
	<li>Если <code>price &lt;= 0</code>, <code>quantity &lt;= 0</code> или <code>discountPercent &lt; 0</code>, метод возвращает <code>0</code></li>
	<li>Если <code>discountPercent &gt; 100</code>, считайте скидку равной <code>100</code></li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🐛 Код с ошибкой для отладки</h2>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #c62828; margin-top: 0;">Скопируйте локально и проверьте через Debug</h4>

<pre style="background: white; border: 1px solid #e9ecef; border-radius: 5px; padding: 12px;"><code class="language-java">class OrderCalculator {
    static int calculateFinalPrice(int price, int quantity, int discountPercent) {
        if (price &lt;= 0 || quantity &lt;= 0 || discountPercent &lt; 0) {
            return 0;
        }

        if (discountPercent &gt; 100) {
            discountPercent = 100;
        }

        int total = price + quantity;
        int discountAmount = total * discountPercent / 100;
        return total - discountAmount;
    }
}</code></pre>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🔍 Как отлаживать</h2>

<div style="background: #e8eaf6; border: 1px solid #3f51b5; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #283593; margin-top: 0;">План Debug</h4>

<ol style="margin: 10px 0;">
	<li>Создайте временный <code>main</code> и вызовите <code>calculateFinalPrice(100, 3, 10)</code>.</li>
	<li>Поставьте breakpoint на строку с расчётом <code>total</code>.</li>
	<li>Запустите Debug.</li>
	<li>Нажмите <strong>Step Over</strong> и проверьте, чему равен <code>total</code>.</li>
	<li>Сравните фактическое значение с ожидаемым: <code>100 * 3 = 300</code>.</li>
	<li>Исправьте первую строку, на которой значение стало неверным.</li>
