<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🔎 Отладить объект и передачу ссылки</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>

<p style="margin: 0;">Научиться через Debug проверять, как метод меняет состояние объекта, переданного по ссылке.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">💼 Контекст и мотивация</h3>

<p style="margin-bottom: 10px;">После темы классов и объектов важно уметь видеть не только значения локальных переменных, но и поля объекта. Debug показывает объект в панели Variables и позволяет раскрыть его поля.</p>

<p style="margin: 0;"><strong>Задача:</strong> исправить начисление бонусов на счёт пользователя и проверить через Debug, что меняется поле уже существующего объекта.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>

<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Объект в Debug можно раскрыть</h4>

<p>Если переменная хранит ссылку на объект, в панели Variables можно раскрыть объект и увидеть его поля:</p>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code class="language-java">UserAccount account = new UserAccount();
account.owner = "Alice";
account.points = 10;</code></pre>

<p style="margin-bottom: 0;">В Debug вы увидите объект <code>account</code>, а внутри него поля <code>owner</code> и <code>points</code>.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;"><strong>Создайте класс:</strong></p>

<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>class UserAccount</code></div>

<h4 style="color: #e65100; margin-top: 16px;">📝 Поля</h4>

<ul style="margin: 10px 0;">
	<li><code>String owner</code> — имя владельца аккаунта</li>
	<li><code>int points</code> — количество бонусных баллов</li>
</ul>

<h4 style="color: #e65100; margin-top: 16px;">⚡ Методы</h4>

<ul style="margin: 10px 0;">
	<li><code>void addPoints(int amount)</code> — увеличивает <code>points</code> на <code>amount</code>, если <code>amount &gt; 0</code></li>
	<li><code>String status()</code> — возвращает <code>"&lt;owner&gt;: &lt;points&gt; points"</code></li>
</ul>

<h4 style="color: #e65100; margin-top: 16px;">🔍 Статический метод</h4>

<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>static boolean reward(UserAccount account, int amount)</code></div>

<p style="margin-bottom: 0;">Метод должен вернуть <code>false</code>, если <code>account == null</code> или <code>amount &lt;= 0</code>. Иначе он должен начислить баллы на переданный объект и вернуть <code>true</code>.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🐛 Код с ошибкой для отладки</h2>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #c62828; margin-top: 0;">Скопируйте локально и проверьте через Debug</h4>

<pre style="background: white; border: 1px solid #e9ecef; border-radius: 5px; padding: 12px;"><code class="language-java">class UserAccount {
    String owner;
    int points;

    void addPoints(int amount) {
        if (amount &gt; 0) {
            points = amount;
        }
    }

    String status() {
        return owner + ": " + points + " points";
    }

    static boolean reward(UserAccount account, int amount) {
        if (account == null || amount &lt;= 0) {
            return false;
        }

        account.addPoints(amount);
        return true;
    }
}</code></pre>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Примеры работы</h2>

<div style="background: linear-gradient(135deg, #e3f2fd 0%, #bbdefb 100%); border-radius: 10px; padding: 20px; margin: 20px 0; border: 1px solid #64b5f6;">
<pre style="background: white; border-radius: 6px; padding: 15px;"><code class="language-java">UserAccount account = new UserAccount();
account.owner = "Alice";
account.points = 10;

UserAccount.reward(account, 5); // true
account.status();              // "Alice: 15 points"

UserAccount.reward(account, -3); // false
account.status();                // "Alice: 15 points"

UserAccount.reward(null, 10); // false</code></pre>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🔍 Как отлаживать</h2>

<div style="background: #e8eaf6; border: 1px solid #3f51b5; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #283593; margin-top: 0;">План Debug</h4>

<ol style="margin: 10px 0;">
	<li>Создайте временный <code>main</code> с объектом <code>UserAccount</code>.</li>
	<li>Поставьте breakpoint на строку <code>account.addPoints(amount)</code> внутри <code>reward</code>.</li>
	<li>Запустите Debug и раскройте объект <code>account</code> в Variables.</li>
	<li>Нажмите <strong>Step Into</strong>, чтобы зайти в <code>addPoints</code>.</li>
	<li>Следите за полем <code>points</code> до и после строки присваивания.</li>
	<li>Определите, почему старые баллы исчезают вместо накопления.</li>
</ol>
</div>

<h2 style="color: #34495e; margin-top: 28px;">✅ Чек-лист проверки</h2>

<div style="background: #f8f9fa; border-left: 4px solid #28a745; padding: 16px; margin: 16px 0;">
<ul style="margin-bottom: 0;">
	<li>☐ <code>addPoints</code> добавляет баллы к текущему значению, а не заменяет его</li>
	<li>☐ <code>reward</code> возвращает <code>true</code> при успешном начислении</li>
	<li>☐ <code>reward</code> возвращает <code>false</code> при <code>null</code></li>
	<li>☐ <code>reward</code> возвращает <code>false</code> при нуле или отрицательном количестве</li>
	<li>☐ Неуспешное начисление не меняет <code>points</code></li>
	<li>☐ <code>status()</code> возвращает строку в точном формате</li>
</ul>
</div>

<div style="background: #fff8e1; border-left: 4px solid #ffc107; padding: 16px; margin: 16px 0;">

<ul style="margin-bottom: 0;">

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #c62828; margin-top: 0;">⚠️ Частые ошибки и антипаттерны</h4>

<p><strong>Ошибка 1: Заменять значение вместо накопления</strong><br />
❌ <code>points = amount;</code><br />
✅ <code>points = points + amount;</code></p>

<p><strong>Ошибка 2: Создавать новый объект внутри reward</strong><br />
❌ <code>account = new UserAccount();</code><br />
✅ Меняйте объект, который пришёл в параметре</p>

<p style="margin-bottom: 0;"><strong>Ошибка 3: Проверять null слишком поздно</strong><br />
❌ <code>account.addPoints(amount);</code> до проверки<br />
✅ Сначала <code>if (account == null)</code></p>
