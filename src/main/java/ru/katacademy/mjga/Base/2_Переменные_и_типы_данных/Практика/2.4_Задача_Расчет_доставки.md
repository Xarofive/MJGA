<h1 style="color: #2c3e50; border-bottom: 3px solid #3498db; padding-bottom: 10px;">Задача 2.4: Расчёт доставки</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться выполнять арифметику с переменными разных числовых типов.</p>
</div>

<h2 style="color: #34495e;">Контекст</h2>
<p>Служба доставки считает стоимость по формуле: <strong>расстояние × цена за километр</strong>.</p>

<h2 style="color: #34495e;">Задание</h2>
<ol>
	<li>Объявите <code>int distance = 15;</code></li>
	<li>Объявите <code>double pricePerKm = 25.50;</code></li>
	<li>Вычислите <code>double cost = pricePerKm * distance;</code></li>
	<li>Выведите результат в указанном формате.</li>
</ol>

<h2 style="color: #34495e;">Ожидаемый вывод</h2>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 15px; border-radius: 8px; overflow-x: auto;"><code>Distance: 15 km
Price per km: 25.5
Delivery cost: 382.5</code></pre>

<h2 style="color: #34495e;">Требования</h2>
<ul>
	<li>Используйте типы <code>int</code> и <code>double</code>.</li>
	<li>Стоимость должна вычисляться через переменные, а не быть записана готовым числом.</li>
	<li>Для вывода используйте <code>System.out.println</code>.</li>
	<li>Не используйте <code>Scanner</code>.</li>
</ul>

<div style="background: #fff3cd; border: 1px solid #ffeaa7; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #856404; margin-top: 0;">Почему без Scanner</h4>
<p style="margin: 0;"><code>Scanner</code> нужен для чтения данных, которые пользователь вводит с клавиатуры. Ввод будет отдельной темой позже. В этой задаче значения уже заданы в условии, поэтому просто объявите переменные в коде.</p>
</div>

<div style="background: #e3f2fd; border: 1px solid #90caf9; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #1565c0; margin-top: 0;">Подсказка про типы</h4>
<p style="margin: 0;">В выражении <code>double * int</code> Java автоматически приведёт <code>int</code> к <code>double</code>, потому что это безопасное преобразование.</p>
</div>
