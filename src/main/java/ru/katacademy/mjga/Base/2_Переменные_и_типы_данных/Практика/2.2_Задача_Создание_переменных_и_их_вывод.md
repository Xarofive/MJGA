<h1 style="color: #2c3e50; border-bottom: 3px solid #3498db; padding-bottom: 10px;">Задача 2.2: Создание переменных и их вывод</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Что нужно сделать</h3>
<p style="margin: 0;">Создайте несколько переменных разных типов и выведите их значения в заданном порядке.</p>
</div>

<h2 style="color: #34495e;">Переменные для объявления</h2>
<ul>
	<li><strong>age</strong> &mdash; <code>int</code> = <code>25</code></li>
	<li><strong>name</strong> &mdash; <code>String</code> = <code>"John"</code></li>
	<li><strong>height</strong> &mdash; <code>double</code> = <code>1.75</code></li>
	<li><strong>isActive</strong> &mdash; <code>boolean</code> = <code>true</code></li>
	<li><strong>balance</strong> &mdash; <code>double</code> = <code>1000.50</code></li>
	<li><strong>birthYear</strong> &mdash; <code>int</code> = <code>1995</code></li>
	<li><strong>country</strong> &mdash; <code>String</code> = <code>"Kazakhstan"</code></li>
	<li><strong>temperature</strong> &mdash; <code>double</code> = <code>36.6</code></li>
	<li><strong>friendsCount</strong> &mdash; <code>int</code> = <code>150</code></li>
	<li><strong>registeredAt</strong> &mdash; <code>String</code> = <code>"2024-09-01"</code></li>
</ul>

<h2 style="color: #34495e;">Ожидаемый вывод</h2>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 15px; border-radius: 8px; overflow-x: auto;"><code>25
John

true
1000.5

Kazakhstan
36.6

2024-09-01</code></pre>

<h2 style="color: #34495e;">Требования</h2>
<ul>
	<li>Программа должна содержать <code>public class Main</code>.</li>
	<li>Все переменные объявлены внутри <code>main</code>.</li>
	<li>Используйте явные типы, без <code>var</code>.</li>
	<li>Значения выведены через <code>System.out.println</code>, каждое на новой строке.</li>
	<li>Порядок вывода должен совпадать с образцом.</li>
</ul>

<div style="background: #e3f2fd; border: 1px solid #90caf9; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #1565c0; margin-top: 0;">Обратите внимание</h4>
<p style="margin: 0;"><code>System.out.println(1000.50)</code> может вывести <code>1000.5</code>. Это нормально: лишний ноль в конце дробной части не влияет на значение.</p>
