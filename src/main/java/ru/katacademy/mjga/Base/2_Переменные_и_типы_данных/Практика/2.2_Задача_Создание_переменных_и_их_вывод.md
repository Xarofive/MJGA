<p style="margin-bottom: 10px;">Вы описываете тестовый профиль пользователя: возраст, имя, рост, баланс, страну и другие параметры.</p>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Тип выбирается по смыслу значения</h4>
<p>Для целых чисел используйте <code>int</code>, для дробных — <code>double</code>, для текста — <code>String</code>, для значений <code>true</code>/<code>false</code> — <code>boolean</code>.</p>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>double balance = 1000.50;
System.out.println(balance);</code></pre>
<p style="margin-bottom: 0;">При выводе <code>1000.50</code> Java может показать <code>1000.5</code>. Последний ноль не меняет значение дробного числа.</p>
</div>
