<h1 style="color: #2c3e50; border-bottom: 3px solid #3498db; padding-bottom: 10px;">Задача 2.5: Расчёт с налогом</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Закрепить вычисления с <code>int</code>, <code>double</code> и <code>String</code>.</p>
</div>

<h2 style="color: #34495e;">Условие</h2>
<p>Рассчитайте итоговую стоимость покупки с налогом 13%.</p>

<h2 style="color: #34495e;">Переменные</h2>
<ul>
	<li><code>int taxPercent = 13;</code></li>
	<li><code>String currency = "RUB";</code></li>
	<li><code>double price = 1500.00;</code></li>
	<li><code>int qty = 2;</code></li>
</ul>

<h2 style="color: #34495e;">Расчёты</h2>
<ul>
	<li><code>double total = price * qty;</code></li>
	<li><code>double tax = total * taxPercent / 100;</code></li>
	<li><code>double finalPrice = total + tax;</code></li>
</ul>

<h2 style="color: #34495e;">Ожидаемый вывод</h2>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 15px; border-radius: 8px; overflow-x: auto;"><code>Price: 1500.0
Qty: 2
Tax: 390.0
Final: 3390.0 RUB</code></pre>

<h2 style="color: #34495e;">Требования</h2>
<ul>
	<li>Все переменные объявите внутри метода <code>main</code>.</li>
	<li>Используйте указанные типы и значения.</li>
	<li>Итоговая стоимость должна вычисляться через переменные.</li>
	<li>Для вывода используйте <code>System.out.println</code>.</li>
	<li>Порядок вывода должен совпадать с образцом.</li>
</ul>

<div style="background: #fff3cd; border-left: 4px solid #ffc107; padding: 15px; margin: 20px 0;">
<h3 style="color: #856404; margin-top: 0;">Важно</h3>
<p style="margin: 0;">В реальных финансовых приложениях для денег часто используют не <code>double</code>, а специальные классы. В этой учебной задаче <code>double</code> подходит, потому что цель &mdash; понять арифметику типов.</p>
