<h1 style="color: #2c3e50; border-bottom: 3px solid #3498db; padding-bottom: 10px;">Задача 2.3: Переменные и типы — быстрый старт</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Закрепить четыре базовых типа Java: <code>int</code>, <code>double</code>, <code>String</code>, <code>boolean</code>.</p>
</div>

<h2 style="color: #34495e;">Сценарий</h2>
<p>Вы создаёте тестовый учебный профиль. Нужно сохранить данные в переменные и вывести значение вместе с названием типа.</p>

<h2 style="color: #34495e;">Данные</h2>
<table style="width: 100%; border-collapse: collapse;">
	<tbody>
		<tr style="background: #6c757d; color: white;"><th style="padding: 10px; border: 1px solid #495057;">Имя</th><th style="padding: 10px; border: 1px solid #495057;">Тип</th><th style="padding: 10px; border: 1px solid #495057;">Значение</th></tr>
		<tr><td style="padding: 10px; border: 1px solid #dee2e6;"><code>n</code></td><td style="padding: 10px; border: 1px solid #dee2e6;"><code>int</code></td><td style="padding: 10px; border: 1px solid #dee2e6;"><code>25</code></td></tr>
		<tr style="background: #e9ecef;"><td style="padding: 10px; border: 1px solid #dee2e6;"><code>p</code></td><td style="padding: 10px; border: 1px solid #dee2e6;"><code>double</code></td><td style="padding: 10px; border: 1px solid #dee2e6;"><code>99.99</code></td></tr>
		<tr><td style="padding: 10px; border: 1px solid #dee2e6;"><code>user</code></td><td style="padding: 10px; border: 1px solid #dee2e6;"><code>String</code></td><td style="padding: 10px; border: 1px solid #dee2e6;"><code>"Alice"</code></td></tr>
		<tr style="background: #e9ecef;"><td style="padding: 10px; border: 1px solid #dee2e6;"><code>ok</code></td><td style="padding: 10px; border: 1px solid #dee2e6;"><code>boolean</code></td><td style="padding: 10px; border: 1px solid #dee2e6;"><code>true</code></td></tr>
	</tbody>
</table>

<h2 style="color: #34495e;">Ожидаемый вывод</h2>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 15px; border-radius: 8px; overflow-x: auto;"><code>25 (int)
99.99 (double)
Alice (String)
true (boolean)</code></pre>

<h2 style="color: #34495e;">Требования</h2>
<ul>
	<li>Используйте только явные типы, без <code>var</code>.</li>
	<li>Выводите строки через <code>System.out.println</code>.</li>
	<li>Названия типов в скобках должны совпадать с образцом.</li>
	<li>Не используйте ввод с клавиатуры.</li>
</ul>
