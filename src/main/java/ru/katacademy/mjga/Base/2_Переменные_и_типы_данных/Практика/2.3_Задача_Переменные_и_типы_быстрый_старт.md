<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">Задача 2.3: Переменные и типы — быстрый старт</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Закрепить четыре базовых типа Java: <code>int</code>, <code>double</code>, <code>String</code>, <code>boolean</code>, и научиться явно показывать тип в выводе.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">🎯 Сценарий</h3>
<p style="margin-bottom: 10px;">Вы создаёте тестовый учебный профиль и подписываете каждое значение его типом.</p>
<p style="margin: 0;"><strong>Ваша задача:</strong> объявить переменные и вывести значение вместе с названием типа в скобках.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Конкатенация строк</h4>
<p>Оператор <code>+</code> может склеивать значение переменной и текст.</p>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>int n = 25;
System.out.println(n + " (int)");</code></pre>
<p style="margin-bottom: 0;">Названия типов в этой задаче выводятся как обычный текст. Java не подставит их сама.</p>
</div>

<p style="margin-bottom: 0;">Что изменится в выводе, если случайно написать <code>"25"</code> вместо <code>25</code>? Значение похоже, но тип станет другим.</p>
