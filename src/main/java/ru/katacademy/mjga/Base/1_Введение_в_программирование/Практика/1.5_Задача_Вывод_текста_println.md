<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">Задача 1.5: Вывод текста в консоль</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться выводить текст в консоль с помощью базовой команды Java <code>System.out.println</code>.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">🎯 Сценарий</h3>
<p style="margin-bottom: 10px;">Вы запускаете первую команду вывода в Java и проверяете, что программа может показать пользователю простое сообщение.</p>
<p style="margin: 0;"><strong>Ваша задача:</strong> вывести одну строку без изменений, лишних пробелов и дополнительных строк.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>

<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Команда println</h4>
<p><code>System.out.println</code> выводит текст в консоль и после вывода переводит курсор на новую строку.</p>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>System.out.println("текст");</code></pre>
<p style="margin-bottom: 0;">Текстовый литерал в Java записывается в двойных кавычках.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<p><strong>Напишите одну команду вывода, которая печатает:</strong></p>

<pre style="background: #2c3e50; color: #ecf0f1; padding: 16px; border-radius: 8px; overflow-x: auto;"><code>Hello Kata Academy</code></pre>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Результат программы</h2>
<p>После запуска в консоли должна быть ровно одна строка:</p>

<pre style="background: #2c3e50; color: #ecf0f1; padding: 16px; border-radius: 8px; overflow-x: auto;"><code>Hello Kata Academy</code></pre>

<div style="background: #fff3e0; border: 1px solid #ff9800; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #e65100; margin-top: 0;">🤔 Момент для размышления</h4>
<p style="margin-bottom: 0;">Почему строка должна быть в кавычках? Что изменится, если убрать кавычки вокруг <code>Hello Kata Academy</code>?</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📝 Структура решения</h2>
<pre style="background: #263238; color: #aed581; padding: 20px; border-radius: 8px; overflow-x: auto;"><code>// Напишите здесь одну команду System.out.println</code></pre>

<p style="margin: 10px 0;">В этой задаче платформа ожидает только команду вывода. Полный класс <code>Main</code> писать не нужно, если редактор задачи уже содержит шаблон программы.</p>

<h2 style="color: #34495e; margin-top: 28px;">✅ Чек-лист проверки</h2>
<div style="background: #f8f9fa; border-left: 4px solid #28a745; padding: 16px; margin: 16px 0;">
<ul style="margin-bottom: 0;">
	<li>☐ Используется <code>System.out.println</code></li>
	<li>☐ Текст записан в двойных кавычках</li>
	<li>☐ Вывод совпадает с образцом символ в символ</li>
	<li>☐ Нет лишних строк до или после текста</li>
</ul>
</div>

<div style="background: #fff8e1; border-left: 4px solid #ffc107; padding: 16px; margin: 16px 0;">
<h4 style="color: #f57c00; margin-top: 0;">⚠️ Технические ограничения</h4>
<ul style="margin-bottom: 0;">
	<li>Не используйте переменные, методы, циклы и условия</li>
	<li>Не пишите полный класс <code>Main</code>, если в задаче уже дан шаблон</li>
	<li>Не заменяйте <code>System.out.println</code> на <code>System.out.print</code></li>
	<li>Откажитесь от любого дополнительного текста, даже если он кажется пояснением</li>
</ul>
</div>
