<h1 style="color: #2c3e50; border-bottom: 3px solid #3498db; padding-bottom: 10px;">Задача 1.5: Вывод текста в консоль</h1>

<p style="font-size: 18px; color: #555; margin-bottom: 25px;">
В этой задаче нужно вывести одну строку текста с помощью команды <code>System.out.println</code>.
</p>

<div style="background: #e8f4fd; border-left: 4px solid #3498db; padding: 15px; margin: 20px 0;">
<h3 style="color: #2980b9; margin-top: 0;">Цель задачи</h3>
<p>Научиться использовать базовую команду вывода текста в Java.</p>
</div>

<h2 style="color: #34495e;">Условие</h2>

<p>Напишите строку кода, которая выводит в консоль:</p>

<pre style="background: #2c3e50; color: #ecf0f1; padding: 15px; border-radius: 8px; overflow-x: auto;"><code>Hello Kata Academy</code></pre>

<h2 style="color: #34495e;">Требования</h2>

<ul style="line-height: 1.8;">
  <li>Используйте <code>System.out.println</code>.</li>
  <li>Текст должен быть выведен без изменений.</li>
  <li>Не добавляйте лишние строки.</li>
  <li>Не используйте переменные.</li>
</ul>

<div style="background: #fff3cd; border-left: 4px solid #ffc107; padding: 15px; margin: 20px 0;">
<h3 style="color: #856404; margin-top: 0;">Важно</h3>
<p>В этой задаче платформа ожидает только одну команду вывода. Полный класс <code>Main</code> писать не нужно, если редактор задачи уже содержит шаблон программы.</p>
</div>

<h2 style="color: #34495e;">Пример решения</h2>

<pre style="background: #2c3e50; color: #ecf0f1; padding: 15px; border-radius: 8px; overflow-x: auto;"><code>System.out.println("Hello Kata Academy");</code></pre>

<h2 style="color: #34495e;">Ожидаемый вывод</h2>

<pre style="background: #2c3e50; color: #ecf0f1; padding: 15px; border-radius: 8px; overflow-x: auto;"><code>Hello Kata Academy</code></pre>

<div style="background: #d4edda; border-left: 4px solid #28a745; padding: 15px; margin: 20px 0;">
<h3 style="color: #155724; margin-top: 0;">Что нужно понять</h3>
<p><code>System.out.println</code> выводит текст в консоль и после этого переводит курсор на новую строку.</p>
</div>
