<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🧮 Поиск первого делителя</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Использовать <code>break</code>, чтобы остановить поиск сразу после первого найденного делителя и не выполнять лишние проверки.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Сценарий</h2>
<p>Представьте, что программа проверяет число и должна быстро найти его первый делитель. Как только делитель найден, дальнейший перебор уже не нужен: можно остановить цикл, вычислить вторую часть разложения и вывести результат.</p>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<ul>
	<li>Оператор <code>%</code> показывает остаток от деления.</li>
	<li>Если <code>n % divisor == 0</code>, значит <code>divisor</code> делит число <code>n</code> без остатка.</li>
	<li><code>break</code> завершает цикл целиком.</li>
	<li>После найденного делителя можно вычислить частное через <code>n / divisor</code>.</li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p><strong>Найдите первый делитель числа 91, кроме 1, и вычислите частное.</strong></p>
<ul style="margin: 10px 0;">
	<li>создайте класс <code>Main</code> и метод <code>public static void main(String[] args)</code>;</li>
	<li>число для анализа: <code>int n = 91;</code>;</li>
	<li>цикл должен начинаться с делителя 2;</li>
	<li>проверяйте <code>n % divisor == 0</code>;</li>
	<li>при первом совпадении сохраните делитель и выполните <code>break</code>;</li>
	<li>вычислите частное <code>n / divisor</code>;</li>
	<li>выведите делитель и частное на отдельных строках;</li>
	<li>не выводите дополнительный текст.</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Результат программы</h2>
<div style="background: #e3f2fd; border: 1px solid #64b5f6; border-radius: 10px; padding: 16px; margin: 20px 0;">
<pre style="margin: 0;"><code>7
13</code></pre>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🤔 Момент для размышления</h2>
<p>Почему после нахождения делителя 7 нет смысла продолжать проверять 8, 9, 10 и остальные числа?</p>

<h2 style="color: #34495e; margin-top: 28px;">📝 Структура программы</h2>
<ol>
	<li>Сохраните число <code>91</code> в переменную.</li>
	<li>Создайте переменную для найденного делителя.</li>
	<li>Запустите цикл перебора делителей, начиная с 2.</li>
	<li>При делении без остатка сохраните делитель и остановите цикл через <code>break</code>.</li>
	<li>Вычислите частное.</li>
	<li>Выведите делитель и частное.</li>
</ol>

<h2 style="color: #34495e; margin-top: 28px;">⚠️ Технические ограничения</h2>
<ul>
	<li>Не начинайте поиск с 1: единица делит любое число и не подходит для этой задачи.</li>
	<li>Не продолжайте цикл после первого найденного делителя.</li>
	<li>Не используйте готовые значения <code>7</code> и <code>13</code> напрямую в выводе.</li>
	<li>Не используйте готовые структуры данных, библиотечные методы и дополнительные методы.</li>
	<li>Откажитесь от проверки всех делителей, если первый уже найден.</li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">🐛 Отладка: если что-то не работает</h2>
<ul>
	<li>Если первым выводится 1, проверьте стартовое значение цикла.</li>
	<li>Если выводится не 7, убедитесь, что условие делимости написано как <code>n % divisor == 0</code>.</li>
	<li>Если частное неверное, проверьте, что деление выполняется после сохранения найденного делителя.</li>
	<li>Если программа выводит лишний текст, оставьте только два вызова <code>System.out.println</code> с числами.</li>
</ul>
