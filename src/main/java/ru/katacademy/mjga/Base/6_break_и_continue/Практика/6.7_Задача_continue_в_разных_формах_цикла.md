<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🔁 Continue в разных формах цикла</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Понять разницу между <code>continue</code> в классическом <code>for</code> и в <code>while</code>.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p><strong>Создайте программу с двумя циклами.</strong></p>
<ul style="margin: 10px 0;">
	<li>первый цикл: классический <code>for (int i = 0; i &lt; 10; i++)</code>;</li>
	<li>если <code>i == 5</code>, выполните <code>continue</code>;</li>
	<li>выведите числа 0-9 без 5;</li>
	<li>второй цикл: <code>while</code> с переменной, объявленной до цикла;</li>
	<li>если значение равно 5, сначала увеличьте счётчик, потом выполните <code>continue</code>;</li>
	<li>после вывода тоже увеличивайте счётчик;</li>
	<li>вывод должен содержать две одинаковые последовательности без разделителей и лишнего текста.</li>
</ul>
