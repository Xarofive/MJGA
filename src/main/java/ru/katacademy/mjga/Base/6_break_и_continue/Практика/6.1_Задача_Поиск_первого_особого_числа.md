<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">Задача 6.1: Поиск первого особого числа</h1>
<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться использовать <code>break</code>, чтобы остановить цикл сразу после первого совпадения.</p>
</div>
<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">🎯 Сценарий</h3>
<p style="margin-bottom: 10px;">Вы ищете первое число, которое подходит под два математических условия. После находки продолжать перебор уже бессмысленно.</p>
<p style="margin: 0;"><strong>Ваша задача:</strong> вывести первое число от 1 до 100, которое делится на 7 и даёт остаток 1 при делении на 3.</p>
</div>
<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">break</h4>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px;"><code>if (condition) {
    System.out.println(n);
    break;
}</code></pre>
<p style="margin-bottom: 0;"><code>break</code> завершает ближайший цикл, поэтому код после цикла продолжит выполняться уже после найденного совпадения.</p>
</div>
<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<ul>
	<li>Создайте полный класс <code>Main</code> с методом <code>public static void main(String[] args)</code>.</li>
	<li>Переберите числа от 1 до 100 включительно.</li>
	<li>Условие поиска: <code>n % 7 == 0 &amp;&amp; n % 3 == 1</code>.</li>
	<li>Как только число найдено, выведите его и выполните <code>break</code>.</li>
	<li>Не используйте <code>continue</code>, <code>switch</code> и лишний вывод.</li>
</ul>
<h2 style="color: #34495e; margin-top: 28px;">🎯 Результат программы</h2>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 16px; border-radius: 8px;"><code>7</code></pre>
<div style="background: #fff3e0; border: 1px solid #ff9800; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #e65100; margin-top: 0;">🤔 Момент для размышления</h4>
<p style="margin-bottom: 0;">Почему после найденного числа не нужно проверять оставшиеся числа до 100?</p>
</div>
<h2 style="color: #34495e; margin-top: 28px;">📝 Структура программы</h2>
<pre style="background: #263238; color: #aed581; padding: 20px; border-radius: 8px;"><code>public class Main {
    public static void main(String[] args) {
        for (int n = 1; n &lt;= 100; n++) {
            // if (...) { print; break; }
        }
    }
}</code></pre>
<h2 style="color: #34495e; margin-top: 28px;">✅ Чек-лист проверки</h2>
<div style="background: #f8f9fa; border-left: 4px solid #28a745; padding: 16px;"><ul style="margin-bottom: 0;">
	<li>☐ Есть класс <code>Main</code> и метод <code>main</code></li>
	<li>☐ Цикл перебирает числа от 1 до 100</li>
	<li>☐ Условие проверяет оба требования через <code>&amp;&amp;</code></li>
	<li>☐ <code>break</code> расположен внутри блока <code>if</code></li>
	<li>☐ Выводится только найденное число</li>
</ul></div>
<div style="background: #fff8e1; border-left: 4px solid #ffc107; padding: 16px; margin: 16px 0;">
<h4 style="color: #f57c00; margin-top: 0;">⚠️ Технические ограничения</h4>
<ul style="margin-bottom: 0;"><li>Не используйте <code>continue</code> и <code>switch</code>.</li><li>Не выводите поясняющий текст.</li><li>Не ищите все подходящие числа, нужен только первый результат.</li></ul>
</div>
<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #c62828; margin-top: 0;">🐛 Отладка: если что-то не работает</h4>
<p><strong>Выводится несколько чисел?</strong> Проверьте, что после вывода стоит <code>break</code>.</p>
<p><strong>Неверное число?</strong> Проверьте оба остатка: <code>% 7</code> и <code>% 3</code>.</p>
<p style="margin-bottom: 0;"><strong>Проверка не принимает?</strong> Уберите весь лишний текст из вывода.</p>
</div>
