<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🔎 Поиск первого особого числа</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться использовать <code>break</code>, чтобы остановить цикл сразу после первого совпадения.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p><strong>Создайте полную Java-программу:</strong></p>
<ul style="margin: 10px 0;">
	<li>класс должен называться <code>Main</code>;</li>
	<li>в программе должна быть только функция <code>public static void main(String[] args)</code>;</li>
	<li>переберите числа от 1 до 100 включительно;</li>
	<li>найдите первое число, которое делится на 7 и даёт остаток 1 при делении на 3;</li>
	<li>условие: <code>n % 7 == 0 &amp;&amp; n % 3 == 1</code>;</li>
	<li>как только число найдено, выведите его и выполните <code>break</code>;</li>
	<li>не используйте <code>continue</code>, <code>switch</code> и лишний вывод.</li>
</ul>
<p><strong>Вывод:</strong> используйте <code>System.out.println</code> или <code>System.out.printf</code>. Нужен только найденный номер без текста.</p>
</div>
