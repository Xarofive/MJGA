<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">Задача 6.3: Вложенные циклы — поиск пары с меткой</h1>
<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;"><h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3><p style="margin: 0;">Научиться выходить сразу из двух вложенных циклов через метку.</p></div>
<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;"><h3 style="color: #2d3436; margin-top: 0;">🎯 Сценарий</h3><p style="margin-bottom: 10px;">Вы перебираете пары значений. Когда первая подходящая пара найдена, нужно остановить оба цикла, а не только внутренний.</p><p style="margin: 0;"><strong>Ваша задача:</strong> найти первую пару <code>i</code> и <code>j</code> от 1 до 9, произведение которой равно 45.</p></div>
<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2><div style="background: #f0f4f8; border-radius: 8px; padding: 16px;"><h4 style="color: #455a64; margin-top: 0;">Метка цикла</h4><pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px;"><code>outer:
for (...) {
    for (...) {
        break outer;
    }
}</code></pre><p style="margin-bottom: 0;">Обычный <code>break</code> завершит только внутренний цикл. <code>break outer</code> завершает цикл с меткой.</p></div>
<div style="background: #fff8e1; border-left: 4px solid #ffc107; padding: 14px; margin: 16px 0;"><p style="margin: 0;"><strong>Перед этой задачей проверьте:</strong> обычный <code>break</code> выходит только из внутреннего цикла. Метка нужна именно для выхода сразу из двух циклов.</p></div>
<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2><ul><li>Создайте полную Java-программу.</li><li>Внешний цикл перебирает <code>i</code> от 1 до 9.</li><li>Внутренний цикл перебирает <code>j</code> от 1 до 9.</li><li>Если пара не подходит, используйте <code>continue</code>.</li><li>Когда найдено <code>i * j == 45</code>, выведите <code>i</code> и <code>j</code> через пробел.</li><li>После вывода используйте <code>break</code> с меткой для выхода из обоих циклов.</li><li>Не выводите ничего лишнего.</li></ul>
<h2 style="color: #34495e; margin-top: 28px;">🎯 Результат программы</h2><pre style="background: #2c3e50; color: #ecf0f1; padding: 16px; border-radius: 8px;"><code>5 9</code></pre>
<div style="background: #fff3e0; border: 1px solid #ff9800; border-radius: 8px; padding: 14px; margin: 16px 0;"><h4 style="color: #e65100; margin-top: 0;">🤔 Момент для размышления</h4><p style="margin-bottom: 0;">Почему без метки программа может продолжить искать пары после <code>5 9</code>?</p></div>
<h2 style="color: #34495e; margin-top: 28px;">📝 Структура программы</h2><pre style="background: #263238; color: #aed581; padding: 20px; border-radius: 8px;"><code>outer:
for (int i = 1; i &lt;= 9; i++) {
    for (int j = 1; j &lt;= 9; j++) {
        if (i * j != 45) continue;
        System.out.println(i + " " + j);
        break outer;
    }
}</code></pre>
<h2 style="color: #34495e; margin-top: 28px;">✅ Чек-лист проверки</h2><div style="background: #f8f9fa; border-left: 4px solid #28a745; padding: 16px;"><ul style="margin-bottom: 0;"><li><label><input type="checkbox"> Перед внешним циклом стоит метка</label></li><li><label><input type="checkbox"> Оба цикла перебирают числа от 1 до 9</label></li><li><label><input type="checkbox"> Для неподходящих пар используется <code>continue</code></label></li><li><label><input type="checkbox"> Для найденной пары используется <code>break outer</code> или аналогичная метка</label></li><li><label><input type="checkbox"> Выводится ровно одна пара</label></li></ul></div>
<div style="background: #fff8e1; border-left: 4px solid #ffc107; padding: 16px; margin: 16px 0;"><h4 style="color: #f57c00; margin-top: 0;">⚠️ Технические ограничения</h4><ul style="margin-bottom: 0;"><li>Не используйте дополнительный флаг вместо метки.</li><li>Не выводите все пары с произведением 45.</li><li>Не используйте другие операторы выбора.</li></ul></div>
<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;"><h4 style="color: #c62828; margin-top: 0;">🐛 Отладка: если что-то не работает</h4><p><strong>Выводится больше одной пары?</strong> Проверьте <code>break</code> с меткой.</p><p><strong>Пара не 5 9?</strong> Проверьте порядок перебора: сначала <code>i</code>, затем <code>j</code>.</p><p style="margin-bottom: 0;"><strong>Ошибка с меткой?</strong> Метка ставится перед внешним циклом и используется как <code>break outer;</code>.</p></div>
<div style="background: #e8f5e9; border: 1px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;"><h4 style="color: #2e7d32; margin-top: 0;">💪 Дополнительный вызов (после сдачи задачи)</h4><p style="margin-bottom: 0;">Поменяйте цель на <code>36</code> и посмотрите, какая первая пара найдётся при таком порядке перебора.</p></div>
