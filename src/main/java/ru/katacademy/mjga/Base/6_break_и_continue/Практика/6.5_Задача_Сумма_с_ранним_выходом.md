<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">Задача 6.5: Сумма с ранним выходом</h1>
<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;"><h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3><p style="margin: 0;">Научиться останавливать цикл через <code>break</code> после достижения порога.</p></div>
<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;"><h3 style="color: #2d3436; margin-top: 0;">🎯 Сценарий</h3><p style="margin-bottom: 10px;">Вы накапливаете сумму натуральных чисел и останавливаетесь сразу после превышения лимита.</p><p style="margin: 0;"><strong>Ваша задача:</strong> вывести последнее добавленное число и итоговую сумму после превышения 100.</p></div>
<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2><div style="background: #f0f4f8; border-radius: 8px; padding: 16px;"><h4 style="color: #455a64; margin-top: 0;">Порог после обновления</h4><pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px;"><code>sum += i;
last = i;
if (sum &gt; 100) {
    break;
}</code></pre><p style="margin-bottom: 0;">Порог проверяется после добавления числа, потому что нужно знать число, которое впервые превысило лимит.</p></div>
<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2><ul><li>Создайте полную Java-программу.</li><li>Перебирайте числа <code>1, 2, 3, ...</code>.</li><li>Накапливайте сумму в отдельной переменной.</li><li>Сохраняйте последнее добавленное число.</li><li>Когда сумма станет больше 100, выйдите через <code>break</code>.</li><li>После цикла выведите два числа: последнее добавленное и итоговую сумму.</li><li>Можно вывести в одной строке через пробел или в двух строках.</li><li>Не выводите дополнительный текст.</li></ul>
<h2 style="color: #34495e; margin-top: 28px;">🎯 Результат программы</h2><pre style="background: #2c3e50; color: #ecf0f1; padding: 16px; border-radius: 8px;"><code>14 105</code></pre>
<div style="background: #fff3e0; border: 1px solid #ff9800; border-radius: 8px; padding: 14px; margin: 16px 0;"><h4 style="color: #e65100; margin-top: 0;">🤔 Момент для размышления</h4><p style="margin-bottom: 0;">Почему последнее добавленное число равно 14, если сумма до 13 ещё не превышала 100?</p></div>
<h2 style="color: #34495e; margin-top: 28px;">📝 Структура программы</h2><pre style="background: #263238; color: #aed581; padding: 20px; border-radius: 8px;"><code>int sum = 0;
int last = 0;
for (int i = 1; ; i++) {
    sum += i;
    last = i;
    if (sum &gt; 100) break;
}</code></pre>
<h2 style="color: #34495e; margin-top: 28px;">✅ Чек-лист проверки</h2><div style="background: #f8f9fa; border-left: 4px solid #28a745; padding: 16px;"><ul style="margin-bottom: 0;"><li>☐ Сумма обновляется до проверки порога</li><li>☐ Последнее добавленное число сохраняется</li><li>☐ При сумме больше 100 выполняется <code>break</code></li><li>☐ Вывод содержит только два числа: 14 и 105</li></ul></div>
<div style="background: #fff8e1; border-left: 4px solid #ffc107; padding: 16px; margin: 16px 0;"><h4 style="color: #f57c00; margin-top: 0;">⚠️ Технические ограничения</h4><ul style="margin-bottom: 0;"><li>Не используйте готовую формулу суммы.</li><li>Не проверяйте порог до добавления числа.</li><li>Не выводите поясняющий текст.</li></ul></div>
<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;"><h4 style="color: #c62828; margin-top: 0;">🐛 Отладка: если что-то не работает</h4><p><strong>Получилось 13 91?</strong> Вы проверили порог слишком рано.</p><p><strong>Нет остановки?</strong> Проверьте <code>break</code> при <code>sum &gt; 100</code>.</p><p style="margin-bottom: 0;"><strong>Вывод не принят?</strong> Оставьте только числа.</p></div>
<div style="background: #e8f5e9; border: 1px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;"><h4 style="color: #2e7d32; margin-top: 0;">💪 Дополнительный вызов (после сдачи задачи)</h4><p style="margin-bottom: 0;">Поменяйте порог на 50 и посмотрите, на каком числе сумма впервые его превысит.</p></div>
