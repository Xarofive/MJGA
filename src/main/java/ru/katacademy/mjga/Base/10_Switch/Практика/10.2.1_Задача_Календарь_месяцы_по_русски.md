<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">📅 Календарь: месяцы по-русски</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться использовать <code>switch expression</code> для выбора строки по числовому коду и обрабатывать некорректные значения через <code>default</code>.</p>
</div>

<div style="background: #e3f2fd; border: 1px solid #90caf9; border-radius: 8px; padding: 12px; margin: 16px 0;">
<p style="margin: 0;">📖 <strong>Пререквизит:</strong> повторите <code>switch expression</code>, стрелочный синтаксис <code>case ... -></code>, <code>return</code> и отличие метода, который возвращает значение, от вывода в консоль.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">🎯 Сценарий</h3>
<p style="margin-bottom: 10px;">В отчёте или календаре хранится номер месяца, но пользователю нужно показать понятное русское название.</p>
<p style="margin: 0;"><strong>Ваша задача:</strong> написать метод, который превращает число от <code>1</code> до <code>12</code> в название месяца, а все остальные значения — в <code>"unknown"</code>.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Switch expression возвращает значение</h4>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code class="language-java">return switch (month) {
    case 1 -> "январь";
    default -> "unknown";
};</code></pre>
<p style="margin-bottom: 0;">После закрывающей фигурной скобки нужна точка с запятой, потому что весь <code>switch</code> является выражением внутри <code>return</code>.</p>
</div>

<div style="background: #fff3e0; border: 1px solid #ff9800; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #e65100; margin-top: 0;">🔀 Почему здесь не нужен break</h4>
<p style="margin-bottom: 0;">В форме <code>case ... -></code> выполнение не проваливается в следующий <code>case</code>, поэтому <code>break</code> не нужен.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;"><strong>Реализуйте метод:</strong></p>
<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>public static String monthName(int month)</code></div>
<h4 style="color: #e65100; margin-top: 16px;">📝 Требования</h4>
<ul style="margin: 10px 0;">
	<li><code>1</code> → <code>"январь"</code>, <code>2</code> → <code>"февраль"</code>, <code>3</code> → <code>"март"</code></li>
	<li><code>4</code> → <code>"апрель"</code>, <code>5</code> → <code>"май"</code>, <code>6</code> → <code>"июнь"</code></li>
	<li><code>7</code> → <code>"июль"</code>, <code>8</code> → <code>"август"</code>, <code>9</code> → <code>"сентябрь"</code></li>
	<li><code>10</code> → <code>"октябрь"</code>, <code>11</code> → <code>"ноябрь"</code>, <code>12</code> → <code>"декабрь"</code></li>
	<li>Для любого другого числа верните <code>"unknown"</code></li>
	<li>Используйте <code>switch expression</code> и форму <code>case ... -></code></li>
	<li>Метод должен возвращать строку, а не печатать её</li>
</ul>
<h4 style="color: #e65100; margin-top: 16px;">🚫 Ограничения</h4>
<ul style="margin: 10px 0;">
	<li>Не используйте массивы, коллекции, <code>Map</code> и дополнительные методы</li>
	<li>Не используйте <code>System.out.println</code></li>
	<li>Не указывайте <code>package</code> внутри решения</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Результат программы</h2>
