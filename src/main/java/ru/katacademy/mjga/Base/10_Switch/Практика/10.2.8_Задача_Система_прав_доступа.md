<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">Система прав доступа</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">Цель задачи</h3>
<p style="margin: 0;">Использовать <code>switch expression</code> для маршрутизации ролей и синонимов ролей к уровню доступа.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">Контекст</h2>
<p>В приложениях часто нужно определить, что может делать пользователь. Роль приходит строкой, а программа должна вернуть понятный набор прав.</p>

<h2 style="color: #34495e; margin-top: 28px;">Техническое задание</h2>
<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;">Создайте метод:</p>
<pre style="background: white; border: 1px solid #e9ecef; border-radius: 5px; padding: 12px;"><code class="language-java">public static String accessLevel(String role)</code></pre>
<p><strong>Правила:</strong></p>
<ul>
	<li><code>"admin"</code>, <code>"administrator"</code>, <code>"root"</code> → <code>"all"</code></li>
	<li><code>"editor"</code>, <code>"moderator"</code> → <code>"write"</code></li>
	<li><code>"viewer"</code>, <code>"reader"</code>, <code>"guest"</code> → <code>"read"</code></li>
	<li><code>"banned"</code>, <code>"blocked"</code> → <code>"none"</code></li>
	<li>Любая другая роль → <code>"unknown"</code></li>
</ul>
<p>Регистр и пробелы по краям не должны влиять на результат: <code>" ADMIN "</code> обрабатывается как <code>"admin"</code>.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">Подсказка</h2>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px;"><code class="language-java">String normalized = role.trim().toLowerCase();

return switch (normalized) {
    case "admin", "administrator", "root" -> "all";
    case "editor", "moderator" -> "write";
    case "viewer", "reader", "guest" -> "read";
    case "banned", "blocked" -> "none";
    default -> "unknown";
};</code></pre>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 16px 0;">
<p style="margin: 0;"><strong>Частая ошибка:</strong> не нормализовать строку. Тогда <code>"Admin"</code>, <code>" admin"</code> и <code>"admin"</code> станут разными ролями.</p>
</div>
