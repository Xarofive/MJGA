<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🔐 Система прав доступа</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Использовать <code>switch</code> для выбора набора прав по роли пользователя.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p>В Java не нужно использовать fall-through для наследования прав. Проще и безопаснее вернуть готовый массив для каждой роли.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px;">
<p><strong>Реализуйте метод:</strong></p>
<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px;"><code>static String[] permissions(String role)</code></div>
<ul>
	<li><code>"admin"</code> → <code>{"read", "write", "delete"}</code></li>
	<li><code>"editor"</code> → <code>{"read", "write"}</code></li>
	<li><code>"viewer"</code> → <code>{"read"}</code></li>
	<li>Неизвестная роль, пустая строка или <code>null</code> → пустой массив</li>
	<li>Регистр роли не важен: <code>"ADMIN"</code> работает как <code>"admin"</code></li>
	<li>Используйте <code>switch</code> по нормализованной роли</li>
	<li>Не используйте <code>ArrayList</code> и другие коллекции</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Примеры</h2>
<div style="background: linear-gradient(135deg, #e3f2fd 0%, #bbdefb 100%); border-radius: 10px; padding: 20px; margin: 20px 0; border: 1px solid #64b5f6;">
<p><code>permissions("admin")</code> → <code>{"read", "write", "delete"}</code></p>
<p><code>permissions("EDITOR")</code> → <code>{"read", "write"}</code></p>
<p><code>permissions("viewer")</code> → <code>{"read"}</code></p>
<p style="margin-bottom: 0;"><code>permissions("guest")</code> → пустой массив</p>
</div>
