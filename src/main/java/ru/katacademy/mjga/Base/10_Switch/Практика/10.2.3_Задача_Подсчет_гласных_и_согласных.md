<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">Подсчёт гласных и согласных</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">Цель задачи</h3>
<p style="margin: 0;">Закрепить <code>switch</code> по символу, нормализацию регистра и аккуратный перебор строки.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">Кратко</h2>
<p>Нужно посчитать количество английских гласных и английских согласных в строке. Остальные символы игнорируются.</p>

<h2 style="color: #34495e; margin-top: 28px;">Техническое задание</h2>
<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;">Создайте метод:</p>
<pre style="background: white; border: 1px solid #e9ecef; border-radius: 5px; padding: 12px;"><code class="language-java">public static String countVowelsAndConsonants(String text)</code></pre>
<p><strong>Метод возвращает строку:</strong> <code>"vowels consonants"</code>.</p>
<p><strong>Правила:</strong></p>
<ul>
	<li>Гласные: <code>a</code>, <code>e</code>, <code>i</code>, <code>o</code>, <code>u</code>.</li>
	<li>Регистр не важен: <code>A</code> считается как <code>a</code>.</li>
	<li>Английские согласные — буквы <code>a..z</code>, которые не являются гласными.</li>
	<li>Цифры, пробелы, знаки препинания и кириллица не учитываются.</li>
	<li>Для определения гласных используйте <code>switch</code> по <code>char</code>.</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">Подсказка</h2>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px;"><code class="language-java">char lower = Character.toLowerCase(ch);

switch (lower) {
    case 'a', 'e', 'i', 'o', 'u' -> vowels++;
    default -> {
        if (lower &gt;= 'a' &amp;&amp; lower &lt;= 'z') {
            consonants++;
        }
    }
}</code></pre>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 16px 0;">
<p style="margin: 0;"><strong>Частая ошибка:</strong> считать любую букву, не попавшую в гласные, согласной. Тогда кириллица попадёт в согласные, хотя по условию её нужно игнорировать.</p>
