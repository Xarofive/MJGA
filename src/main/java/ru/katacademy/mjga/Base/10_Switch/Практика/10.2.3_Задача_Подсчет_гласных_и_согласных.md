<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px;">
<p><strong>Реализуйте метод:</strong></p>
<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px;"><code>static int[] countVowels(String text)</code></div>
<ul>
	<li>Верните массив из двух чисел: <code>{vowels, consonants}</code></li>
	<li>Считайте только английские буквы</li>
	<li>Гласные: <code>a</code>, <code>e</code>, <code>i</code>, <code>o</code>, <code>u</code></li>
	<li>Регистр не важен: <code>A</code> считается как <code>a</code></li>
	<li>Не буквы игнорируются</li>
	<li>Если <code>text == null</code>, верните <code>{0, 0}</code></li>
	<li>Используйте <code>switch</code> для определения гласных</li>
</ul>
</div>
