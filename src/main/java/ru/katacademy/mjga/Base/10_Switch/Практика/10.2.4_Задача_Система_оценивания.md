<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">Система оценивания</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">Цель задачи</h3>
<p style="margin: 0;">Понять границу применения <code>switch</code> в Java: диапазоны сначала превращаются в категорию, а затем обрабатываются через <code>switch</code>.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">Что важно</h2>
<p>В Java нельзя написать <code>case score &gt;= 90 -></code>. Поэтому в этой задаче нужно сначала проверить диапазон <code>0..100</code>, затем вычислить <code>bucket = score / 10</code> и использовать <code>switch</code> по <code>bucket</code>.</p>

<h2 style="color: #34495e; margin-top: 28px;">Техническое задание</h2>
<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;">Создайте метод:</p>
<pre style="background: white; border: 1px solid #e9ecef; border-radius: 5px; padding: 12px;"><code class="language-java">public static String grade(int score)</code></pre>
<p><strong>Правила:</strong></p>
<ul>
	<li><code>90..100</code> → <code>"A"</code></li>
	<li><code>80..89</code> → <code>"B"</code></li>
	<li><code>70..79</code> → <code>"C"</code></li>
	<li><code>60..69</code> → <code>"D"</code></li>
	<li><code>0..59</code> → <code>"F"</code></li>
	<li>Любое значение меньше <code>0</code> или больше <code>100</code> → <code>"invalid"</code></li>
</ul>
<p>После проверки валидности используйте <code>switch expression</code>.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">Подсказка</h2>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px;"><code class="language-java">if (score &lt; 0 || score &gt; 100) {
    return "invalid";
}

int bucket = score / 10;

return switch (bucket) {
    case 10, 9 -> "A";
    case 8 -> "B";
    case 7 -> "C";
    case 6 -> "D";
    default -> "F";
};</code></pre>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 16px 0;">
<p style="margin: 0;"><strong>Частая ошибка:</strong> забыть отдельно обработать <code>100</code>. При делении на 10 оно даёт <code>10</code>, поэтому <code>case 10, 9 -> "A"</code>.</p>
