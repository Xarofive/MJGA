<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">📊 Система оценивания</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Понять границу применения <code>switch</code>: диапазоны удобнее сначала превратить в код категории.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p>В Java классический <code>switch</code> не проверяет условия вида <code>score &gt;= 90</code>. Поэтому сначала вычислите десяток: <code>int bucket = score / 10</code>, а затем используйте <code>switch</code> по <code>bucket</code>.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px;">
<p><strong>Реализуйте метод:</strong></p>
<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px;"><code>static String grade(int score)</code></div>
<ul>
	<li><code>90..100</code> → <code>"A"</code></li>
	<li><code>80..89</code> → <code>"B"</code></li>
	<li><code>70..79</code> → <code>"C"</code></li>
	<li><code>60..69</code> → <code>"D"</code></li>
	<li><code>0..59</code> → <code>"F"</code></li>
	<li>Вне диапазона <code>0..100</code> → <code>"invalid"</code></li>
	<li>Используйте <code>switch</code> после проверки валидного диапазона</li>
</ul>
</div>
