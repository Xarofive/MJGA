<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🔢 Парсер байта с классификацией результата</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться разбирать строку в число диапазона <code>0..255</code> и классифицировать результат через <code>switch</code>.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px;">
<p><strong>Реализуйте метод:</strong></p>
<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px;"><code>static String parseByteInfo(String text)</code></div>
<ul>
	<li>Если строка успешно парсится и число в диапазоне <code>0..255</code>, верните <code>"ok"</code></li>
	<li>Если <code>text == null</code> или строка пустая после <code>trim()</code>, верните <code>"empty"</code></li>
	<li>Если строка не является целым числом, верните <code>"bad syntax"</code></li>
	<li>Если число меньше 0 или больше 255, верните <code>"out of range"</code></li>
	<li>Используйте <code>Integer.parseInt</code> и <code>switch</code> для финальной классификации состояния</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Примеры</h2>
<div style="background: linear-gradient(135deg, #e3f2fd 0%, #bbdefb 100%); border-radius: 10px; padding: 20px; margin: 20px 0; border: 1px solid #64b5f6;">
<p><code>parseByteInfo("0")</code>, <code>parseByteInfo("255")</code> → <code>"ok"</code></p>
<p><code>parseByteInfo("")</code> → <code>"empty"</code></p>
<p><code>parseByteInfo("abc")</code> → <code>"bad syntax"</code></p>
<p style="margin-bottom: 0;"><code>parseByteInfo("-1")</code>, <code>parseByteInfo("256")</code> → <code>"out of range"</code></p>
</div>
