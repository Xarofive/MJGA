<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">Классификация HTTP-кодов</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">Цель задачи</h3>
<p style="margin: 0;">Потренировать классификацию числовых кодов через предварительную группу и <code>switch expression</code>.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">Контекст</h2>
<p>HTTP-коды используются веб-серверами и клиентами. По первой цифре кода можно понять общий тип ответа: успех, перенаправление, ошибка клиента или ошибка сервера.</p>

<h2 style="color: #34495e; margin-top: 28px;">Техническое задание</h2>
<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;">Создайте метод:</p>
<pre style="background: white; border: 1px solid #e9ecef; border-radius: 5px; padding: 12px;"><code class="language-java">public static String httpCategory(int code)</code></pre>
<p><strong>Верните:</strong></p>
<ul>
	<li><code>100..199</code> → <code>"informational"</code></li>
	<li><code>200..299</code> → <code>"success"</code></li>
	<li><code>300..399</code> → <code>"redirection"</code></li>
	<li><code>400..499</code> → <code>"client error"</code></li>
	<li><code>500..599</code> → <code>"server error"</code></li>
	<li>Все остальные значения → <code>"unknown"</code></li>
</ul>
<p>Используйте <code>int group = code / 100</code> и <code>switch expression</code> по <code>group</code>.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">Подсказка</h2>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px;"><code class="language-java">if (code &lt; 100 || code &gt; 599) {
    return "unknown";
}

int group = code / 100;

return switch (group) {
    case 1 -> "informational";
    case 2 -> "success";
    case 3 -> "redirection";
    case 4 -> "client error";
    case 5 -> "server error";
    default -> "unknown";
};</code></pre>
