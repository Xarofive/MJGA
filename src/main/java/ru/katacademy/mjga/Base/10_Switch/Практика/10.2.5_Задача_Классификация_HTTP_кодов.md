<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🌐 Классификация HTTP-кодов</h1>
<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;"><h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3><p style="margin: 0;">Потренировать классификацию числовых кодов через предварительную группу и <code>switch expression</code>.</p></div>
<div style="background: #e3f2fd; border: 1px solid #90caf9; border-radius: 8px; padding: 12px; margin: 16px 0;"><p style="margin: 0;">📖 <strong>Пререквизит:</strong> повторите целочисленное деление, диапазоны, HTTP-коды на уровне общей идеи и <code>switch expression</code>.</p></div>
<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;"><h3 style="color: #2d3436; margin-top: 0;">🎯 Сценарий</h3><p style="margin-bottom: 10px;">Сервер вернул числовой HTTP-код. По первой цифре нужно понять общий тип ответа: успех, перенаправление, ошибка клиента или ошибка сервера.</p><p style="margin: 0;"><strong>Ваша задача:</strong> вернуть текстовую категорию HTTP-кода.</p></div>
<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2><div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;"><h4 style="color: #455a64; margin-top: 0;">Группа по первой цифре</h4><pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code class="language-java">int group = code / 100;</code></pre><p style="margin-bottom: 0;">Для <code>404</code> группа равна <code>4</code>, для <code>200</code> — <code>2</code>.</p></div>
<div style="background: #fff3e0; border: 1px solid #ff9800; border-radius: 8px; padding: 14px; margin: 16px 0;"><h4 style="color: #e65100; margin-top: 0;">🚧 Сначала проверьте диапазон</h4><p style="margin-bottom: 0;">HTTP-коды в этой задаче считаются валидными только от <code>100</code> до <code>599</code>. Всё остальное — <code>"unknown"</code>.</p></div>
<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2><div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;"><p style="margin-top: 0;"><strong>Реализуйте метод:</strong></p><div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>public static String httpCategory(int code)</code></div><h4 style="color: #e65100; margin-top: 16px;">📝 Правила</h4><ul style="margin: 10px 0;"><li><code>100..199</code> → <code>"informational"</code></li><li><code>200..299</code> → <code>"success"</code></li><li><code>300..399</code> → <code>"redirection"</code></li><li><code>400..499</code> → <code>"client error"</code></li><li><code>500..599</code> → <code>"server error"</code></li><li>Все остальные значения → <code>"unknown"</code></li><li>Используйте <code>int group = code / 100</code> и <code>switch expression</code></li></ul><h4 style="color: #e65100; margin-top: 16px;">🚫 Ограничения</h4><ul style="margin: 10px 0;"><li>Не перечисляйте все HTTP-коды вручную</li><li>Не используйте массивы, коллекции или <code>Map</code></li><li>Не выводите результат в консоль</li></ul></div>
<h2 style="color: #34495e; margin-top: 28px;">🎯 Результат программы</h2><div style="background: linear-gradient(135deg, #e3f2fd 0%, #bbdefb 100%); border-radius: 10px; padding: 20px; margin: 20px 0; border: 1px solid #64b5f6;"><div style="background: white; border-radius: 6px; padding: 15px; box-shadow: 0 2px 5px rgba(0,0,0,0.1);"><code>httpCategory(200)</code> → <code>"success"</code><br><code>httpCategory(301)</code> → <code>"redirection"</code><br><code>httpCategory(404)</code> → <code>"client error"</code><br><code>httpCategory(599)</code> → <code>"server error"</code><br><code>httpCategory(700)</code> → <code>"unknown"</code></div></div>
<div style="background: #fff3e0; border: 1px solid #ff9800; border-radius: 8px; padding: 14px; margin: 16px 0;"><h4 style="color: #e65100; margin-top: 0;">🤔 Момент для размышления</h4><p style="margin-bottom: 0;">Почему в этой задаче важна группа кода, а не конкретное значение <code>404</code> или <code>500</code>?</p></div>

        return "unknown";
    }

    };

<h2 style="color: #34495e; margin-top: 28px;">✅ Чек-лист проверки</h2><div style="background: #f8f9fa; border-left: 4px solid #28a745; padding: 16px; margin: 16px 0;"><ul style="margin-bottom: 0;"><li>☐ Метод называется <code>httpCategory</code></li><li>☐ Диапазон <code>100..599</code> проверяется</li><li>☐ Используется <code>code / 100</code></li><li>☐ Используется <code>switch expression</code></li><li>☐ Не перечисляются все коды</li><li>☐ Нет вывода в консоль</li></ul></div>
<div style="background: #fff8e1; border-left: 4px solid #ffc107; padding: 16px; margin: 16px 0;"><h4 style="color: #f57c00; margin-top: 0;">⚠️ Технические ограничения</h4><ul style="margin-bottom: 0;"><li>Не используйте массивы и коллекции</li><li>Не используйте <code>switch</code> по каждому конкретному коду</li><li>Не возвращайте пустую строку для неизвестного кода</li></ul></div>
