<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🌐 Классификация HTTP-кодов</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Использовать <code>switch</code> для классификации числового кода по группе.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px;">
<p><strong>Реализуйте метод:</strong></p>
<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px;"><code>static String httpClass(int code)</code></div>
<ul>
	<li>Валидные HTTP-коды: <code>100..599</code></li>
	<li>Верните <code>"1xx"</code>, <code>"2xx"</code>, <code>"3xx"</code>, <code>"4xx"</code> или <code>"5xx"</code></li>
	<li>Для невалидного кода верните <code>"unknown"</code></li>
	<li>Используйте <code>switch</code> по значению <code>code / 100</code></li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Примеры</h2>
<div style="background: linear-gradient(135deg, #e3f2fd 0%, #bbdefb 100%); border-radius: 10px; padding: 20px; margin: 20px 0; border: 1px solid #64b5f6;">
<p><code>httpClass(200)</code> → <code>"2xx"</code></p>
<p><code>httpClass(404)</code> → <code>"4xx"</code></p>
<p><code>httpClass(599)</code> → <code>"5xx"</code></p>
<p style="margin-bottom: 0;"><code>httpClass(99)</code>, <code>httpClass(600)</code> → <code>"unknown"</code></p>
</div>
