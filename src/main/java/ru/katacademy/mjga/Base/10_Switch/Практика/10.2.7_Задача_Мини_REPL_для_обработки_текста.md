<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">Мини-REPL для обработки текста</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">Цель задачи</h3>
<p style="margin: 0;">Научиться обрабатывать поток текстовых команд через <code>switch</code> и нормализацию пользовательского ввода.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">Контекст</h2>
<p>REPL — это цикл «прочитать команду, выполнить действие, вывести ответ». Такие обработчики встречаются в консольных утилитах, интерактивных оболочках и админских инструментах.</p>

<h2 style="color: #34495e; margin-top: 28px;">Техническое задание</h2>
<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;">Создайте метод:</p>
<pre style="background: white; border: 1px solid #e9ecef; border-radius: 5px; padding: 12px;"><code class="language-java">public static void processTextCommands(BufferedReader reader, PrintWriter writer) throws IOException</code></pre>
<p><strong>Команды:</strong></p>
<ul>
	<li><code>upper text</code> — вывести <code>text</code> в верхнем регистре.</li>
	<li><code>lower text</code> — вывести <code>text</code> в нижнем регистре.</li>
	<li><code>trim text</code> — вывести <code>text</code> без пробелов по краям.</li>
	<li><code>len text</code> — вывести длину <code>text</code> через <code>text.length()</code>.</li>
	<li><code>help</code> или <code>?</code> — вывести <code>commands: upper, lower, trim, len, help, quit</code>.</li>
	<li><code>quit</code>, <code>exit</code> или <code>q</code> — вывести <code>bye</code> и завершить метод.</li>
	<li>Неизвестная команда — вывести <code>UNKNOWN COMMAND</code>.</li>
</ul>
<p>Команда регистронезависима: <code>UPPER text</code> и <code>upper text</code> работают одинаково.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">Пример</h2>
<pre style="background: #f5f5f5; padding: 12px; border-radius: 6px;"><code>Ввод:
upper hello
lower JAVA
len test
?
unknown data
quit

Вывод:
HELLO
java
4
commands: upper, lower, trim, len, help, quit
UNKNOWN COMMAND
bye</code></pre>

<h2 style="color: #34495e; margin-top: 28px;">Подсказка</h2>
<p>Нужно отделить первое слово от остального текста. Для этого можно найти первый пробел через <code>indexOf(' ')</code>. Если пробела нет, вся строка является командой, а текстовая часть пустая.</p>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px;"><code class="language-java">String line = reader.readLine();
int spaceIndex = line.indexOf(' ');

String command;
String text;

if (spaceIndex == -1) {
    command = line.trim().toLowerCase();
    text = "";
} else {
    command = line.substring(0, spaceIndex).trim().toLowerCase();
    text = line.substring(spaceIndex + 1);
}</code></pre>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 16px 0;">
<p style="margin: 0;"><strong>Частая ошибка:</strong> использовать <code>split(" ")</code> и потерять часть текста после второго пробела. Здесь текстовая часть может содержать пробелы, поэтому лучше отделять только первое слово.</p>
</div>
