<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">⌨️ Мини-REPL для обработки текста</h1>
<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;"><h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3><p style="margin: 0;">Научиться обрабатывать поток текстовых команд через <code>switch</code> и нормализацию пользовательского ввода.</p></div>
<div style="background: #e3f2fd; border: 1px solid #90caf9; border-radius: 8px; padding: 12px; margin: 16px 0;"><p style="margin: 0;">📖 <strong>Пререквизит:</strong> повторите <code>BufferedReader</code>, <code>PrintWriter</code>, <code>throws IOException</code>, <code>indexOf</code>, <code>substring</code>, <code>trim</code>, <code>toLowerCase</code> и <code>switch</code> по строке.</p></div>
<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;"><h3 style="color: #2d3436; margin-top: 0;">🎯 Сценарий</h3><p style="margin-bottom: 10px;">REPL — это цикл “прочитать команду, выполнить действие, вывести ответ”. Такие обработчики есть в консольных утилитах, админских инструментах и интерактивных оболочках.</p><p style="margin: 0;"><strong>Ваша задача:</strong> читать команды до <code>quit</code> и применять простые операции к тексту.</p></div>
<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2><div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;"><h4 style="color: #455a64; margin-top: 0;">Команда и хвост строки</h4><p>Нужно отделить первое слово от остального текста. Остальная часть может содержать пробелы, поэтому <code>split(" ")</code> здесь неудобен.</p><pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code class="language-java">int spaceIndex = line.indexOf(' ');
String command = spaceIndex == -1 ? line.trim() : line.substring(0, spaceIndex).trim();
String text = spaceIndex == -1 ? "" : line.substring(spaceIndex + 1);</code></pre></div>
<div style="background: #fff3e0; border: 1px solid #ff9800; border-radius: 8px; padding: 14px; margin: 16px 0;"><h4 style="color: #e65100; margin-top: 0;">🔡 Команда регистронезависима</h4><p style="margin-bottom: 0;">Команду нужно привести к нижнему регистру. Текстовую часть при этом не меняйте заранее: разные команды работают с ней по-разному.</p></div>
<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2><div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;"><p style="margin-top: 0;"><strong>Реализуйте метод:</strong></p><div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>public static void processTextCommands(BufferedReader reader, PrintWriter writer) throws IOException</code></div><h4 style="color: #e65100; margin-top: 16px;">📝 Команды</h4><ul style="margin: 10px 0;"><li><code>upper text</code> — вывести <code>text</code> в верхнем регистре</li><li><code>lower text</code> — вывести <code>text</code> в нижнем регистре</li><li><code>trim text</code> — вывести <code>text</code> без пробелов по краям</li><li><code>len text</code> — вывести длину <code>text</code> через <code>text.length()</code></li><li><code>help</code> или <code>?</code> — вывести <code>commands: upper, lower, trim, len, help, quit</code></li><li><code>quit</code>, <code>exit</code> или <code>q</code> — вывести <code>bye</code> и завершить метод</li><li>Неизвестная команда — вывести <code>UNKNOWN COMMAND</code></li></ul><h4 style="color: #e65100; margin-top: 16px;">🚫 Ограничения</h4><ul style="margin: 10px 0;"><li>Не используйте <code>split(" ")</code> так, чтобы терялась часть текста</li><li>Не закрывайте <code>reader</code> и <code>writer</code></li><li>Не используйте <code>System.out</code></li></ul></div>
