<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🔄 Мини-REPL для обработки текста</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Закрепить чтение строк из <code>BufferedReader</code>, вывод через <code>PrintWriter</code> и маршрутизацию команд через <code>switch</code>.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p>REPL — это цикл Read-Eval-Print-Loop: прочитать команду, обработать, напечатать результат, повторить. Для тестируемости метод получает <code>BufferedReader</code> и <code>PrintWriter</code> параметрами.</p>
<p style="margin-bottom: 0;"><code>PrintWriter</code> — это объект для записи текста. В этой задаче воспринимайте его как место, куда нужно печатать результат через <code>writer.println(...)</code>. Подробная работа с потоками вывода будет позже.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px;">
<p><strong>Реализуйте метод:</strong></p>
<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px;"><code>static void miniRepl(BufferedReader reader, PrintWriter writer) throws IOException</code></div>
<ul>
	<li>Читайте команды построчно до конца потока или команды <code>QUIT</code></li>
	<li>Команды регистронезависимы</li>
	<li><code>UPPER text</code> → вывести <code>text.toUpperCase()</code></li>
	<li><code>LOWER text</code> → вывести <code>text.toLowerCase()</code></li>
	<li><code>LEN text</code> → вывести длину текста через <code>text.length()</code></li>
	<li><code>QUIT</code> → завершить метод без вывода</li>
	<li>Неизвестная команда → вывести <code>ERR unknown command</code></li>
	<li>Используйте <code>switch</code> по команде</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Пример</h2>
<div style="background: linear-gradient(135deg, #e3f2fd 0%, #bbdefb 100%); border-radius: 10px; padding: 20px; margin: 20px 0; border: 1px solid #64b5f6;">
<p>Ввод: <code>UPPER hello</code> → вывод: <code>HELLO</code></p>
<p>Ввод: <code>LOWER Java</code> → вывод: <code>java</code></p>
<p>Ввод: <code>LEN test</code> → вывод: <code>4</code></p>
<p style="margin-bottom: 0;">Ввод: <code>REVERSE abc</code> → вывод: <code>ERR unknown command</code></p>
</div>
