<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🔤 Классификация символов</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться использовать <code>switch</code> по символу и стандартные проверки <code>Character</code>.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p><code>Character.isLetter(ch)</code>, <code>Character.isDigit(ch)</code> и <code>Character.isWhitespace(ch)</code> классифицируют символы. Для явного закрепления <code>switch</code> сначала обработайте специальные пробельные символы через <code>switch</code>, а затем используйте проверки <code>Character</code>.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px;">
<p><strong>Реализуйте метод:</strong></p>
<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px;"><code>static String classifyChar(char ch)</code></div>
<ul>
	<li>Если символ — пробел, табуляция или перевод строки, верните <code>"space"</code></li>
	<li>Если символ — буква, верните <code>"letter"</code></li>
	<li>Если символ — цифра, верните <code>"digit"</code></li>
	<li>Иначе верните <code>"other"</code></li>
	<li>Используйте <code>switch</code> для явной обработки <code>' '</code>, <code>'\t'</code>, <code>'\n'</code></li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Примеры</h2>
<div style="background: linear-gradient(135deg, #e3f2fd 0%, #bbdefb 100%); border-radius: 10px; padding: 20px; margin: 20px 0; border: 1px solid #64b5f6;">
<p><code>classifyChar('A')</code> → <code>"letter"</code></p>
<p><code>classifyChar('7')</code> → <code>"digit"</code></p>
<p><code>classifyChar(' ')</code> → <code>"space"</code></p>
<p style="margin-bottom: 0;"><code>classifyChar('@')</code> → <code>"other"</code></p>
</div>
