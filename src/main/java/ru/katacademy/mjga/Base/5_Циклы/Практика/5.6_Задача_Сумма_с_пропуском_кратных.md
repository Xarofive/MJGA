<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">Задача 5.6: Сумма с пропуском кратных</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться фильтровать значения внутри цикла без <code>break</code> и <code>continue</code>.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">🎯 Сценарий</h3>
<p style="margin-bottom: 10px;">В отчёте нужно сложить числа диапазона, но исключить значения, которые кратны заданному делителю.</p>
<p style="margin: 0;"><strong>Ваша задача:</strong> пройти от <code>1</code> до <code>limit</code> и суммировать только числа, не кратные <code>skip</code>.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Проверка кратности</h4>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>6 % 3 == 0 // 6 кратно 3
7 % 3 == 1 // 7 не кратно 3</code></pre>
<p style="margin-bottom: 0;">Чтобы добавить только некратные числа, используйте условие <code>i % skip != 0</code>.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<p><strong>Создайте метод с точной сигнатурой:</strong></p>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 16px; border-radius: 8px; overflow-x: auto;"><code>static int sumSkipMultiples(int limit, int skip)</code></pre>
<ul>
	<li>суммировать числа от <code>1</code> до <code>limit</code> включительно;</li>
	<li>не добавлять числа, кратные <code>skip</code>;</li>
	<li>если <code>limit &lt; 1</code>, вернуть <code>0</code>;</li>
	<li>если <code>skip &lt;= 1</code>, вернуть <code>0</code>;</li>
	<li>использовать классический цикл <code>for</code>;</li>
	<li>кратность проверять через <code>%</code>;</li>
	<li>не использовать <code>break</code>, <code>continue</code>, вспомогательные методы и вывод в консоль.</li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Результат программы</h2>

sumSkipMultiples(6, 2)  -> 9
sumSkipMultiples(0, 5)  -> 0
sumSkipMultiples(10, 1) -> 0</code></pre>

<h4 style="color: #e65100; margin-top: 0;">🤔 Момент для размышления</h4>
<p style="margin-bottom: 0;">Почему при <code>skip &lt;= 1</code> нужно сразу вернуть <code>0</code>? Что значит пропускать числа, кратные <code>1</code>?</p>
</div>
