<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">➕ Сумма с пропуском кратных</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться фильтровать значения внутри цикла без <code>break</code> и <code>continue</code>.</p>
</div>

<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Проверка кратности</h4>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>6 % 3 == 0 // 6 кратно 3
7 % 3 == 1 // 7 не кратно 3</code></pre>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #e65100; margin-top: 0;">🔧 Метод sumSkipMultiples</h4>
<p><strong>Сигнатура:</strong></p>
<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>static int sumSkipMultiples(int limit, int skip)</code></div>
<ul style="margin: 10px 0;">
	<li>суммировать числа от 1 до <code>limit</code> включительно;</li>
	<li>не добавлять числа, кратные <code>skip</code>;</li>
	<li>если <code>limit &lt; 1</code>, вернуть 0;</li>
	<li>если <code>skip &lt;= 1</code>, вернуть 0;</li>
	<li>использовать классический цикл <code>for</code>;</li>
	<li>кратность проверять через <code>%</code>;</li>
	<li>не использовать <code>break</code>, <code>continue</code>, вспомогательные методы и вывод в консоль.</li>
</ul>
<details style="margin: 15px 0;"><summary style="cursor: pointer; color: #e65100; font-weight: bold;">Подсказка по реализации</summary>
<div style="background: #fff8e1; padding: 12px; border-radius: 4px; margin-top: 10px;">
<p style="margin: 0;">Добавляйте число только если <code>i % skip != 0</code>.</p>
</div>
</details>
</div>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #c62828; margin-top: 0;">⚠️ Частые ошибки</h4>
<p><strong>Деление на 1:</strong> при <code>skip &lt;= 1</code> нужно сразу вернуть 0.</p>
<p><strong>Использование continue:</strong> в этой задаче фильтрация должна быть через <code>if</code>.</p>
<p style="margin-bottom: 0;"><strong>Добавлены кратные числа:</strong> проверяйте условие <code>i % skip != 0</code>.</p>
</div>
