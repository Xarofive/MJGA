<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">Задача 5.1: Сумма чисел от 1 до N</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Освоить итеративное накопление суммы с помощью цикла <code>for</code>.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">🎯 Сценарий</h3>
<p style="margin-bottom: 10px;">В аналитике и отчётах часто нужно пройти по последовательности значений и накопить общий итог.</p>
<p style="margin: 0;"><strong>Ваша задача:</strong> вернуть сумму всех чисел от <code>1</code> до <code>n</code> включительно.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Паттерн накопления</h4>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>int sum = 0;

for (int i = 1; i &lt;= n; i++) {
    sum += i;
}</code></pre>
<p style="margin-bottom: 0;">Аккумулятор суммы обычно начинается с <code>0</code>, потому что это нейтральное значение для сложения.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<p><strong>Создайте метод с точной сигнатурой:</strong></p>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 16px; border-radius: 8px; overflow-x: auto;"><code>static int sumToN(int n)</code></pre>
<ul>
	<li>вернуть сумму всех чисел от <code>1</code> до <code>n</code> включительно;</li>
	<li>использовать цикл <code>for</code>;</li>
	<li>при <code>n = 0</code> вернуть <code>0</code>;</li>
	<li>в тестах <code>n</code> не будет отрицательным;</li>
	<li>отправьте только метод, без класса <code>Main</code> и без <code>import</code>.</li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Результат программы</h2>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 16px; border-radius: 8px; overflow-x: auto;"><code>sumToN(5)  -> 15
sumToN(10) -> 55
sumToN(1)  -> 1
sumToN(0)  -> 0</code></pre>

<div style="background: #fff3e0; border: 1px solid #ff9800; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #e65100; margin-top: 0;">🤔 Момент для размышления</h4>
<p style="margin-bottom: 0;">Почему условие цикла должно быть <code>i &lt;= n</code>, а не <code>i &lt; n</code>?</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📝 Структура метода</h2>
<pre style="background: #263238; color: #aed581; padding: 20px; border-radius: 8px; overflow-x: auto;"><code>static int sumToN(int n) {
    int sum = 0;
    // цикл от 1 до n включительно
    return sum;
}</code></pre>

<h2 style="color: #34495e; margin-top: 28px;">✅ Чек-лист проверки</h2>
<div style="background: #f8f9fa; border-left: 4px solid #28a745; padding: 16px; margin: 16px 0;">
<ul style="margin-bottom: 0;">
	<li>☐ Метод имеет сигнатуру <code>static int sumToN(int n)</code></li>
	<li>☐ Цикл идёт от <code>1</code> до <code>n</code> включительно</li>
	<li>☐ Сумма начинается с <code>0</code></li>
	<li>☐ При <code>n = 0</code> возвращается <code>0</code></li>
	<li>☐ Нет вывода в консоль</li>
</ul>
</div>

<div style="background: #fff8e1; border-left: 4px solid #ffc107; padding: 16px; margin: 16px 0;">
<h4 style="color: #f57c00; margin-top: 0;">⚠️ Технические ограничения</h4>
<ul style="margin-bottom: 0;">
	<li>Не используйте готовую формулу суммы вместо цикла</li>
	<li>Не используйте <code>System.out.println</code></li>
	<li>Не добавляйте класс <code>Main</code>, если платформа ждёт только метод</li>
	<li>Откажитесь от цикла, который пропускает последнее число</li>
</ul>
</div>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
