<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">➕ Сумма чисел от 1 до N</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Освоить итеративное накопление суммы с помощью цикла.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">💼 Контекст</h3>
<p style="margin: 0;">Суммирование последовательностей используется в статистике, аналитике и расчёте агрегированных значений. Здесь нужно пройти по числам от 1 до N и накопить сумму.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Паттерн накопления</h4>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>int sum = 0;

for (int i = 1; i &lt;= n; i++) {
    sum += i;
}</code></pre>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #e65100; margin-top: 0;">🔧 Метод sumToN</h4>
<p><strong>Сигнатура:</strong></p>
<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>static int sumToN(int n)</code></div>
<ul style="margin: 10px 0;">
	<li>вернуть сумму всех чисел от 1 до <code>n</code> включительно;</li>
	<li>использовать цикл <code>for</code>;</li>
	<li>при <code>n = 0</code> вернуть 0;</li>
	<li>в тестах <code>n</code> не будет отрицательным;</li>
	<li>отправьте только метод, без класса <code>Main</code> и без <code>import</code>;</li>
	<li>не используйте вывод в консоль.</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Примеры работы</h2>
<div style="background: #e3f2fd; border: 1px solid #64b5f6; border-radius: 10px; padding: 16px; margin: 20px 0;">
<pre style="margin: 0;"><code>sumToN(5)  → 15
sumToN(10) → 55
sumToN(1)  → 1
sumToN(0)  → 0</code></pre>
</div>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #c62828; margin-top: 0;">⚠️ Частые ошибки</h4>
<p><strong>Пропуск последнего числа:</strong> условие должно включать <code>n</code>: <code>i &lt;= n</code>.</p>
<p><strong>Неверный аккумулятор:</strong> для суммы начальное значение должно быть <code>0</code>.</p>
<p style="margin-bottom: 0;"><strong>Печать вместо возврата:</strong> метод должен вернуть число через <code>return</code>.</p>
</div>
