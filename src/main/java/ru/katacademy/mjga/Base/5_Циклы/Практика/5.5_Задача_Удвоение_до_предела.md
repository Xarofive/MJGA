<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">Задача 5.5: Удвоение до предела</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Потренироваться использовать цикл <code>while</code>, который завершается естественно по условию.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">🎯 Сценарий</h3>
<p style="margin-bottom: 10px;">Значение растёт в два раза на каждом шаге, пока не достигнет заданного предела.</p>
<p style="margin: 0;"><strong>Ваша задача:</strong> вернуть первое значение, которое стало больше или равно <code>limit</code>.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Цикл while</h4>
<p><code>while</code> удобен, когда заранее неизвестно точное количество повторений, но известно условие продолжения.</p>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>int value = start;
while (value &lt; limit) {
    value *= 2;
}</code></pre>
<p style="margin-bottom: 0;">Если <code>start</code> уже не меньше <code>limit</code>, цикл не выполнится ни разу.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<p><strong>Создайте метод с точной сигнатурой:</strong></p>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 16px; border-radius: 8px; overflow-x: auto;"><code>static int doubleUntilLimit(int start, int limit)</code></pre>
<ul>
	<li>удваивать значение <code>start</code>, пока оно меньше <code>limit</code>;</li>
	<li>вернуть первое значение, которое больше или равно <code>limit</code>;</li>
	<li>если <code>start &gt;= limit</code> изначально, вернуть <code>start</code> без цикла;</li>
	<li>использовать цикл <code>while</code>;</li>
	<li>внутри цикла удваивать значение через <code>*= 2</code>;</li>
	<li>не использовать <code>break</code>, <code>continue</code> и вывод в консоль.</li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Результат программы</h2>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 16px; border-radius: 8px; overflow-x: auto;"><code>doubleUntilLimit(3, 50)    -> 96
doubleUntilLimit(100, 50)  -> 100
doubleUntilLimit(10, 25)   -> 40
doubleUntilLimit(1, 1000)  -> 1024</code></pre>

<div style="background: #fff3e0; border: 1px solid #ff9800; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #e65100; margin-top: 0;">🤔 Момент для размышления</h4>
<p style="margin-bottom: 0;">Почему метод возвращает значение, которое может быть больше <code>limit</code>, а не обязательно равно ему?</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📝 Структура метода</h2>
<pre style="background: #263238; color: #aed581; padding: 20px; border-radius: 8px; overflow-x: auto;"><code>static int doubleUntilLimit(int start, int limit) {
    int value = start;
    while (value &lt; limit) {
        value *= 2;
    }
    return value;
}</code></pre>

<h2 style="color: #34495e; margin-top: 28px;">✅ Чек-лист проверки</h2>
<div style="background: #f8f9fa; border-left: 4px solid #28a745; padding: 16px; margin: 16px 0;">
<ul style="margin-bottom: 0;">
	<li>☐ Метод имеет сигнатуру <code>static int doubleUntilLimit(int start, int limit)</code></li>
	<li>☐ Используется цикл <code>while</code></li>
	<li>☐ Значение удваивается через <code>*= 2</code></li>
	<li>☐ Если <code>start &gt;= limit</code>, метод возвращает <code>start</code></li>
	<li>☐ Нет <code>break</code>, <code>continue</code> и вывода в консоль</li>
</ul>
</div>

<div style="background: #fff8e1; border-left: 4px solid #ffc107; padding: 16px; margin: 16px 0;">
<h4 style="color: #f57c00; margin-top: 0;">⚠️ Технические ограничения</h4>
<ul style="margin-bottom: 0;">
	<li>Не используйте <code>break</code> и <code>continue</code></li>
	<li>Не используйте цикл <code>for</code></li>
	<li>Не используйте вывод в консоль</li>
	<li>Откажитесь от ручного подбора количества удвоений</li>
</ul>
</div>
