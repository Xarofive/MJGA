<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Потренироваться использовать цикл <code>while</code>, который завершается естественно по условию.</p>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #e65100; margin-top: 0;">🔧 Метод doubleUntilLimit</h4>
<p><strong>Сигнатура:</strong></p>
<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>static int doubleUntilLimit(int start, int limit)</code></div>
<ul style="margin: 10px 0;">
	<li>удваивать значение <code>start</code>, пока оно меньше <code>limit</code>;</li>
	<li>вернуть первое значение, которое больше или равно <code>limit</code>;</li>
	<li>если <code>start &gt;= limit</code> изначально, вернуть <code>start</code> без цикла;</li>
	<li>использовать цикл <code>while</code>;</li>
	<li>внутри цикла удваивать значение через <code>*= 2</code>;</li>
	<li>не использовать <code>break</code>, <code>continue</code> и вывод в консоль.</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Примеры работы</h2>
<div style="background: #e3f2fd; border: 1px solid #64b5f6; border-radius: 10px; padding: 16px; margin: 20px 0;">
<pre style="margin: 0;"><code>doubleUntilLimit(3, 50)    → 96
doubleUntilLimit(100, 50)  → 100
doubleUntilLimit(10, 25)   → 40
doubleUntilLimit(1, 1000)  → 1024</code></pre>
</div>

<p style="margin-bottom: 0;">Создайте переменную <code>value</code>, присвойте ей <code>start</code>, затем выполняйте <code>while (value &lt; limit)</code>.</p>
