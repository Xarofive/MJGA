<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">Задача 5.4: Сумма цифр числа</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Освоить извлечение цифр числа через деление и остаток от деления.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">🎯 Сценарий</h3>
<p style="margin-bottom: 10px;">Вместо работы со строкой вы разбираете число арифметически: берёте последнюю цифру и отбрасываете её.</p>
<p style="margin: 0;"><strong>Ваша задача:</strong> вернуть сумму всех цифр числа, игнорируя знак.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Извлечение цифр</h4>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>1234 % 10 = 4   // последняя цифра
1234 / 10 = 123 // отбросили последнюю цифру</code></pre>
<p style="margin-bottom: 0;">В Java целочисленное деление отбрасывает дробную часть.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<p><strong>Создайте метод с точной сигнатурой:</strong></p>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 16px; border-radius: 8px; overflow-x: auto;"><code>static int sumDigits(int n)</code></pre>
<ul>
	<li>вернуть сумму всех цифр числа;</li>
	<li>для отрицательных чисел знак игнорируется;</li>
	<li>для нуля вернуть <code>0</code>;</li>
	<li>использовать цикл и арифметические операции;</li>
	<li>запрещено преобразовывать число в строку;</li>
	<li>не используйте рекурсию, импорт и вывод в консоль.</li>
</ul>
