<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🔢 Сумма цифр числа</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Освоить извлечение цифр числа через деление и остаток от деления.</p>
</div>

<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Извлечение цифр</h4>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>1234 % 10 = 4   // последняя цифра
1234 / 10 = 123 // отбросили последнюю цифру</code></pre>
<p style="margin-bottom: 0;">В Java целочисленное деление отбрасывает дробную часть.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #e65100; margin-top: 0;">🔧 Метод sumDigits</h4>
<p><strong>Сигнатура:</strong></p>
<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>static int sumDigits(int n)</code></div>
<ul style="margin: 10px 0;">
	<li>вернуть сумму всех цифр числа;</li>
	<li>для отрицательных чисел знак игнорируется;</li>
	<li>для нуля вернуть 0;</li>
	<li>использовать цикл и арифметические операции;</li>
	<li>запрещено преобразовывать число в строку;</li>
	<li>не используйте рекурсию, импорт и вывод в консоль.</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Примеры работы</h2>
<div style="background: #e3f2fd; border: 1px solid #64b5f6; border-radius: 10px; padding: 16px; margin: 20px 0;">
<pre style="margin: 0;"><code>sumDigits(1234) → 10
sumDigits(567)  → 18
sumDigits(0)    → 0
sumDigits(-123) → 6</code></pre>
</div>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #c62828; margin-top: 0;">⚠️ Частые ошибки</h4>
<p><strong>Бесконечный цикл:</strong> не забывайте делить число на 10 на каждой итерации.</p>
<p><strong>Отрицательный остаток:</strong> перед обработкой отрицательное число нужно сделать положительным.</p>
<p style="margin-bottom: 0;"><strong>Преобразование в строку:</strong> в этой задаче оно запрещено.</p>
</div>

<div style="background: #e8f5e9; border: 1px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">

<ul style="margin-bottom: 0; list-style: none; padding-left: 0;">

</ul>
