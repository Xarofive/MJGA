<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">⚡ Возведение числа в степень</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться реализовывать математическую операцию через цикл и накопление результата.</p>
</div>

<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Математическая основа</h4>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>x^n = x * x * ... * x  // n раз
x^0 = 1
x^1 = x</code></pre>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #e65100; margin-top: 0;">🔧 Метод power</h4>
<p><strong>Сигнатура:</strong></p>
<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>static int power(int x, int n)</code></div>
<ul style="margin: 10px 0;">
	<li>вернуть результат возведения <code>x</code> в степень <code>n</code>;</li>
	<li><code>n</code> всегда неотрицательное;</li>
	<li>использовать цикл <code>for</code>;</li>
	<li>не использовать <code>Math.pow</code> и рекурсию;</li>
	<li>отправьте только метод, без класса и импортов;</li>
	<li>не используйте вывод в консоль.</li>
</ul>
<p><strong>Пояснение:</strong> <code>Math.pow</code> &mdash; готовый метод стандартной библиотеки для возведения в степень. Рекурсия &mdash; это приём, когда метод вызывает сам себя. В этой задаче они запрещены, потому что нужно потренировать именно цикл.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Примеры работы</h2>
<div style="background: #e3f2fd; border: 1px solid #64b5f6; border-radius: 10px; padding: 16px; margin: 20px 0;">
<pre style="margin: 0;"><code>power(2, 3)   → 8
power(5, 0)   → 1
power(7, 1)   → 7
power(1, 100) → 1</code></pre>
</div>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #c62828; margin-top: 0;">⚠️ Частые ошибки</h4>
<p><strong>Начальное значение 0:</strong> для умножения аккумулятор должен начинаться с <code>1</code>.</p>
<p><strong>Умножение на степень:</strong> нужно умножать на <code>x</code>, а не на <code>n</code>.</p>
<p style="margin-bottom: 0;"><strong>Нулевая степень:</strong> при <code>n = 0</code> цикл не должен выполняться, результат должен быть 1.</p>
</div>

</ul>
