<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">Задача 5.3: Возведение числа в степень</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться реализовывать математическую операцию через цикл и накопление результата.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">🎯 Сценарий</h3>
<p style="margin-bottom: 10px;">Вместо готового библиотечного метода вы вручную моделируете степень как повторное умножение.</p>
<p style="margin: 0;"><strong>Ваша задача:</strong> вернуть <code>x</code> в степени <code>n</code>, используя цикл <code>for</code>.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Повторное умножение</h4>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>x^n = x * x * ... * x  // n раз
x^0 = 1
x^1 = x</code></pre>
<p style="margin-bottom: 0;">Аккумулятор для умножения должен начинаться с <code>1</code>, а не с <code>0</code>.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<p><strong>Создайте метод с точной сигнатурой:</strong></p>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 16px; border-radius: 8px; overflow-x: auto;"><code>static int power(int x, int n)</code></pre>
<ul>
	<li>вернуть результат возведения <code>x</code> в степень <code>n</code>;</li>
	<li><code>n</code> всегда неотрицательное;</li>
	<li>использовать цикл <code>for</code>;</li>
	<li>не использовать <code>Math.pow</code> и рекурсию;</li>
	<li>отправьте только метод, без класса и импортов.</li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Результат программы</h2>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 16px; border-radius: 8px; overflow-x: auto;"><code>power(2, 3)   -> 8
power(5, 0)   -> 1
power(7, 1)   -> 7
power(1, 100) -> 1</code></pre>

<div style="background: #fff3e0; border: 1px solid #ff9800; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #e65100; margin-top: 0;">🤔 Момент для размышления</h4>
<p style="margin-bottom: 0;">Почему при <code>n = 0</code> результат должен быть <code>1</code>, даже если цикл не выполнится ни разу?</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📝 Структура метода</h2>
<pre style="background: #263238; color: #aed581; padding: 20px; border-radius: 8px; overflow-x: auto;"><code>static int power(int x, int n) {
    int result = 1;
    // умножьте result на x ровно n раз
    return result;
}</code></pre>

<h2 style="color: #34495e; margin-top: 28px;">✅ Чек-лист проверки</h2>
<div style="background: #f8f9fa; border-left: 4px solid #28a745; padding: 16px; margin: 16px 0;">
<ul style="margin-bottom: 0;">
	<li>☐ Метод имеет сигнатуру <code>static int power(int x, int n)</code></li>
	<li>☐ Результат начинается с <code>1</code></li>
	<li>☐ Цикл выполняется ровно <code>n</code> раз</li>
	<li>☐ Не используется <code>Math.pow</code></li>
	<li>☐ Метод возвращает результат через <code>return</code></li>
</ul>
</div>

<div style="background: #fff8e1; border-left: 4px solid #ffc107; padding: 16px; margin: 16px 0;">
<h4 style="color: #f57c00; margin-top: 0;">⚠️ Технические ограничения</h4>
<ul style="margin-bottom: 0;">
	<li>Не используйте <code>Math.pow</code></li>
	<li>Не используйте рекурсию</li>
	<li>Не используйте вывод в консоль</li>
	<li>Откажитесь от аккумулятора <code>0</code>: при умножении он обнулит результат</li>
</ul>
</div>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #c62828; margin-top: 0;">🐛 Отладка: если что-то не работает</h4>
<table style="width: 100%; margin-top: 10px;">
	<tbody>
		<tr><td style="padding: 8px; vertical-align: top;"><strong><code>power(5, 0)</code> не 1?</strong></td><td style="padding: 8px;">→ Начальное значение результата должно быть <code>1</code></td></tr>

		<tr><td style="padding: 8px; vertical-align: top;"><strong>Умножаете не на то?</strong></td><td style="padding: 8px;">→ На каждой итерации умножайте на <code>x</code>, а не на <code>n</code></td></tr>
