<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🔢 Сумма чётных чисел в диапазоне</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться применять условие внутри цикла и суммировать только подходящие числа.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Проверка чётности</h4>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>if (i % 2 == 0) {
    sum += i;
}</code></pre>
<p style="margin-bottom: 0;">Оператор <code>%</code> возвращает остаток от деления. Чётное число делится на 2 без остатка.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #e65100; margin-top: 0;">🔧 Метод sumEven</h4>
<p><strong>Сигнатура:</strong></p>
<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>static int sumEven(int n)</code></div>
<ul style="margin: 10px 0;">
	<li>вернуть сумму всех чётных чисел от 1 до <code>n</code> включительно;</li>
	<li>использовать цикл <code>for</code>;</li>
	<li>обязательно использовать оператор <code>%</code> для проверки чётности;</li>
	<li>при <code>n &lt; 1</code> вернуть 0;</li>
	<li>отправьте только метод, без класса и импортов;</li>
	<li>не используйте вывод в консоль.</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Примеры работы</h2>
<div style="background: #e3f2fd; border: 1px solid #64b5f6; border-radius: 10px; padding: 16px; margin: 20px 0;">
<pre style="margin: 0;"><code>sumEven(5)  → 6
sumEven(10) → 30
sumEven(2)  → 2
sumEven(1)  → 0
sumEven(0)  → 0</code></pre>
</div>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #c62828; margin-top: 0;">⚠️ Частые ошибки</h4>
<p><strong>Суммируются все числа:</strong> добавляйте число только внутри проверки <code>i % 2 == 0</code>.</p>
<p><strong>Диапазон не включает <code>n</code>:</strong> используйте <code>i &lt;= n</code>.</p>
<p style="margin-bottom: 0;"><strong>Нет return:</strong> метод должен вернуть накопленную сумму.</p>
</div>
