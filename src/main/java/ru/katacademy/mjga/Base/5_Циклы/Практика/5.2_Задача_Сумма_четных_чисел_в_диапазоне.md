<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">Задача 5.2: Сумма чётных чисел в диапазоне</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться применять условие внутри цикла и суммировать только подходящие числа.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">🎯 Сценарий</h3>
<p style="margin-bottom: 10px;">В отчёте нужно учитывать не все значения подряд, а только те, которые проходят фильтр.</p>
<p style="margin: 0;"><strong>Ваша задача:</strong> пройти по числам от <code>1</code> до <code>n</code> и сложить только чётные.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Проверка чётности</h4>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>if (i % 2 == 0) {
    sum += i;
}</code></pre>
<p style="margin-bottom: 0;">Оператор <code>%</code> возвращает остаток от деления. Чётное число делится на <code>2</code> без остатка.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<p><strong>Создайте метод с точной сигнатурой:</strong></p>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 16px; border-radius: 8px; overflow-x: auto;"><code>static int sumEven(int n)</code></pre>
<ul>
	<li>вернуть сумму всех чётных чисел от <code>1</code> до <code>n</code> включительно;</li>
	<li>использовать цикл <code>for</code>;</li>
	<li>обязательно использовать оператор <code>%</code> для проверки чётности;</li>
	<li>при <code>n &lt; 1</code> вернуть <code>0</code>;</li>
	<li>отправьте только метод, без класса и импортов.</li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Результат программы</h2>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 16px; border-radius: 8px; overflow-x: auto;"><code>sumEven(5)  -> 6
sumEven(10) -> 30
sumEven(2)  -> 2
sumEven(1)  -> 0
sumEven(0)  -> 0</code></pre>

<div style="background: #fff3e0; border: 1px solid #ff9800; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #e65100; margin-top: 0;">🤔 Момент для размышления</h4>
<p style="margin-bottom: 0;">Почему число нужно добавлять в сумму только внутри <code>if</code>, а не на каждой итерации?</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📝 Структура метода</h2>
<pre style="background: #263238; color: #aed581; padding: 20px; border-radius: 8px; overflow-x: auto;"><code>static int sumEven(int n) {
    int sum = 0;
    // цикл и проверка i % 2 == 0
    return sum;
}</code></pre>
