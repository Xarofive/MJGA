<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">Частотный анализ категорий</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">Цель задачи</h3>
<p style="margin: 0;">Посчитать частоту строк через <code>HashMap</code> и вывести результат в отсортированном порядке.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">Кратко</h2>

<p>На вход приходит список категорий. Нужно вывести каждую уникальную категорию и количество её появлений. Категории выводятся лексикографически.</p>

<h2 style="color: #34495e; margin-top: 28px;">Техническое задание</h2>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;">Создайте метод:</p>

<pre style="background: white; border: 1px solid #e9ecef; border-radius: 5px; padding: 12px; margin: 10px 0;"><code class="language-java">public static ArrayList&lt;String&gt; frequencyReport(String[] categories)</code></pre>

<p><strong>Метод должен вернуть:</strong></p>
<ul>
	<li><code>ArrayList&lt;String&gt;</code>, где каждая строка имеет формат <code>"category count"</code>.</li>
	<li>Если входной массив пустой, вернуть список с одной строкой <code>"EMPTY"</code>.</li>
	<li>Категории должны быть отсортированы через <code>Collections.sort</code>.</li>
	<li>Регистр важен: <code>Food</code>, <code>food</code> и <code>FOOD</code> — разные категории.</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">Пример</h2>

<pre style="background: #f5f5f5; padding: 12px; border-radius: 6px;"><code>Вход:
{"apple", "apple", "pear", "berry", "pear"}

Результат:
["apple 2", "berry 1", "pear 2"]</code></pre>

<h2 style="color: #34495e; margin-top: 28px;">Алгоритм</h2>

<ol>
	<li>Создайте <code>HashMap&lt;String, Integer&gt;</code> для счётчиков.</li>
	<li>Для каждой категории увеличьте счётчик через <code>getOrDefault</code>.</li>
	<li>Скопируйте ключи в <code>ArrayList&lt;String&gt;</code>.</li>
	<li>Отсортируйте ключи через <code>Collections.sort(keys)</code>.</li>
	<li>Соберите строки ответа в новый <code>ArrayList&lt;String&gt;</code>.</li>
</ol>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code class="language-java">ArrayList&lt;String&gt; keys = new ArrayList&lt;&gt;(counts.keySet());
Collections.sort(keys);</code></pre>
