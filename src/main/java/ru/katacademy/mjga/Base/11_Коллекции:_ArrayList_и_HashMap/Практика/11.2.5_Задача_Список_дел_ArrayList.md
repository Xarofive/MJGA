<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">Список дел на ArrayList</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">Цель задачи</h3>
<p style="margin: 0;">Закрепить базовые операции <code>ArrayList</code>: добавление, получение, замена, удаление и просмотр размера.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">Техническое задание</h2>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;">Создайте метод:</p>

<pre style="background: white; border: 1px solid #e9ecef; border-radius: 5px; padding: 12px; margin: 10px 0;"><code class="language-java">public static void processTodoList(BufferedReader reader, PrintWriter writer) throws IOException</code></pre>

<p><strong>Команды:</strong></p>
<ul>
	<li><code>ADD text</code> — добавить задачу в конец списка.</li>
	<li><code>GET index</code> — вывести задачу по индексу или <code>INVALID</code>.</li>
	<li><code>SET index text</code> — заменить задачу по индексу, вывести <code>OK</code> или <code>INVALID</code>.</li>
	<li><code>REMOVE index</code> — удалить задачу по индексу, вывести <code>OK</code> или <code>INVALID</code>.</li>
	<li><code>COUNT</code> — вывести количество задач.</li>
</ul>

<p>Индексы начинаются с <code>0</code>. Текст задачи в этой версии не содержит пробелов.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">Пример</h2>

<pre style="background: #f5f5f5; padding: 12px; border-radius: 6px;"><code>Ввод:
8
ADD read
ADD practice
GET 0
SET 1 solve
GET 1
REMOVE 0
COUNT
GET 3

Вывод:
read
OK
solve
OK
1
INVALID</code></pre>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code class="language-java">if (index &gt;= 0 &amp;&amp; index &lt; tasks.size()) {
    writer.println(tasks.get(index));
} else {
    writer.println("INVALID");
}</code></pre>
