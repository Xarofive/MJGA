<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">Учёт товаров по категориям</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">Цель задачи</h3>
<p style="margin: 0;">Освоить вложенный <code>HashMap</code>: категория хранит отдельную карту товаров, а каждый товар хранит накопленную стоимость.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">Кратко</h2>

<p>Реализуйте командный учёт товаров. Данные хранятся в структуре:</p>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code class="language-java">HashMap&lt;String, HashMap&lt;String, Integer&gt;&gt; inventory = new HashMap&lt;&gt;();</code></pre>

<p>Внешний ключ — категория, внутренний ключ — товар, значение — суммарная стоимость товара.</p>

<h2 style="color: #34495e; margin-top: 28px;">Техническое задание</h2>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;">Создайте метод:</p>

<pre style="background: white; border: 1px solid #e9ecef; border-radius: 5px; padding: 12px; margin: 10px 0;"><code class="language-java">public static void processInventory(BufferedReader reader, PrintWriter writer) throws IOException</code></pre>

<p><strong>Формат входных данных:</strong></p>
<ul>
	<li>Первая строка: количество команд <code>N</code>.</li>
	<li>Следующие <code>N</code> строк: команды.</li>
</ul>

<p><strong>Команды:</strong></p>
<ul>
	<li><code>ADD category item price quantity</code> — добавить к товару стоимость <code>price * quantity</code>.</li>
	<li><code>DELETE category item</code> — удалить товар.</li>
	<li><code>TOTAL category</code> — вывести сумму по категории.</li>
	<li><code>SUMMARY</code> — вывести все категории с суммами в лексикографическом порядке.</li>
</ul>

<p><strong>Поведение:</strong></p>
<ul>
	<li><code>ADD</code> ничего не выводит.</li>
	<li><code>DELETE</code> выводит <code>OK</code>, если товар был удалён, иначе <code>NOT FOUND</code>.</li>
	<li>После удаления последнего товара категория тоже удаляется.</li>
	<li><code>TOTAL</code> выводит сумму категории или <code>NOT FOUND</code>.</li>
	<li><code>SUMMARY</code> выводит строки <code>category sum</code> или <code>EMPTY</code>, если категорий нет.</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">Пример</h2>

<pre style="background: #f5f5f5; padding: 12px; border-radius: 6px;"><code>Ввод:
10
ADD groceries milk 50 2
ADD groceries bread 30 1
TOTAL groceries
DELETE groceries milk
TOTAL groceries
DELETE groceries milk
ADD tech phone 200 3
SUMMARY
TOTAL tech
TOTAL groceries

Вывод:
130
OK
30
NOT FOUND
groceries 30
tech 600
600
30</code></pre>

<h2 style="color: #34495e; margin-top: 28px;">Подсказки</h2>

<ul>
	<li>Перед добавлением товара проверьте, есть ли категория. Если нет — положите туда новый <code>HashMap&lt;String, Integer&gt;</code>.</li>
	<li>Для накопления стоимости используйте <code>getOrDefault(item, 0)</code>.</li>
	<li>Для <code>SUMMARY</code> скопируйте категории в <code>ArrayList</code> и отсортируйте через <code>Collections.sort</code>.</li>
	<li>Для разбора чисел используйте <code>Integer.parseInt</code>.</li>
</ul>
