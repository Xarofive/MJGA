<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">Удаление дубликатов с сохранением порядка</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">Цель задачи</h3>
<p style="margin: 0;">Использовать <code>HashMap</code> как множество для быстрой проверки повторов и <code>ArrayList</code> для сохранения порядка первого появления.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">Кратко</h2>

<p>Дан массив целых чисел. Нужно вернуть список уникальных чисел в том порядке, в котором они впервые встретились.</p>

<pre style="background: #f5f5f5; padding: 12px; border-radius: 6px;"><code>Вход:  [1, 2, 3, 2, 4, 1, 5]
Выход: [1, 2, 3, 4, 5]</code></pre>

<h2 style="color: #34495e; margin-top: 28px;">Техническое задание</h2>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;">Создайте метод:</p>

<pre style="background: white; border: 1px solid #e9ecef; border-radius: 5px; padding: 12px; margin: 10px 0;"><code class="language-java">public static ArrayList&lt;Integer&gt; uniqueInOrder(int[] values)</code></pre>

<p><strong>Требования:</strong></p>
<ul>
	<li>Используйте <code>HashMap&lt;Integer, Boolean&gt;</code> для проверки, встречалось ли число.</li>
	<li>Используйте <code>ArrayList&lt;Integer&gt;</code> для результата.</li>
	<li>Не сортируйте результат.</li>
	<li>Если массив пустой, верните пустой <code>ArrayList</code>.</li>
	<li>Алгоритм должен работать за <code>O(n)</code> по времени.</li>
</ul>
</div>
