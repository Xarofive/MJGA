<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">Фильтрация списка на ArrayList</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">Цель задачи</h3>
<p style="margin: 0;">Научиться строить новый <code>ArrayList</code> на основе старого, не изменяя исходный список.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">Кратко</h2>

<p>Дан массив чисел и граница <code>limit</code>. Нужно вернуть <code>ArrayList</code> со всеми числами, которые больше или равны <code>limit</code>, сохранив исходный порядок.</p>

<h2 style="color: #34495e; margin-top: 28px;">Техническое задание</h2>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;">Создайте метод:</p>

<pre style="background: white; border: 1px solid #e9ecef; border-radius: 5px; padding: 12px; margin: 10px 0;"><code class="language-java">public static ArrayList&lt;Integer&gt; filterAtLeast(int[] values, int limit)</code></pre>

<p><strong>Требования:</strong></p>
<ul>
	<li>Создайте новый <code>ArrayList&lt;Integer&gt;</code> для результата.</li>
	<li>Добавляйте в него только числа <code>&gt;= limit</code>.</li>
	<li>Сохраняйте исходный порядок подходящих чисел.</li>
	<li>Если подходящих чисел нет, верните пустой список.</li>
	<li>Не сортируйте результат.</li>
</ul>
</div>
