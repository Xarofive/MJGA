<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">☎️ Телефонная книга</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться хранить пары ключ-значение в <code>HashMap</code>, обновлять данные по ключу и проверять наличие контакта перед чтением или удалением.</p>
</div>

<div style="background: #e3f2fd; border: 1px solid #90caf9; border-radius: 8px; padding: 12px; margin: 16px 0;">
<p style="margin: 0;">📖 <strong>Пререквизит:</strong> повторите <code>HashMap</code>, методы <code>put</code>, <code>get</code>, <code>containsKey</code>, <code>remove</code>, <code>size</code>, чтение строк и вывод через <code>PrintWriter</code>.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">🎯 Сценарий</h3>
<p style="margin-bottom: 10px;">Вы пишете простой обработчик команд для телефонной книги. Контакты добавляются, обновляются, удаляются и ищутся по имени.</p>
<p style="margin: 0;"><strong>Ваша задача:</strong> обработать заданное количество команд и выводить ответы только для команд, которые этого требуют.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">HashMap как телефонная книга</h4>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code class="language-java">HashMap&lt;String, String&gt; phonebook = new HashMap&lt;&gt;();
phonebook.put(name, phone);
String phone = phonebook.get(name);</code></pre>
<p style="margin-bottom: 0;">Ключ — имя контакта. Значение — телефон. Если выполнить <code>put</code> с уже существующим именем, телефон обновится.</p>
</div>

<div style="background: #fff3e0; border: 1px solid #ff9800; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #e65100; margin-top: 0;">🔍 Проверка перед чтением и удалением</h4>
<p style="margin-bottom: 0;">Для <code>GET</code> и <code>DEL</code> сначала проверяйте <code>containsKey</code>. Так вы точно знаете, выводить телефон, <code>OK</code> или <code>NOT FOUND</code>.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;"><strong>Реализуйте метод:</strong></p>
<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>public static void processPhonebook(BufferedReader reader, PrintWriter writer) throws IOException</code></div>
<h4 style="color: #e65100; margin-top: 16px;">📝 Формат команд</h4>
<ul style="margin: 10px 0;">
	<li>Первая строка: число <code>Q</code> — количество команд</li>
	<li><code>PUT name phone</code> — добавить или обновить контакт, ничего не выводит</li>
	<li><code>GET name</code> — вывести телефон или <code>NOT FOUND</code></li>
	<li><code>DEL name</code> — удалить контакт, вывести <code>OK</code> или <code>NOT FOUND</code></li>
	<li><code>COUNT</code> — вывести количество контактов</li>
</ul>
<h4 style="color: #e65100; margin-top: 16px;">🚫 Ограничения</h4>
<ul style="margin: 10px 0;">
	<li>Используйте <code>HashMap&lt;String, String&gt;</code></li>
	<li>Не создавайте переменные под конкретные имена вроде <code>Alice</code></li>
	<li>Не используйте <code>System.out</code>, выводите через <code>writer</code></li>
	<li>Не закрывайте <code>reader</code> и <code>writer</code></li>
</ul>
</div>
