<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">Телефонная книга</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">Цель задачи</h3>
<p style="margin: 0;">Научиться хранить пары ключ-значение в <code>HashMap</code>, обновлять данные по ключу и проверять наличие контакта перед чтением или удалением.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">Кратко</h2>

<p>Реализуйте метод <code>processPhonebook</code>, который читает команды и управляет телефонной книгой:</p>

<ul>
	<li><code>PUT name phone</code> — добавить или обновить контакт;</li>
	<li><code>GET name</code> — вывести телефон или <code>NOT FOUND</code>;</li>
	<li><code>DEL name</code> — удалить контакт, вывести <code>OK</code> или <code>NOT FOUND</code>;</li>
	<li><code>COUNT</code> — вывести количество контактов.</li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">Что нужно знать</h2>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code class="language-java">HashMap&lt;String, String&gt; phonebook = new HashMap&lt;&gt;();

phonebook.put(name, phone);

if (phonebook.containsKey(name)) {
    writer.println(phonebook.get(name));
} else {
    writer.println("NOT FOUND");
}</code></pre>

<p>Ключ в этой задаче — имя контакта. Это динамический ключ: он приходит из ввода, а не прописан заранее в коде.</p>

<h2 style="color: #34495e; margin-top: 28px;">Техническое задание</h2>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;">Создайте метод:</p>

<pre style="background: white; border: 1px solid #e9ecef; border-radius: 5px; padding: 12px; margin: 10px 0;"><code class="language-java">public static void processPhonebook(BufferedReader reader, PrintWriter writer) throws IOException</code></pre>

<p><strong>Формат входных данных:</strong></p>
<ul>
	<li>Первая строка: число <code>Q</code> — количество команд.</li>
	<li>Следующие <code>Q</code> строк: команды <code>PUT</code>, <code>GET</code>, <code>DEL</code> или <code>COUNT</code>.</li>
</ul>

<p><strong>Формат вывода:</strong></p>
<ul>
	<li><code>PUT</code> ничего не выводит.</li>
	<li><code>GET</code> выводит телефон или <code>NOT FOUND</code>.</li>
	<li><code>DEL</code> выводит <code>OK</code>, если контакт был удалён, иначе <code>NOT FOUND</code>.</li>
	<li><code>COUNT</code> выводит количество контактов.</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">Пример</h2>

<pre style="background: #f5f5f5; padding: 12px; border-radius: 6px;"><code>Ввод:
7
PUT Alice 123456
PUT Bob 789012
GET Alice
GET Charlie
COUNT
DEL Bob
COUNT

Вывод:
123456
NOT FOUND
2
OK
1</code></pre>

<h2 style="color: #34495e; margin-top: 28px;">Подсказки</h2>

<ul>
	<li>Используйте <code>HashMap&lt;String, String&gt;</code>.</li>
	<li>Перед <code>GET</code> и <code>DEL</code> проверяйте ключ через <code>containsKey</code>.</li>
	<li>Для разбора команды можно использовать <code>line.split("\\s+")</code>.</li>
	<li>Не создавайте отдельные переменные для конкретных имён. Телефонная книга должна работать с любым именем из ввода.</li>
</ul>
