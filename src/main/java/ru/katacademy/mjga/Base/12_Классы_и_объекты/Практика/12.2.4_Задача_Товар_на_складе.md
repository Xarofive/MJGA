<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">📦 Товар на складе</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться создавать класс с несколькими методами: расчёт, добавление товара и безопасная продажа.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">🎯 Сценарий</h3>
<p style="margin-bottom: 10px;">Вы пишете простую складскую систему. У товара есть название, цена и количество. Склад должен уметь принимать поставку и продавать товар, если остатка хватает.</p>
<p style="margin: 0;"><strong>Ваша задача:</strong> описать товар как объект с состоянием и поведением.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Метод расчёта</h4>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code class="language-java">int totalPrice() {
    return price * quantity;
}</code></pre>
<p style="margin-bottom: 0;">Метод может ничего не менять, а только считать значение на основе текущего состояния.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;"><strong>Создайте класс:</strong></p>
<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>class Product</code></div>
<h4 style="color: #e65100; margin-top: 16px;">📝 Поля</h4>
<ul style="margin: 10px 0;">
	<li><code>String name</code> — название товара</li>
	<li><code>int price</code> — цена одной единицы</li>
	<li><code>int quantity</code> — количество на складе</li>
</ul>
<h4 style="color: #e65100; margin-top: 16px;">⚡ Методы</h4>
<ul style="margin: 10px 0;">
	<li><code>int totalPrice()</code> — возвращает <code>price * quantity</code></li>
	<li><code>void add(int amount)</code> — увеличивает количество, если <code>amount &gt; 0</code></li>
	<li><code>boolean sell(int amount)</code> — уменьшает количество и возвращает <code>true</code>, если <code>amount &gt; 0</code> и товара хватает</li>
	<li><code>String getLabel()</code> — возвращает <code>"&lt;name&gt; (&lt;quantity&gt; pcs)"</code></li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Результат программы</h2>
<div style="background: linear-gradient(135deg, #e3f2fd 0%, #bbdefb 100%); border-radius: 10px; padding: 20px; margin: 20px 0; border: 1px solid #64b5f6;">
<pre style="background: white; border-radius: 6px; padding: 15px;"><code class="language-java">Product product = new Product();
product.name = "Notebook";
product.price = 100;
product.quantity = 3;

product.totalPrice(); // 300
product.add(2);       // quantity = 5
product.sell(4);      // true, quantity = 1
product.sell(10);     // false, quantity = 1
product.getLabel();   // "Notebook (1 pcs)"</code></pre>
</div>

<h2 style="color: #34495e; margin-top: 28px;">✅ Чек-лист проверки</h2>
<div style="background: #f8f9fa; border-left: 4px solid #28a745; padding: 16px; margin: 16px 0;">
<ul style="margin-bottom: 0;">
	<li>☐ <code>totalPrice()</code> считает цену по текущему количеству</li>
	<li>☐ <code>add</code> игнорирует ноль и отрицательные значения</li>
	<li>☐ <code>sell</code> возвращает <code>true</code> при успешной продаже</li>
	<li>☐ <code>sell</code> возвращает <code>false</code>, если товара не хватает</li>
