<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🎓 Карточка студента</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться создавать простой класс с полями и методом, который собирает данные объекта в строку.</p>
</div>

<div style="background: #e3f2fd; border: 1px solid #90caf9; border-radius: 8px; padding: 12px; margin: 16px 0;">
<p style="margin: 0;">📖 <strong>Пререквизит:</strong> повторите поля класса, создание объекта через <code>new</code>, обращение к полям через точку и возвращение строки из метода.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">🎯 Сценарий</h3>
<p style="margin-bottom: 10px;">Вы делаете внутреннюю систему курса. Для каждого ученика нужна короткая карточка: имя, возраст и средний балл.</p>
<p style="margin: 0;"><strong>Ваша задача:</strong> описать класс студента и метод, который возвращает карточку в заданном формате.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Класс как модель данных</h4>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code class="language-java">class Student {
    String name;
    int age;
    double averageGrade;
}</code></pre>
<p style="margin-bottom: 0;">Поля хранят состояние конкретного студента. У разных объектов <code>Student</code> могут быть разные значения.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;"><strong>Создайте класс:</strong></p>
<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>class Student</code></div>
<h4 style="color: #e65100; margin-top: 16px;">📝 Поля</h4>
<ul style="margin: 10px 0;">
	<li><code>String name</code> — имя студента</li>
	<li><code>int age</code> — возраст</li>
	<li><code>double averageGrade</code> — средний балл</li>
</ul>
<h4 style="color: #e65100; margin-top: 16px;">⚡ Метод</h4>
<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>String getCard()</code></div>
<p style="margin-bottom: 0;">Метод должен вернуть строку формата: <code>Student: &lt;name&gt;, age: &lt;age&gt;, grade: &lt;averageGrade&gt;</code></p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Результат программы</h2>
<div style="background: linear-gradient(135deg, #e3f2fd 0%, #bbdefb 100%); border-radius: 10px; padding: 20px; margin: 20px 0; border: 1px solid #64b5f6;">
<pre style="background: white; border-radius: 6px; padding: 15px;"><code class="language-java">Student student = new Student();
student.name = "Alice";
student.age = 20;
student.averageGrade = 4.7;

student.getCard();</code></pre>
<pre style="background: white; border-radius: 6px; padding: 15px;"><code>Student: Alice, age: 20, grade: 4.7</code></pre>
</div>

<div style="background: #fff3e0; border: 1px solid #ff9800; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #e65100; margin-top: 0;">🤔 Момент для размышления</h4>
<p style="margin-bottom: 0;">Почему удобнее хранить имя, возраст и балл в одном объекте, а не в трёх отдельных массивах?</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📝 Структура</h2>

    String name;
    int age;
    double averageGrade;

    String getCard() {
        // верните строку по формату
    }
}</code></pre>

<h2 style="color: #34495e; margin-top: 28px;">✅ Чек-лист проверки</h2>
<div style="background: #f8f9fa; border-left: 4px solid #28a745; padding: 16px; margin: 16px 0;">
<ul style="margin-bottom: 0;">
	<li>☐ Класс называется ровно <code>Student</code></li>
	<li>☐ Поля имеют указанные имена и типы</li>
	<li>☐ Метод называется <code>getCard</code></li>
	<li>☐ Метод возвращает строку, а не печатает её</li>
	<li>☐ Формат строки совпадает с примером</li>
</ul>
</div>

<div style="background: #fff8e1; border-left: 4px solid #ffc107; padding: 16px; margin: 16px 0;">
