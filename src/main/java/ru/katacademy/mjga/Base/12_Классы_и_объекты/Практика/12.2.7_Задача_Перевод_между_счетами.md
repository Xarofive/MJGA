<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">💸 Перевод между счетами</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться передавать объекты в статический метод и менять состояние уже существующих объектов.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">🎯 Сценарий</h3>
<p style="margin-bottom: 10px;">В банковской системе нужно перевести деньги с одного счёта на другой. Метод получает две ссылки: откуда списать деньги и куда зачислить.</p>
<p style="margin: 0;"><strong>Ваша задача:</strong> выполнить перевод так, чтобы изменились именно переданные объекты, а не новые временные счета.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">В метод передаётся копия ссылки</h4>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code class="language-java">static void addBonus(BankAccount account) {
    account.balance = account.balance + 100;
}</code></pre>
<p style="margin-bottom: 0;">Метод получает ссылку на тот же объект. Если изменить поле через эту ссылку, изменение будет видно снаружи метода.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;"><strong>Создайте класс:</strong></p>
<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>class BankAccount</code></div>
<h4 style="color: #e65100; margin-top: 16px;">📝 Поля</h4>
<ul style="margin: 10px 0;">
	<li><code>String owner</code> — владелец счёта</li>
	<li><code>int balance</code> — текущий баланс</li>
</ul>
<h4 style="color: #e65100; margin-top: 16px;">⚡ Методы экземпляра</h4>
<ul style="margin: 10px 0;">
	<li><code>void deposit(int amount)</code> — увеличивает баланс на <code>amount</code>, если <code>amount &gt; 0</code></li>
	<li><code>boolean withdraw(int amount)</code> — снимает деньги и возвращает <code>true</code>, если <code>amount &gt; 0</code> и денег хватает</li>
	<li><code>String getInfo()</code> — возвращает <code>"&lt;owner&gt;: &lt;balance&gt;"</code></li>
</ul>
<h4 style="color: #e65100; margin-top: 16px;">🔍 Статический метод</h4>
<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>static boolean transfer(BankAccount from, BankAccount to, int amount)</code></div>
<p style="margin-bottom: 0;">Метод должен перевести деньги со счёта <code>from</code> на счёт <code>to</code> и вернуть <code>true</code>, если <code>amount &gt; 0</code>, оба счёта не равны <code>null</code> и на счёте <code>from</code> хватает денег. Иначе метод должен вернуть <code>false</code> и не менять балансы.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Результат программы</h2>
<div style="background: linear-gradient(135deg, #e3f2fd 0%, #bbdefb 100%); border-radius: 10px; padding: 20px; margin: 20px 0; border: 1px solid #64b5f6;">
<pre style="background: white; border-radius: 6px; padding: 15px;"><code class="language-java">BankAccount alice = new BankAccount();
alice.owner = "Alice";
alice.balance = 100;

BankAccount bob = new BankAccount();
bob.owner = "Bob";
bob.balance = 20;

BankAccount.transfer(alice, bob, 30);  // true
alice.getInfo();                       // "Alice: 70"
bob.getInfo();                         // "Bob: 50"

BankAccount.transfer(alice, bob, 1000); // false
alice.getInfo();                        // "Alice: 70"
bob.getInfo();                          // "Bob: 50"</code></pre>
</div>

<div style="background: #fff3e0; border: 1px solid #ff9800; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #e65100; margin-top: 0;">🤔 Момент для размышления</h4>
<p style="margin-bottom: 0;">Почему внутри <code>transfer</code> нельзя создавать новые счета вместо изменения объектов <code>from</code> и <code>to</code>?</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">✅ Чек-лист проверки</h2>
<div style="background: #f8f9fa; border-left: 4px solid #28a745; padding: 16px; margin: 16px 0;">
<ul style="margin-bottom: 0;">
	<li>☐ Успешный перевод уменьшает баланс <code>from</code></li>
	<li>☐ Успешный перевод увеличивает баланс <code>to</code></li>
	<li>☐ При нехватке денег метод возвращает <code>false</code></li>
	<li>☐ При некорректной сумме метод возвращает <code>false</code></li>
	<li>☐ Если один из счетов равен <code>null</code>, метод возвращает <code>false</code></li>
	<li>☐ При неуспешном переводе оба баланса остаются прежними</li>
</ul>
</div>

<div style="background: #fff8e1; border-left: 4px solid #ffc107; padding: 16px; margin: 16px 0;">
<h4 style="color: #f57c00; margin-top: 0;">⚠️ Технические ограничения</h4>
<ul style="margin-bottom: 0;">
	<li>Не используйте <code>System.out</code> внутри методов</li>
	<li>Не делайте поля <code>static</code></li>
	<li>Не создавайте новые объекты внутри <code>transfer</code></li>
	<li>Не меняйте балансы, если перевод выполнить нельзя</li>
	<li>Проверяйте <code>null</code> до обращения к полям и методам счетов</li>
</ul>
</div>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
