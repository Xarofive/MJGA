<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🔗 Счётчик и ссылки</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться видеть разницу между двумя объектами и двумя ссылками на один объект.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">🎯 Сценарий</h3>
<p style="margin-bottom: 10px;">Вы пишете простой счётчик событий. Иногда два имени переменных могут ссылаться на один и тот же счётчик, и тогда изменение через одну переменную видно через другую.</p>
<p style="margin: 0;"><strong>Ваша задача:</strong> создать класс счётчика и методы, которые помогут проверить работу ссылок.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">== сравнивает ссылки</h4>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code class="language-java">Counter first = new Counter();
Counter second = first;

first == second; // true</code></pre>
<p style="margin-bottom: 0;">Если две переменные указывают на один объект, изменение через одну переменную меняет тот же объект.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;"><strong>Создайте класс:</strong></p>
<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>class Counter</code></div>
<h4 style="color: #e65100; margin-top: 16px;">📝 Поле</h4>
<ul style="margin: 10px 0;">
	<li><code>int value</code> — текущее значение счётчика</li>
</ul>
<h4 style="color: #e65100; margin-top: 16px;">⚡ Методы экземпляра</h4>
<ul style="margin: 10px 0;">
	<li><code>void increment()</code> — увеличивает <code>value</code> на 1</li>
	<li><code>void add(int amount)</code> — увеличивает <code>value</code> на <code>amount</code>, если <code>amount &gt; 0</code></li>
	<li><code>String status()</code> — возвращает <code>"Counter: &lt;value&gt;"</code></li>
</ul>
<h4 style="color: #e65100; margin-top: 16px;">🔍 Статический метод</h4>
<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>static boolean sameObject(Counter first, Counter second)</code></div>
<p style="margin-bottom: 0;">Метод должен вернуть результат сравнения ссылок через <code>==</code>.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Результат программы</h2>
<div style="background: linear-gradient(135deg, #e3f2fd 0%, #bbdefb 100%); border-radius: 10px; padding: 20px; margin: 20px 0; border: 1px solid #64b5f6;">
<pre style="background: white; border-radius: 6px; padding: 15px;"><code class="language-java">Counter first = new Counter();
Counter second = first;
Counter third = new Counter();

first.increment();
second.add(4);

first.status();                    // "Counter: 5"
Counter.sameObject(first, second); // true
Counter.sameObject(first, third);  // false</code></pre>
</div>

<div style="background: #fff3e0; border: 1px solid #ff9800; border-radius: 8px; padding: 14px; margin: 16px 0;">

<p style="margin-bottom: 0;">Почему после <code>Counter second = first</code> изменение через <code>second</code> видно через <code>first</code>?</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">✅ Чек-лист проверки</h2>
<div style="background: #f8f9fa; border-left: 4px solid #28a745; padding: 16px; margin: 16px 0;">
<ul style="margin-bottom: 0;">
	<li>☐ Новый счётчик начинается со значения <code>0</code></li>
	<li>☐ <code>increment()</code> увеличивает значение на 1</li>
	<li>☐ <code>add</code> игнорирует ноль и отрицательные значения</li>
	<li>☐ <code>status()</code> возвращает строку по формату</li>
	<li>☐ <code>sameObject</code> сравнивает ссылки через <code>==</code></li>
	<li>☐ Две ссылки на один объект дают <code>true</code></li>

<div style="background: #fff8e1; border-left: 4px solid #ffc107; padding: 16px; margin: 16px 0;">
<h4 style="color: #f57c00; margin-top: 0;">⚠️ Технические ограничения</h4>
<ul style="margin-bottom: 0;">
	<li>Не используйте <code>System.out</code> внутри методов</li>
	<li>Не делайте поле <code>value</code> статическим</li>
	<li>Не создавайте новые объекты внутри <code>sameObject</code></li>
	<li>Не сравнивайте <code>value</code> в <code>sameObject</code>; нужен именно <code>first == second</code></li>
</ul>
</div>
