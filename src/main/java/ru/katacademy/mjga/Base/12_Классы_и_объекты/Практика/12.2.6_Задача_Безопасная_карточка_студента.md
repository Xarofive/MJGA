<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🛡️ Безопасная карточка студента</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться безопасно работать с объектной ссылкой, которая может быть равна <code>null</code>.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">🎯 Сценарий</h3>
<p style="margin-bottom: 10px;">Вы дорабатываете систему курса. Иногда студент ещё не выбран, и вместо объекта в метод приходит <code>null</code>.</p>
<p style="margin: 0;"><strong>Ваша задача:</strong> вернуть карточку студента, если объект есть, и понятную заглушку, если ссылки на объект нет.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">null означает отсутствие объекта</h4>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code class="language-java">Student student = null;

if (student == null) {
    return "No student";
}</code></pre>
<p style="margin-bottom: 0;">Перед обращением к полям или методам объекта проверьте, что ссылка не равна <code>null</code>.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;"><strong>Создайте класс:</strong></p>
<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>class Student</code></div>
<h4 style="color: #e65100; margin-top: 16px;">📝 Поля</h4>
<ul style="margin: 10px 0;">
	<li><code>String name</code> — имя студента</li>
	<li><code>int age</code> — возраст</li>
	<li><code>double averageGrade</code> — средний балл</li>
</ul>
<h4 style="color: #e65100; margin-top: 16px;">⚡ Метод экземпляра</h4>
<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>String getCard()</code></div>
<p>Метод должен вернуть строку формата: <code>Student: &lt;name&gt;, age: &lt;age&gt;, grade: &lt;averageGrade&gt;</code></p>
<h4 style="color: #e65100; margin-top: 16px;">🔍 Статический метод</h4>
<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>static String getCardOrDefault(Student student)</code></div>
<p style="margin-bottom: 0;">Метод должен вернуть <code>"No student"</code>, если <code>student == null</code>. Иначе метод должен вернуть результат вызова <code>student.getCard()</code>.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Результат программы</h2>
<div style="background: linear-gradient(135deg, #e3f2fd 0%, #bbdefb 100%); border-radius: 10px; padding: 20px; margin: 20px 0; border: 1px solid #64b5f6;">
<pre style="background: white; border-radius: 6px; padding: 15px;"><code class="language-java">Student student = new Student();
student.name = "Alice";
student.age = 20;
student.averageGrade = 4.7;

Student.getCardOrDefault(student); // "Student: Alice, age: 20, grade: 4.7"
Student.getCardOrDefault(null);    // "No student"</code></pre>
</div>

<div style="background: #fff3e0; border: 1px solid #ff9800; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #e65100; margin-top: 0;">🤔 Момент для размышления</h4>
<p style="margin-bottom: 0;">Почему проверку <code>student == null</code> нужно сделать до вызова <code>student.getCard()</code>?</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📝 Структура</h2>
<pre style="background: #263238; color: #aed581; padding: 20px; border-radius: 8px; overflow-x: auto;"><code class="language-java">class Student {
    String name;
    int age;
    double averageGrade;

    String getCard() {
        // ...
    }

    static String getCardOrDefault(Student student) {
        // ...
    }
}</code></pre>

<h2 style="color: #34495e; margin-top: 28px;">✅ Чек-лист проверки</h2>
<div style="background: #f8f9fa; border-left: 4px solid #28a745; padding: 16px; margin: 16px 0;">
<ul style="margin-bottom: 0;">
	<li>☐ Класс называется ровно <code>Student</code></li>
	<li>☐ Поля имеют указанные имена и типы</li>
	<li>☐ <code>getCard()</code> возвращает строку в точном формате</li>
	<li>☐ <code>getCardOrDefault(null)</code> возвращает <code>"No student"</code></li>
	<li>☐ Для существующего объекта <code>getCardOrDefault</code> возвращает его карточку</li>
	<li>☐ Метод не падает с <code>NullPointerException</code></li>
</ul>
</div>

<div style="background: #fff8e1; border-left: 4px solid #ffc107; padding: 16px; margin: 16px 0;">
<h4 style="color: #f57c00; margin-top: 0;">⚠️ Технические ограничения</h4>
<ul style="margin-bottom: 0;">
	<li>Не используйте <code>System.out</code> внутри методов</li>
	<li>Не добавляйте <code>package</code> в решение</li>
	<li>Не делайте поля <code>static</code></li>
	<li>Не вызывайте <code>student.getCard()</code> до проверки на <code>null</code></li>
</ul>
</div>
