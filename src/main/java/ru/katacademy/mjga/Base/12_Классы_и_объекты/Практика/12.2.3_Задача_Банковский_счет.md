<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🏦 Банковский счёт</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться писать методы объекта, которые проверяют условия перед изменением состояния.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">🎯 Сценарий</h3>
<p style="margin-bottom: 10px;">В банковской системе нельзя просто менять баланс как угодно. Пополнение должно увеличивать баланс, а снятие должно проходить только если денег хватает.</p>
<p style="margin: 0;"><strong>Ваша задача:</strong> описать банковский счёт и безопасные методы работы с балансом.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Метод как правило изменения</h4>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code class="language-java">boolean withdraw(int amount) {
    if (amount &gt; balance) {
        return false;
    }

    balance = balance - amount;
    return true;
}</code></pre>
<p style="margin-bottom: 0;">Метод не только меняет поле, но и защищает объект от некорректного состояния.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;"><strong>Создайте класс:</strong></p>
<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>class BankAccount</code></div>
<h4 style="color: #e65100; margin-top: 16px;">📝 Поля</h4>
<ul style="margin: 10px 0;">
	<li><code>String owner</code> — владелец счёта</li>
	<li><code>int balance</code> — текущий баланс</li>
</ul>
<h4 style="color: #e65100; margin-top: 16px;">⚡ Методы</h4>
<ul style="margin: 10px 0;">
	<li><code>void deposit(int amount)</code> — увеличивает баланс на <code>amount</code>, если <code>amount &gt; 0</code></li>
	<li><code>boolean withdraw(int amount)</code> — снимает деньги и возвращает <code>true</code>, если <code>amount &gt; 0</code> и денег хватает</li>
	<li><code>int getBalance()</code> — возвращает текущий баланс</li>
	<li><code>String getInfo()</code> — возвращает <code>"&lt;owner&gt;: &lt;balance&gt;"</code></li>
</ul>
<p style="margin-bottom: 0;">Если сумма некорректная или денег не хватает, баланс не должен измениться.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Результат программы</h2>

<pre style="background: white; border-radius: 6px; padding: 15px;"><code class="language-java">BankAccount account = new BankAccount();
account.owner = "Alice";
account.balance = 100;

account.deposit(50);      // balance = 150
account.withdraw(70);     // true, balance = 80
account.withdraw(1000);   // false, balance = 80
account.getInfo();        // "Alice: 80"</code></pre>
</div>

<h2 style="color: #34495e; margin-top: 28px;">✅ Чек-лист проверки</h2>
<div style="background: #f8f9fa; border-left: 4px solid #28a745; padding: 16px; margin: 16px 0;">
<ul style="margin-bottom: 0;">
	<li>☐ Пополнение положительной суммой увеличивает баланс</li>
	<li>☐ Пополнение нулём или отрицательной суммой ничего не меняет</li>
	<li>☐ Снятие при достаточном балансе возвращает <code>true</code></li>
	<li>☐ Снятие при нехватке денег возвращает <code>false</code></li>
	<li>☐ Баланс никогда не становится отрицательным из-за <code>withdraw</code></li>
	<li>☐ <code>getInfo()</code> возвращает строку в точном формате</li>
</ul>
</div>

<div style="background: #fff8e1; border-left: 4px solid #ffc107; padding: 16px; margin: 16px 0;">
<h4 style="color: #f57c00; margin-top: 0;">⚠️ Технические ограничения</h4>
<ul style="margin-bottom: 0;">
	<li>Не используйте <code>System.out</code> в методах</li>
	<li>Не делайте поля <code>static</code></li>
	<li>Не создавайте новый счёт внутри методов</li>
	<li>Не меняйте баланс при некорректной операции</li>
</ul>
</div>
