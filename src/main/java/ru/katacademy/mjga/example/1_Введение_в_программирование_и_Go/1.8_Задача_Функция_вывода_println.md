# Задача: Функция вывода println

## Описание:
Теперь тебе нужно использовать функцию println для того, чтобы вывести текст "Hello Kata Academy".

Внимание! Функция println вызывается без пакета fmt.

## Подсказка:
Функция — это специальная команда, которая выполняет определенные действия.
println — это функция, которая выводит текст на экран. Всё, что ты передашь в скобках после println, будет напечатано.
Чтобы вывести текст, нужно передать его в скобках и обязательно заключить в двойные кавычки ("").

## Пример:
Твой ответ должен выглядеть так:
``
```
println("требуемый текст задачи")
```

## Реши задачу в окошке ниже:
Просто добавь использование функции println для вывода текста "Hello Kata Academy". Не забудь заключить текст в двойные кавычки.
