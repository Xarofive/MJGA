# Базовая структура программы на Go

## Что такое программа на Go?

Программа на языке Go — это набор инструкций, который сообщает компьютеру, какие действия необходимо выполнить. Программы могут быть разной сложности, но любая программа на Go состоит из нескольких обязательных элементов:

- **Пакет (package)**
- **Импорт пакетов**
- **Функции**
- **Вывод информации на экран**

---

## Основные элементы программы

### 1. Package main

Каждая исполняемая программа на Go должна начинаться с объявления пакета `main`. Это говорит компилятору, что программа предназначена для запуска.

```go
package main
```

### 2. Импорт пакетов (import)

Для расширения функциональности программы используют пакеты. Например, пакет `fmt` предоставляет функции для работы с вводом-выводом текста.

```go
import "fmt"
```

### 3. Функция main()

`main()` — это точка входа в программу, с неё начинается выполнение кода.

```go
func main() {
    // код программы
}
```

### 4. Вывод данных

Для вывода текста используются две основные функции:

- `fmt.Println()` — стандартная функция из пакета `fmt`, которая выводит текст с переносом строки:

```go
fmt.Println("Hello Kata Academy")
```

- `println()` — упрощённая встроенная функция, не требующая импорта пакетов:

```go
println("Hello Kata Academy")
```

---

## Простая программа на Go

Рассмотрим готовый пример простой программы на Go:

```go
package main

import "fmt"

func main() {
    fmt.Println("Hello Kata Academy!")
    fmt.Println("Hello My First Program with fmt!")
}
```

### Что происходит в программе:

- Определяем `package main`, чтобы программа была исполняемой.
- Подключаем пакет `fmt` для работы с выводом информации.
- Создаём функцию `main()`, которая является точкой входа.
- Выводим две строки текста с помощью функции `fmt.Println`.
