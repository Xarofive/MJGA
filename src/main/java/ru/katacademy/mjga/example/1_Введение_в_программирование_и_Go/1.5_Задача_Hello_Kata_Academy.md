# Задача: "Hello Kata Academy"

## Описание:
Тебе нужно просто написать текст в двойных кавычках: "Hello Kata Academy".

## Подсказка:
В Golang, когда ты заключаешь текст в двойные кавычки (""), компьютер понимает, что это строка — набор символов, таких как буквы и знаки, которые нужно вывести на экран.

## Пример:
Твой ответ должен выглядеть так:

```
"Hello Kata Academy"
```

## Реши задачу в окошке ниже:
Просто напиши "Hello Kata Academy". Не забудь заключить текст в двойные кавычки.
