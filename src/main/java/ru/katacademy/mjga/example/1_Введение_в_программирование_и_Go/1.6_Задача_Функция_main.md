# Задача: Функция main

## Описание:
Теперь тебе нужно написать функцию main(), внутри которой разместить два вызова println для вывода двух строк:

- "Hello Kata Academy!"
- "Hello My First Main function!"

Каждое предложение должно выводиться на новой строке.

Внимание! Ничего не нужно импортировать, так как все необходимые пакеты уже подключены.

## Ожидаемый выво``д:
```
Hello Kata Academy!
Hello My First Main function!
```

## Критерии приемки:
- Код написан в функции main().
- Используйте функцию println для вывода каждой строки.
- Убедитесь, что каждое предложение выводится на новой строке и программа не выводит ничего лишнего.
