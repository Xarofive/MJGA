# Задача: Полная программа

## Описание:
Теперь тебе нужно написать полную программу, используя ключевые элементы Go:

- пакет fmt для вывода текста,
- package main для обозначения основного пакета программы,
- import, чтобы подключить пакет fmt,
- и функцию main, внутри которой будет весь код.

## Что такое fmt и зачем он нужен:
fmt — это специальный ``пакет в Go, который предоставляет полезные функции для работы с вводом и выводом данных.
Одна из этих функций — Println, которая выводит текст на экран. Когда ты используешь fmt.Println, Go понимает, что ты обращаешься к функции Println из пакета fmt.
В программе ты сначала подключаешь пакет fmt с помощью ключевого слова import, а потом используешь функцию.

## Что нужно сделать:
Напиши программу, которая выводит следующие две строки:

- "Hello Kata Academy!"
- "Hello My First Program with fmt!"

## Ожидаемый вывод:
```
Hello Kata Academy!
Hello My First Program with fmt!
```

## Критерии приемки:
- Твой код должен быть в функции main().
- Ты должен подключить пакет fmt с помощью import.
- Используй fmt.Println для вывода каждой строки.
- Убедись, что каждая строка выводится отдельно и ничего лишнего не выводится.
