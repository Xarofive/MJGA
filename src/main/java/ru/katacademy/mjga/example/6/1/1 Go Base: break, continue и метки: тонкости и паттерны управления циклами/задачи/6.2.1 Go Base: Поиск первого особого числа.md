<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🔄 Поиск первого особого числа</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>

<p style="margin: 0;">Отработать перебор с досрочным выходом: найти первое число диапазона, которое удовлетворяет составному условию делимости, используя оператор <code>%</code>, логическое И <code>&amp;&amp;</code> и <code>break</code>.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">💼 Контекст</h3>

<p style="margin-bottom: 10px;">Ранний выход из цикла экономит время: найдено значение &mdash; завершаем поиск и не тратим ресурсы на остальные элементы. Такой паттерн нужен при анализе транзакций, построении расписаний, фильтрации потоков данных.</p>

<p style="margin: 0;"><strong>Ваша задача:</strong> перебрать числа от 1 до 100, остановиться на первом, которое одновременно делится на 7 и даёт остаток 1 при делении на 3.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📚 Что пригодится</h2>

<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Остаток от деления</h4>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code>n % d == 0  // n делится на d
n % d == 1  // остаток 1, значит n = d * k + 1</code></pre>
</div>

<div style="background: #e3f2fd; border: 1px solid #90caf9; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #1565c0; margin-top: 0;">Паттерн &laquo;нашёл &mdash; выходи&raquo;</h4>

<ul style="margin: 10px 0;">
	<li>перебор кандидатов в цикле <code>for</code></li>
	<li>проверка составного условия внутри <code>if</code></li>
	<li><code>break</code> внутри <code>if</code>, чтобы остановить цикл сразу после успеха</li>
	<li>никаких лишних итераций после найденного результата</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #e65100; margin-top: 0;">Структура программы</h4>

<ul style="margin: 10px 0;">
	<li>файл начинается с <code>package main</code></li>
	<li>импортируется только пакет <code>&quot;fmt&quot;</code> без псевдонима (используйте <code>import &quot;fmt&quot;</code>, форма <code>import f &quot;fmt&quot;</code> не допускается; ни одного лишнего импорта)</li>
	<li>единственная функция <code>func main()</code>; открывающая <code>{</code> стоит на той же строке</li>
	<li>дополнительные функции, глобальные переменные и сокращённые объявления (<code>:=</code>) не нужны</li>
</ul>

<h4 style="color: #e65100; margin-top: 16px;">Алгоритм</h4>

<ul style="margin: 10px 0;">
	<li>переберите числа от 1 до 100 включительно (любой удобной формой цикла <code>for</code>)</li>
	<li>внутри цикла используйте <code>if</code> с условием <code>n % 7 == 0 &amp;&amp; n % 3 == 1</code> (порядок проверок не важен)</li>
	<li>как только условие выполняется &mdash; напечатайте найденное число и вызовите <code>break</code> внутри блока <code>if</code></li>
	<li>не применяйте <code>continue</code> и <code>switch</code>, а также не проверяйте остальные элементы после совпадения</li>
</ul>

<h4 style="color: #e65100; margin-top: 16px;">Вывод</h4>

<ul style="margin: 10px 0;">
	<li>используйте <code>fmt.Println</code> или <code>fmt.Printf</code> для вывода найденного числа</li>
	<li>передайте в него найденное число (дополнительный текст не требуется)</li>
	<li>не используйте встроенные <code>print</code>/<code>println</code> без пакета <code>fmt</code></li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">⚠️ Частые ошибки</h2>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<ul style="margin: 0;">
	<li><strong>Отсутствует <code>break</code> внутри <code>if</code>:</strong> программа выводит несколько чисел.</li>
	<li><strong>Используется <code>continue</code> вместо <code>break</code>:</strong> цикл не останавливается на совпадении.</li>
	<li><strong>Печать строк или нескольких значений:</strong> требуется вывести только найденное число.</li>
	<li><strong>Дополнительные функции или импорты:</strong> оставьте только минимально необходимый код.</li>
	<li><strong>Неверный диапазон:</strong> начинайте перебор с 1 и ограничивайте верхней границей 100.</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">✅ Чек-лист</h2>

<div style="background: #e8f5e9; border: 1px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<ul style="margin: 0;">
	<li>Есть <code>package main</code> и импорт только <code>&quot;fmt&quot;</code>.</li>
	<li>Только одна функция <code>main()</code> с <code>{</code> на той же строке.</li>
	<li>Используется цикл <code>for</code> с перебором 1..100.</li>
	<li>Условие проверяет и делимость на 7, и остаток 1 при делении на 3.</li>
	<li><code>break</code> расположен внутри блока <code>if</code>.</li>
	<li>В коде нет <code>switch</code>, <code>continue</code>, дополнительных функций и лишних печатей.</li>
	<li>Программа выводит корректный результат.</li>
</ul>
</div>
