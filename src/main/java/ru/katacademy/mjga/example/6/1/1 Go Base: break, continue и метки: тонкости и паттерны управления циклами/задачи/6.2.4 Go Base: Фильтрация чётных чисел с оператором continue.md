<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🎯 Фильтрация чётных чисел с оператором continue</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>

<p style="margin: 0;">Научиться использовать оператор <code>continue</code> для управления потоком выполнения цикла, пропуская ненужные итерации вместо вложенности условных операторов.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">💼 Контекст и мотивация</h3>

<p style="margin-bottom: 10px;">В реальной разработке часто требуется обработать только элементы, удовлетворяющие определённым условиям. Оператор <code>continue</code> позволяет элегантно пропускать ненужные итерации, делая код более читаемым и эффективным. Это критично при работе с большими наборами данных, валидации пользовательского ввода и фильтрации результатов.</p>

<p style="margin: 0;"><strong>Задача:</strong> перебрать числа от 1 до 20, пропуская чётные с помощью <code>continue</code>, и вывести только нечётные.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>

<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Классический цикл for в Go</h4>

<p>Полная форма цикла <code>for</code> содержит три компонента:</p>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code class="language-go">for инициализация; условие; шаг {
    // тело цикла
}

// Пример:
for i := 0; i &lt; 10; i++ {
    fmt.Println(i)
}</code></pre>

<ul style="margin: 10px 0;">
	<li><strong>Инициализация:</strong> выполняется один раз перед началом цикла</li>
	<li><strong>Условие:</strong> проверяется перед каждой итерацией</li>
	<li><strong>Шаг:</strong> выполняется после каждой итерации тела цикла</li>
</ul>
</div>

<div style="background: #e3f2fd; border: 1px solid #90caf9; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #1565c0; margin-top: 0;">🎯 Оператор continue</h4>

<p>Оператор <code>continue</code> прерывает текущую итерацию цикла и переходит к следующей:</p>

<ul style="margin: 10px 0;">
	<li>Пропускает оставшийся код в теле цикла</li>
	<li>Выполняет шаг цикла (например, <code>i++</code>)</li>
	<li>Проверяет условие и продолжает, если оно истинно</li>
	<li>Позволяет избежать глубокой вложенности условий</li>
</ul>
</div>

<div style="background: #fff3e0; border: 1px solid #ff9800; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #f57c00; margin-top: 0;">🔢 Проверка чётности</h4>

<p>Для определения чётности числа используется оператор остатка от деления <code>%</code>:</p>

<ul style="margin: 10px 0;">
	<li><code>число % 2 == 0</code> &mdash; число чётное</li>
	<li><code>число % 2 != 0</code> или <code>число % 2 == 1</code> &mdash; число нечётное</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;"><strong>Создайте программу, которая перебирает числа от 1 до 20 и выводит только нечётные:</strong></p>

<h4 style="color: #e65100; margin-top: 16px;">📝 Структура программы</h4>

<ul style="margin: 10px 0;">
	<li>Начинается с <code>package main</code></li>
	<li>Импортирует только пакет <code>&quot;fmt&quot;</code> без псевдонима (используйте <code>import &quot;fmt&quot;</code>, форма <code>import f &quot;fmt&quot;</code> не допускается)</li>
	<li>Содержит единственную функцию <code>main()</code></li>
</ul>

<h4 style="color: #e65100; margin-top: 16px;">🔧 Требования к циклу</h4>

<ul style="margin: 10px 0;">
	<li>Используйте классический цикл <code>for</code> с тремя компонентами (инициализация, условие, шаг)</li>
	<li>Инициализируйте счётчик числовым литералом</li>
	<li>Условие должно обеспечивать перебор чисел до 20 включительно</li>
	<li>Шаг должен увеличивать счётчик на 1 (<code>i++</code> или <code>i += 1</code>)</li>
</ul>

<h4 style="color: #e65100; margin-top: 16px;">⚡ Логика фильтрации</h4>

<ul style="margin: 10px 0;">
	<li>Внутри цикла проверяйте, является ли текущее число чётным</li>
	<li>Если число чётное &mdash; используйте <code>continue</code> для пропуска итерации</li>
	<li>Если число нечётное &mdash; выведите его с помощью <code>fmt.Println()</code> или <code>fmt.Printf()</code></li>
	<li>Каждое число выводится на отдельной строке</li>
	<li>Используйте <code>i % 2</code> непосредственно внутри условия <code>if</code>, без вынесения в отдельную переменную (валидатор проверяет именно эту форму)</li>
</ul>

<h4 style="color: #e65100; margin-top: 16px;">🚫 Ограничения</h4>

<ul style="margin: 10px 0;">
	<li>Запрещено использовать оператор <code>switch</code></li>
	<li>Не создавайте дополнительные функции</li>
	<li>Не используйте сокращённые формы цикла <code>for</code></li>
</ul>

<h4 style="color: #c62828; margin-top: 0;">⚠️ Частые ошибки и антипаттерны</h4>

<p><strong>Ошибка 1: Неправильные границы цикла</strong><br />
❌ <code>for i := 0; i &lt; 20; i++</code> &mdash; начнёт с 0, выведет чётное<br />
❌ <code>for i := 1; i &lt; 20; i++</code> &mdash; не включит 20<br />
✅ Убедитесь, что цикл начинается с 1 и доходит до 20 включительно</p>

<p><strong>Ошибка 2: Неверная проверка чётности</strong><br />
❌ <code>if i % 2 == 1 { continue }</code> &mdash; пропустит нечётные<br />
❌ <code>if i % 2 &gt; 0 { continue }</code> &mdash; пропустит нечётные<br />
✅ Пропускайте итерацию, когда остаток от деления на 2 равен 0</p>

<p><strong>Ошибка 3: Отсутствие continue</strong><br />
❌ <code>if i % 2 != 0 { fmt.Println(i) }</code> &mdash; работает, но не использует continue<br />
✅ Обязательно используйте <code>continue</code> для пропуска чётных</p>

<p><strong>Ошибка 4: Бесконечный цикл</strong><br />
❌ Забыли указать шаг цикла<br />
❌ Неправильное условие продолжения<br />
✅ Проверьте все три компонента цикла</p>

<p><strong>Ошибка 5: Использование цикла без всех компонентов</strong><br />
❌ <code>for i &lt; 20 { ... }</code> &mdash; упрощённая форма<br />
✅ Используйте полную форму: <code>for init; condition; post { ... }</code></p>

<p style="margin-bottom: 0;"><strong>Ошибка 6: Вывод лишней информации</strong><br />
❌ <code>fmt.Println(&quot;Число:&quot;, i)</code> &mdash; добавляет текст<br />
✅ Выводите только само число</p>

<div style="background: #fff8e1; border-left: 4px solid #ffc107; padding: 16px; margin: 16px 0;">
<h4 style="color: #f57c00; margin-top: 0;">💡 Подсказка через паттерн</h4>

<p>Представьте турникет на входе в метро: он проверяет каждого пассажира. Если у человека нет билета (чётное число), турникет не открывается, и человек не проходит (continue). Если билет есть (нечётное число), турникет открывается, и пассажир проходит на платформу (вывод числа). Цикл &mdash; это очередь людей, а continue &mdash; это механизм, который экономит время, не обрабатывая тех, кто не может пройти.</p>
