<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🔄 Сумма с ранним выходом</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>

<p style="margin: 0;">Научиться использовать оператор <code>break</code> для раннего выхода из цикла по условию и накапливать значения в переменной-аккумуляторе.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">💼 Контекст и мотивация</h3>

<p style="margin-bottom: 10px;">В реальных системах часто нужно накапливать данные до достижения определённого порога: расчёт бюджета до лимита, подсчёт баллов до победы, заполнение буфера до максимального размера. Умение правильно организовать цикл с ранним выходом &mdash; базовый навык для работы с потоками данных, финансовыми расчётами и игровой логикой.</p>

<p style="margin: 0;"><strong>Задача:</strong> суммировать натуральные числа до превышения заданного порога и зафиксировать последнее добавленное число.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>

<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Оператор break в циклах</h4>

<p>Оператор <code>break</code> немедленно прерывает выполнение текущего цикла:</p>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code class="language-go">for i := 1; ; i++ {  // Бесконечный цикл со счётчиком
    if условие {
        break  // Выход из цикла
    }
    // код после break не выполнится в этой итерации
}
// код здесь выполнится после выхода из цикла</code></pre>

<p style="margin-bottom: 0;">После <code>break</code> управление передаётся первой инструкции после цикла (не из функции!).</p>
</div>

<div style="background: #e3f2fd; border: 1px solid #90caf9; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #1565c0; margin-top: 0;">🎯 Паттерн накопления с проверкой</h4>

<p>Типичная структура для суммирования с условием выхода:</p>

<ul style="margin: 10px 0;">
	<li><strong>Инициализация:</strong> создать переменные для суммы и последнего значения</li>
	<li><strong>Цикл:</strong> добавлять очередное число к сумме</li>
	<li><strong>Проверка:</strong> если превысили порог &mdash; выйти через <code>break</code></li>
	<li><strong>Фиксация:</strong> запомнить последнее добавленное число</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;"><strong>Создайте программу, которая суммирует натуральные числа до превышения порога 100:</strong></p>

<h4 style="color: #e65100; margin-top: 16px;">📝 Структура программы</h4>

<ul style="margin: 10px 0;">
	<li>Начинается с <code>package main</code></li>
	<li>Импортирует только пакет <code>&quot;fmt&quot;</code> без псевдонима (используйте <code>import &quot;fmt&quot;</code>, форма <code>import f &quot;fmt&quot;</code> не допускается)</li>
	<li>Содержит только функцию <code>main()</code></li>
</ul>

<h4 style="color: #e65100; margin-top: 16px;">🔧 Алгоритмические требования</h4>

<ul style="margin: 10px 0;">
	<li>Используйте цикл <code>for</code> для перебора натуральных чисел (1, 2, 3, ...)</li>
	<li>Накапливайте сумму в отдельной переменной</li>
	<li>Сохраняйте последнее добавленное число в переменной</li>
	<li>Когда сумма превысит 100 &mdash; выйдите из цикла через <code>break</code></li>
	<li>После цикла выведите два числа: последнее добавленное и итоговую сумму</li>
</ul>

<h4 style="color: #e65100; margin-top: 16px;">⚡ Формат вывода</h4>

<ul style="margin: 10px 0;">
	<li>Используйте <code>fmt.Println</code> или <code>fmt.Printf</code> для вывода</li>
	<li>Выведите ровно два числа: последнее добавленное и итоговую сумму</li>
	<li>Можно вывести в одной строке через пробел или в двух строках</li>
	<li>Никакого дополнительного текста не должно быть</li>
</ul>
</div>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #c62828; margin-top: 0;">⚠️ Частые ошибки и антипаттерны</h4>

<p><strong>Ошибка 1: Проверка до добавления</strong><br />
❌ Проверять сумму ДО добавления числа (получите 13 вместо 14)<br />
✅ Сначала добавьте число, потом проверяйте условие</p>

<p><strong>Ошибка 2: Неправильное условие</strong><br />
❌ <code>if sum == 100</code> &mdash; может никогда не выполниться<br />
✅ <code>if sum &gt; 100</code> или <code>if sum &gt;= 100</code></p>

<p><strong>Ошибка 3: Забыть сохранить последнее число</strong><br />
❌ Вывести только сумму<br />
✅ Сохранить последнее добавленное число в отдельной переменной</p>

<p><strong>Ошибка 4: Лишний текст в выводе</strong><br />
❌ <code>fmt.Println(&quot;Последнее:&quot;, 14, &quot;Сумма:&quot;, 105)</code><br />
✅ <code>fmt.Println(14, 105)</code> &mdash; только числа</p>

<p style="margin-bottom: 0;"><strong>Ошибка 5: Бесконечный цикл</strong><br />
❌ Забыть увеличивать счётчик или неправильное условие<br />
✅ Убедитесь, что цикл гарантированно достигнет условия выхода</p>
</div>
