<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🎯 Фильтрация и подсчёт: управление потоком</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>

<p style="margin: 0;">Освоить комбинированное управление циклом через <code>continue</code> и <code>break</code>, реализуя фильтрацию с подсчётом и ограничением результатов.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">💼 Контекст</h3>

<p style="margin-bottom: 10px;">В обработке потоков данных часто нужно: пропускать неподходящие элементы, подсчитывать валидные и останавливаться при достижении лимита. Это базовый паттерн для пагинации, выборки, анализа логов.</p>

<p style="margin: 0;"><strong>Задача:</strong> найти и подсчитать числа с особыми свойствами, остановившись при достижении целевого количества.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>

<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Управление циклом</h4>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code class="language-go">continue  // пропускает остаток итерации, переходит к следующей
break     // полностью прерывает цикл
for i := 1; i &lt;= 100; i++ {
if условие_пропуска {
continue  // к следующему i
}
// обработка
if условие_выхода {
break     // выход из цикла
}
}</code></pre>
</div>

<div style="background: #e3f2fd; border: 1px solid #90caf9; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #1565c0; margin-top: 0;">🎯 Паттерн фильтр-счётчик-лимит</h4>

<ul style="margin: 10px 0;">
	<li><strong>Фильтрация:</strong> пропуск элементов, не соответствующих критериям</li>
	<li><strong>Накопление:</strong> подсчёт или сбор подходящих элементов</li>
	<li><strong>Ограничение:</strong> остановка при достижении целевого количества</li>
	<li><strong>Эффективность:</strong> не проверяем лишние элементы после достижения цели</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;"><strong>Создайте программу для поиска и подсчёта особых чисел:</strong></p>

<h4 style="color: #e65100; margin-top: 16px;">📝 Структура программы</h4>

<ul style="margin: 10px 0;">
	<li>Начинается с <code>package main</code></li>
	<li>Импортирует только пакет <code>&quot;fmt&quot;</code> без псевдонима (используйте <code>import &quot;fmt&quot;</code>, форма <code>import f &quot;fmt&quot;</code> не допускается)</li>
	<li>Содержит единственную функцию <code>main()</code> с <code>{</code> на той же строке</li>
	<li>Не содержит дополнительных функций</li>
</ul>

<h4 style="color: #e65100; margin-top: 16px;">🎯 Алгоритм</h4>

<p><strong>Задача:</strong> Найти первые 5 чисел в диапазоне от 1 до 100, которые:</p>

<ul style="margin: 10px 0;">
	<li>Делятся на 3 без остатка <strong>И</strong></li>
	<li>НЕ делятся на 2 (то есть нечётные)</li>
</ul>

<p><strong>Требования к реализации:</strong></p>

<ul style="margin: 10px 0;">
	<li>Используйте цикл <code>for</code> для перебора чисел от 1 до 100</li>
	<li>Используйте счётчик для подсчёта найденных чисел (инициализируйте до цикла)</li>
	<li>Используйте минимум 2 условных оператора <code>if</code></li>
	<li>Используйте <code>continue</code> для пропуска чисел, не удовлетворяющих условиям</li>
	<li>Используйте <code>break</code> для выхода после нахождения 5 подходящих чисел</li>
	<li>Используйте оператор <code>%</code> для проверки делимости</li>
	<li>Используйте логический оператор (<code>&amp;&amp;</code>, <code>||</code> или <code>!</code>)</li>
	<li>Не используйте <code>switch</code></li>
</ul>

<h4 style="color: #e65100; margin-top: 16px;">⚡ Вывод результата</h4>

<ul style="margin: 10px 0;">
	<li>Выведите количество найденных чисел с помощью <code>fmt.Println()</code></li>
	<li>Должен быть ровно один вывод</li>
	<li>Используйте <code>fmt.Println</code> или <code>fmt.Printf</code></li>
</ul>
</div>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #c62828; margin-top: 0;">⚠️ Частые ошибки</h4>

<p><strong>Ошибка 1: Неправильное размещение continue</strong><br />
<code>continue</code> должен быть в теле цикла, обычно внутри <code>if</code> для пропуска</p>

<p><strong>Ошибка 2: break вне условия счётчика</strong><br />
<code>break</code> должен срабатывать только когда счётчик достиг 5</p>

<p><strong>Ошибка 3: Забыт инкремент счётчика</strong><br />
Не забудьте увеличивать счётчик (<code>++</code> или <code>+= 1</code>) для каждого найденного числа</p>

<p><strong>Ошибка 4: Вывод чисел вместо количества</strong><br />
Выводите финальное значение счётчика, а не сами числа</p>

<p style="margin-bottom: 0;"><strong>Ошибка 5: Один if вместо двух</strong><br />
Требуется минимум 2 оператора <code>if</code> &mdash; для фильтрации и для проверки лимита</p>
</div>

<h4 style="color: #2e7d32; margin-top: 0;">✅ Проверьте себя</h4>

<ul style="margin-bottom: 0;">
	<li>Программа начинается с <code>package main</code>?</li>
	<li>Импортирован только пакет <code>&quot;fmt&quot;</code>?</li>
	<li>Есть единственная функция <code>main()</code>?</li>
	<li>Объявлен счётчик до начала цикла?</li>
	<li>Используется цикл <code>for</code> для перебора 1-100?</li>
	<li>Есть минимум 2 условных оператора <code>if</code>?</li>
	<li>Используется <code>continue</code> для пропуска неподходящих чисел?</li>
	<li>Используется <code>break</code> при достижении счётчиком 5?</li>
	<li>Счётчик увеличивается для каждого подходящего числа?</li>
	<li>Программа выводит корректное количество найденных чисел?</li>
</ul>
</div>

<div style="background: #f3e5f5; border: 1px solid #ce93d8; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #6a1b9a; margin-top: 0;">💡 Подсказка</h4>

<p style="margin: 0;">Организуйте логику так: сначала проверьте, НЕ подходит ли число (и используйте <code>continue</code>), затем обработайте подходящее (увеличьте счётчик), и наконец проверьте, не пора ли остановиться (<code>break</code>). Помните: числа вида 3&times;(2k+1) &mdash; это нечётные кратные тройки.</p>
