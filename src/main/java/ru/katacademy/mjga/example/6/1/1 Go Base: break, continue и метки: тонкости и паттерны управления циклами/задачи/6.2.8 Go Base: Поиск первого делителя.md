<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🔍 Поиск первого делителя</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>

<p style="margin: 0;">Научиться использовать паттерн break-guard для поиска первого подходящего элемента и эффективного выхода из цикла при нахождении результата.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">💼 Контекст и мотивация</h3>

<p style="margin-bottom: 10px;">Поиск делителей &mdash; фундаментальная операция в криптографии, оптимизации алгоритмов и теории чисел. Умение находить первый делитель с ранним выходом критично для проверки простоты чисел, факторизации в системах шифрования RSA и оптимизации вычислений в математических библиотеках.</p>

<p style="margin: 0;"><strong>Задача:</strong> найти наименьший делитель числа 91 (кроме 1) и вычислить частное от деления.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>

<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Делимость и поиск делителей</h4>

<p>Число a является делителем числа n, если остаток от деления n на a равен нулю:</p>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code class="language-go">if n % divisor == 0 {
    // divisor является делителем n
    quotient := n / divisor
    // n = divisor * quotient
}</code></pre>

<p style="margin-bottom: 0;">Наименьший делитель (кроме 1) всегда находится в диапазоне от 2 до n.</p>
</div>

<div style="background: #e3f2fd; border: 1px solid #90caf9; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #1565c0; margin-top: 0;">🎯 Паттерн раннего выхода</h4>

<p>При поиске первого подходящего элемента используйте <code>break</code> сразу после нахождения:</p>

<ul style="margin: 10px 0;">
	<li><strong>Эффективность:</strong> не проверяем лишние элементы после нахождения результата</li>
	<li><strong>Детерминизм:</strong> всегда находим именно первый (наименьший) делитель</li>
	<li><strong>Ясность кода:</strong> явный выход лучше флагов и дополнительных условий</li>
</ul>
</div>

<h4 style="color: #c62828; margin-top: 0;">⚠️ Частые ошибки и антипаттерны</h4>

<p><strong>Ошибка 1: Начать с 1 вместо 2</strong><br />
❌ Начинать цикл с <code>divisor := 1</code> (1 &mdash; тривиальный делитель)<br />
✅ Начинайте с <code>divisor := 2</code></p>

<p><strong>Ошибка 2: Забыть break</strong><br />
❌ Продолжать поиск после нахождения первого делителя<br />
✅ Используйте <code>break</code> сразу после нахождения</p>

<p><strong>Ошибка 3: Вывод с текстом</strong><br />
❌ <code>fmt.Println(&quot;Делитель:&quot;, 7)</code><br />
✅ <code>fmt.Println(7)</code> &mdash; только числа</p>

<p><strong>Ошибка 4: Неправильный порядок вывода</strong><br />
❌ Сначала частное, потом делитель<br />
✅ Сначала делитель (7), потом частное (13)</p>

<p><strong>Ошибка 5: Проверка до корня</strong><br />
❌ Ограничивать цикл корнем из n (может пропустить делитель)<br />
✅ Проверяйте все числа до n или до первого найденного</p>

<p style="margin-bottom: 0;"><strong>Ошибка 6: Использование флагов</strong><br />
❌ <code>found := false; if ... { found = true }</code><br />
✅ Просто <code>break</code> при нахождении</p>
