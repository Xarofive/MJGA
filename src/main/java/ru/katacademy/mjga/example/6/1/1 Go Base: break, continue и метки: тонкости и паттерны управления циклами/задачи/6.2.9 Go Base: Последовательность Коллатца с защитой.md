<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🔢 Числовая цепочка с защитой от зацикливания</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>

<p style="margin: 0;">Научиться использовать <code>break</code> с защитным условием, чтобы цикл не работал бесконечно, и комбинировать <code>if/else</code> с циклом <code>for</code>.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">💼 Контекст</h3>

<p style="margin-bottom: 10px;">Есть простая числовая игра: берём число и применяем к нему одно из двух правил &mdash; если оно чётное, делим на 2, если нечётное &mdash; умножаем на 3 и прибавляем 1. Повторяем, пока не получим 1. Для числа 27 цепочка довольно длинная (111 шагов), поэтому важно поставить защиту &mdash; если вдруг цепочка не завершится за разумное число шагов, программа должна остановиться. Никакой дополнительной математики здесь не нужно: просто точно реализуйте правила и цикл.</p>

<p style="margin: 0;"><strong>Задача:</strong> вычислить числовую цепочку для числа 27 с защитой от превышения 1000 шагов.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>

<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Правила числовой цепочки</h4>

<p>Берём число <code>n</code> и на каждом шаге применяем одно из двух правил:</p>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code class="language-go">if n%2 == 0 {
    n = n / 2       // чётное — делим на 2
} else {
    n = 3*n + 1     // нечётное — умножаем на 3 и прибавляем 1
}</code></pre>

<p style="margin-bottom: 0;">Повторяем, пока <code>n</code> не станет равным 1.</p>
</div>

<div style="background: #e3f2fd; border: 1px solid #90caf9; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #1565c0; margin-top: 0;">🎯 Защита от зацикливания</h4>

<p>Когда цикл может работать очень долго (или даже бесконечно), полезно добавить счётчик шагов и остановить программу, если шагов стало слишком много:</p>

<ul style="margin: 10px 0;">
	<li><strong>Счётчик шагов:</strong> переменная, которая увеличивается на каждой итерации</li>
	<li><strong>Лимит:</strong> максимальное количество допустимых шагов</li>
	<li><strong>Проверка + break:</strong> если счётчик превысил лимит &mdash; выводим сообщение и выходим из цикла</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;"><strong>Реализуйте числовую цепочку с защитой от зацикливания:</strong></p>

<h4 style="color: #e65100; margin-top: 16px;">📝 Структура программы</h4>

<ul style="margin: 10px 0;">
	<li>Начинается с <code>package main</code></li>
	<li>Импортирует только пакет <code>&quot;fmt&quot;</code> без псевдонима (используйте <code>import &quot;fmt&quot;</code>, форма <code>import f &quot;fmt&quot;</code> не допускается)</li>
	<li>Содержит только функцию <code>main()</code></li>
</ul>

<h4 style="color: #e65100; margin-top: 16px;">🔧 Алгоритм</h4>

<ul style="margin: 10px 0;">
	<li>Начальное значение: <code>n = 27</code></li>
	<li>Счётчик шагов: начинается с 0</li>
	<li>Используйте цикл <code>for</code> до достижения <code>n == 1</code></li>
	<li>На каждой итерации выводите текущее значение n</li>
	<li>Если <code>steps &gt; 1000</code>: вывести <code>&quot;limit exceeded&quot;</code> и <code>break</code></li>
	<li>Применяйте правила цепочки: чётное &mdash; делим на 2, нечётное &mdash; умножаем на 3 и прибавляем 1</li>
	<li>Увеличивайте счётчик после каждой итерации</li>
</ul>

<h4 style="color: #e65100; margin-top: 16px;">📤 Формат вывода</h4>

<ul style="margin: 10px 0;">
	<li>Используйте <code>fmt.Println</code> или <code>fmt.Printf</code></li>
	<li>Каждое число последовательности на отдельной строке</li>
	<li>Начинайте с вывода 27</li>
	<li>Заканчивайте выводом 1</li>
	<li>Если сработала защита: последняя строка <code>&quot;limit exceeded&quot;</code></li>
</ul>
</div>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #c62828; margin-top: 0;">⚠️ Частые ошибки и антипаттерны</h4>

<p><strong>Ошибка 1: Забыть инициализировать счётчик</strong><br />
❌ Объявить steps без начального значения<br />
✅ <code>steps := 0</code> перед циклом</p>

<p><strong>Ошибка 2: Неправильное место проверки лимита</strong><br />
❌ Проверять после изменения n<br />
✅ Проверять в начале каждой итерации</p>

<p><strong>Ошибка 3: Забыть увеличить счётчик</strong><br />
❌ Цикл без <code>steps++</code><br />
✅ Увеличивать счётчик на каждой итерации</p>

<p><strong>Ошибка 4: Неправильное сообщение</strong><br />
❌ <code>&quot;Limit Exceeded&quot;</code> или <code>&quot;limit_exceeded&quot;</code><br />
✅ Точно <code>&quot;limit exceeded&quot;</code> (нижний регистр)</p>

<p><strong>Ошибка 5: Лишний вывод</strong><br />
❌ Выводить счётчик или промежуточные сообщения<br />
✅ Только числа последовательности (и защитное сообщение при необходимости)</p>

<p style="margin-bottom: 0;"><strong>Ошибка 6: Неправильное условие цикла</strong><br />
❌ <code>for n &gt; 1</code> (не выведет последнюю 1)<br />
✅ <code>for n != 1</code> с выводом перед проверкой</p>
</div>

<h4 style="color: #6a1b9a; margin-top: 0;">🎨 Где пригодится защита от зацикливания</h4>

<p><strong>Чтение данных:</strong> Программа читает строки из файла &mdash; нужен лимит, чтобы не застрять на бесконечном вводе</p>

<p><strong>Повторные попытки:</strong> Отправляем запрос на сервер &mdash; если 5 попыток не сработали, останавливаемся</p>

<p style="margin-bottom: 0;"><strong>Любой цикл с неизвестным числом итераций:</strong> Если заранее не ясно, сколько раз выполнится тело цикла, счётчик + break &mdash; надёжная страховка</p>

<div style="background: #fff8e1; border-left: 4px solid #ffc107; padding: 16px; margin: 16px 0;">
<h4 style="color: #f57c00; margin-top: 0;">💡 Подсказка</h4>

<p>Структура программы: объявите <code>n</code> и <code>steps</code>, затем один цикл <code>for n != 1</code>. Внутри цикла сначала выведите <code>n</code>, потом проверьте лимит, потом примените правило (чётное/нечётное), потом увеличьте счётчик. После цикла не забудьте вывести финальную единицу.</p>
