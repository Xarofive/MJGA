<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🌡️ Советчик по температуре</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>

<p style="margin: 0;">Научиться использовать цепочки условных операторов для классификации данных по нескольким диапазонам и возвращать соответствующие текстовые рекомендации.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">💼 Контекст и мотивация</h3>

<p style="margin-bottom: 10px;">Классификация числовых данных по диапазонам &mdash; фундаментальная задача в системах умного дома, метеостанциях, климат-контроле и IoT-устройствах. От корректной интерпретации температурных показаний зависит комфорт пользователей, энергоэффективность зданий и правильная работа систем оповещения.</p>

<p style="margin: 0;"><strong>Задача:</strong> создать функцию для выдачи рекомендаций на основе температуры воздуха.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>

<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Цепочки условных операторов</h4>

<p>Конструкция <code>if - else if - else</code> позволяет проверять несколько условий последовательно:</p>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code class="language-go">if условие1 {
    // выполняется если условие1 истинно
} else if условие2 {
    // выполняется если условие1 ложно, но условие2 истинно
} else if условие3 {
    // выполняется если условия 1 и 2 ложны, но условие3 истинно
} else {
    // выполняется если все предыдущие условия ложны
}</code></pre>

<p style="margin-bottom: 0;">Проверки идут сверху вниз, выполняется первый подходящий блок, остальные игнорируются.</p>
</div>

<div style="background: #e3f2fd; border: 1px solid #90caf9; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #1565c0; margin-top: 0;">🎯 Работа с диапазонами</h4>

<p>При классификации по диапазонам важен порядок проверок:</p>

<ul style="margin: 10px 0;">
	<li><strong>Операторы сравнения:</strong> <code>&lt;</code>, <code>&lt;=</code>, <code>&gt;</code>, <code>&gt;=</code></li>
	<li><strong>Граничные значения:</strong> внимательно следите, куда относится граница</li>
	<li><strong>Полнота покрытия:</strong> все возможные значения должны быть обработаны</li>
	<li><strong>Непересекающиеся диапазоны:</strong> каждое значение попадает только в один диапазон</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;"><strong>Создайте функцию для выдачи температурных рекомендаций:</strong></p>

<h4 style="color: #e65100; margin-top: 16px;">🔧 Функция temperatureAdvice</h4>

<p><strong>Сигнатура:</strong></p>

<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>func temperatureAdvice(temp int) string</code></div>

<p><strong>Параметры:</strong></p>

<ul style="margin: 10px 0;">
	<li><code>temp</code> (int) &mdash; температура воздуха в градусах Цельсия</li>
</ul>

<p><strong>Температурные диапазоны и рекомендации:</strong></p>

<ul style="margin: 10px 0;">
	<li><strong>&quot;freezing&quot;</strong> &mdash; температура ниже 0&deg;C (не включая 0)</li>
	<li><strong>&quot;chilly&quot;</strong> &mdash; от 0&deg;C до 17&deg;C включительно</li>
	<li><strong>&quot;warm&quot;</strong> &mdash; от 18&deg;C до 27&deg;C включительно</li>
	<li><strong>&quot;hot&quot;</strong> &mdash; выше 27&deg;C</li>
</ul>

<p><strong>Требования к реализации:</strong></p>

<ul style="margin: 10px 0;">
	<li>Используйте одну из допустимых форм классификации: (1) полная цепочка <code>if</code> &rarr; <code>else if</code> &rarr; <code>else</code>, (2) последовательные <code>if cond { return ... }</code> с финальным <code>return</code>, или (3) <code>if</code> &rarr; <code>else if</code> без финального <code>else</code> плюс финальный <code>return</code></li>
	<li>Функция должна возвращать строку с рекомендацией</li>
	<li>Запрещено использовать циклы (<code>for</code>) и оператор <code>switch</code></li>
	<li>Каждое значение температуры должно попадать ровно в один диапазон</li>
</ul>

<h4 style="color: #e65100; margin-top: 16px;">📝 Формат решения</h4>

<ul style="margin: 10px 0;">
	<li>Включите в ответ полную сигнатуру функции, например <code>func temperatureAdvice(temp int) string { ... }</code></li>
	<li>НЕ указывайте <code>package main</code></li>
	<li>НЕ добавляйте <code>import</code> директивы</li>
</ul>
</div>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #c62828; margin-top: 0;">⚠️ Частые ошибки и антипаттерны</h4>

<p><strong>Ошибка 1: Неправильные границы диапазонов</strong><br />
❌ 0&deg;C относится к &quot;freezing&quot;<br />
✅ 0&deg;C относится к &quot;chilly&quot;</p>

<p><strong>Ошибка 2: Пропущенные значения</strong><br />
❌ Граница 17/18 не обработана<br />
✅ 17&deg;C &mdash; &quot;chilly&quot;, 18&deg;C &mdash; &quot;warm&quot;</p>

<p><strong>Ошибка 3: Пересекающиеся диапазоны</strong><br />
❌ Одна температура попадает в два диапазона<br />
✅ Каждое значение &mdash; только в один диапазон</p>

<p><strong>Ошибка 4: Неверный порядок проверок</strong><br />
❌ Сначала проверять широкие диапазоны<br />
✅ Упорядочить проверки логично (например, от холода к жаре)</p>

<p><strong>Ошибка 5: Все условия &mdash; цепочка <code>if</code> без раннего возврата</strong><br />
❌ Несколько <code>if</code>-блоков подряд без <code>else if</code> и без раннего <code>return</code> &mdash; не охватывают все диапазоны<br />
✅ Используйте либо цепочку <code>if</code> &rarr; <code>else if</code> &rarr; <code>else</code>, либо последовательные <code>if cond { return ... }</code> с финальным <code>return</code></p>

<p style="margin-bottom: 0;"><strong>Ошибка 6: Опечатки в строках</strong><br />
❌ <code>&quot;Freezing&quot;</code>, <code>&quot;WARM&quot;</code>, <code>&quot;chill&quot;</code><br />
✅ Точные значения: <code>&quot;freezing&quot;</code>, <code>&quot;chilly&quot;</code>, <code>&quot;warm&quot;</code>, <code>&quot;hot&quot;</code></p>
</div>

<h4 style="color: #f57c00; margin-top: 0;">💡 Подсказка</h4>

<p>Представьте термометр с цветными зонами: синяя (мороз), голубая (прохлада), зелёная (комфорт), красная (жара). Границы между зонами чёткие &mdash; каждый градус относится только к одной зоне.
