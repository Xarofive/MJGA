<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🎭 Контроль доступа на мероприятие</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>

<p style="margin: 0;">Научиться использовать логические операторы &amp;&amp; (И) и || (ИЛИ) для создания составных условий проверки доступа.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">💼 Контекст и мотивация</h3>

<p style="margin-bottom: 10px;">Системы контроля доступа используются повсеместно: от концертных площадок до корпоративных мероприятий. Правильная реализация логики доступа обеспечивает безопасность, соблюдение возрастных ограничений и управление привилегированными пользователями. Комбинирование нескольких условий через логические операторы позволяет создавать гибкие правила допуска.</p>

<p style="margin: 0;"><strong>Задача:</strong> создать функцию для определения права доступа на мероприятие.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>

<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Логические операторы</h4>

<p>Go предоставляет операторы для комбинирования булевых условий:</p>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code class="language-go">// Оператор &amp;&amp; (И) - оба условия должны быть true
if age &gt;= 18 &amp;&amp; hasTicket {
    // доступ только для взрослых с билетом
}
// Оператор || (ИЛИ) - хотя бы одно условие true
if hasTicket || isVIP {
// доступ с билетом ИЛИ VIP-статусом
}
// Комбинирование операторов
if (age &gt;= 18 &amp;&amp; hasTicket) || vip {
// взрослый с билетом ИЛИ VIP любого возраста
}</code></pre>

<p style="margin-bottom: 0;">Скобки помогают управлять приоритетом операций.</p>
</div>

<div style="background: #e3f2fd; border: 1px solid #90caf9; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #1565c0; margin-top: 0;">🎯 Приоритет логических операций</h4>

<p>Важные правила при работе с логическими операторами:</p>

<ul style="margin: 10px 0;">
	<li><strong>&amp;&amp; имеет приоритет над ||:</strong> <code>a || b &amp;&amp; c</code> = <code>a || (b &amp;&amp; c)</code></li>
	<li><strong>Используйте скобки:</strong> для ясности и управления порядком</li>
	<li><strong>Короткое замыкание:</strong> если первое условие в &amp;&amp; false, второе не проверяется</li>
	<li><strong>VIP обходит ограничения:</strong> типичный паттерн для привилегий</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;"><strong>Создайте функцию для проверки доступа на мероприятие:</strong></p>

<h4 style="color: #e65100; margin-top: 16px;">🔧 Функция accessStatus</h4>

<p><strong>Сигнатура:</strong></p>

<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>func accessStatus(age int, hasTicket bool, vip bool) string</code></div>

<p><strong>Параметры:</strong></p>

<ul style="margin: 10px 0;">
	<li><code>age</code> (int) &mdash; возраст посетителя</li>
	<li><code>hasTicket</code> (bool) &mdash; наличие билета (<code>true</code> &mdash; есть билет)</li>
	<li><code>vip</code> (bool) &mdash; VIP-статус (<code>true</code> &mdash; VIP-гость)</li>
</ul>

<p><strong>Правила доступа:</strong></p>

<ul style="margin: 10px 0;">
	<li><strong>&quot;allowed&quot;</strong> &mdash; доступ разрешён, если:

	<ul>
		<li>Возраст &ge; 18 И есть билет, ИЛИ</li>
		<li>Есть VIP-статус (независимо от возраста и билета)</li>
	</ul>
	</li>
	<li><strong>&quot;denied&quot;</strong> &mdash; доступ запрещён во всех остальных случаях</li>
</ul>

<p><strong>Требования к реализации:</strong></p>

<ul style="margin: 10px 0;">
	<li>Обязательно использовать оператор <code>&amp;&amp;</code> (И)</li>
	<li>Обязательно использовать оператор <code>||</code> (ИЛИ)</li>
	<li>Запрещено использовать циклы (<code>for</code>) и оператор <code>switch</code></li>
	<li>Запрещено использовать функции вывода (<code>fmt.Print*</code>)</li>
</ul>

<h4 style="color: #e65100; margin-top: 16px;">📝 Формат решения</h4>

<ul style="margin: 10px 0;">
	<li>Включите в ответ полную сигнатуру функции, например <code>func accessStatus(age int, hasTicket bool, vip bool) string { ... }</code></li>
	<li>НЕ указывайте <code>package main</code></li>
	<li>НЕ добавляйте <code>import</code> директивы</li>
</ul>
</div>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #c62828; margin-top: 0;">⚠️ Частые ошибки и антипаттерны</h4>

<p><strong>Ошибка 1: Неправильный порядок операций</strong><br />
❌ <code>age &gt;= 18 || hasTicket &amp;&amp; vip</code><br />
✅ <code>(age &gt;= 18 &amp;&amp; hasTicket) || vip</code></p>

<p><strong>Ошибка 2: Избыточные проверки VIP</strong><br />
❌ <code>if vip &amp;&amp; age &gt;= 0</code> &mdash; лишнее условие<br />
✅ <code>if vip</code> &mdash; VIP обходит все ограничения</p>

<p><strong>Ошибка 3: Неиспользование операторов</strong><br />
❌ Вложенные if без &amp;&amp; и ||<br />
✅ Обязательно использовать оба оператора</p>

<p><strong>Ошибка 4: Неверная граница возраста</strong><br />
❌ <code>age &gt; 18</code> &mdash; исключает 18-летних<br />
✅ <code>age &gt;= 18</code> &mdash; включает 18 лет</p>

<p><strong>Ошибка 5: Опечатки в возвращаемых строках</strong><br />
❌ <code>&quot;Allowed&quot;</code>, <code>&quot;DENIED&quot;</code>, <code>&quot;allow&quot;</code><br />
✅ Точные значения: <code>&quot;allowed&quot;</code> и <code>&quot;denied&quot;</code></p>

<p style="margin-bottom: 0;"><strong>Ошибка 6: Лишняя логика</strong><br />
❌ Отдельная проверка <code>!hasTicket &amp;&amp; !vip</code><br />
✅ Простой else для всех остальных случаев</p>
</div>

<div style="background: #fff8e1; border-left: 4px solid #ffc107; padding: 16px; margin: 16px 0;">
<h4 style="color: #f57c00; margin-top: 0;">💡 Подсказка</h4>

<p>Подумайте о двух путях получения доступа: стандартный (взрослый с билетом) и привилегированный (VIP). Оператор &amp;&amp; создаёт строгое условие &mdash; оба критерия должны выполняться. Оператор || даёт альтернативу &mdash; достаточно одного пути. VIP-статус &mdash; это &quot;золотой ключ&quot;, который открывает все двери независимо от других факторов. Правильное сочетание операторов создаёт элегантное решение в одном условии.</p>
