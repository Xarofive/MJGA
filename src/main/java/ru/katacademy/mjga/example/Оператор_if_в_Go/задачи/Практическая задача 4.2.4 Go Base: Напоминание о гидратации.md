<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">💧 Напоминание о гидратации ⭐</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>

<p style="margin: 0;">Научиться использовать составные условия с несколькими параметрами для принятия решений и обрабатывать недопустимые входные данные.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">💼 Контекст и мотивация</h3>

<p style="margin-bottom: 10px;">В фитнес-приложениях, системах здоровья и спортивных трекерах важно напоминать пользователям о необходимости пить воду. Риск обезвоживания зависит от температуры окружающей среды и длительности физической активности. Правильные рекомендации помогают предотвратить тепловой удар и поддерживать оптимальную работоспособность.</p>

<p style="margin: 0;"><strong>Задача:</strong> создать функцию для определения необходимости брать воду на прогулку.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>

<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Комбинирование условий</h4>

<p>Для принятия решений на основе нескольких факторов используйте логические операторы:</p>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code class="language-go">// Проверка нескольких условий одновременно
if температура &gt; 30 &amp;&amp; время &gt; 60 {
    // оба условия должны выполняться
}
// Проверка хотя бы одного условия
if температура &gt; 35 || время &gt; 120 {
// достаточно одного истинного условия
}
// Цепочка проверок
if условие1 {
// первый случай
} else if условие2 {
// второй случай
} else {
// остальные случаи
}</code></pre>

<p style="margin-bottom: 0;">Порядок проверок важен &mdash; выполняется первый подходящий блок.</p>
</div>

<div style="background: #e3f2fd; border: 1px solid #90caf9; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #1565c0; margin-top: 0;">🎯 Валидация данных</h4>

<p>Всегда проверяйте корректность входных параметров:</p>

<ul style="margin: 10px 0;">
	<li><strong>Границы значений:</strong> температура и время не могут быть произвольными</li>
	<li><strong>Приоритет проверок:</strong> сначала валидация, потом бизнес-логика</li>
	<li><strong>Явное указание ошибки:</strong> возвращайте специальное значение для некорректных данных</li>
	<li><strong>Защитное программирование:</strong> не доверяйте входным данным</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;"><strong>Создайте функцию для рекомендаций по гидратации:</strong></p>

<h4 style="color: #e65100; margin-top: 16px;">🔧 Функция hydrationReminder</h4>

<p><strong>Сигнатура:</strong></p>

<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>func hydrationReminder(temp int, minutesOutside int) string</code></div>

<p><strong>Параметры:</strong></p>

<ul style="margin: 10px 0;">
	<li><code>temp</code> (int) &mdash; температура воздуха в градусах Цельсия</li>
	<li><code>minutesOutside</code> (int) &mdash; планируемое время на улице в минутах</li>
</ul>

<p><strong>Логика рекомендаций:</strong></p>

<ul style="margin: 10px 0;">
	<li><strong>&quot;invalid&quot;</strong> &mdash; недопустимые параметры (температура ниже -30&deg;C или время отрицательное)</li>
	<li><strong>&quot;bring bottle&quot;</strong> &mdash; обязательно взять воду (температура &gt; 30&deg;C ИЛИ время &gt; 90 минут)</li>
	<li><strong>&quot;optional&quot;</strong> &mdash; желательно взять воду (температура от 20&deg;C до 30&deg;C включительно ИЛИ время от 30 до 90 минут включительно)</li>
	<li><strong>&quot;no need&quot;</strong> &mdash; вода не требуется (все остальные допустимые случаи)</li>
</ul>

<p><strong>Требования к реализации:</strong></p>

<ul style="margin: 10px 0;">
	<li>Обязательно использовать конструкцию <code>if</code> - <code>else if</code> - <code>else</code></li>
	<li>Запрещено использовать циклы (<code>for</code>) и оператор <code>switch</code></li>
	<li>Запрещено использовать функции вывода (<code>fmt.Print*</code>)</li>
	<li>Сначала проверять валидность данных, затем применять бизнес-логику</li>
</ul>

<h4 style="color: #e65100; margin-top: 16px;">📝 Формат решения</h4>

<ul style="margin: 10px 0;">
	<li>Включите в ответ полную сигнатуру функции, например <code>func hydrationReminder(temp int, minutesOutside int) string { ... }</code></li>
	<li>НЕ указывайте <code>package main</code></li>
	<li>НЕ добавляйте <code>import</code> директивы</li>
</ul>
</div>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #c62828; margin-top: 0;">⚠️ Частые ошибки и антипаттерны</h4>

<p><strong>Ошибка 1: Неправильный порядок проверок</strong><br />
❌ Сначала бизнес-логика, потом валидация<br />
✅ Всегда проверяйте валидность данных первой</p>

<p><strong>Ошибка 2: Неверная логика для &quot;optional&quot;</strong><br />
❌ Требуются оба условия одновременно (И)<br />
✅ Достаточно одного из условий в диапазоне (ИЛИ), но не критичные значения</p>

<p><strong>Ошибка 3: Пропущенная граница -30&deg;C</strong><br />
❌ temp &lt; -30 считается допустимым<br />
✅ Только temp &ge; -30 является валидным</p>

<p><strong>Ошибка 4: Неправильная логика &quot;bring bottle&quot;</strong><br />
❌ Требуются оба условия (И)<br />
✅ Достаточно одного условия (ИЛИ)</p>

<p><strong>Ошибка 5: Отсутствие else if</strong><br />
❌ Использование только вложенных if<br />
✅ Обязательно использовать <code>else if</code> для цепочки проверок</p>

<p style="margin-bottom: 0;"><strong>Ошибка 6: Опечатки в строках</strong><br />
❌ <code>&quot;Invalid&quot;</code>, <code>&quot;bring water&quot;</code>, <code>&quot;not needed&quot;</code><br />
✅ Точные значения: <code>&quot;invalid&quot;</code>, <code>&quot;bring bottle&quot;</code>, <code>&quot;optional&quot;</code>, <code>&quot;no need&quot;</code></p>
</div>
