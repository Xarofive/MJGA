<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🔍 Определение знака числа</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>

<p style="margin: 0;">Научиться использовать условные операторы для классификации данных и возвращать результат проверки из функции в виде строки.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">💼 Контекст и мотивация</h3>

<p style="margin-bottom: 10px;">Определение знака числа &mdash; фундаментальная операция в программировании, используемая повсеместно: от финансовых расчётов (прибыль/убыток) до физических симуляций (направление движения). Эта задача учит принимать решения на основе числовых данных и корректно обрабатывать граничные случаи, включая ноль.</p>

<p style="margin: 0;"><strong>Задача:</strong> создать функцию для текстового описания знака числа.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>

<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Условный оператор if</h4>

<p>Оператор <code>if</code> позволяет выполнять разный код в зависимости от условия:</p>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code class="language-go">if условие {
    // выполняется, если условие истинно
} else {
    // выполняется, если условие ложно
}
// Можно проверять несколько условий:
if условие1 {
// первый вариант
} else if условие2 {
// второй вариант
} else {
// если ни одно условие не выполнено
}</code></pre>

<p style="margin-bottom: 0;">Условия проверяются последовательно сверху вниз, выполняется первый подходящий блок.</p>
</div>

<div style="background: #e3f2fd; border: 1px solid #90caf9; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #1565c0; margin-top: 0;">🎯 Возврат значения из функции</h4>

<p>Функции могут возвращать результат с помощью оператора <code>return</code>:</p>

<ul style="margin: 10px 0;">
	<li><strong>Объявление типа возврата:</strong> указывается после параметров функции</li>
	<li><strong>Оператор return:</strong> немедленно завершает функцию и возвращает значение</li>
	<li><strong>Пример:</strong> <code>func getName() string { return &quot;Alice&quot; }</code></li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;"><strong>Создайте функцию для текстового описания знака числа:</strong></p>

<h4 style="color: #e65100; margin-top: 16px;">🔧 Функция describeSign</h4>

<p><strong>Сигнатура:</strong></p>

<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>func describeSign(n int) string</code></div>

<p><strong>Требования:</strong></p>

<ul style="margin: 10px 0;">
	<li>Принимает один параметр типа <code>int</code></li>
	<li>Возвращает строку (<code>string</code>)</li>
	<li>Для положительных чисел (n &gt; 0) возвращает <code>&quot;positive&quot;</code></li>
	<li>Для нуля (n == 0) возвращает <code>&quot;zero&quot;</code></li>
	<li>Для отрицательных чисел (n &lt; 0) возвращает <code>&quot;negative&quot;</code></li>
	<li>Обязательно использовать цепочку <code>if</code> / <code>else if</code> / <code>else</code></li>
	<li>Запрещено использовать циклы (<code>for</code>) и оператор <code>switch</code></li>
</ul>

<h4 style="color: #e65100; margin-top: 16px;">📝 Формат решения</h4>

<ul style="margin: 10px 0;">
	<li>Включите в ответ полную сигнатуру функции, например <code>func describeSign(n int) string { ... }</code></li>
	<li>НЕ указывайте <code>package main</code></li>
	<li>НЕ добавляйте <code>import</code> директивы</li>
</ul>
</div>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #c62828; margin-top: 0;">⚠️ Частые ошибки и антипаттерны</h4>

<p><strong>Ошибка 1: Неправильная классификация нуля</strong><br />
❌ Ноль считается положительным или отрицательным числом<br />
✅ Ноль &mdash; это отдельная категория: <code>&quot;zero&quot;</code></p>

<p><strong>Ошибка 2: Опечатки в возвращаемых строках</strong><br />
❌ <code>&quot;Positive&quot;</code>, <code>&quot;Negative&quot;</code>, <code>&quot;Zero&quot;</code><br />
✅ Точные значения: <code>&quot;positive&quot;</code>, <code>&quot;negative&quot;</code> и <code>&quot;zero&quot;</code></p>

<p><strong>Ошибка 3: Забытый return</strong><br />
❌ Присваивание результата переменной без возврата<br />
✅ Используйте <code>return</code> для возврата строки</p>

<p><strong>Ошибка 4: Использование запрещённых конструкций</strong><br />
❌ <code>switch</code>, <code>for</code>, тернарный оператор<br />
✅ Только <code>if</code> и <code>else</code></p>

<p><strong>Ошибка 5: Неправильная сигнатура</strong><br />
❌ <code>func describeSign(number int)</code> &mdash; забыт тип возврата<br />
✅ <code>func describeSign(n int) string</code> &mdash; имя параметра может быть любым</p>

<p style="margin-bottom: 0;"><strong>Ошибка 6: Лишний код вне функции</strong><br />
❌ Добавление <code>package main</code> или <code>import</code><br />
✅ Полное определение <code>func describeSign(n int) string { ... }</code> без <code>package</code> и <code>import</code></p>
</div>
