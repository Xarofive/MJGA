# Задача: Возведение в квадрат
> **Соответствует задаче:** `/Platform/go_base/4_Функции/4.3_Возвращаемые_значения/4.3.2 ЗАДАЧА - Go base: Возведение в квадрат.md`

## Описание:
Создай функцию `square`, которая принимает одно число `x` типа `int` и возвращает его квадрат.

## Что нужно сделать:
- Объяви функцию `square` с одним аргументом `x` типа `int`.
- Функция должна возвращать квадрат этого числа.

## Критерии приемки:
- Не нужно писать полную программу, только саму функцию `square`.
- Функция должна корректно возвращать квадрат переданного числа.

## Пример:
При вызове функции с `x = 4` ожидаемый результат: `16`
