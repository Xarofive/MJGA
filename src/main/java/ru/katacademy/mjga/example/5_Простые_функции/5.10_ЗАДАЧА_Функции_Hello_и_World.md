# Задача: Функции Hello и World
> **Соответствует задаче:** `/Platform/go_base/4_Функции/4.1_Объявление_функций/4.1.4 ЗАДАЧА - Go base: Функции Hello и World.md`

## Описание:
Создай функции `Hello` и `World`. Функция `Hello` должна вызывать функцию `World`, а функция `World` должна выводить строку `"Привет, мир!"`. Таким образом при вызове функции `Hello` должно выводиться сообщение `"Привет, мир!"`.

## Что нужно сделать:
- Объяви функцию `World` без параметров, которая выводит строку `"Привет, мир!"`.
- Объяви функцию `Hello` без параметров, которая вызывает функцию `World` внутри себя.

## Критерии приемки:
- Не нужно писать полную программу, только сами функции `Hello` и `World`.
- Функция `World` должна выводить строку `"Привет, мир!"`.
- Функция `Hello` должна вызывать функцию `World`.
- При вызове функции `Hello` должно происходить то же самое, что и при вызове функции `World` — вывод строки `"Привет, мир!"`.

## Пример:
```go
func main() {
    Hello() // При выполнении этой строки будет выведено "Привет, мир!"
}
```
