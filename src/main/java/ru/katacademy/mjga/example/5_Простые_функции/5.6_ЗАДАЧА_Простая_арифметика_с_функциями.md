# Задача: Арифметическая операция
> **Соответствует задаче:** `/Platform/go_base/4_Функции/4.2_Аргументы_функций/4.2.3 ЗАДАЧА - Go base: Арифметическая операция.md`

## Описание:
Создай функцию `calculate`, которая принимает два аргумента `a` и `b` типа `int` и выводит на экран результат их сложения, вычитания, умножения и деления.

## Что нужно сделать:
- Объяви функцию `calculate` с двумя аргументами `a` и `b` типа `int`.
- Внутри функции `calculate` выполни и выведи результаты следующих операций:
  - Сложение `a + b`
  - Вычитание `a - b`
  - Умножение `a * b`
  - Деление `a / b`

## Ожидаемый вывод при `a = 10` и `b = 2`: 
```
Сумма: 12
Разность: 8
Произведение: 20
Частное: 5
```

## Критерии приемки:
- Не нужно писать полную программу, только саму функцию calculate.
- Функция должна корректно выполнять и выводить все указанные операции.
