# Задача: Вычисление дискриминанта
> **Соответствует задаче:** `/Platform/go_base/4_Функции/4.3_Возвращаемые_значения/4.3.3 ЗАДАЧА - Go base: Вычисление дискриминанта.md`

## Описание:
Создай функцию `discriminant`, которая принимает три аргумента `a`, `b` и `c` типа `float64` и возвращает дискриминант квадратного уравнения.

## Что нужно сделать:
- Объяви функцию `discriminant` с тремя аргументами `a`, `b` и `c` типа `float64`.
- Функция должна вычислять дискриминант по формуле: `b*b - 4*a*c`.
- Функция должна возвращать результат типа `float64`.

## Критерии приемки:
- Не нужно писать полную программу, только саму функцию `discriminant`.
- Функция должна корректно вычислять дискриминант квадратного уравнения.

## Пример:
При вызове функции с параметрами `1`, `5`, `6` ожидаемый результат: `1.0`

Пояснение:
Дискриминант вычисляется как `b² - 4*a*c`, где `a`, `b` и `c` — коэффициенты квадратного уравнения `ax² + bx + c = 0`.
Для уравнения `1x² + 5x + 6 = 0` дискриминант равен `5² - 4*1*6 = 25 - 24 = 1`.
