# Задача: Функции Foo и Bar
> **Соответствует задаче:** `/Platform/go_base/4_Функции/4.1_Объявление_функций/4.1.3 ЗАДАЧА - Go base: Функции Foo и Bar.md`

## Описание:
Создай две функции: `Foo` и `Bar`. Функция `Foo` должна выводить на экран строку `"Вызвана функция Foo"`, а функция `Bar` должна выводить строку `"Вызвана функция Bar"`.

## Что нужно сделать:
- Объяви функцию `Foo` без параметров, которая выводит сообщение `"Вызвана функция Foo"`.
- Объяви функцию `Bar` без параметров, которая выводит сообщение `"Вызвана функция Bar"`.

## Критерии приемки:
- Не нужно писать полную программу, только сами функции `Foo` и `Bar`.
- Каждая функция должна выводить соответствующее сообщение.
- Функции должны быть объявлены отдельно друг от друга.
