# Задача: Объявление функции greet
> **Соответствует задаче:** `/Platform/go_base/4_Функции/4.1_Объявление_функций/4.1.2 ЗАДАЧА - Go base: Объявление функции greet.md`

## Описание:
Создай функцию `greet`, которая выводит на экран строку `"Привет, мир!"`. 

## Что нужно сделать:
- Объяви функцию `greet` без параметров.
- Внутри функции используй `fmt.Println()` для вывода строки `"Привет, мир!"`.

## Критерии приемки:
- Не нужно писать полную программу, только саму функцию `greet`.
- Функция должна выводить строку `"Привет, мир!"`.
- Не забудь использовать `fmt.Println()` для вывода строки.
