# Возвращаемые значения функций в Go

Функции в Go могут не только выполнять какие-либо действия, но и возвращать результаты этих действий. Возвращаемые значения позволяют использовать результаты работы функции в других частях программы.

## Синтаксис функции с возвращаемым значением

Общий синтаксис функции, возвращающей значение:

```go
func имяФункции(аргументы) типВозвращаемогоЗначения {
    // тело функции
    return значение
}
```

## Пример функции с возвращаемым значением

Рассмотрим простую функцию, вычисляющую квадрат числа и возвращающую результат:

```go
package main

import "fmt"

// Функция square возвращает квадрат числа n
func square(n int) int {
    return n * n
}

func main() {
    result := square(5)
    fmt.Println("Квадрат числа 5 равен:", result)
}
```

### Объяснение примера:
- Функция `square` принимает одно целое число `n` и возвращает его квадрат (`n * n`).
- Внутри функции используется ключевое слово `return`, которое указывает на возвращаемое значение.
- В функции `main()` результат выполнения функции `square(5)` сохраняется в переменную `result`.
- Затем результат выводится на экран.

Вывод программы будет следующим:
```
Квадрат числа 5 равен: 25
```

## Возврат нескольких значений

Функции в Go могут возвращать сразу несколько значений.

```go
func имяФункции(аргументы) (тип1, тип2) {
    // тело функции
    return значение1, значение2
}
```

### Пример функции с несколькими возвращаемыми значениями:

Функция возвращает сумму и разность двух чисел:

```go
package main

import "fmt"

// Функция addAndSubtract возвращает сумму и разность двух чиселunc addAndSubtract(a, b int) (int, int) {
    return a + b, a - b
}

func main() {
    sum, difference := addAndSubtract(10, 5)
    fmt.Println("Сумма:", sum)
    fmt.Println("Разность:", difference)
}
```

Вывод программы будет таким:
```
Сумма: 15
Разность: 5
```

## Именованные возвращаемые значения

В Go можно использовать именованные возвращаемые значения, что делает код понятнее:

```go
func имяФункции(аргументы) (result int) {
    result = вычисление
    return
}
```

### Пример функции с именованным возвращаемым значением:

```go
package main

import "fmt"

// Функция multiply возвращает произведение двух чисел
func multiply(a, b int) (result int) {
    result = a * b
    return
}

func main() {
    product := multiply(4, 7)
    fmt.Println("Произведение:", product)
}
```

Вывод программы будет таким:
```
Произведение: 28
```

## Важные выводы:
- Функции в Go могут возвращать одно или несколько значений.
- Для возврата значений используется ключевое слово `return`.
- Можно использовать именованные возвращаемые значения для повышения читаемости кода.
