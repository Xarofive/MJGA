# Задача: Приветствие по имени
> **Соответствует задаче:** `/Platform/go_base/4_Функции/4.2_Аргументы_функций/4.2.2 ЗАДАЧА - Go base: Приветствие по имени.md`

## Описание:
Создай функцию `greet`, которая принимает один аргумент `name` типа `string` и выводит на экран приветствие в формате `"Привет, <n>!"`.

Для соединения строки `"Привет, "` и переданного имени используй конкатенацию строк.

```go
fmt.Println("строка1" + "строка2")
```

## Что нужно сделать:
- Объяви функцию `greet` с одним аргументом типа `string`.
- Внутри функции используй `fmt.Println` для вывода строки `"Привет, <n>!"`, заменяя `<n>` на переданное значение.

## Критерии приемки:
- Не нужно писать полную программу, только саму функцию `greet`.
- Функция должна корректно выводить переданное имя в формате `"Привет, <n>!"`.
- Тебе нужно объявить только саму функцию `greet` с использованием `fmt.Println` для вывода строки.
