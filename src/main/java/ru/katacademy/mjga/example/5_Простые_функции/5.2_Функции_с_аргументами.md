# Функции с аргументами в Go

Функции в языке Go могут принимать аргументы — это позволяет передавать в функцию разные значения и получать соответствующий результат. В этой лекции разберём, как создавать функции с аргументами и правильно их использовать.

## Что такое аргументы функции?

Аргументы (или параметры) функции — это входные данные, которые мы передаём внутрь функции для её работы. Используя аргументы, одна и та же функция может быть применена к разным входным значениям.

### Синтаксис объявления функций с аргументами

Общий синтаксис выглядит так:

```go
func имяФункции(аргумент1 тип, аргумент2 тип) {
    // тело функции
}
```

### Пример простой функции с аргументами

Создадим функцию, которая выводит приветствие для переданного ей имени:

```go
package main

import "fmt"

// greet принимает один аргумент name типа string
func greet(name string) {
    fmt.Println("Привет,", name)
}

func main() {
    greet("Алексей") // Вызов функции с аргументом "Алексей"
    greet("Мария")   // Вызов функции с аргументом "Мария"
}
```

### Вывод программы:
```
Привет, Алексей
Привет, Мария
```

## Несколько аргументов функции

Функции могут принимать несколько аргументов разных типов. Вот пример:

```go
package main

import "fmt"

// Функция принимает два аргумента и выводит их сумму
func add(a int, b int) {
    fmt.Println("Сумма равна:", a+b)
}

func main() {
    add(5, 10)    // Вызов функции с аргументами 5 и 10
    add(20, 30)   // Вызов функции с аргументами 20 и 30
}
```

### Вывод программы:
```
Сумма равна: 15
Сумма равна: 50
```

### Сокращённая запись нескольких аргументов одного типа

Если аргументы имеют одинаковый тип, можно использовать сокращённую запись:

```go
func add(a, b int) {
    fmt.Println("Сумма равна:", a+b)
}
```

## Важные моменты при использовании аргументов

- Аргументы передаются в функцию по значению. То есть функция получает копию значения аргумента, а исходная переменная не меняется.
- Имена аргументов должны быть понятными и отражать их предназначение.

## Заключение

Использование аргументов делает функции в Go универсальными и гибкими, позволяя создавать код, который легко переиспользовать и поддерживать.
