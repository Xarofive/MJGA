# 5.1 Создание и вызов функций в Go

Функции — это один из важнейших элементов программирования, позволяющий разбивать код на логические части и повторно использовать его. В Go функции используются очень часто, поэтому важно понять их устройство и правила использования.

## Что такое функция?

Функция — это блок кода, который выполняет определённую задачу. Функции помогают упрощать код, избегать повторений и делать программу более читаемой.

## Как создать функцию

Функция в Go создаётся с помощью ключевого слова `func`. У функции обязательно должно быть имя и круглые скобки `()`. Вот базовый синтаксис:

```go
func имяФункции() {
    // код функции
}
```

## Пример простой функции

Создадим простую функцию, которая выводит сообщение на экран:

```go
package main

import "fmt"

func greet() {
    fmt.Println("Привет от функции greet!")
}

func main() {
    greet() // вызов функции
}
```

### Объяснение примера:
- Мы создали функцию `greet`, которая не принимает аргументов и просто выводит текст.
- В функции `main` мы вызвали нашу функцию `greet`.

Вывод:
```
Привет от функции greet!
```

## Правила именования функций

- Имя функции должно начинаться с буквы и может включать буквы, цифры и подчёркивания.
- Название функции должно быть осмысленным и отражать её предназначение (например, `calculateSum`, `printMessage`).

## Функция с несколькими вызовами

Функции можно вызывать сколько угодно раз. Это одно из основных преимуществ функций:

```go
package main

import "fmt"

func sayHello() {
    fmt.Println("Hello!")
}

func main() {
    sayHello()
    sayHello()
    sayHello()
}
```

Вывод:
```
Hello!
Hello!
Hello!
```

## Итоги

Функции:
- Повышают читаемость и организованность кода.
- Позволяют избежать повторения кода.
- Могут быть вызваны многократно из разных мест программы.
