<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">⚡ Возведение числа в степень</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>

<p style="margin: 0;">Научиться реализовывать математические операции через базовые конструкции языка, понимая механику накопления результата в цикле.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">💼 Контекст</h3>

<p style="margin-bottom: 10px;">Возведение в степень &mdash; фундаментальная операция в криптографии, компьютерной графике и научных вычислениях. Понимание её реализации через цикл развивает навык построения алгоритмов накопления.</p>

<p style="margin: 0;"><strong>Задача:</strong> создать функцию для возведения целого числа в неотрицательную степень без использования встроенных математических функций.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>

<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Математическая основа</h4>

<p>Возведение в степень &mdash; это многократное умножение числа на себя:</p>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code>x^n = x × x × x × ... × x (n раз)
x^0 = 1 (для любого x)
x^1 = x</code></pre>

<p style="margin-bottom: 0;">Ключевая идея: начинаем с единицы и последовательно умножаем на основание нужное количество раз.</p>
</div>

<div style="background: #e3f2fd; border: 1px solid #90caf9; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #1565c0; margin-top: 0;">🎯 Паттерн накопления</h4>

<p>Многие алгоритмы строятся на накоплении результата в переменной:</p>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code><span style="color: #546e7a;">// Общий паттерн накопления</span>
result := initialValue
<span style="color: #c792ea;">for</span> условие {
    result = обновление(result)
}</code></pre>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;"><strong>Реализуйте функцию возведения в степень:</strong></p>

<h4 style="color: #e65100; margin-top: 16px;">📝 Сигнатура функции</h4>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code><span style="color: #c792ea;">func</span> <span style="color: #82aaff;">power</span>(x, n <span style="color: #82aaff;">int</span>) <span style="color: #82aaff;">int</span></code></pre>

<h4 style="color: #e65100; margin-top: 16px;">🎯 Требования</h4>

<ul style="margin: 10px 0;">
	<li><strong>Входные данные:</strong> два целых числа &mdash; основание <code>x</code> и показатель степени <code>n</code></li>
	<li><strong>Выходные данные:</strong> результат возведения <code>x</code> в степень <code>n</code></li>
	<li><strong>Ограничения:</strong> <code>n &ge; 0</code> (степень всегда неотрицательная)</li>
	<li><strong>Метод решения:</strong> использовать цикл <code>for</code> для вычисления (рекурсия запрещена)</li>
	<li><strong>Формат решения:</strong> отправляйте только функцию без <code>package</code> и <code>import</code></li>
	<li><strong>Запрещено:</strong> вывод в консоль через <code>print</code>, <code>println</code> и <code>fmt.*</code></li>
</ul>

<h4 style="color: #e65100; margin-top: 16px;">⚡ Особые случаи</h4>

<ul style="margin: 10px 0;">
	<li>Любое число в степени 0 равно 1</li>
	<li>Любое число в степени 1 равно самому себе</li>
	<li>1 в любой степени равно 1</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Примеры работы</h2>

<div style="background: #ffffff; border: 2px solid #3498db; border-radius: 8px; padding: 16px; margin: 16px 0;">

<pre style="background: #f4f4f4; color: #000000; padding: 12px; border-radius: 4px; border-left: 4px solid #3498db; font-family: monospace; margin: 0; font-size: 14px; line-height: 1.6;">
power(2, 3) &rarr; 8

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #c62828; margin-top: 0;">⚠️ Частые ошибки</h4>

<p><strong>Ошибка 1: Начальное значение 0</strong><br />
Если начать с 0, результат всегда будет 0 (0 &times; x = 0).</p>

<p><strong>Ошибка 2: Умножение на степень</strong><br />
Не путайте: нужно умножать на <code>x</code> ровно <code>n</code> раз, а не на <code>n</code>.</p>

<p><strong>Ошибка 3: Лишняя итерация</strong><br />
Проверьте граничные условия цикла &mdash; сколько раз он выполняется?</p>

<p style="margin-bottom: 0;"><strong>Ошибка 4: Игнорирование n=0</strong><br />
Не забудьте, что при n=0 цикл не должен выполняться, а результат должен быть 1.</p>
</div>

<div style="background: #e8f5e9; border: 1px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #2e7d32; margin-top: 0;">✅ Проверьте себя</h4>

<ul style="margin-bottom: 0;">
	<li>Функция возвращает 1 для любого x при n=0?</li>
	<li>Функция возвращает x при n=1?</li>
	<li>Результат корректен для обычных случаев (например, 2^3 = 8)?</li>
	<li>Используется цикл for для вычисления?</li>
	<li>Переменная-накопитель инициализирована правильным значением?</li>
	<li>В решении нет <code>package</code>/<code>import</code> и вызовов <code>print</code>, <code>println</code>, <code>fmt.*</code>?</li>
</ul>
</div>
