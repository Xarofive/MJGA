<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🔢 Сумма чётных чисел в диапазоне</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>

<p style="margin: 0;">Научиться применять условную логику внутри цикла для обработки подмножества значений, используя оператор остатка от деления для проверки чётности.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">💼 Контекст</h3>

<p style="margin-bottom: 10px;">Фильтрация и агрегация данных по условию &mdash; базовый паттерн в аналитике, статистике и обработке массивов. Чётность числа определяется остатком от деления на 2, что широко используется в алгоритмах распределения нагрузки и оптимизации.</p>

<p style="margin: 0;"><strong>Задача:</strong> создать функцию для вычисления суммы всех чётных чисел в диапазоне от 1 до n включительно.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>

<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Проверка чётности</h4>

<p>Число является чётным, если делится на 2 без остатка:</p>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code>Чётное: x % 2 == 0
Нечётное: x % 2 == 1 (или != 0)

Примеры:
4 % 2 = 0 → чётное
7 % 2 = 1 → нечётное</code></pre>

<p style="margin-bottom: 0;">Оператор <code>%</code> (остаток от деления) &mdash; ключевой инструмент для проверки делимости и классификации чисел.</p>
</div>

<div style="background: #e3f2fd; border: 1px solid #90caf9; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #1565c0; margin-top: 0;">🎯 Паттерн селективного накопления</h4>

<p>Накопление с условием &mdash; частый алгоритмический приём:</p>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code><span style="color: #546e7a;">// Общий паттерн</span>
sum := 0
<span style="color: #c792ea;">for</span> i := начало; i &lt;= конец; i++ {
    <span style="color: #c792ea;">if</span> условие(i) {
        sum += i
    }
}</code></pre>

<p style="margin-bottom: 0;">Этот паттерн позволяет обрабатывать только элементы, удовлетворяющие критерию.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;"><strong>Реализуйте функцию суммирования чётных чисел:</strong></p>

<h4 style="color: #e65100; margin-top: 16px;">📝 Сигнатура функции</h4>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code><span style="color: #c792ea;">func</span> <span style="color: #82aaff;">sumEven</span>(n <span style="color: #82aaff;">int</span>) <span style="color: #82aaff;">int</span></code></pre>

<h4 style="color: #e65100; margin-top: 16px;">🎯 Требования</h4>

<ul style="margin: 10px 0;">
	<li><strong>Входные данные:</strong> целое число <code>n</code> &mdash; верхняя граница диапазона</li>
	<li><strong>Выходные данные:</strong> сумма всех чётных чисел от 1 до <code>n</code> включительно</li>
	<li><strong>Метод решения:</strong> использовать цикл <code>for</code> для перебора чисел</li>
	<li><strong>Проверка чётности:</strong> обязательно использовать оператор <code>%</code> (остаток от деления)</li>
	<li><strong>Накопление:</strong> суммировать только чётные числа в переменной-аккумуляторе</li>
</ul>

<h4 style="color: #e65100; margin-top: 16px;">⚡ Особые случаи</h4>

<ul style="margin: 10px 0;">
	<li>При <code>n &lt; 1</code> результат должен быть 0 (пустой диапазон)</li>
	<li>При <code>n = 1</code> результат 0 (единственное число 1 &mdash; нечётное)</li>
	<li>При <code>n = 2</code> результат 2 (единственное чётное число)</li>
</ul>

<h4 style="color: #e65100; margin-top: 16px;">🚫 Ограничения</h4>

<ul style="margin: 10px 0;">
	<li>Запрещены импорты и пакеты &mdash; только базовые конструкции языка</li>
	<li>Запрещена печать (fmt.Print и т.д.) &mdash; только вычисления</li>
	<li>Функция должна возвращать значение через <code>return</code> &mdash; никакого вывода и аварийной остановки программы</li>
	<li>Решение должно содержать только одну функцию <code>sumEven</code></li>
</ul>
</div>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #c62828; margin-top: 0;">⚠️ Частые ошибки</h4>

<p><strong>Ошибка 1: Неверная проверка чётности</strong><br />
Не путайте: <code>x % 2 == 0</code> для чётных, а не <code>x / 2 == 0</code>.</p>

<p><strong>Ошибка 2: Забыли инициализировать сумму</strong><br />
Переменная-аккумулятор должна начинаться с 0, иначе результат будет неверным.</p>

<p><strong>Ошибка 3: Неверные границы цикла</strong><br />
Убедитесь, что перебираете числа от 1 до n включительно (при n &ge; 1).</p>

<p><strong>Ошибка 4: Накопление всех чисел</strong><br />
Проверяйте чётность ПЕРЕД добавлением к сумме, а не добавляйте все подряд.</p>

<p style="margin-bottom: 0;"><strong>Ошибка 5: Отсутствие обработки n &lt; 1</strong><br />
При отрицательном или нулевом n цикл не должен выполняться, верните 0.</p>
</div>
