<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">➕ Сумма с пропуском кратных</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>

<p style="margin: 0;">Научиться фильтровать значения внутри цикла с помощью условий, работать с оператором остатка от деления (%) для проверки кратности и накапливать результаты с проверкой граничных условий.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">💼 Контекст и мотивация</h3>

<p style="margin-bottom: 10px;">Фильтрация данных по условиям &mdash; базовая операция в программировании. Она используется в статистических расчётах (исключение выбросов), финансовых системах (пропуск нерабочих дней), игровой логике (бонусы на определённых уровнях), и обработке сигналов (фильтрация шумов). Умение селективно обрабатывать элементы последовательности критично для создания эффективных алгоритмов.</p>

<p style="margin: 0;"><strong>Задача:</strong> создать функцию, суммирующую числа от 1 до предела, исключая числа, кратные заданному значению.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>

<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Фильтрация значений внутри цикла</h4>

<p>Чтобы обработать только некоторые значения в цикле, используйте <code>if</code> с нужным условием:</p>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code class="language-go">// Пример: суммирование только нечётных чисел
sum := 0
for i := 1; i &lt;= 10; i++ {
    if i%2 != 0 {
        sum += i // добавляем только нечётные
    }
}</code></pre>

<p style="margin-bottom: 0;">Условие <code>if</code> внутри цикла позволяет выбирать, какие значения обрабатывать, а какие пропускать.</p>
</div>

<div style="background: #e3f2fd; border: 1px solid #90caf9; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #1565c0; margin-top: 0;">🎯 Оператор остатка и проверка кратности</h4>

<p>Оператор <code>%</code> (остаток от деления) позволяет проверять кратность:</p>

<ul style="margin: 10px 0;">
	<li><strong>Кратность:</strong> число A кратно B, если <code>A % B == 0</code></li>
	<li><strong>Примеры:</strong> <code>6 % 3 == 0</code> (6 кратно 3), <code>7 % 3 == 1</code> (7 не кратно 3)</li>
	<li><strong>Паттерн накопления:</strong> <code>sum += value</code> эквивалентно <code>sum = sum + value</code></li>
	<li><strong>Начальное значение:</strong> аккумулятор суммы инициализируется нулём</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;"><strong>Создайте функцию для суммирования чисел с пропуском кратных:</strong></p>

<h4 style="color: #e65100; margin-top: 16px;">🔧 Функция sumSkipMultiples</h4>

<p><strong>Сигнатура:</strong></p>

<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>func sumSkipMultiples(limit, skip int) int</code></div>

<p><strong>Требования:</strong></p>

<ul style="margin: 10px 0;">
	<li>Принимает два параметра: <code>limit</code> (верхняя граница) и <code>skip</code> (делитель для проверки кратности)</li>
	<li>Суммирует все числа от 1 до <code>limit</code> включительно, КРОМЕ чисел, кратных <code>skip</code></li>
	<li><strong>Формат решения:</strong> отправляйте только функцию без <code>package</code> и <code>import</code></li>
	<li><strong>Только функция <code>sumSkipMultiples</code></strong> &mdash; без вспомогательных функций (helpers), без <code>const</code>, без <code>type</code>, без переменных на верхнем уровне. Все вычисления должны быть внутри тела <code>sumSkipMultiples</code>.</li>
	<li><strong>Запрещено:</strong> вывод в консоль через <code>print</code>, <code>println</code> и <code>fmt.*</code></li>
	<li><strong>Граничные условия:</strong>
	<ul>
		<li>Если <code>limit &lt; 1</code>, вернуть 0</li>
		<li>Если <code>skip &lt;= 1</code>, вернуть 0</li>
	</ul>
	</li>
	<li>Использовать классический цикл <code>for</code> для перебора чисел от 1 до <code>limit</code></li>
	<li>Внутри цикла проверять кратность через оператор <code>%</code> и накапливать сумму подходящих чисел</li>
	<li>НЕ использовать <code>break</code> и <code>continue</code> &mdash; фильтрация должна делаться через условие</li>
	<li>Вернуть накопленную сумму</li>
</ul>

<details style="margin: 15px 0;"><summary style="cursor: pointer; color: #e65100; font-weight: bold;">Подсказка по реализации</summary>

<div style="background: #fff8e1; padding: 12px; border-radius: 4px; margin-top: 10px;">
<ul style="margin: 0;">
	<li>Цикл: <code>for i := 1; i &lt;= limit; i++</code></li>
	<li>Фильтрация: <code>if i%skip != 0 { sum += i }</code></li>
</ul>
</div>
</details>

<h4 style="color: #e65100; margin-top: 16px;">📐 Математическая модель</h4>

<p>Результат: &Sigma;(i) для i от 1 до limit, где i mod skip &ne; 0</p>

<p>Пример: sumSkipMultiples(10, 3) = 1+2+4+5+7+8+10 = 37 (пропущены 3, 6, 9)</p>
</div>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #c62828; margin-top: 0;">⚠️ Частые ошибки и антипаттерны</h4>

<p><strong>Ошибка 1: Отсутствие граничных проверок</strong><br />
❌ Сразу начинать цикл без проверки <code>limit &lt; 1</code> или <code>skip &lt;= 1</code><br />
✅ Сначала проверить оба граничных условия и вернуть 0</p>

<p><strong>Ошибка 2: Неправильная структура цикла</strong><br />
❌ <code>for i := 0; i &lt; limit; i++</code> &mdash; начинается с 0, не включает limit<br />
✅ <code>for i := 1; i &lt;= limit; i++</code> &mdash; от 1 до limit включительно</p>

<p><strong>Ошибка 3: Неверная проверка кратности</strong><br />
❌ <code>if i/skip == 0</code> &mdash; это деление, не остаток<br />
✅ <code>if i%skip != 0</code> &mdash; проверка остатка от деления</p>

<p><strong>Ошибка 4: Перевёрнутое условие</strong><br />
❌ <code>if i%skip == 0 { sum += i }</code> &mdash; суммирует кратные вместо НЕкратных<br />
✅ <code>if i%skip != 0 { sum += i }</code> &mdash; суммирует только НЕкратные</p>

<p><strong>Ошибка 5: Забыть накопление суммы</strong><br />
❌ Проверять условия, но не добавлять числа к сумме<br />
✅ <code>sum += i</code> внутри блока <code>if</code> для всех чисел, не кратных skip</p>

<p style="margin-bottom: 0;"><strong>Ошибка 6: Неправильная инициализация</strong><br />
❌ Не инициализировать переменную суммы<br />
✅ <code>sum := 0</code> перед циклом</p>
