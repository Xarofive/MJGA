# Цикл for в Go

Циклы используются в программировании, чтобы многократно выполнять один и тот же набор действий, пока выполняется некоторое условие. В языке Go основной и единственный тип цикла — это цикл `for`. Однако он универсален и способен заменить все остальные типы циклов, знакомые по другим языкам программирования.

## Синтаксис цикла for

Базовый синтаксис цикла for выглядит следующим образом:

```go
for начальное_значение; условие; изменение_значения {
    // код, который нужно повторить
}
```

### Пример цикла

Давайте рассмотрим простой пример, который выводит числа от 1 до 5:

```go
package main

import "fmt"

func main() {
    for i := 1; i <= 5; i++ {
        fmt.Println(i)
    }
}
```

В этом примере:

- Переменная `i` объявляется и инициализируется значением 1.
- Условие `i <= 5` проверяется перед каждой итерацией цикла.
- `i++` увеличивает значение переменной `i` на 1 после каждой итерации.
- Цикл выполняется, пока условие верно (`true`).

Вывод:
```
1
2
3
4
5
```

## Бесконечный цикл

В Go можно создать бесконечный цикл, опустив все три части условия:

```go
for {
    fmt.Println("Бесконечный цикл")
}
```

Такой цикл будет выполняться без остановки, пока не будет прерван вручную или с помощью оператора `break`.

## Цикл с условием (аналог while)

В Go нет отдельного цикла типа `while`. Вместо него используется сокращённая форма цикла `for`:

```go
i := 1
for i <= 5 {
    fmt.Println(i)
    i++
}
```

Это полностью эквивалентно конструкции `while` в других языках программирования.

## Итоги

- Цикл `for` в Go универсален и поддерживает различные сценарии использования.
- Он может использоваться как стандартный цикл с счётчиком, цикл с условием или бесконечный цикл.
- Это единственный цикл в языке Go, что делает его изучение особенно важным для работы с повторяющимися операциями.

Используйте циклы, чтобы выполнять действия столько раз, сколько вам необходимо, избегая дублирования кода и улучшая его читаемость.
