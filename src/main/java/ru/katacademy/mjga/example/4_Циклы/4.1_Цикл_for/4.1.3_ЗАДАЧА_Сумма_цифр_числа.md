# Задача: Сумма цифр числа
> **Соответствует задаче:** `/Platform/go_base/6_Циклы/6.1_Цикл_с_условием/6.1.4 ЗАДАЧА - Go base: Сумма цифр числа.md`

## Описание:
Напиши функцию `SumDigits`, которая принимает целое число `n` и возвращает сумму его цифр.

## Что нужно сделать:
1. Создай функцию `SumDigits`, которая принимает один аргумент `n` типа `int`.
2. Функция должна вычислять сумму всех цифр числа `n` и возвращать эту сумму.
3. Используй цикл `for` для последовательного извлечения и суммирования цифр.

## Критерии приемки:
- Функция должна принимать одно целое число `n` и возвращать сумму всех его цифр.
- Например, для числа 1234 результат должен быть 10 (1 + 2 + 3 + 4 = 10).
- Используй цикл `for` для решения задачи.
