# Операторы break и continue в Go

При работе с циклами иногда нужно прервать цикл или пропустить его текущую итерацию. Для этого в языке Go используются специальные операторы:

- `break`
- `continue`

## Оператор break

Оператор `break` используется для полного выхода из цикла раньше, чем это предусмотрено условием самого цикла.

### Пример:
```go
package main

import "fmt"

func main() {
	for i := 0; i < 10; i++ {
		if i == 5 {
			break // выход из цикла, когда i становится равным 5
		}
		fmt.Println(i)
	}
}
```

### Результат выполнения программы:
```
0
1
2
3
4
```

Когда значение переменной `i` достигает 5, цикл полностью останавливается.

---

## Оператор continue

Оператор `continue` используется, чтобы пропустить текущую итерацию цикла и перейти сразу к следующей итерации.

### Пример:
```go
package main

import "fmt"

func main() {
	for i := 0; i < 10; i++ {
		if i%2 == 0 {
			continue // пропускаем четные числа
		}
		fmt.Println(i)
	}
}
```

### Результат выполнения программы:
```
1
3
5
7
9
```

В данном примере оператор `continue` пропускает выполнение оставшейся части тела цикла для четных чисел, и цикл сразу переходит к следующей итерации.

---

## Сравнение break и continue

| Оператор | Действие                          | Пример использования                         |
|----------|-----------------------------------|----------------------------------------------|
| break    | Полностью завершает цикл          | Выход из цикла при достижении какого-то условия |
| continue | Пропускает текущую итерацию цикла | Игнорирование определённых итераций цикла (например, чётных чисел) |

---

## Когда использовать?

- Используйте `break`, когда дальнейшее выполнение цикла бессмысленно (например, нужный элемент уже найден).
- Используйте `continue`, если нужно пропустить итерацию и перейти к следующей без выполнения оставшейся логики текущей итерации.

Эти операторы делают ваш код более гибким и легко читаемым.
