# Задача: Поиск первого числа, кратного 3, но не кратного 5
> **Соответствует задаче:** `/Platform/go_base/6_Циклы/6.2_Операторы_цикла_continue_break/6.2.3 ЗАДАЧА - Go base: Поиск первого числа, кратного 3, но не кратного 5.md`

## Описание:
Создай функцию `findSpecialNumber`, которая принимает срез целых чисел и находит первое число, которое делится на 3 без остатка, но не делится на 5. Функция должна возвращать найденное число и его индекс в срезе. Если такого числа нет, функция должна вернуть 0 и -1.

## Что нужно сделать:
1. Создай функцию `findSpecialNumber(numbers []int) (int, int)`, которая:
   - Принимает срез целых чисел `numbers`
   - Перебирает все числа из среза и ищет первое число, которое удовлетворяет условию: делится на 3 без остатка, но не делится на 5
   - Использует оператор `continue` для пропуска чисел, не удовлетворяющих условию
   - Использует оператор `break` для завершения поиска, как только подходящее число найдено
   - Возвращает найденное число и его индекс или 0 и -1, если такого числа нет

## Критерии приемки:
1. Функция должна корректно находить первое число, которое делится на 3 без остатка, но не делится на 5
2. Функция должна возвращать найденное число и его индекс
3. Если такого числа нет, функция должна вернуть 0 и -1
4. Функция должна использовать операторы `continue` и `break` для решения задачи

## Пример использования:
```go
func main() {
    numbers := []int{1, 2, 3, 5, 6, 9, 10, 12, 15}
    num, index := findSpecialNumber(numbers)
    fmt.Printf("Число: %d, Индекс: %d\n", num, index)
}
```

Ожидаемый вывод:
```
Число: 3, Индекс: 2
```

В этом примере:
- 1 и 2 пропускаются, так как не делятся на 3
- 3 - первое число, которое делится на 3 (3 / 3 = 1) и не делится на 5 (3 % 5 = 3)
- Поиск прекращается, и функция возвращает 3 и его индекс 2
