# Задача: Возведение числа в степень
> **Соответствует задаче:** `/Platform/go_base/6_Циклы/6.1_Цикл_с_условием/6.1.2 ЗАДАЧА - Go base: Возведение числа в степень.md`

## Описание:
Создай функцию `power`, которая принимает два аргумента: число `x` типа `int` и степень `n` типа `int`. Функция должна возвращать результат возведения числа `x` в степень `n`. Для решения задачи используй цикл `for`.

## Что нужно сделать:
1. Создай функцию `power`, которая:
   - Принимает два аргумента: число `x` и степень `n` типа `int`.
   - Возвращает результат возведения числа `x` в степень `n`.
   - Если степень равна 0, функция должна возвращать 1 (любое число в степени 0 равно 1).
   - Для отрицательных значений степени пока не нужно обрабатывать (степень всегда положительная).

## Критерии приемки:
1. Функция должна корректно вычислять значение числа в степени.
2. Используй цикл `for` для вычисления результата.
