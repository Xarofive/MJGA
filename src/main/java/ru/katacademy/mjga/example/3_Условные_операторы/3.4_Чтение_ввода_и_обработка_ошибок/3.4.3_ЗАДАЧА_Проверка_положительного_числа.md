# Задача: Проверка положительного числа
> **Соответствует задаче:** `/Platform/go_base/5_Условные_операторы/5.4_Возврат_и_обработка_ошибок_Go/5.4.3 ЗАДАЧА - Go base: Проверка положительного числа.md`

## Описание:
Создай функцию checkPositive, которая принимает одно число n типа int и возвращает это число и возможную ошибку. Если число отрицательное, функция должна вернуть ошибку, в противном случае вернуть само число и nil.

## Что нужно сделать:
- Объяви функцию checkPositive с одним аргументом n типа int.
- Если n меньше нуля, функция должна вернуть 0 и ошибку с текстом "отрицательное число" с использованием errors.New.
- Если n не отрицательное, функция должна вернуть его и nil.

## Критерии приемки:
- Не нужно писать полную программу, только саму функцию checkPositive.
- Функция должна корректно возвращать ошибку при отрицательном значении.
- Функция должна возвращать само число, если оно не отрицательное.

## Пример:
При вызове функции с n = 5, ожидаемый результат:
```
5 и nil (ошибки нет)
```

При вызове функции с n = -3, ожидаемый результат:
```
0 и ошибка "отрицательное число"
```
