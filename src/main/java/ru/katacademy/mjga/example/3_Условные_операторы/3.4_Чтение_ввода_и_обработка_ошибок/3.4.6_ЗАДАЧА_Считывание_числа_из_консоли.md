# Задача: Считывание числа из консоли и преобразование
> **Соответствует задаче:** `/Platform/go_base/5_Условные_операторы/5.5_Стандартные_библиотеки_Go/5.5.4 ЗАДАЧА - Go base: Считывание числа из консоли и преобразование.md`

## Описание:
Создай функцию readNumberFromConsole, которая запрашивает у пользователя ввод числа через консоль, преобразует введенную строку в целое число и возвращает это число или ошибку.

## Что нужно сделать:
- Объяви функцию readNumberFromConsole без аргументов.
- Используй пакет bufio и os.Stdin для чтения строки из консоли.
- Используй пакет strconv для преобразования строки в целое число.
- Обработай все возможные ошибки:
  - Ошибку при чтении ввода
  - Ошибку при преобразовании строки в число
- Верни преобразованное число и nil в случае успешного чтения и преобразования, либо 0 и соответствующую ошибку, если произошла ошибка.

## Критерии приемки:
- Не нужно писать полную программу, только саму функцию readNumberFromConsole.
- Функция должна корректно считывать ввод пользователя, преобразовывать его в число и возвращать это число.
- Функция должна корректно обрабатывать ошибки как при чтении ввода, так и при преобразовании строки в число.

## Пример:
Если пользователь вводит "42", функция должна вернуть:
```
42 и nil (ошибки нет)
```

Если пользователь вводит "abc", функция должна вернуть:
```
0 и ошибку "не удалось преобразовать в число: strconv.Atoi: parsing "abc": invalid syntax"
```

## Подсказки:
- Для чтения ввода с новой строки можно использовать `reader.ReadString('\n')`.
- Для удаления символа новой строки из считанной строки можно использовать `strings.TrimSpace()` или другие методы пакета strings.
- Для преобразования строки в число используй `strconv.Atoi()`.
- Не забудь проверить и обработать ошибки на каждом шаге.
