# Оператор if в Go

## Введение

В программировании часто возникают ситуации, когда необходимо выполнять различные действия в зависимости от того, верно ли некоторое условие или нет. Для таких случаев в языке программирования Go используется оператор **if**.

Оператор **if** позволяет проверить условие и выполнить определенный блок кода только в случае истинности этого условия.

## Синтаксис оператора if

Общий синтаксис оператора if выглядит следующим образом:

```go
if условие {
    // блок кода, который выполняется, если условие истинно
}
```

Здесь:
- `условие` — логическое выражение, результатом которого является значение типа `bool` (истина — `true` или ложь — `false`).
- Если результат условия равен `true`, выполняется код внутри фигурных скобок `{}`.
- Если результат условия равен `false`, то этот код пропускается.

## Пример использования if

Рассмотрим пример:

```go
package main

import "fmt"

func main() {
    age := 20

    if age >= 18 {
        fmt.Println("Вы совершеннолетний.")
    }
}
```

**Вывод программы:**

```
Вы совершеннолетний.
```

В этом примере условие `age >= 18` проверяется на истинность:
- Так как переменная `age` равна `20`, условие истинно (`20 >= 18`), и программа выводит сообщение.

Если бы переменная `age` была, например, равна `16`, код внутри блока `{}` не выполнился бы, и программа не вывела бы ничего.

## Оператор if с else

Часто необходимо выполнять одно действие, если условие истинно, и другое действие, если оно ложно. В таких случаях используется расширенная форма оператора if с использованием ключевого слова `else`:

```go
if условие {
    // код, если условие истинно
} else {
    // код, если условие ложно
}
```

### Пример использования if-else

```go
package main

import "fmt"

func main() {
    age := 15

    if age >= 18 {
        fmt.Println("Вы совершеннолетний.")
    } else {
        fmt.Println("Вы еще не достигли совершеннолетия.")
    }
}
```

**Вывод программы:**

```
Вы еще не достигли совершеннолетия.
```

В этом примере, так как `age` равен `15`, выполняется второй блок (`else`).

## Вложенные if

Вы можете использовать вложенные операторы if для проверки нескольких условий одновременно:

```go
package main

import "fmt"

func main() {
    age := 25
    hasPassport := true

    if age >= 18 {
        if hasPassport {
            fmt.Println("Вы можете путешествовать за границу.")
        } else {
            fmt.Println("Вам нужен паспорт для путешествия за границу.")
        }
    } else {
        fmt.Println("Вы еще не достигли совершеннолетия для путешествия за границу.")
    }
}
```

**Вывод программы:**

```
Вы можете путешествовать за границу.
```

## Короткая форма объявления переменных в if

Go позволяет объявлять переменные непосредственно в операторе if. Эти переменные доступны только внутри этого оператора:

```go
package main

import "fmt"

func main() {
	if temp := 10; temp <= 0 {
		fmt.Println("При температуре", temp, "°C вода замерзает.")
	} else {
		fmt.Println("При температуре", temp, "°C вода в жидком состоянии.")
	}
}
```

**Вывод программы:**

```
При температуре 10 °C вода в жидком состоянии.
```

Обратите внимание, что переменная `temp` объявлена и доступна только внутри оператора if и его блока else.

## Итоги

- Оператор **if** позволяет программе выполнять разные действия в зависимости от условий.
- Использование **else** позволяет обработать альтернативный случай.
- Вложенные **if** помогают в проверке нескольких условий.
- Объявление переменных внутри условия позволяет писать более лаконичный и читаемый код.
