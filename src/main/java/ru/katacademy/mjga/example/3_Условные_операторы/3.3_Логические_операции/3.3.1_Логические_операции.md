# Логические операции в Go

В языке Go логические операции позволяют комбинировать условия для более сложных проверок и ветвлений. Основные логические операторы в Go:

## Основные логические операторы:

| Оператор | Описание                  | Пример использования      |
|----------|---------------------------|---------------------------|
| `&&`     | Логическое «И» (AND)      | `(a > 5 && a < 10)`       |
| `||`     | Логическое «ИЛИ» (OR)     | `(b == 5 || b == 10)`     |
| `!`      | Логическое «НЕ» (NOT)     | `!(c > 100)`              |

## Как работают логические операторы:

### 1. Логическое «И» (`&&`)
Оператор `&&` возвращает `true`, если оба условия одновременно истинны.

```go
package main

import "fmt"

func main() {
    temp := 15

    if temp > 0 && temp < 100 {
        fmt.Println("Температура находится в нормальном диапазоне.")
    } else {
        fmt.Println("Температура за пределами нормального диапазона.")
    }
}
```

Вывод:
```
Температура находится в нормальном диапазоне.
```

### 2. Логическое «ИЛИ» (`||`)
Оператор `||` возвращает `true`, если хотя бы одно из условий истинно.

```go
package main

import "fmt"

func main() {
    weather := "дождь"

    if weather == "дождь" || weather == "снег" {
        fmt.Println("Не забудь взять зонт!")
    } else {
        fmt.Println("Зонт не нужен.")
    }
}
```

Вывод:
```
Не забудь взять зонт!
```

### 3. Логическое «НЕ» (`!`)
Оператор `!` инвертирует условие, превращая `true` в `false` и наоборот.

```go
package main

import "fmt"

func main() {
    isWeekend := false

    if !isWeekend {
        fmt.Println("Сегодня рабочий день.")
    } else {
        fmt.Println("Сегодня выходной.")
    }
}
```

Вывод:
```
Сегодня рабочий день.
```

## Комбинация операторов

Логические операторы можно комбинировать для создания более сложных условий:

```go
package main

import "fmt"

func main() {
    hour := 9
    isWorkingDay := true

    if (hour >= 9 && hour <= 18) && isWorkingDay {
        fmt.Println("Время работать!")
    } else {
        fmt.Println("Сейчас отдых.")
    }
}
```

Вывод:
```
Время работать!
```

## Приоритет выполнения

Операторы выполняются в следующем порядке (от большего приоритета к меньшему):

1. `!` (НЕ)
2. `&&` (И)
3. `||` (ИЛИ)

Скобки `()` могут использоваться для изменения приоритета операций.

## Заключение

Логические операторы — это мощный инструмент для создания более сложных условий и ветвлений. Используя их, вы сможете писать гибкий и понятный код, эффективно решающий разнообразные задачи.
