# Задача: Проверка деления с обработкой ошибок
> **Соответствует задаче:** `/Platform/go_base/5_Условные_операторы/5.4_Возврат_и_обработка_ошибок_Go/5.4.2 ЗАДАЧА - Go base: Проверка деления с обработкой ошибок.md`

## Описание:
Создай функцию safeDivide, которая принимает два аргумента a и b типа float64 и возвращает результат деления и возможную ошибку. Функция должна возвращать ошибку, если b равно нулю, иначе она должна вернуть результат деления и nil.

## Что нужно сделать:
- Объяви функцию safeDivide с двумя аргументами a и b типа float64.
- Если b равно нулю, функция должна вернуть ошибку с текстом "деление на ноль" с использованием errors.New.
- В остальных случаях функция должна возвращать результат деления a / b и nil.

## Критерии приемки:
- Не нужно писать полную программу, только саму функцию safeDivide.
- Функция должна корректно возвращать ошибку при попытке деления на ноль.
- Функция должна возвращать результат деления при корректных входных данных.

## Пример:
При вызове функции с a = 10 и b = 2, ожидаемый результат:
```
5 и nil (ошибки нет)
```

При вызове функции с a = 10 и b = 0, ожидаемый результат:
```
0 и ошибка "деление на ноль"
```
