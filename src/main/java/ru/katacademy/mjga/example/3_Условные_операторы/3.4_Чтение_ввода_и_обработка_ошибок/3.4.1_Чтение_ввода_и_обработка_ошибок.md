# Чтение ввода и обработка ошибок в Go

## Чтение данных из консоли

В Go для чтения данных из стандартного ввода (консоли) часто используется пакет `fmt` с функциями `Scan`, `Scanf` и `Scanln`.

```go
package main

import (
    "fmt"
)

func main() {
    var name string
    fmt.Print("Введите ваше имя: ")
    fmt.Scan(&name)
    fmt.Printf("Привет, %s!\n", name)
}
```

## Преобразование строк в числа

Для преобразования строк в числа в Go используется пакет `strconv`:

```go
package main

import (
    "fmt"
    "strconv"
)

func main() {
    str := "123"
    
    // Преобразование строки в целое число
    num, err := strconv.Atoi(str)
    if err != nil {
        fmt.Println("Ошибка при преобразовании:", err)
        return
    }
    
    fmt.Printf("Преобразованное число: %d\n", num)
}
```

## Обработка ошибок

В Go ошибки представляют собой обычные значения, которые часто возвращаются в качестве последнего значения из функций:

```go
package main

import (
    "fmt"
    "strconv"
)

func main() {
    // Пример с преобразованием строки в число
    str := "abc"
    num, err := strconv.Atoi(str)
    
    // Проверка на наличие ошибки
    if err != nil {
        fmt.Println("Произошла ошибка:", err)
    } else {
        fmt.Println("Число:", num)
    }
    
    // Пример деления с проверкой на ноль
    divisor := 0
    if divisor == 0 {
        fmt.Println("Ошибка: деление на ноль невозможно")
    } else {
        result := 10 / divisor
        fmt.Println("Результат деления:", result)
    }
}
```

В Go принято всегда проверять ошибки, возвращаемые функциями. Обычно это делается с помощью условного оператора `if`, который проверяет, не является ли возвращенная ошибка `nil`.

## Чтение чисел из консоли

Комбинируя чтение из консоли и преобразование строк, можно читать числовые значения:

```go
package main

import (
    "fmt"
    "strconv"
)

func main() {
    var input string
    
    fmt.Print("Введите число: ")
    fmt.Scan(&input)
    
    num, err := strconv.Atoi(input)
    if err != nil {
        fmt.Println("Ошибка при преобразовании:", err)
        return
    }
    
    fmt.Printf("Вы ввели число: %d\n", num)
}
```
