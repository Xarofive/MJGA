# Задача: Определение типа символа
> **Соответствует задаче:** `/Platform/go_base/5_Условные_операторы/5.3_Оператор_Switch/5.3.2 ЗАДАЧА - Go base: Определение типа символа.md`

## Описание:
Создай функцию characterType, которая принимает символ char типа rune и определяет его тип: буква, цифра, пробел или специальный символ. Для этого используй оператор switch.

Использование оператора switch в этой задаче обязательно, так как необходимо проверять множество различных диапазонов символов, и if-else конструкция была бы неудобной.

## Что нужно сделать:
- Объяви функцию characterType, которая принимает символ char типа rune.
- Используй оператор switch для определения типа символа:
  - Если это буква (строчные и заглавные латинские буквы), верни "Буква".
  - Если это цифра (символы '0'–'9'), верни "Цифра".
  - Если это пробел (' '), верни "Пробел".
  - Если это любой другой символ (например, пунктуация), верни "Специальный символ".
- Верни соответствующую строку в зависимости от типа символа.

## Критерии приемки:
- Использование switch обязательно для проверки диапазонов символов.
- Не нужно писать полную программу, только саму функцию characterType.

## Пример:
При вызове characterType('a') ожидается результат: "Буква"

При вызове characterType('5') ожидается результат: "Цифра"

При вызове characterType(' ') ожидается результат: "Пробел"

При вызове characterType('!') ожидается результат: "Специальный символ"

## Подсказки:
- Используй проверку диапазонов в switch, например, case 'a' <= char && char <= 'z':.
- Для символов можно использовать руну (rune), так как это числовое представление символов.
- Это задание специально создано так, что без оператора switch решение было бы либо менее эффективным, либо избыточным.
