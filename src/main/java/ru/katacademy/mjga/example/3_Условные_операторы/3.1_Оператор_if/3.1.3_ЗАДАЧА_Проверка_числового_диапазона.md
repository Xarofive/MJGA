# Задача: Проверка числового диапазона
> **Соответствует задаче:** `/Platform/go_base/5_Условные_операторы/5.2_Условие_Если-Иначе/5.2.2 ЗАДАЧА - Go base: Проверка числового диапазона.md`

## Описание:
Создай функцию checkRange, которая принимает одно целое число n и определяет, в какой из трех диапазонов оно находится:

- Если число меньше 0, функция должна вернуть "Отрицательное".
- Если число равно 0, функция должна вернуть "Ноль".
- Если число больше 0, функция должна вернуть "Положительное".

## Требование:
Используй else в своей реализации. Решение задачи без else не должно быть возможным.

## Что нужно сделать:
- Объяви функцию checkRange, которая принимает один аргумент n типа int.
- Используй условный оператор if для проверки каждого случая.
- Используй else if и else для проверки всех вариантов.

## Критерии приемки:
- Не нужно писать полную программу, только саму функцию checkRange.
- Функция должна корректно определять диапазон числа.

## Пример:
При вызове checkRange(-5) ожидается: "Отрицательное"

При вызове checkRange(0) ожидается: "Ноль"

При вызове checkRange(10) ожидается: "Положительное"

## Примечание:
Эта задача требует использования else, потому что:
- Мы проверяем все возможные варианты: меньше, равно, и больше нуля.
- Исключить else в этом случае было бы избыточным и менее эффективно, так как все три варианта охватывают все возможные значения переменной n.
