# Задача: Математические операции с помощью функций
> **Соответствует задаче:** `/Platform/go_base/5_Условные_операторы/5.3_Оператор_Switch/5.3.3 ЗАДАЧА - Go base: Математические операции с помощью функций.md`

## Описание:
Создай функцию calculate, которая принимает три аргумента: первое число a типа int, строку operator, представляющую оператор (+, -, *, /), и второе число b типа int. Функция должна использовать оператор switch для выбора математической операции и вызвать соответствующую функцию (sum, subtract, multiply, divide) для выполнения операции. Каждая функция должна возвращать результат или ошибку.

## Что нужно сделать:
Создай четыре функции:
- sum(a, b int) int: для сложения.
- subtract(a, b int) int: для вычитания.
- multiply(a, b int) int: для умножения.
- divide(a, b int) (int, error): для деления (деление на ноль должно возвращать ошибку).

Объяви функцию calculate, которая принимает три аргумента: два числа и оператор.
Используй оператор switch в calculate для вызова соответствующей функции на основе оператора.
Верни результат или ошибку, если произошла ошибка (например, деление на ноль).

## Критерии приемки:
- Не нужно писать полную программу, только саму функцию calculate и математические функции.
- Функция должна корректно обрабатывать все четыре операции и возвращать ошибки в случае некорректного оператора или деления на ноль.

## Пример:
При вызове calculate(1, "+", 1) ожидается результат: 2, nil

При вызове calculate(6, "/", 0) ожидается ошибка: "деление на ноль"
