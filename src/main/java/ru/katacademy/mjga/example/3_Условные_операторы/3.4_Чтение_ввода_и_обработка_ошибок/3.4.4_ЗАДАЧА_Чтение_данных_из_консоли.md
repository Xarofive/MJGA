# Задача: Чтение данных из консоли
> **Соответствует задаче:** `/Platform/go_base/5_Условные_операторы/5.5_Стандартные_библиотеки_Go/5.5.2 ЗАДАЧА - Go base: Чтение данных из консоли.md`

## Описание:
Создай функцию readUserInput, которая запрашивает у пользователя ввод строки через консоль и возвращает введенную строку или ошибку.

## Что нужно сделать:
- Объяви функцию readUserInput без аргументов.
- Используй пакет bufio и os.Stdin для чтения строки из консоли.
- Обработай возможную ошибку при чтении ввода.
- Верни введенную пользователем строку без символа новой строки и nil в случае успешного чтения, либо пустую строку и ошибку, если чтение не удалось.

## Критерии приемки:
- Не нужно писать полную программу, только саму функцию readUserInput.
- Функция должна корректно считывать ввод пользователя и возвращать введенную строку.
- Функция должна корректно обрабатывать ошибки при чтении ввода.

## Пример:
Если пользователь вводит "Привет, мир!", функция должна вернуть:
```
"Привет, мир!" и nil (ошибки нет)
```

В случае ошибки функция должна вернуть:
```
"" и соответствующую ошибку
```

## Подсказки:
- Для чтения ввода с новой строки можно использовать `reader.ReadString('\n')`.
- Для удаления символа новой строки из считанной строки можно использовать `strings.TrimSpace()` или другие методы пакета strings.
