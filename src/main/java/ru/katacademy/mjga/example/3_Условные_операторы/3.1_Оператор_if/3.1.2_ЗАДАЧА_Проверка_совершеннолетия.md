# Задача: Проверка совершеннолетия
> **Соответствует задаче:** `/Platform/go_base/5_Условные_операторы/5.1_Условие_Если/5.1.2 ЗАДАЧА - Go base: Проверка совершеннолетия.md`

## Описание:
Создай функцию checkAdult, которая принимает один аргумент age типа int и с помощью оператора if проверяет, достиг ли человек совершеннолетия (18 лет и старше). Если да, функция должна возвращать строку "Совершеннолетний".

## Что нужно сделать:
- Объяви функцию checkAdult, которая принимает аргумент age типа int.
- Используй оператор if для проверки возраста.
- Если age больше или равен 18, функция должна вернуть строку "Совершеннолетний".
- Не используй else.

## Критерии приемки:
- Не нужно писать полную программу, только саму функцию checkAdult.
- Функция должна корректно обрабатывать возраст и возвращать строку "Совершеннолетний" только при выполнении условия.

## Пример:
При вызове функции с age = 20 ожидается:
```
"Совершеннолетний"
```

При вызове функции с age = 16 ничего не должно возвращаться.
