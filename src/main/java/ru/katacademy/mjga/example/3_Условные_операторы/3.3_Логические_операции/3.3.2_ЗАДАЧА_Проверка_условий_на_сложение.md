# Задача: Проверка условий на сложение
> **Соответствует разделу:** `/Platform/go_base/5_Условные_операторы/5.3_Логические_операции`

## Описание:
Создай функцию checkAddition, которая принимает три аргумента: a, b и result типа int. Функция должна проверить, равна ли сумма a и b значению result, и при этом хотя бы одно из чисел a или b должно быть положительным.

## Что нужно сделать:
- Объяви функцию checkAddition, которая принимает три аргумента: a, b и result типа int.
- Используй логические операторы && и || для проверки условий:
  - Сумма a и b должна быть равна result.
  - Хотя бы одно из чисел a или b должно быть положительным.
- Функция должна возвращать булево значение (true или false) в зависимости от результата проверки.

## Критерии приемки:
- Не нужно писать полную программу, только саму функцию checkAddition.
- Функция должна использовать логические операторы && и ||.
- Функция должна возвращать true только если оба условия выполняются.

## Пример:
При вызове checkAddition(2, 3, 5) ожидается результат: true
(сумма 2 + 3 = 5 и хотя бы одно из чисел положительное)

При вызове checkAddition(-2, 3, 1) ожидается результат: true
(сумма -2 + 3 = 1 и хотя бы одно из чисел положительное)

При вызове checkAddition(-2, -3, -5) ожидается результат: false
(сумма равна, но оба числа отрицательные)

При вызове checkAddition(2, 3, 6) ожидается результат: false
(хотя бы одно из чисел положительное, но сумма не равна result)
