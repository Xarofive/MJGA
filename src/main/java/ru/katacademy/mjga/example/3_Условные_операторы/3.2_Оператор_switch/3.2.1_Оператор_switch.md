# Оператор switch в Go

В предыдущей лекции мы разобрали оператор `if`, который позволяет принимать решения на основе условий. Однако иногда нужно проверить сразу несколько условий или вариантов значений переменной. Для таких задач лучше всего подходит оператор **`switch`**.

## Что такое оператор switch?

Оператор `switch` в языке Go используется, когда необходимо сравнить одно значение с несколькими возможными вариантами и выполнить код, соответствующий одному из них.

### Основной синтаксис

```go
switch переменная {
case значение1:
    // действия для значения1
case значение2:
    // действия для значения2
default:
    // действия по умолчанию, если ни одно значение не подошло
}
```

## Классический switch с выражением

В классическом виде оператор switch сравнивает значение выражения с каждым из предложенных case.

```go
package main

import "fmt"

func main() {
    day := 3
    
    switch day {
    case 1:
        fmt.Println("Понедельник")
    case 2:
        fmt.Println("Вторник")
    case 3:
        fmt.Println("Среда")
    case 4:
        fmt.Println("Четверг")
    case 5:
        fmt.Println("Пятница")
    case 6:
        fmt.Println("Суббота")
    case 7:
        fmt.Println("Воскресенье")
    default:
        fmt.Println("Неизвестный день недели")
    }
}
```

**Вывод программы:**
```
Среда
```

## Switch без выражения

В Go можно использовать switch без указания выражения. В этом случае, он работает как более структурированная форма if-else-if:

```go
package main

import "fmt"

func main() {
    temp := -5

    switch {
    case temp <= 0:
        fmt.Println("При температуре", temp, "°C вода замерзает.")
    case temp > 0 && temp < 100:
        fmt.Println("При температуре", temp, "°C вода в жидком состоянии.")
    case temp >= 100:
        fmt.Println("При температуре", temp, "°C вода кипит.")
    default:
        fmt.Println("Неизвестное состояние.")
    }
}
```

**Вывод программы:**
```
При температуре -5 °C вода замерзает.
```

## Особенности switch в Go

### 1. Автоматический break

В отличие от других языков программирования (например, C, C++, Java), в Go после выполнения подходящего case **не требуется** ставить оператор `break`. Go автоматически добавляет его, что предотвращает случайное выполнение следующего case блока:

```go
package main

import "fmt"

func main() {
    num := 2
    
    switch num {
    case 1:
        fmt.Println("Один")
        // Здесь не нужен break
    case 2:
        fmt.Println("Два")
        // Здесь тоже не нужен break
    case 3:
        fmt.Println("Три")
    }
}
```

**Вывод программы:**
```
Два
```

### 2. Использование fallthrough

Если вы хотите, чтобы выполнение продолжилось в следующем case (как в других языках), используйте ключевое слово `fallthrough`:

```go
package main

import "fmt"

func main() {
    num := 2
    
    switch num {
    case 1:
        fmt.Println("Один")
    case 2:
        fmt.Println("Два")
        fallthrough // Выполнение перейдет в следующий case
    case 3:
        fmt.Println("Три")
    case 4:
        fmt.Println("Четыре")
    }
}
```

**Вывод программы:**
```
Два
Три
```

Обратите внимание, что `fallthrough` всегда переходит к следующему case, игнорируя его условие.

### 3. Множественные значения в одном case

Один case может содержать несколько значений, разделенных запятыми:

```go
package main

import "fmt"

func main() {
    day := "суббота"
    
    switch day {
    case "понедельник", "вторник", "среда", "четверг", "пятница":
        fmt.Println("Рабочий день")
    case "суббота", "воскресенье":
        fmt.Println("Выходной день")
    default:
        fmt.Println("Неизвестный день")
    }
}
```

**Вывод программы:**
```
Выходной день
```

### 4. Инициализация переменных в switch

Как и в if, в switch можно инициализировать переменную перед проверкой условия:

```go
package main

import "fmt"
import "time"

func main() {
    switch currentHour := time.Now().Hour(); {
    case currentHour < 12:
        fmt.Println("Доброе утро!")
    case currentHour < 17:
        fmt.Println("Добрый день!")
    default:
        fmt.Println("Добрый вечер!")
    }
}
```

### 5. Type switch для проверки типов

В Go есть специальный вариант switch, называемый "type switch", который проверяет тип значения интерфейса:

```go
package main

import "fmt"

func printType(i interface{}) {
    switch v := i.(type) {
    case int:
        fmt.Printf("Переменная типа int: %d\n", v)
    case string:
        fmt.Printf("Переменная типа string: %s\n", v)
    case bool:
        fmt.Printf("Переменная типа bool: %t\n", v)
    default:
        fmt.Printf("Неизвестный тип: %T\n", v)
    }
}

func main() {
    printType(42)
    printType("Привет")
    printType(true)
    printType(3.14)
}
```

**Вывод программы:**
```
Переменная типа int: 42
Переменная типа string: Привет
Переменная типа bool: true
Неизвестный тип: float64
```

Type switch особенно полезен при работе с интерфейсами и обработке разных типов данных.

## Когда лучше использовать switch?

Использование оператора `switch` позволяет сделать код:
- Чётче и удобнее для восприятия.
- Удобным для сравнения множества значений одной переменной.
- Легко расширяемым и поддерживаемым.

Используйте оператор `switch`:
- Когда нужно проверить переменную на соответствие многим значениям.
- Вместо длинной цепочки if-else-if для улучшения читаемости.
- Для проверки типа переменной в интерфейсе.

## Заключение

Оператор `switch` в Go имеет ряд особенностей, которые делают его более мощным и гибким, чем в других языках:
- Автоматический break после каждого case.
- Возможность использования fallthrough для продолжения выполнения.
- Поддержка нескольких значений в одном case.
- Возможность работы без выражения (как цепочка if-else).
- Инициализация переменных в switch.
- Type switch для проверки типов интерфейсов.

Эти возможности делают оператор switch в Go мощным инструментом для создания четкой логики ветвления и обработки различных условий в вашем коде.
