# Задача: Максимум и минимум
> **Соответствует задаче:** `/Platform/go_base/5_Условные_операторы/5.1_Условие_Если/5.1.3 ЗАДАЧА - Go base: Максимум и минимум.md`

## Описание:
Напиши функцию MaxMin, которая принимает три числа типа int и возвращает два значения — максимальное и минимальное число среди них.

## Что нужно сделать:
- Создай функцию MaxMin, которая принимает три числа a, b, и c.
- Определи максимальное и минимальное числа среди трех.
- Функция должна возвращать два значения: максимальное и минимальное.

## Критерии приемки:
- Не нужно писать полную программу, только саму функцию MaxMin.
- Функция должна принимать три числа и возвращать два значения — максимальное и минимальное число.
- Функция должна корректно определять максимальное и минимальное число среди трех.

## Пример:
При вызове функции MaxMin(3, 1, 2) ожидается:
```
3, 1
```

Пример использования в main:
```go
func main() {
    max, min := MaxMin(3, 1, 2)
    fmt.Println("Max:", max)
    fmt.Println("Min:", min)
}
```

Пример вывода:
```
Max: 3
Min: 1
```
