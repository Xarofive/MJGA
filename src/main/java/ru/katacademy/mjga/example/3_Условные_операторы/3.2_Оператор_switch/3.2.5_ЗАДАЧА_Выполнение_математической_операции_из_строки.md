# Задача: Выполнение математической операции из строки
> **Соответствует задаче:** `/Platform/go_base/5_Условные_операторы/5.3_Оператор_Switch/5.3.5 ЗАДАЧА - Go base: Выполнение математической операции из строки.md`

## Описание:
Создай функцию calculate, которая принимает строку с математическим выражением (например, "1 + 1", "3 - 1", или "6 / 2"). Функция должна выполнить следующие шаги:

1. Разбить строку на три части: первое число, оператор, и второе число, используя strings.Fields.
2. Преобразовать числа из строк в тип int с помощью strconv.Atoi. Если первое или второе число некорректное, функция должна вернуть соответствующую ошибку.
3. Выполнить соответствующую операцию в зависимости от оператора (+, -, *, /). Если оператор неверный, функция должна вернуть ошибку.
4. Если операция — деление, и второе число равно нулю, вернуть ошибку "деление на ноль".
5. Вернуть результат операции или ошибку, если произошла ошибка (некорректные числа или оператор).

## Что нужно сделать:
- Объяви функцию calculate, которая принимает строку expression типа string.
- Используй strings.Fields для разделения строки на три части: числа и оператор.
- Преобразуй строки чисел в целые числа с помощью strconv.Atoi.
- Если оператор неверный или операция деления производится на ноль, верни соответствующую ошибку.
- Выполни математическую операцию и верни результат или ошибку в случае некорректного ввода.

## Критерии приемки:
- Функция должна корректно обрабатывать все четыре операции (+, -, *, /) и возвращать ошибки в случае некорректного оператора, неверного числа или деления на ноль.
- Возвращаемая ошибка должна содержать текст, объясняющий, что пошло не так: "некорректное первое число", "некорректное второе число", "некорректный оператор", "деление на ноль".

## Пример:
При вызове calculate("1 + 1") ожидается результат: 2, nil

При вызове calculate("3 - 1") ожидается результат: 2, nil

При вызове calculate("6 / 0") ожидается ошибка: "деление на ноль"

При вызове calculate("abc + 1") ожидается ошибка: "некорректное первое число"

При вызове calculate("1 + abc") ожидается ошибка: "некорректное второе число"

При вызове calculate("1 # 1") ожидается ошибка: "некорректный оператор"

## Подсказки:
- Используй пакет strings для работы со строками (разделение на части).
- Используй пакет strconv для преобразования строк в числа.
