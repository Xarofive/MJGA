# Задача: Подсчет частоты встречаемости слов

## Цель

Написать функцию `CountWordFrequency`, которая принимает срез строк и возвращает карту, где ключами являются слова, а значениями — количество их появлений в исходном срезе.

## Описание

Создайте функцию с сигнатурой:

```go
func CountWordFrequency(words []string) map[string]int
```

Функция должна:
1. Принимать срез строк `words`, содержащий различные слова.
2. Возвращать карту (`map[string]int`), где:
   - Ключами являются уникальные слова из среза.
   - Значениями являются количества появлений каждого слова в исходном срезе.

## Пример использования

```go
package main

import "fmt"

func main() {
    words := []string{"apple", "banana", "apple", "orange", "banana", "apple"}
    frequency := CountWordFrequency(words)
    
    for word, count := range frequency {
        fmt.Printf("%s: %d\n", word, count)
    }
    // Ожидаемый вывод (порядок может отличаться):
    // apple: 3
    // banana: 2
    // orange: 1
}
```

## Советы по решению

* Создайте пустую карту для хранения результатов.
* Используйте цикл для перебора всех слов в исходном срезе.
* Для каждого слова увеличивайте соответствующее значение в карте.
* Если слово встречается впервые, добавьте его в карту со значением 1.

## Что оценивается

* Правильность работы функции для различных входных данных.
* Корректное создание и заполнение карты.
* Обработка повторяющихся слов.
* Обработка пустого среза.
* Соответствие сигнатуры функции заданию.
