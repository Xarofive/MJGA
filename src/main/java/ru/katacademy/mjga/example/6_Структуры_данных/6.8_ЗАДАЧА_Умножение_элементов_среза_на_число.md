# Задача: Умножение элементов среза на число

## Цель

Написать функцию `MultiplySlice`, которая принимает срез целых чисел и множитель, и возвращает новый срез, в котором каждый элемент умножен на этот множитель.

## Описание

Создайте функцию с сигнатурой:

```go
func MultiplySlice(numbers []int, multiplier int) []int
```

Функция должна:
1. Принимать срез целых чисел `numbers` и целое число `multiplier`.
2. Возвращать новый срез, где каждый элемент исходного среза умножен на `multiplier`.
3. Корректно обрабатывать пустые срезы (возвращать пустой срез).

## Пример использования

```go
package main

import "fmt"

func main() {
    numbers := []int{1, 2, 3, 4, 5}
    multiplied := MultiplySlice(numbers, 2)
    fmt.Println("Результат умножения:", multiplied) // Вывод: [2 4 6 8 10]
    
    // Пример с пустым срезом
    emptySlice := []int{}
    result := MultiplySlice(emptySlice, 3)
    fmt.Println("Результат для пустого среза:", result) // Вывод: []
}
```

## Советы по решению

* Создайте новый срез, чтобы не изменять исходный.
* Используйте цикл для перебора всех элементов в исходном срезе.
* Умножайте каждый элемент на указанный множитель и добавляйте в новый срез.
* Не забудьте о граничных случаях, таких как пустой срез.

## Что оценивается

* Правильность работы функции для различных входных данных.
* Корректное создание и заполнение нового среза.
* Обработка пустого среза.
* Соответствие сигнатуры функции заданию.
