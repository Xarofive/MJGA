# Задача: Фильтрация слов в срезах

## Цель

Написать функцию `FilterWords`, которая принимает срез строк и целочисленное значение длины, и возвращает новый срез, содержащий только те слова, которые длиннее указанной длины.

## Описание

Создайте функцию с сигнатурой:

```go
func FilterWords(words []string, length int) []string
```

Функция должна:
1. Принимать срез строк `words` и целое число `length`.
2. Возвращать новый срез, содержащий только те строки из исходного среза, длина которых строго больше указанного параметра `length`.

## Пример использования

```go
package main

import "fmt"

func main() {
    words := []string{"apple", "banana", "kiwi", "orange", "grape", "pineapple"}
    
    // Фильтрация слов длиннее 5 символов
    longWords := FilterWords(words, 5)
    fmt.Println(longWords) // Вывод: [banana orange pineapple]
    
    // Фильтрация слов длиннее 6 символов
    veryLongWords := FilterWords(words, 6)
    fmt.Println(veryLongWords) // Вывод: [pineapple]
}
```

## Советы по решению

* Используйте цикл для перебора всех слов в исходном срезе.
* Для каждого слова проверяйте его длину с помощью функции `len()`.
* Если длина слова больше указанного параметра, добавляйте это слово в результирующий срез.
* Используйте функцию `append()` для добавления элементов в результирующий срез.

## Что оценивается

* Правильность работы функции для различных входных данных.
* Корректное создание и заполнение нового среза.
* Соответствие сигнатуры функции заданию.
