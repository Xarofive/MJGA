# Задача: Уникальные слова в срезе

## Цель

Написать функцию `UniqueWords`, которая принимает срез строк и возвращает срез, содержащий только уникальные слова из исходного среза в порядке их первого появления.

## Описание

Создайте функцию со следующей сигнатурой:

```go
func UniqueWords(words []string) []string
```

Функция должна:
1. Принимать срез строк `words`.
2. Возвращать новый срез, содержащий только уникальные слова из исходного среза, в порядке их первого появления.
3. Использовать карту (map) для эффективного отслеживания повторяющихся слов.

## Пример использования

```go
package main

import "fmt"

func main() {
    words := []string{"apple", "banana", "apple", "orange", "banana", "grape"}
    uniqueWords := UniqueWords(words)
    fmt.Println("Уникальные слова:", uniqueWords) // Вывод: [apple banana orange grape]
    
    // Пример с пустым срезом
    emptySlice := []string{}
    result := UniqueWords(emptySlice)
    fmt.Println("Результат для пустого среза:", result) // Вывод: []
}
```

## Советы по решению

* Создайте пустую карту для отслеживания слов, которые уже встречались.
* Создайте пустой срез для хранения результата.
* Пройдитесь по исходному срезу, проверяя каждое слово:
  * Если слово еще не встречалось (его нет в карте), добавьте его в результирующий срез и отметьте в карте.
  * Если слово уже встречалось, пропустите его.
* Возвратите результирующий срез.

## Что оценивается

* Правильность работы функции для различных входных данных.
* Сохранение порядка первого появления элементов.
* Корректное создание и заполнение карты и результирующего среза.
* Обработка пустого среза и среза с повторяющимися элементами.
* Соответствие сигнатуры функции заданию.
