# Массивы в Go

## Что такое массив?

Массив в языке Go — это упорядоченный набор элементов одного типа, который имеет фиксированную длину. Размер массива определяется при его создании и не может быть изменён в процессе работы программы.

Массивы полезны, когда заранее известно, сколько именно элементов будет храниться.

---

## Создание массива

Общий синтаксис объявления массива:

```go
var имя_массива [размер]тип_данных
```

### Примеры создания массивов:

```go
var numbers [5]int               // массив из 5 целых чисел
var names [3]string              // массив из 3 строк
var flags [4]bool                // массив из 4 булевых значений
```

Можно сразу инициализировать массив значениями:

```go
var nums = [3]int{1, 2, 3}       // инициализация массива числами
var letters = [4]string{"a", "b", "c", "d"}
```

Размер массива может быть выведен автоматически при инициализации:

```go
var nums = [...]int{1, 2, 3, 4}  // размер будет определён автоматически (4 элемента)
```

---

## Доступ к элементам массива

Элементы массива доступны по индексу. Индексация начинается с 0:

```go
var numbers = [3]int{10, 20, 30}
fmt.Println(numbers[0]) // вывод: 10
fmt.Println(numbers[2]) // вывод: 30
```

Можно также изменять элементы:

```go
numbers[1] = 50
fmt.Println(numbers) // вывод: [10 50 30]
```

---

## Длина массива

Узнать длину массива можно с помощью встроенной функции `len()`:

```go
fmt.Println(len(numbers)) // вывод: 3
```

---

## Итерация по элементам массива

Для перебора элементов массива используется цикл `for`:

```go
var nums = [5]int{1, 2, 3, 4, 5}

for i := 0; i < len(nums); i++ {
    fmt.Println(nums[i])
}
```

Или удобная форма с использованием `range`:

```go
for index, value := range nums {
    fmt.Println("Индекс", index, "значение", value)
}
```

## Важные особенности массивов

- Размер массива неизменяем после объявления.
- Массивы передаются в функции по значению (создаётся копия массива). Это важно учитывать, так как большие массивы могут замедлить программу. Для передачи по ссылке используют указатели или срезы.

Пример передачи массива по значению:

```go
func modifyArray(arr [3]int) {
    arr[0] = 100
}

func main() {
    nums := [3]int{1, 2, 3}
    modifyArray(nums)
    fmt.Println(nums) // вывод: [1 2 3], не изменился
}
```

Если нужно изменить исходный массив, используют указатель:

```go
func modifyArray(arr *[3]int) {
    arr[0] = 100
}

func main() {
    nums := [3]int{1, 2, 3}
    modifyArray(&nums)
    fmt.Println(nums) // вывод: [100 2 3], массив изменён
}
```

## Когда использовать массивы

Используйте массивы:
- Когда нужно хранить фиксированное количество данных.
- Когда важна производительность и низкие затраты памяти.

Однако в большинстве ситуаций, когда заранее неизвестен размер набора данных или нужна гибкость, лучше использовать срезы (`slices`), о которых мы поговорим в следующей лекции.
