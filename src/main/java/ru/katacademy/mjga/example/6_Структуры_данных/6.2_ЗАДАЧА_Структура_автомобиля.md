# Задача: Структура автомобиля

## Цель

Создать структуру `Car`, представляющую автомобиль, и функцию для расчета стоимости автомобиля с учетом дополнительных опций.

## Описание

1. Создайте структуру `Car` со следующими полями:
   - `Make` (марка, тип `string`)
   - `Model` (модель, тип `string`)
   - `Year` (год выпуска, тип `int`)
   - `BasePrice` (базовая стоимость, тип `float64`)
   - `Options` (список дополнительных опций, тип `[]string`)

2. Реализуйте метод `TotalPrice` для структуры `Car`, который рассчитывает полную стоимость автомобиля с учетом дополнительных опций. Каждая дополнительная опция добавляет к базовой стоимости 5%.

## Пример использования

```go
package main

import "fmt"

func main() {
    car := Car{
        Make:      "Toyota",
        Model:     "Camry",
        Year:      2023,
        BasePrice: 25000.0,
        Options:   []string{"Кожаный салон", "Панорамная крыша", "Премиум аудио"},
    }

    totalPrice := car.TotalPrice()
    fmt.Printf("Автомобиль: %s %s %d\n", car.Make, car.Model, car.Year)
    fmt.Printf("Базовая стоимость: $%.2f\n", car.BasePrice)
    fmt.Printf("Опции: %v\n", car.Options)
    fmt.Printf("Полная стоимость: $%.2f\n", totalPrice)
}
```

## Ожидаемый вывод

```
Автомобиль: Toyota Camry 2023
Базовая стоимость: $25000.00
Опции: [Кожаный салон Панорамная крыша Премиум аудио]
Полная стоимость: $28750.00
```

## Советы по решению

* Создайте структуру `Car` с указанными полями.
* Реализуйте метод `TotalPrice`, который вычисляет полную стоимость автомобиля: базовая стоимость + 5% за каждую опцию.
* Для метода `TotalPrice` используйте следующую формулу: `BasePrice * (1 + 0.05 * len(Options))`.

## Что оценивается

* Правильное объявление структуры с требуемыми полями.
* Корректная реализация метода расчета полной стоимости.
* Правильная работа с дополнительными опциями.
* Точность расчетов.
