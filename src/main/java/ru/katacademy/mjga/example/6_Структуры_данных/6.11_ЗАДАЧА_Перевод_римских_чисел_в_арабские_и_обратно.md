# Задача: Перевод римских чисел в арабские и обратно

## Цель

Написать две функции:
1. `RomanToInt` - для перевода римских чисел в арабские
2. `IntToRoman` - для перевода арабских чисел в римские

Обе функции должны работать с числами от 1 до 10.

## Описание

Создайте функции со следующими сигнатурами:

```go
func RomanToInt(roman string) int
func IntToRoman(num int) string
```

Функции должны:
1. `RomanToInt` - принимать строку, представляющую римское число от 1 до 10, и возвращать соответствующее целое число.
2. `IntToRoman` - принимать целое число от 1 до 10 и возвращать строку, представляющую соответствующее римское число.

Для реализации используйте карты (map) для сопоставления римских и арабских чисел.

## Римские цифры

Для этой задачи вам понадобятся следующие римские цифры:
- I = 1
- V = 5
- X = 10

Особые случаи:
- IV = 4 (5-1)
- IX = 9 (10-1)

## Пример использования

```go
package main

import "fmt"

func main() {
    // Пример использования функции RomanToInt
    roman := "IX"
    number := RomanToInt(roman)
    fmt.Println("Римское число", roman, "равно", number) // Вывод: Римское число IX равно 9
    
    // Пример использования функции IntToRoman
    num := 5
    romanNumber := IntToRoman(num)
    fmt.Println("Число", num, "в римской записи:", romanNumber) // Вывод: Число 5 в римской записи: V
}
```

## Советы по решению

* Для функции `RomanToInt`:
  * Создайте карту для сопоставления римских цифр с их числовыми значениями.
  * Учитывайте правила сложения и вычитания в римской системе счисления (например, IV = 4).
  * Проверяйте каждый символ и его отношение к следующему символу (если он существует).

* Для функции `IntToRoman`:
  * Создайте карту или массив для сопоставления арабских чисел с их римскими эквивалентами.
  * Учтите, что некоторые числа представляются комбинацией символов (например, 4 = IV).

## Что оценивается

* Правильность работы функций для всех чисел от 1 до 10.
* Корректное использование карт для сопоставления.
* Обработка специальных случаев (IV, IX).
* Соответствие сигнатур функций заданию.
