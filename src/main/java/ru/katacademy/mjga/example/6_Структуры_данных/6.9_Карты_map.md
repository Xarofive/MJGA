# Карты (map) в Go

## Что такое map?

**Карта (map)** — это структура данных, которая позволяет хранить пары ключ-значение. Значения хранятся по ключам, и по ключу можно быстро получить соответствующее значение.

### Особенности карт в Go:

- Все ключи должны быть уникальными.
- Тип ключей должен быть сравниваемым (`comparable`). Например, допустимы ключи типа `string`, `int`, но недопустимы ключи типа `slice`.
- Значения могут быть любого типа, включая структуры и даже другие карты.

## Создание карт

Карты создаются с помощью встроенной функции `make` или через литерал карты.

### Создание с помощью `make`:

```go
// создание пустой карты с ключами типа string и значениями типа int
ages := make(map[string]int)

// добавление элементов
ages["Alice"] = 25
ages["Bob"] = 30
```

### Создание с помощью литерала карты:

```go
ages := map[string]int{
    "Alice": 25,
    "Bob":   30,
}
```

## Операции с картами

### Добавление и изменение элементов

```go
ages["Charlie"] = 35 // добавление
ages["Alice"] = 26   // изменение
```

### Получение элементов

```go
age := ages["Alice"] // получение значения по ключу
fmt.Println("Возраст Alice:", age)
```

### Проверка наличия ключа

При получении значения по ключу можно проверить, существует ли ключ:

```go
age, ok := ages["Bob"]
if ok {
    fmt.Println("Возраст Bob:", age)
} else {
    fmt.Println("Ключ не найден")
}
```

### Удаление элементов

```go
delete(ages, "Charlie")
```

## Перебор элементов карты

Используем цикл `range`:

```go
for key, value := range ages {
    fmt.Printf("%s: %d\n", key, value)
}
```

## Пример использования карт

```go
package main

import "fmt"

func main() {
    // создаём карту
    countries := map[string]string{
        "RU": "Россия",
        "KZ": "Казахстан",
        "US": "США",
    }

    // добавляем новую страну
    countries["CN"] = "Китай"

    // перебираем элементы карты
    for code, country := range countries {
        fmt.Printf("Код: %s, Страна: %s\n", code, country)
    }

    // удаляем элемент по ключу
    delete(countries, "US")

    fmt.Println("После удаления США:")
    for code, country := range countries {
        fmt.Printf("Код: %s, Страна: %s\n", code, country)
    }
}
```

### Вывод:
```
Код: RU, Страна: Россия
Код: KZ, Страна: Казахстан
Код: US, Страна: США
Код: CN, Страна: Китай
После удаления США:
Код: RU, Страна: Россия
Код: KZ, Страна: Казахстан
Код: CN, Страна: Китай
```

## Вложенные карты

В Go можно создавать вложенные карты для более сложных структур данных:

```go
package main

import "fmt"

func main() {
    // Карта, в которой значения - тоже карты
    users := map[string]map[string]string{
        "user1": {
            "name":  "Иван",
            "email": "ivan@example.com",
        },
        "user2": {
            "name":  "Мария",
            "email": "maria@example.com",
        },
    }

    // Доступ к вложенным данным
    fmt.Println("Имя user1:", users["user1"]["name"])
    fmt.Println("Email user2:", users["user2"]["email"])
}
```

## Карты как множества

Карты можно использовать для реализации множеств, где важно только наличие элемента:

```go
package main

import "fmt"

func main() {
    // Используем карту как множество
    set := map[string]bool{
        "apple":  true,
        "banana": true,
        "orange": true,
    }

    // Проверка наличия элемента в множестве
    if set["apple"] {
        fmt.Println("Apple входит в множество")
    }

    // Добавление элемента в множество
    set["grape"] = true

    // Удаление элемента из множества
    delete(set, "orange")

    // Перечисление элементов множества
    fmt.Println("Элементы множества:")
    for fruit := range set {
        fmt.Println(fruit)
    }
}
```

## Когда использовать карты?

Карты полезны, когда нужно:
- Быстро получать доступ к данным по ключу.
- Хранить данные с уникальными ключами.
- Подсчитывать частоту встречаемости элементов.

## Итоги

- Карты — структура данных для хранения пар ключ-значение.
- Используйте `make` или литерал для создания карт.
- Проверяйте наличие ключа, используя второе возвращаемое значение при получении элемента.
- Удаляйте элементы с помощью функции `delete()`.
- Карты можно использовать для реализации более сложных структур данных и алгоритмов.
