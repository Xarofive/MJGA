# Задача: Поиск уникальных чисел в срезе

## Цель

Написать функцию `UniqueNumbers`, которая принимает срез целых чисел и возвращает срез, содержащий только уникальные элементы из исходного среза в порядке их первого появления.

## Описание

Создайте функцию со следующей сигнатурой:

```go
func UniqueNumbers(numbers []int) []int
```

Функция должна:
1. Принимать срез целых чисел `numbers`.
2. Возвращать новый срез, содержащий только уникальные числа из исходного среза, в порядке их первого появления.
3. Использовать карту (map) для эффективного отслеживания повторяющихся чисел.

## Пример использования

```go
package main

import "fmt"

func main() {
    numbers := []int{1, 2, 3, 2, 1, 4, 5, 5}
    uniqueNumbers := UniqueNumbers(numbers)
    fmt.Println("Уникальные числа:", uniqueNumbers) // Вывод: [1 2 3 4 5]
    
    // Пример с пустым срезом
    emptySlice := []int{}
    result := UniqueNumbers(emptySlice)
    fmt.Println("Результат для пустого среза:", result) // Вывод: []
}
```

## Советы по решению

* Создайте пустую карту для отслеживания чисел, которые уже встречались.
* Создайте пустой срез для хранения результата.
* Пройдитесь по исходному срезу, проверяя каждое число:
  * Если число еще не встречалось (его нет в карте), добавьте его в результирующий срез и отметьте в карте.
  * Если число уже встречалось, пропустите его.
* Возвратите результирующий срез.

## Что оценивается

* Правильность работы функции для различных входных данных.
* Сохранение порядка первого появления элементов.
* Корректное создание и заполнение карты и результирующего среза.
* Обработка пустого среза и среза с повторяющимися элементами.
* Соответствие сигнатуры функции заданию.
