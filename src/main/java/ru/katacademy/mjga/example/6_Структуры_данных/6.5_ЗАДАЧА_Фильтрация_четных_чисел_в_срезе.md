# Задача: Фильтрация четных чисел в срезе
> **Соответствует задаче:** `/Platform/go_base/7_Структуры_данных/7.2_Срезы/7.2.2 ЗАДАЧА - Go base: Фильтрация четных чисел в срезе.md`

## Описание:
Напиши функцию `FilterEvenNumbers`, которая принимает срез целых чисел и возвращает новый срез, содержащий только четные числа из исходного среза.
Для определения четности числа используй оператор `%` (остаток от деления).

## Что нужно сделать:
1. Создай функцию `FilterEvenNumbers`, которая принимает срез целых чисел.
2. Используй цикл для итерации по элементам среза.
3. Если элемент четный, добавь его в новый срез.
4. Верни новый срез, содержащий только четные числа.

## Критерии приемки:
1. **Функция `FilterEvenNumbers`:** Проходит по каждому элементу исходного среза, проверяя, является ли число четным (условие `num%2 == 0`). Четные числа добавляются в новый срез с помощью функции `append`.
2. **Автопроверка:** Тестирует функцию на нескольких входных данных, проверяя, что функция правильно фильтрует и возвращает только четные числа.
