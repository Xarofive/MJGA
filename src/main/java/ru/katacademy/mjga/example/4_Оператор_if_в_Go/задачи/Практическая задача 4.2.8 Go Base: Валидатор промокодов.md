<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🎟️ Валидатор промокодов</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>

<p style="margin: 0;">Научиться использовать короткую форму условного оператора if с инициализацией и комбинировать несколько условий для валидации данных.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">💼 Контекст и мотивация</h3>

<p style="margin-bottom: 10px;">Промокоды &mdash; важный инструмент маркетинга в e-commerce, позволяющий привлекать новых клиентов и удерживать существующих. Правильная валидация промокодов предотвращает мошенничество, обеспечивает корректное применение скидок и защищает бизнес от финансовых потерь. Система должна учитывать формат кода, членство пользователя и другие бизнес-правила.</p>

<p style="margin: 0;"><strong>Задача:</strong> создать функцию для проверки статуса промокода с учётом членства пользователя.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>

<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Короткая форма if с несколькими условиями</h4>

<p>Короткая форма if позволяет вычислить значение один раз и использовать его в нескольких проверках:</p>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code class="language-go">if n := len(code); n == 0 {
    // код пустой
} else if n != 6 {
    // код неверной длины
} else {
    // длина правильная, дальнейшие проверки
}</code></pre>

<p style="margin-bottom: 0;">Переменная <code>n</code> видна во всей цепочке <code>if/else if/else</code>, но не снаружи.</p>
</div>

<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Сравнение строк в Go</h4>

<p>В Go строки можно сравнивать оператором <code>==</code>, так же как числа:</p>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code class="language-go">if name == "Alice" {
    fmt.Println("Привет, Алиса!")
}
if code != "" {
    fmt.Println("Код не пустой")
}</code></pre>

<p style="margin-bottom: 0;">Сравнение регистрозависимое: <code>&quot;Hello&quot; != &quot;hello&quot;</code>.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;"><strong>Создайте функцию для проверки промокода:</strong></p>

<h4 style="color: #e65100; margin-top: 16px;">🔧 Функция promoStatus</h4>

<p><strong>Сигнатура:</strong></p>

<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>func promoStatus(code string, isMember bool) string</code></div>

<p><strong>Параметры:</strong></p>

<ul style="margin: 10px 0;">
	<li><code>code</code> (string) &mdash; промокод для проверки</li>
	<li><code>isMember</code> (bool) &mdash; статус членства пользователя (<code>true</code> &mdash; участник программы)</li>
</ul>

<p><strong>Правила валидации (в порядке проверки):</strong></p>

<ol style="margin: 10px 0;">
	<li><strong>&quot;missing&quot;</strong> &mdash; код пустой (длина 0)</li>
	<li><strong>&quot;invalid length&quot;</strong> &mdash; длина кода не равна 6 символам</li>
	<li><strong>&quot;members only&quot;</strong> &mdash; код &quot;GOSALE&quot;, но пользователь не участник</li>
	<li><strong>&quot;accepted&quot;</strong> &mdash; код прошёл все проверки</li>
</ol>

<p><strong>Требования к реализации:</strong></p>

<ul style="margin: 10px 0;">
	<li>Обязательно использовать короткую форму <code>if</code> с инициализацией</li>
	<li>Промокоды передаются в верхнем регистре без пробелов &mdash; дополнительная обработка не нужна</li>
	<li>Проверять условия в указанном порядке</li>
	<li>Запрещено использовать циклы (<code>for</code>) и оператор <code>switch</code></li>
</ul>

<h4 style="color: #e65100; margin-top: 16px;">📝 Формат решения</h4>

<ul style="margin: 10px 0;">
	<li>Включите в ответ полную сигнатуру функции, например <code>func promoStatus(code string, isMember bool) string { ... }</code></li>
	<li>НЕ указывайте <code>package main</code></li>
	<li>НЕ добавляйте <code>import</code> директивы</li>
</ul>
</div>

<h4 style="color: #c62828; margin-top: 0;">⚠️ Частые ошибки и антипаттерны</h4>

<p><strong>Ошибка 1: Неиспользование короткой формы if</strong><br />
❌ <code>n := len(code); if n == 0 {...}</code><br />
✅ <code>if n := len(code); n == 0 {...}</code></p>

<p><strong>Ошибка 2: Неверный порядок проверок</strong><br />
❌ Проверка GOSALE до проверки длины<br />
✅ Порядок: пустота &rarr; длина &rarr; GOSALE &rarr; accepted</p>

<p><strong>Ошибка 3: Проверка GOSALE без учёта isMember</strong><br />
❌ Возвращать &quot;members only&quot; для всех кодов<br />
✅ Только код &quot;GOSALE&quot; при <code>!isMember</code> возвращает &quot;members only&quot;</p>

<p style="margin-bottom: 0;"><strong>Ошибка 4: Опечатки в возвращаемых строках</strong><br />
❌ <code>&quot;Missing&quot;</code>, <code>&quot;invalid&quot;</code>, <code>&quot;member only&quot;</code><br />
✅ Точные значения: <code>&quot;missing&quot;</code>, <code>&quot;invalid length&quot;</code>, <code>&quot;members only&quot;</code>, <code>&quot;accepted&quot;</code></p>
</div>

<div style="background: #fff8e1; border-left: 4px solid #ffc107; padding: 16px; margin: 16px 0;">
<h4 style="color: #f57c00; margin-top: 0;">💡 Подсказка</h4>

<p>Используйте короткую форму if, чтобы вычислить длину кода один раз: <code>if n := len(code); n == 0 {...}</code>. Переменная <code>n</code> будет видна во всей цепочке <code>else if</code>. Каждая проверка отсеивает невалидные случаи, и только коды, прошедшие все фильтры, считаются принятыми. Проверку на GOSALE делайте после проверки длины &mdash; к этому моменту вы уже знаете, что код ровно из 6 символов.</p>
