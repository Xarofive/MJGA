# Задача: Вычитание двух чисел
> **Соответствует задаче:** `/Platform/go_base/3_Переменные/3.3_Операции_с_типами/3.3.3 ЗАДАЧА - Go base: Вычитание двух чисел.md`

## Описание:
Твоя задача — создать программу, которая объявляет две переменные с определенными типами и затем выводит результат вычитания одной переменной из другой.

## Что нужно сделать:
- Объяви две переменные a и b типа int.
- Объяви переменную c типа int, которая будет хранить результат вычитания b из a.
- Выведи значение c на экран с помощью fmt.Println.

## Ожидаемый вывод:
Если a = 10 и b = 4, то программа должна вывести: 6

## Критерии приемки:
- Переменные должны быть объявлены строго с указанием типа int.
- Задай переменным значения a = 10 и b = 4.
- Используй fmt.Println для вывода результата.
- Программа должна быть полной, включающей package main, import "fmt" и функцию main().
