# Объявление переменных в Go

## Введение

Переменные — это фундаментальная концепция любого языка программирования, включая Go. Переменные позволяют сохранять и использовать различные данные в программе, такие как числа, строки или булевые значения.

В этой лекции ты научишься объявлять переменные двумя основными способами в языке Go и узнаешь, какой из способов лучше использовать в различных ситуациях.

## Объявление переменных с помощью var

Самый базовый и универсальный способ объявления переменных в Go — это использование ключевого слова `var`. Синтаксис выглядит следующим образом:

```go
var имя_переменной тип = значение
```

### Пример объявления переменных с использованием var:

```go
var age int = 25
var name string = "John"
var height float64 = 1.75
```

Если значение при объявлении не указано, переменная инициализируется значением по умолчанию:

```go
var count int // по умолчанию будет 0
var message string // пустая строка ""
```

## Короткая декларация переменных через :=

В Go также существует удобный способ объявления и инициализации переменных — короткая декларация с помощью оператора `:=`. Она позволяет объявлять переменные без явного указания типа, так как тип выводится автоматически из присваиваемого значения.

### Синтаксис:

```go
имя_переменной := значение
```

### Пример короткой декларации переменных:

```go
age := 30
name := "Иван"
isActive := true
```

Обрати внимание, что короткую декларацию переменных можно использовать только внутри функций.

## Логическое именование переменных

Очень важно давать переменным осмысленные названия, отражающие их предназначение в программе. Это помогает легко читать и понимать код. Например:

Плохо:

```go
var x int = 25
var y string = "Россия"
```

Хорошо:

```go
var age int = 25
var country string = "Казахстан"
```

Использование логически корректных имен переменных упрощает чтение кода, помогает избежать ошибок и делает код более профессиональным.

## Итог

В Go есть два основных способа объявления переменных:

- Использование `var` для общего объявления.
- Использование короткой декларации `:=` для краткого объявления внутри функций.

Всегда выбирай осмысленные названия, чтобы код был понятным и удобным для работы и сопровождения.
