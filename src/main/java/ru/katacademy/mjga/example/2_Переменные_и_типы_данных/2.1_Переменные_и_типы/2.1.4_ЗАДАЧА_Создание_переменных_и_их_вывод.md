# Задача: Создание переменных и их вывод
> **Соответствует задаче:** `/Platform/go_base/3_Переменные/3.2_Типы_данных/3.2.4 ЗАДАЧА - Go base: Создание переменных и их вывод.md`

## Описание:
Тебе нужно написать полную программу на Go, начиная с package main, объявив переменные с использованием ключевого слова var. После объявления переменных, ты должен вывести их значения в функции main().

## Переменные для объявления:
- Возраст: age (целое число — int) = 25.
- Имя: name (строка — string) = "John".
- Рост: height (с плавающей точкой — float64) = 1.75.
- Активный пользователь: isActive (булевое значение — bool) = true.
- Баланс: balance (с плавающей точкой — float64) = 1000.50.
- Год рождения: birthYear (целое число — int) = 1995.
- Страна: country (строка — string) = "Kazakhstan".
- Температура: temperature (с плавающей точкой — float64) = 36.6.
- Количество друзей: friendsCount (целое число — int) = 150.
- Зарегистрирован: registeredAt (строка для даты — string) = "2024-09-01".

## Ожидаемый вывод:
Программа должна вывести следующие значения:

```
25
John
1.75
true
1000.50
1995
Kazakhstan
36.6
150
2024-09-01
```

## Критерии приемки:
- Программа должна начинаться с package main.
- Переменные должны быть объявлены с использованием var.
- Программа должна выводить значения переменных в функции main().
- Порядок вывода и значения должны быть точно такими же, как указано выше.
