<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">Создание переменных и их вывод</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Что нужно сделать</h3>

<p style="margin: 0;">Напишите полную программу на Go, начиная с <code>package main</code>. Объявите переменные с использованием <code>var</code> и выведите их значения в <code>main()</code>, по одному значению на каждой новой строке.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">Переменные для объявления</h2>

<ul>
	<li><strong>age</strong> &mdash; <code>int</code> = <code>25</code></li>
	<li><strong>name</strong> &mdash; <code>string</code> = <code>&quot;John&quot;</code></li>
	<li><strong>height</strong> &mdash; <code>float64</code> = <code>1.75</code></li>
	<li><strong>isActive</strong> &mdash; <code>bool</code> = <code>true</code></li>
	<li><strong>balance</strong> &mdash; <code>float64</code> = <code>1000.50</code></li>
	<li><strong>birthYear</strong> &mdash; <code>int</code> = <code>1995</code></li>
	<li><strong>country</strong> &mdash; <code>string</code> = <code>&quot;Kazakhstan&quot;</code></li>
	<li><strong>temperature</strong> &mdash; <code>float64</code> = <code>36.6</code></li>
	<li><strong>friendsCount</strong> &mdash; <code>int</code> = <code>150</code></li>
	<li><strong>registeredAt</strong> &mdash; <code>string</code> = <code>&quot;2024-09-01&quot;</code></li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">Ожидаемый вывод</h2>

<pre>
<code class="language-text">25
John
1.75
true
1000.5
1995
Kazakhstan
36.6
150
2024-09-01
</code></pre>

<h2 style="color: #34495e; margin-top: 28px;">Требования</h2>

<ul>
	<li>Программа начинается с <code>package main</code>.</li>
	<li>Переменные объявлены с использованием <code>var</code> (на уровне пакета или внутри <code>main()</code> &mdash; по вашему выбору).</li>
	<li>Значения переменных выведены в функции <code>main()</code>, каждое на новой строке.</li>
	<li>Порядок и значения вывода совпадают с разделом &laquo;Ожидаемый вывод&raquo;.</li>
	<li>Лишний вывод отсутствует.</li>
</ul>

<div style="background: #e3f2fd; border: 1px solid #90caf9; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #1565c0; margin-top: 0;">💡 Обратите внимание</h4>

<p style="margin: 0;"><code>fmt.Println</code> выводит числа в компактном виде: <code>1000.50</code> станет <code>1000.5</code> (Go убирает лишние нули в конце). Это нормальное поведение &mdash; значение то же самое.</p>
</div>

<div style="background: #fff3cd; border: 1px solid #ffeaa7; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #856404; margin-top: 0;">⚠️ Частые ошибки</h4>

<ul style="margin: 0;">
	<li>Опечатки в строках или изменение порядка вывода.</li>
	<li>Вывод нескольких значений в одной строке вместо отдельных строк.</li>
	<li>Использование другого синтаксиса вместо <code>var</code> при объявлении.</li>
	<li>Десятичная запятая вместо точки в вещественных числах.</li>
</ul>
</div>
