# Задача: Деление двух чисел
> **Соответствует задаче:** `/Platform/go_base/3_Переменные/3.3_Операции_с_типами/3.3.4 ЗАДАЧА - Go base: Деление двух чисел.md`

## Описание:
Твоя задача — создать программу, которая объявляет две переменные с определенными типами и затем выводит результат деления одной переменной на другую.

## Что нужно сделать:
- Объяви две переменные a и b типа float64.
- Объяви переменную c типа float64, которая будет хранить результат деления a на b.
- Выведи значение c на экран с помощью fmt.Println.

## Ожидаемый вывод:
Если a = 10 и b = 4, то программа должна вывести: 2.5

## Критерии приемки:
- Переменные a и b должны быть объявлены строго с указанием типа float64.
- Задай переменным значения a = 10 и b = 4.
- Используй fmt.Println для вывода результата.
- Программа должна быть полной, включающей package main, import "fmt" и функцию main().
