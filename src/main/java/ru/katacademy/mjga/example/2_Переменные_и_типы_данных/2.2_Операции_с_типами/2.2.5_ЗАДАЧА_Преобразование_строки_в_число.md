# Задача: Преобразование строки в число
> **Соответствует задаче:** `/Platform/go_base/5_Условные_операторы/5.5_Стандартные_библиотеки_Go/5.5.3 ЗАДАЧА - Go base: Преобразование строки в число.md`

## Описание:
Создай функцию convertToNumber, которая принимает одну строку input и преобразует её в целое число типа int с использованием strconv.Atoi. Если строка не может быть преобразована в число, функция должна вернуть 0 и соответствующую ошибку.

## Что нужно сделать:
- Объяви функцию convertToNumber с одним аргументом input типа string.
- Используй strconv.Atoi для преобразования строки в целое число.
- Если преобразование успешно, верни это число и nil.
- Если ввод не является валидным числом, верни 0 и соответствующую ошибку.

## Критерии приемки:
- Не нужно писать полную программу, только саму функцию convertToNumber.
- Функция должна корректно обрабатывать ввод и возвращать либо число, либо ошибку при некорректном вводе.

## Пример:
При вызове функции с аргументом "123" ожидается:
```
123 и nil (ошибки нет)
```

При вызове функции с аргументом "abc" ожидается:
```
0 и ошибка "invalid syntax"
```
