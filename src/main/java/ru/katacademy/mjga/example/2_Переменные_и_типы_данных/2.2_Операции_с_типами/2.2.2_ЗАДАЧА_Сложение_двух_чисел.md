# Задача: Сложение двух чисел
> **Соответствует задаче:** `/Platform/go_base/3_Переменные/3.3_Операции_с_типами/3.3.2 ЗАДАЧА - Go base: Сложение двух чисел.md`

## Описание:
Твоя задача — создать программу, которая объявляет две переменные с определенными типами и затем выводит сумму этих переменных.

## Что нужно сделать:
- Объяви две переменные a и b типа int.
- Объяви переменную c типа int, которая будет хранить результат сложения a и b.
- Для объявления переменных используй синтаксис var.
- Выведи значение c на экран с помощью fmt.Println.

## Ожидаемый вывод:
Если a = 3 и b = 5, то программа должна вывести: 8

## Критерии приемки:
- Переменные должны быть объявлены строго с указанием типа int.
- Задай переменным значения a = 3 и b = 5.
- Используй fmt.Println для вывода результата.
- Программа должна быть полной, включающей package main, import "fmt" и функцию main().
