# Лекция 2.3: Арифметические операции в Go

## Введение

Арифметические операции — это базовые математические действия, которые используются в программировании для выполнения различных вычислений. В этой лекции мы рассмотрим основные арифметические операции в языке Go и научимся применять их на практике.

---

## Основные арифметические операции

Go поддерживает стандартные арифметические операции:

| Операция | Описание                 | Пример     | Результат |
|----------|--------------------------|------------|-----------|
| `+`      | Сложение                 | `4 + 2`    | `6`       |
| `-`      | Вычитание                | `4 - 2`    | `2`       |
| `*`      | Умножение                | `4 * 2`    | `8`       |
| `/`      | Деление                  | `4 / 2`    | `2`       |
| `%`      | Остаток от деления       | `5 % 2`    | `1`       |

---

## Примеры арифметических операций в Go

Рассмотрим каждую операцию подробнее.

### Сложение (`+`)

Сложение используется для получения суммы двух чисел:

```go
var sum int = 5 + 3
fmt.Println(sum) // 8
```

---

### Вычитание (`-`)

Вычитание используется для определения разницы между числами:

```go
var difference int = 10 - 7
fmt.Println(difference) // 3
```

---

### Умножение (`*`)

Умножение позволяет получить произведение чисел:

```go
var product int = 4 * 5
fmt.Println(product) // 20
```

---

### Деление (`/`)

Деление используется для вычисления частного двух чисел:

* При делении целых чисел результат также будет целым числом:

```go
var quotient int = 9 / 2
fmt.Println(quotient) // 4 (дробная часть отброшена)
```

* Чтобы получить точный результат с дробной частью, нужно использовать числа типа float64:

```go
var preciseQuotient float64 = 9.0 / 2.0
fmt.Println(preciseQuotient) // 4.5
```

---

### Остаток от деления (`%`)

Оператор `%` возвращает остаток от целочисленного деления. Например:

```go
var remainder int = 9 % 2
fmt.Println(remainder) // 1
```

Полезно для определения четности числа (остаток от деления на 2):

```go
var number int = 8
isEven := number % 2 == 0
fmt.Println(isEven) // true
```

---

## Приоритет арифметических операций

Как и в математике, арифметические операции выполняются в определённом порядке:
1. Скобки `()`
2. Умножение `*`, деление `/` и остаток от деления `%`
3. Сложение `+` и вычитание `-`

Пример:

```go
var result int = 3 + 5 * 2
fmt.Println(result) // 13 (сначала 5*2, затем +3)
```

Если нужно изменить порядок выполнения операций, используй скобки:

```go
var result int = (3 + 5) * 2
fmt.Println(result) // 16 (сначала 3+5, затем *2)
```

## Сокращённая форма арифметических операций

В Go есть удобные сокращения для изменения значения переменных:

```go
var x int = 10
x += 5  // аналогично x = x + 5 (теперь x = 15)
x -= 3  // аналогично x = x - 3 (теперь x = 12)
x *= 2  // аналогично x = x * 2 (теперь x = 24)
x /= 4  // аналогично x = x / 4 (теперь x = 6)
x %= 2  // аналогично x = x % 2 (теперь x = 0)
```

---

## Инкремент и декремент

Go также поддерживает инкремент (`++`) и декремент (`--`):

```go
var count int = 0
count++ // аналогично count = count + 1 (теперь count = 1)
count-- // аналогично count = count - 1 (теперь count = 0)
```
