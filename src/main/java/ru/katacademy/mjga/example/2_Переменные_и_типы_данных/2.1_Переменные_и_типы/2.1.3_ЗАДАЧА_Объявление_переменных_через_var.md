# Задача: Объявление переменных через var
> **Соответствует задаче:** `/Platform/go_base/3_Переменные/3.2_Типы_данных/3.2.3 ЗАДАЧА - Go base: Объявление переменных через var.md`

## Описание:
Тебе нужно объявить 10 переменных с использованием ключевого слова var. Каждая переменная должна быть строго типизирована и сразу объявлена со значением.

## Переменные для объявления:
- Возраст: age (целое число — int) = 25.
- Имя: name (строка — string) = "John".
- Рост: height (с плавающей точкой — float64) = 1.75.
- Активный пользователь: isActive (булевое значение — bool) = true.
- Баланс: balance (с плавающей точкой — float64) = 1000.50.
- Год рождения: birthYear (целое число — int) = 1995.
- Страна: country (строка — string) = "Kazakhstan".
- Температура: temperature (с плавающей точкой — float64) = 36.6.
- Количество друзей: friendsCount (целое число — int) = 150.
- Зарегистрирован: registeredAt (строка для даты — string) = "2024-09-01".

## Критерии приемки:
- Все переменные должны быть объявлены с использованием var.
- Каждая переменная должна иметь строгий тип.
- Каждая переменная должна быть объявлена сразу со значением, как указано выше.
- Внимание писать полностью программу не нужно, только объявление переменных.
