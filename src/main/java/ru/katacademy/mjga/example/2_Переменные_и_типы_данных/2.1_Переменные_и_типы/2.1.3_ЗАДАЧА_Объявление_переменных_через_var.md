<h2 id="what-are-variables" style="color: #34495e; margin-top: 30px;">Что такое переменные?</h2>

<div style="background: #f0f9ff; border: 1px solid #0284c7; border-radius: 8px; padding: 15px; margin: 16px 0;">
<h4 style="color: #0c4a6e; margin-top: 0;">📦 Переменные &mdash; это коробочки для данных</h4>

<p>Представьте, что переменная &mdash; это подписанная коробка, в которой лежит что-то одно:</p>

<div style="display: grid; grid-template-columns: repeat(auto-fit, minmax(150px, 1fr)); gap: 15px; margin: 20px 0;">
<div style="text-align: center; background: #fff; padding: 15px; border-radius: 8px; border: 2px solid #0284c7;">
<div style="font-size: 32px;">📦</div>
<strong>age</strong><br />
<small>Число: 25</small></div>

<div style="text-align: center; background: #fff; padding: 15px; border-radius: 8px; border: 2px solid #0284c7;">
<div style="font-size: 32px;">📦</div>
<strong>name</strong><br />
<small>Текст: &quot;Алиса&quot;</small></div>

<div style="text-align: center; background: #fff; padding: 15px; border-radius: 8px; border: 2px solid #0284c7;">
<div style="font-size: 32px;">📦</div>
<strong>isStudent</strong><br />
<small>Да/Нет: true</small></div>
</div>

<p><strong>Зачем нужны переменные?</strong></p>

<ul>
	<li>💾 <strong>Сохранять данные</strong> &mdash; имя пользователя, возраст, результаты вычислений</li>
	<li>🔄 <strong>Изменять значения</strong> &mdash; счётчики, суммы, статусы</li>
	<li>📊 <strong>Использовать повторно</strong> &mdash; одно значение в разных местах программы</li>
</ul>
</div>

<h2 id="var-declaration" style="color: #34495e; margin-top: 30px;">Способ 1: Объявление через var</h2>

<div style="background: #e8f5e9; border: 1px solid #4caf50; border-radius: 8px; padding: 20px; margin: 20px 0;">
<h4 style="color: #2e7d32; margin-top: 0;">📝 Полная форма объявления</h4>

<p><strong>Формула:</strong></p>

<pre style="background: #1e293b; color: #e2e8f0; padding: 15px; border-radius: 5px; font-size: 16px;">
<code>var имя_переменной тип = значение</code></pre>

<p><strong>Примеры:</strong></p>

<pre style="background: #1e293b; color: #e2e8f0; padding: 15px; border-radius: 5px;">
<code>var age int = 25           // Возраст - целое число
var name string = "Алиса"   // Имя - текст
var height float64 = 1.75   // Рост - дробное число
var isStudent bool = true   // Студент? - да/нет</code></pre>

<div style="background: white; padding: 15px; border-radius: 5px; margin-top: 15px;">
<h5 style="color: #2e7d32; margin-top: 0;">🎯 Что здесь происходит:</h5>

<ol>
	<li><code>var</code> &mdash; говорим &quot;создай переменную&quot;</li>
	<li><code>age</code> &mdash; даём имя коробочке</li>
	<li><code>int</code> &mdash; указываем тип (целое число)</li>
	<li><code>= 25</code> &mdash; кладём значение в коробочку</li>
</ol>
</div>
</div>

<div style="background: #fff3e0; border: 1px solid #ff9800; border-radius: 8px; padding: 20px; margin: 20px 0;">
<h4 style="color: #e65100; margin-top: 0;">⚡ Значения по умолчанию</h4>

<p>Если не указать значение, Go положит в коробочку &quot;пустышку&quot;:</p>

<pre style="background: #1e293b; color: #e2e8f0; padding: 15px; border-radius: 5px;">
<code>var count int          // 0 (ноль для чисел)
var message string     // "" (пустая строка для текста)
var isValid bool       // false (ложь для да/нет)</code></pre>

<p><strong>Полная программа с переменными:</strong></p>

<pre style="background: #1e293b; color: #e2e8f0; padding: 15px; border-radius: 5px;">
<code>package main

import "fmt"

func main() {
    var age int = 25
    var name string = "Алиса"

    fmt.Println("Имя:", name)
    fmt.Println("Возраст:", age)
}</code></pre>
</div>

<h2 id="short-declaration" style="color: #34495e; margin-top: 30px;">Способ 2: Короткое объявление :=</h2>

<div style="background: #e3f2fd; border: 1px solid #90caf9; border-radius: 8px; padding: 20px; margin: 20px 0;">
<h4 style="color: #1565c0; margin-top: 0;">🚀 Быстрый способ (самый популярный!)</h4>

<p><strong>Формула:</strong></p>

<pre style="background: #1e293b; color: #e2e8f0; padding: 15px; border-radius: 5px; font-size: 16px;">
<code>имя := значение</code></pre>

<p><strong>Go сам догадается о типе!</strong></p>

<pre style="background: #1e293b; color: #e2e8f0; padding: 15px; border-radius: 5px;">
<code>age := 25             // Go понимает: это int
name := "Алиса"       // Go понимает: это string
height := 1.75        // Go понимает: это float64
isStudent := true     // Go понимает: это bool</code></pre>

<div style="background: white; padding: 15px; border-radius: 5px; margin-top: 15px;">
<h5 style="color: #1565c0; margin-top: 0;">🎯 Сравнение двух способов:</h5>

<table style="width: 100%; border-collapse: collapse;">
	<tbody>
		<tr style="background: #f0f9ff;">
			<th style="padding: 10px; border: 1px solid #90caf9;">Полный (var)</th>
			<th style="padding: 10px; border: 1px solid #90caf9;">Короткий (:=)</th>
		</tr>
		<tr>
			<td style="padding: 10px; border: 1px solid #90caf9;"><code>var age int = 25</code></td>
			<td style="padding: 10px; border: 1px solid #90caf9;"><code>age := 25</code></td>
		</tr>
		<tr>
			<td style="padding: 10px; border: 1px solid #90caf9;">✅ Явно видно тип</td>
			<td style="padding: 10px; border: 1px solid #90caf9;">✅ Короче и проще</td>
		</tr>
		<tr>
			<td style="padding: 10px; border: 1px solid #90caf9;">📍 Можно использовать везде</td>
			<td style="padding: 10px; border: 1px solid #90caf9;">⚠️ Только внутри функций</td>
		</tr>
	</tbody>
</table>
</div>

<div style="background: #d4edda; padding: 10px; border-radius: 5px; margin-top: 15px;">
<p style="margin: 0;"><strong>💡 Совет:</strong> В реальном Go-коде <code>:=</code> встречается чаще, потому что он короче. Но в этом модуле мы тренируем обе формы &mdash; особенно <code>var</code>, потому что она показывает тип явно и работает на уровне пакета.</p>
</div>
</div>

<div style="background: #f8f9fa; border: 1px solid #dee2e6; border-radius: 8px; padding: 20px; margin: 20px 0;">
<h4 style="color: #495057; margin-top: 0;">🎨 4 главных типа для начала</h4>

<div style="display: grid; grid-template-columns: repeat(auto-fit, minmax(200px, 1fr)); gap: 15px; margin-top: 15px;">
<div style="background: #e8f5e9; padding: 15px; border-radius: 8px;">

<pre style="margin: 10px 0;">
<code>age := 25

<p style="margin: 0; font-size: 14px;">Для: возраст, количество, счёт</p>
</div>

<div style="background: #e3f2fd; padding: 15px; border-radius: 8px;">

<p style="margin: 0; font-size: 14px;">Для: имена, адреса, сообщения</p>

<h5 style="color: #880e4f; margin-top: 0;">✅ bool (да/нет)</h5>

<h2 id="multiple-variables" style="color: #34495e; margin-top: 30px;">Несколько переменных сразу</h2>

<div style="background: #f0fdf4; border: 1px solid #22c55e; border-radius: 8px; padding: 20px; margin: 20px 0;">
<h4 style="color: #166534; margin-top: 0;">🎯 Объявление нескольких переменных</h4>

<p><strong>Способ 1: В одну строку</strong></p>

<pre style="background: #1e293b; color: #e2e8f0; padding: 15px; border-radius: 5px;">
<code>// Полная форма
var width, height int = 100, 200
var firstName, lastName string = "Иван", "Петров"

// Короткая форма (рекомендуется)
x, y := 10, 20
name, age := "Алексей", 28</code></pre>

<p><strong>Способ 2: Блоком (несколько переменных вместе)</strong></p>

<pre style="background: #1e293b; color: #e2e8f0; padding: 15px; border-radius: 5px;">
<code>var (
    username = "admin"
    password = "12345"
    attempts = 3
)</code></pre>

<div style="background: white; padding: 10px; border-radius: 5px; margin-top: 10px;">
<p style="margin: 0;"><strong>💡 Когда использовать:</strong> Блочная форма работает и на уровне пакета, и внутри функции &mdash; удобна, когда у вас несколько связанных переменных (координаты x,y или имя и фамилия).</p>
</div>
</div>

<h2 id="constants" style="color: #34495e; margin-top: 30px;">Константы</h2>

<div style="background: #f3e5f5; border: 1px solid #ce93d8; border-radius: 8px; padding: 20px; margin: 20px 0;">
<h4 style="color: #6a1b9a; margin-top: 0;">🔒 Константы &mdash; неизменяемые значения</h4>

<p>Константа &mdash; это переменная, которую <strong>нельзя изменить</strong> после создания:</p>

<pre style="background: #1e293b; color: #e2e8f0; padding: 15px; border-radius: 5px;">
<code>const pi = 3.14159
const maxUsers = 1000
const appName = "MyApp"
const daysInWeek = 7

// Попытка изменить вызовет ошибку!
// pi = 3.14  // ❌ ОШИБКА!</code></pre>

<div style="background: white; padding: 10px; border-radius: 5px; margin-top: 10px;">
<p style="margin: 0;"><strong>Имена констант</strong> в Go подчиняются тем же правилам, что и имена переменных &mdash; в наших задачах это <code>camelCase</code>.</p>

<p style="margin: 10px 0 0 0;"><strong>Когда использовать константы:</strong></p>

<ul style="margin: 5px 0;">
	<li>Математические константы (Pi, E)</li>
	<li>Настройки программы (максимум попыток, размер буфера)</li>
	<li>Фиксированные значения (дни недели, месяцы)</li>
</ul>
</div>
</div>

<h2 id="common-mistakes" style="color: #34495e; margin-top: 30px;">Частые ошибки</h2>

<div style="background: #fee2e2; border: 1px solid #ef4444; border-radius: 8px; padding: 20px; margin: 20px 0;">
<h4 style="color: #991b1b; margin-top: 0;">⚠️ ТОП-5 ошибок с переменными</h4>

<div style="background: white; padding: 15px; border-radius: 5px; margin: 10px 0;">
<h5 style="color: #dc2626; margin-top: 0;">❌ Ошибка 1: := вне функции</h5>

<pre style="background: #fef2f2; color: #991b1b; padding: 10px; border-radius: 5px;">
<code>package main

name := "Alice"  // ❌ ОШИБКА! := только внутри функций

func main() {
    // ...
}</code></pre>

<p><strong>✅ Решение:</strong> Используйте <code>var name = &quot;Alice&quot;</code> для глобальных переменных</p>
</div>

<div style="background: white; padding: 15px; border-radius: 5px; margin: 10px 0;">
<h5 style="color: #dc2626; margin-top: 0;">❌ Ошибка 2: Неиспользуемая переменная</h5>

<pre style="background: #fef2f2; color: #991b1b; padding: 10px; border-radius: 5px;">
<code>func main() {
    name := "Alice"  // ❌ ОШИБКА если не используется
}</code></pre>

<p><strong>✅ Решение:</strong> Используйте переменную или удалите её</p>
</div>

<div style="background: white; padding: 15px; border-radius: 5px; margin: 10px 0;">
<h5 style="color: #dc2626; margin-top: 0;">❌ Ошибка 3: Повторное объявление</h5>

<pre style="background: #fef2f2; color: #991b1b; padding: 10px; border-radius: 5px;">
<code>age := 25
age := 30  // ❌ ОШИБКА! Уже объявлена</code></pre>

<p><strong>✅ Решение:</strong> Используйте <code>age = 30</code> для изменения значения</p>
</div>
</div>

<h2 id="practice" style="color: #34495e; margin-top: 30px;">Практические упражнения</h2>

<div style="background: #e8f5e9; border: 1px solid #4caf50; border-radius: 8px; padding: 20px; margin: 20px 0;">
<h4 style="color: #2e7d32; margin-top: 0;">📝 Упражнение 1: Первые переменные</h4>

<p>Создайте программу с переменными о себе:</p>

<pre style="background: #1e293b; color: #e2e8f0; padding: 15px; border-radius: 5px;">
<code class="language-go">package main

import "fmt"

func main() {
    // Используйте var
    var myName string = "Ваше имя"
    var myAge int = 20

    // Используйте :=
    city := "Ваш город"
    isStudent := true

    fmt.Println("Имя:", myName)
    fmt.Println("Возраст:", myAge)
    fmt.Println("Город:", city)
    fmt.Println("Студент?", isStudent)
}</code></pre>
</div>

<div style="background: #e3f2fd; border: 1px solid #90caf9; border-radius: 8px; padding: 20px; margin: 20px 0;">
<h4 style="color: #1565c0; margin-top: 0;">📝 Упражнение 2: Магазин</h4>

<p>Создайте переменные для товара в магазине:</p>

<details><summary style="cursor: pointer; color: #1565c0; font-weight: bold;">Показать решение</summary>

<pre style="background: #1e293b; color: #e2e8f0; padding: 15px; border-radius: 5px; margin-top: 10px;">
<code>package main

import "fmt"

func main() {
    productName := "Ноутбук"
    price := 49999.99
    quantity := 10
    inStock := true

    const discount = 0.15  // 15% скидка
    discountAmount := price * discount
    finalPrice := price - discountAmount

    fmt.Println("Товар:", productName)
    fmt.Println("Цена:", price)
    fmt.Println("Скидка:", discountAmount)
    fmt.Println("Со скидкой:", finalPrice)
    fmt.Println("Количество:", quantity)
    fmt.Println("В наличии:", inStock)
