<h2 id="why-types" style="color: #34495e; margin-top: 30px;">Зачем нужны типы данных?</h2>

<div style="background: #f0f9ff; border: 1px solid #0284c7; border-radius: 8px; padding: 20px; margin: 20px 0;">
<h4 style="color: #0c4a6e; margin-top: 0;">🎯 Типы &mdash; это правила игры</h4>

<p>Представьте, что данные &mdash; это разные виды транспорта:</p>

<div style="display: grid; grid-template-columns: repeat(auto-fit, minmax(200px, 1fr)); gap: 15px; margin: 20px 0;">
<div style="text-align: center; background: #fff; padding: 15px; border-radius: 8px; border: 2px solid #0284c7;">
<div style="font-size: 32px;">🚗</div>
<strong>int (целые числа)</strong><br />
<small>Для счёта штук: 1, 2, 100</small></div>

<div style="text-align: center; background: #fff; padding: 15px; border-radius: 8px; border: 2px solid #0284c7;">
<div style="font-size: 32px;">✈️</div>
<strong>float64 (дробные)</strong><br />
<small>Для измерений: 1.5, 3.14</small></div>

<div style="text-align: center; background: #fff; padding: 15px; border-radius: 8px; border: 2px solid #0284c7;">
<div style="font-size: 32px;">📝</div>
<strong>string (текст)</strong><br />
<small>Для слов: &quot;Привет&quot;</small></div>

<div style="text-align: center; background: #fff; padding: 15px; border-radius: 8px; border: 2px solid #0284c7;">
<div style="font-size: 32px;">🚦</div>
<strong>bool (да/нет)</strong><br />
<small>Для выбора: true/false</small></div>
</div>

<div style="background: #fff3cd; padding: 15px; border-radius: 5px;">
<p style="margin: 0;"><strong>⚠️ Важно:</strong> Нельзя посадить пассажира самолёта в машину без &quot;пересадки&quot; (преобразования). Так же и в Go &mdash; нельзя смешивать типы!</p>
</div>
</div>

<h2 id="int-type" style="color: #34495e; margin-top: 30px;">Целые числа (int)</h2>

<div style="background: #e8f5e9; border: 1px solid #4caf50; border-radius: 8px; padding: 20px; margin: 20px 0;">
<h4 style="color: #2e7d32; margin-top: 0;">🔢 int &mdash; для целых чисел</h4>

<p><strong>Что это:</strong> Числа без дробной части (без точки)</p>

<p><strong>Когда использовать:</strong> Возраст, количество, счётчики, индексы</p>

<pre style="background: #f8f9fa; border: 1px solid #e9ecef; border-radius: 5px; padding: 15px; margin: 10px 0; overflow-x: auto;">
<code>age := 25           // возраст
count := 100        // количество
score := -50        // может быть отрицательным
level := 1          // уровень в игре</code></pre>

<h2 id="float-type" style="color: #34495e; margin-top: 30px;">Дробные числа (float)</h2>

<div style="background: #e3f2fd; border: 1px solid #90caf9; border-radius: 8px; padding: 20px; margin: 20px 0;">
<h4 style="color: #1565c0; margin-top: 0;">💰 float64 &mdash; для дробных чисел</h4>

<p><strong>Что это:</strong> Числа с дробной частью (с точкой)</p>

<p><strong>Когда использовать:</strong> Деньги, измерения, проценты, координаты</p>

<pre style="background: #f8f9fa; border: 1px solid #e9ecef; border-radius: 5px; padding: 15px; margin: 10px 0; overflow-x: auto;">
<code>price := 99.99      // цена
height := 1.75      // рост в метрах
pi := 3.14159       // число Пи
temperature := -5.5 // температура</code></pre>

<div style="background: white; padding: 15px; border-radius: 5px; margin-top: 15px;">
<h5 style="color: #1565c0; margin-top: 0;">🎯 float32 vs float64</h5>

<ul>
	<li><strong>float32</strong> &mdash; меньше памяти, меньше точность (редко нужен)</li>
	<li><strong>float64</strong> &mdash; больше точности, стандарт в Go ✅</li>
</ul>

<pre style="background: #f8f9fa; border: 1px solid #e9ecef; border-radius: 5px; padding: 10px; margin: 10px 0; overflow-x: auto;">
<code class="language-go">// Go автоматически выберет float64
price := 99.99

// Явно указываем float32 (редко)
var smallNumber float32 = 1.5</code></pre>
</div>
</div>

<h2 id="string-type" style="color: #34495e; margin-top: 30px;">Текст (string)</h2>

<div style="background: #fff3e0; border: 1px solid #ff9800; border-radius: 8px; padding: 20px; margin: 20px 0;">
<h4 style="color: #e65100; margin-top: 0;">📝 string &mdash; для текста</h4>

<p><strong>Что это:</strong> Любой текст в двойных кавычках <code>&quot; &quot;</code></p>

<p><strong>Когда использовать:</strong> Имена, сообщения, адреса, любой текст</p>

<pre style="background: #f8f9fa; border: 1px solid #e9ecef; border-radius: 5px; padding: 15px; margin: 10px 0; overflow-x: auto;">
<code class="language-go">name := "Алиса"
message := "Привет, мир!"
empty := ""  // пустая строка
number := "123"  // это ТЕКСТ, не число!</code></pre>

<div style="background: white; padding: 15px; border-radius: 5px; margin-top: 15px;">
<h5 style="color: #e65100; margin-top: 0;">➕ Склеивание строк</h5>

<pre style="background: #f8f9fa; border: 1px solid #e9ecef; border-radius: 5px; padding: 10px; margin: 10px 0; overflow-x: auto;">
<code class="language-go">firstName := "Иван"
lastName := "Петров"
space := " "

// Склеиваем через +
fullName := firstName + space + lastName
// Результат: "Иван Петров"

// Можно сразу
greeting := "Привет, " + firstName + "!"
// Результат: "Привет, Иван!"</code></pre>
</div>

<div style="background: #fef3c7; padding: 10px; border-radius: 5px; margin-top: 15px;">
<p style="margin: 0;"><strong>⚠️ Важно:</strong> <code>&quot;123&quot;</code> &mdash; это текст! Чтобы превратить в число, нужно преобразование.</p>
</div>
</div>

<h2 id="bool-type" style="color: #34495e; margin-top: 30px;">Логический тип (bool)</h2>

<div style="background: #fce4ec; border: 1px solid #e91e63; border-radius: 8px; padding: 20px; margin: 20px 0;">
<h4 style="color: #880e4f; margin-top: 0;">✅ bool &mdash; только да или нет</h4>

<p><strong>Что это:</strong> Только два значения: <code>true</code> (да) или <code>false</code> (нет)</p>

<p><strong>Когда использовать:</strong> Флаги, статусы, условия, проверки</p>

<pre style="background: #f8f9fa; border: 1px solid #e9ecef; border-radius: 5px; padding: 15px; margin: 10px 0; overflow-x: auto;">
<code class="language-go">isStudent := true       // студент?
isMarried := false      // женат/замужем?
hasAccess := true       // есть доступ?
isReady := false        // готово?</code></pre>

<p style="margin: 15px 0 5px 0;">Значение <code>bool</code> можно получить как результат <strong>сравнения</strong>. Go использует знакомые из математики знаки: <code>&gt;=</code> (больше или равно), <code>==</code> (равно), <code>!=</code> (не равно), <code>&gt;</code>, <code>&lt;</code>, <code>&lt;=</code>. Результат сравнения &mdash; всегда <code>true</code> или <code>false</code>:</p>

<pre style="background: #f8f9fa; border: 1px solid #e9ecef; border-radius: 5px; padding: 15px; margin: 10px 0; overflow-x: auto;">
<code class="language-go">age := 20
isAdult := age &gt;= 18    // true, потому что 20 &gt;= 18</code></pre>

<div style="background: white; padding: 15px; border-radius: 5px; margin-top: 15px;">
<h5 style="color: #880e4f; margin-top: 0;">🚦 Где используется bool</h5>

<pre style="background: #f8f9fa; border: 1px solid #e9ecef; border-radius: 5px; padding: 10px; margin: 10px 0; overflow-x: auto;">
<code class="language-go">isLoggedIn := true
isBlocked := false

fmt.Printf("Вошли в систему: %v\n", isLoggedIn)  // true
fmt.Printf("Заблокирован: %v\n", isBlocked)       // false</code></pre>

<p style="margin: 10px 0 0 0;"><strong>💡</strong> В следующих модулях мы научимся использовать bool в условиях (<code>if/else</code>), чтобы программа принимала решения.</p>
</div>
</div>

<h2 id="operations" style="color: #34495e; margin-top: 30px;">Операции с типами</h2>

<div style="background: #f8f9fa; border: 1px solid #dee2e6; border-radius: 8px; padding: 20px; margin: 20px 0;">
<h4 style="color: #495057; margin-top: 0;">🔧 Что можно делать с разными типами</h4>

<div style="display: grid; grid-template-columns: 1fr 1fr; gap: 20px;">
<div style="background: white; padding: 15px; border-radius: 5px;">
<h5 style="color: #495057; margin-top: 0;">🔢 С числами (int, float)</h5>

<pre style="background: #f8f9fa; border: 1px solid #e9ecef; border-radius: 5px; padding: 10px; margin: 10px 0; overflow-x: auto;">
<code class="language-go">a := 10
b := 3

sum := a + b     // 13
diff := a - b    // 7
mult := a * b    // 30
div := a / b     // 3 (целочисленное!)

// Для дробного результата
result := 10.0 / 3.0  // 3.333...</code></pre>
</div>

<div style="background: white; padding: 15px; border-radius: 5px;">
<h5 style="color: #495057; margin-top: 0;">📝 Со строками</h5>

<pre style="background: #f8f9fa; border: 1px solid #e9ecef; border-radius: 5px; padding: 10px; margin: 10px 0; overflow-x: auto;">
<code class="language-go">s1 := "Hello"
s2 := "World"

// Склеивание
result := s1 + " " + s2  // "Hello World"

// Можно склеивать с пробелами и знаками

</div>
</div>

<div style="background: white; padding: 15px; border-radius: 5px; margin-top: 20px;">
<h5 style="color: #495057; margin-top: 0;">⚖️ Сравнения (результат &mdash; bool)</h5>

<pre style="background: #f8f9fa; border: 1px solid #e9ecef; border-radius: 5px; padding: 10px; margin: 10px 0; overflow-x: auto;">
<code class="language-go">a := 10
b := 5

isEqual := a == b       // false (равны?)
isNotEqual := a != b    // true (не равны?)
isGreater := a &gt; b       // true (больше?)
isLess := a &lt; b          // false (меньше?)
isGreaterOrEq := a &gt;= b  // true (больше или равно?)
isLessOrEq := a &lt;= b     // false (меньше или равно?)</code></pre>
</div>

<h2 id="type-conversion" style="color: #34495e; margin-top: 30px;">Преобразование типов</h2>

<div style="background: #f3e5f5; border: 1px solid #ce93d8; border-radius: 8px; padding: 20px; margin: 20px 0;">
<h4 style="color: #6a1b9a; margin-top: 0;">🔄 Как превратить один тип в другой</h4>

<div style="background: #fee2e2; padding: 15px; border-radius: 5px; margin-bottom: 15px;">
<p style="margin: 0;"><strong>❌ Go не смешивает типы автоматически!</strong></p>

<pre style="background: #f8f9fa; border: 1px solid #e9ecef; border-radius: 5px; padding: 10px; margin-top: 10px; overflow-x: auto;">
<code class="language-go">// ОШИБКА!
var a int = 10
var b float64 = 5.5
// result := a + b  // ❌ Нельзя!</code></pre>
</div>

<div style="background: #d4edda; padding: 15px; border-radius: 5px;">
<p style="margin: 0 0 10px 0;"><strong>✅ Нужно явное преобразование:</strong></p>

<h5 style="color: #155724; margin: 10px 0;">1️⃣ int &harr; float</h5>

<pre style="background: #f8f9fa; border: 1px solid #e9ecef; border-radius: 5px; padding: 10px; margin: 10px 0; overflow-x: auto;">
<code class="language-go">// int → float
var a int = 10
var b float64 = float64(a)  // 10.0

// float → int (теряем дробную часть!)
var price float64 = 99.99
var roundPrice int = int(price)  // 99</code></pre>

<h5 style="color: #155724; margin: 10px 0;">2️⃣ Число в тексте вывода</h5>

<p style="margin: 5px 0;">Для вывода числа внутри текста используйте <code>fmt.Printf</code> с форматными глаголами (их краткая шпаргалка &mdash; ниже, в разделе &laquo;Как узнать тип?&raquo;; подробно <code>fmt.Printf</code> разбирается в следующей лекции этого модуля):</p>

<pre style="background: #f8f9fa; border: 1px solid #e9ecef; border-radius: 5px; padding: 10px; margin: 10px 0; overflow-x: auto;">
<code class="language-go">age := 25
fmt.Printf("Возраст: %d лет\n", age)  // Возраст: 25 лет

price := 99.9
fmt.Printf("Цена: %.1f руб.\n", price)  // Цена: 99.9 руб.</code></pre>

<div style="background: #fff3cd; padding: 10px; border-radius: 5px; margin: 10px 0;">
<p style="margin: 0;"><strong>Заметка:</strong> Получение числа именно как строки (для склейки через <code>+</code>) разберём в модуле 9 вместе с пакетом <code>strconv</code>.</p>
</div>

<h5 style="color: #155724; margin: 10px 0;">3️⃣ string &rarr; int (разберём в модуле 9)</h5>

<div style="background: #fff3cd; padding: 10px; border-radius: 5px; margin: 10px 0;">
<p style="margin: 0;"><strong>Заметка:</strong> Прямого приведения <code>int(text)</code> для строк в Go нет &mdash; такая запись не скомпилируется. Если число пришло как текст (например, из ввода), его нужно отдельно преобразовать функцией из пакета <code>strconv</code>. Эту тему вместе с обработкой ошибок мы изучим в модуле 9.</p>
</div>
</div>
</div>

<h2 id="check-type" style="color: #34495e; margin-top: 30px;">Как узнать тип?</h2>

<div style="background: #e8f5e9; border: 1px solid #4caf50; border-radius: 8px; padding: 20px; margin: 20px 0;">
<h4 style="color: #2e7d32; margin-top: 0;">🔍 Printf с %T покажет тип</h4>

<pre style="background: #f8f9fa; border: 1px solid #e9ecef; border-radius: 5px; padding: 15px; margin: 10px 0; overflow-x: auto;">
<code class="language-go">package main

import "fmt"

func main() {
    // Создаём разные переменные
    age := 25
    price := 99.99
    name := "Алиса"
    isStudent := true

    // Проверяем типы
    fmt.Printf("age: значение = %v, тип = %T\n", age, age)
    fmt.Printf("price: значение = %v, тип = %T\n", price, price)
    fmt.Printf("name: значение = %v, тип = %T\n", name, name)
    fmt.Printf("isStudent: значение = %v, тип = %T\n", isStudent, isStudent)
}</code></pre>

<div style="background: white; padding: 10px; border-radius: 5px; margin-top: 10px;">
<h5 style="color: #2e7d32; margin-top: 0;">📋 Результат:</h5>

<pre style="background: #f8f9fa; border: 1px solid #e9ecef; border-radius: 5px; padding: 10px; margin: 0; overflow-x: auto;">
<code class="language-go">age: значение = 25, тип = int
price: значение = 99.99, тип = float64
name: значение = Алиса, тип = string
isStudent: значение = true, тип = bool</code></pre>
</div>

<p style="margin: 15px 0 0 0;"><strong>💡 Запомните:</strong> <code>%v</code> &mdash; значение, <code>%T</code> &mdash; тип</p>
</div>

<div style="background: #f0f9ff; border: 1px solid #0284c7; border-radius: 8px; padding: 20px; margin: 20px 0;">
<h4 style="color: #0c4a6e; margin-top: 0;">📊 Шпаргалка: форматные глаголы Printf</h4>

<p>Функция <code>fmt.Printf</code> использует специальные символы (глаголы), чтобы подставлять значения в текст. Вот самые нужные:</p>

<h2 id="common-mistakes" style="color: #34495e; margin-top: 30px;">Частые ошибки</h2>

<div style="background: #fee2e2; border: 1px solid #ef4444; border-radius: 8px; padding: 20px; margin: 20px 0;">
<h4 style="color: #991b1b; margin-top: 0;">⚠️ ТОП-5 ошибок с типами</h4>

<div style="background: white; padding: 15px; border-radius: 5px; margin: 10px 0;">
<h5 style="color: #dc2626; margin-top: 0;">❌ Ошибка 1: Смешивание типов</h5>

<pre style="background: #fef2f2; border: 1px solid #fca5a5; border-radius: 5px; padding: 10px; margin: 10px 0; overflow-x: auto;">
<code class="language-go">age := 25
text := " лет"
// result := age + text  // ❌ ОШИБКА!</code></pre>

<p><strong>✅ Решение:</strong> Выведите через <code>Printf</code> с нужными форматами</p>

<pre style="background: #d4edda; border: 1px solid #a3d9a5; border-radius: 5px; padding: 10px; margin: 10px 0; overflow-x: auto;">
<code class="language-go">fmt.Printf("%d%s\n", age, text)  // 25 лет</code></pre>
</div>

<div style="background: white; padding: 15px; border-radius: 5px; margin: 10px 0;">
<h5 style="color: #dc2626; margin-top: 0;">❌ Ошибка 2: Деление целых чисел</h5>

<pre style="background: #fef2f2; border: 1px solid #fca5a5; border-radius: 5px; padding: 10px; margin: 10px 0; overflow-x: auto;">
<code class="language-go">result := 10 / 3  // 3 (не 3.333!)</code></pre>

<p><strong>✅ Решение:</strong> Используйте float</p>

<pre style="background: #d4edda; border: 1px solid #a3d9a5; border-radius: 5px; padding: 10px; margin: 10px 0; overflow-x: auto;">
<code class="language-go">result := 10.0 / 3.0  // 3.333...</code></pre>
</div>

<div style="background: white; padding: 15px; border-radius: 5px; margin: 10px 0;">
<h5 style="color: #dc2626; margin-top: 0;">❌ Ошибка 3: &quot;123&quot; &mdash; это не число!</h5>

<pre style="background: #fef2f2; border: 1px solid #fca5a5; border-radius: 5px; padding: 10px; margin: 10px 0; overflow-x: auto;">
<code class="language-go">number := "123"
// result := number + 1  // ❌ ОШИБКА!</code></pre>

<p><strong>✅ Решение:</strong> Для вычислений используйте числовой тип</p>

<pre style="background: #d4edda; border: 1px solid #a3d9a5; border-radius: 5px; padding: 10px; margin: 10px 0; overflow-x: auto;">
<code class="language-go">realNumber := 123
result := realNumber + 1  // 124</code></pre>

<p style="margin: 10px 0 0 0;"><strong>Если значение пришло строкой:</strong> преобразование текста в число разберём в модуле 9.</p>
</div>
</div>

<h2 id="practice" style="color: #34495e; margin-top: 30px;">Практические упражнения</h2>

<div style="background: #e8f5e9; border: 1px solid #4caf50; border-radius: 8px; padding: 20px; margin: 20px 0;">
<h4 style="color: #2e7d32; margin-top: 0;">📝 Упражнение 1: Калькулятор возраста</h4>

<p>Создайте программу, которая вычисляет возраст в днях:</p>

<pre style="background: #f8f9fa; border: 1px solid #e9ecef; border-radius: 5px; padding: 15px; margin: 10px 0; overflow-x: auto;">
<code class="language-go">package main

import "fmt"

func main() {
    var age int = 25
    var daysInYear int = 365

    ageInDays := age * daysInYear

    fmt.Printf("Возраст: %d лет\n", age)
    fmt.Printf("Это примерно %d дней\n", ageInDays)
}</code></pre>
</div>

<div style="background: #e3f2fd; border: 1px solid #90caf9; border-radius: 8px; padding: 20px; margin: 20px 0;">
<h4 style="color: #1565c0; margin-top: 0;">📝 Упражнение 2: Конвертер температуры</h4>

<p>Переведите температуру из Цельсия в Фаренгейты:</p>

<details><summary style="cursor: pointer; color: #1565c0; font-weight: bold;">Показать решение</summary>

<pre style="background: #f8f9fa; border: 1px solid #e9ecef; border-radius: 5px; padding: 15px; margin-top: 10px; overflow-x: auto;">
<code>package main

import "fmt"

func main() {
    celsius := 25.0
    fahrenheit := celsius * 9/5 + 32

    fmt.Printf("%.1f°C = %.1f°F\n", celsius, fahrenheit)

    // Проверим типы
    fmt.Printf("Тип celsius: %T\n", celsius)
    fmt.Printf("Тип fahrenheit: %T\n", fahrenheit)
}</code></pre>

<div style="background: #fff3cd; padding: 12px; border-radius: 5px; margin-top: 15px;">
<p style="margin: 0 0 8px 0;"><strong>Почему <code>celsius * 9/5 + 32</code> работает без ошибки?</strong></p>

<p style="margin: 0 0 8px 0;">Выше мы говорили, что Go не смешивает типы. Но здесь <code>celsius</code> &mdash; это <code>float64</code>, а <code>9</code>, <code>5</code>, <code>32</code> выглядят как целые числа. Секрет в том, что числовые литералы (<code>9</code>, <code>5</code>, <code>32</code>) в Go &mdash; это <strong>нетипизированные константы</strong>. У них нет жёсткого типа, и Go автоматически подстраивает их под тип переменной <code>celsius</code> (<code>float64</code>).</p>

<p style="margin: 0;">Но если записать число в переменную, у неё уже будет тип <code>int</code>. Тогда без явного преобразования компилятор выдаст ошибку:</p>

<pre style="background: #fef2f2; border: 1px solid #fca5a5; border-radius: 5px; padding: 10px; margin: 8px 0 0 0; overflow-x: auto;">
<code class="language-go">coefficient := 32                          // тип int
// result := celsius*9/5 + coefficient     // ❌ ошибка: mismatched types float64 and int
result := celsius*9/5 + float64(coefficient) // ✅ нужно явное преобразование</code></pre>
</div>
</details>
</div>

<div style="background: #fff3cd; border: 1px solid #ffeaa7; border-radius: 8px; padding: 20px; margin: 20px 0;">
<h4 style="color: #856404; margin-top: 0;">📝 Упражнение 3: Магазин с типами</h4>

<p>Создайте программу для расчёта покупки с правильными типами:</p>

<details><summary style="cursor: pointer; color: #856404; font-weight: bold;">Показать решение</summary>

<pre style="background: #f8f9fa; border: 1px solid #e9ecef; border-radius: 5px; padding: 15px; margin-top: 10px; overflow-x: auto;">
<code>package main

import "fmt"

func main() {
    // Разные типы данных
    productName := "Ноутбук"   // string
    price := 49999.99          // float64
    quantity := 2              // int

    // float64(quantity) — преобразуем int в float64
    total := price * float64(quantity)

    // Вывод с разными форматами
    fmt.Println("=== ЧЕК ===")
    fmt.Printf("Товар: %s\n", productName)
    fmt.Printf("Цена: %.2f руб.\n", price)
    fmt.Printf("Количество: %d шт.\n", quantity)
    fmt.Printf("ИТОГО: %.2f руб.\n", total)
}</code></pre>
</details>
</div>
