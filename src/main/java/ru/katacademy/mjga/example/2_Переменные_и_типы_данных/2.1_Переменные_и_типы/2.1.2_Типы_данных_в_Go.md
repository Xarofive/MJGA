# Типы данных в Go

Типы данных — это ключевая концепция программирования, на основе которой строятся все программы. Они определяют, какие значения может принимать переменная и какие операции можно выполнять с этими значениями.

В языке Go типы данных строго определены и контролируются на этапе компиляции. Это означает, что ты всегда точно знаешь, с какими типами данных работаешь, и не сможешь случайно смешать несовместимые типы.

## Основные категории типов данных в Go

В языке Go есть несколько категорий типов данных:
1. Простые типы (базовые типы)
2. Составные типы
3. Ссылочные типы

Сегодня мы подробно разберём только самые базовые простые типы данных. Остальные будут изучаться позже по мере усложнения курса.

> **Примечание:** Помимо простых типов в Go существуют более сложные структуры данных, такие как массивы, слайсы, карты, структуры и интерфейсы. Чтобы не перегружать тебя на начальном этапе, мы подробно рассмотрим их в следующих модулях курса, когда ты уверенно освоишь основы языка.

## Простые типы данных

К простым типам данных относятся типы, которые представляют собой одно значение, а не группу или структуру значений. Рассмотрим самые распространённые простые типы в Go:

### 1. Целочисленные типы (int)

Целые числа — это числа без дробной части.

Примеры:

```go
var age int = 25
var year int = 2025
```

Также существуют конкретные типы с фиксированной разрядностью:

| Тип   | Диапазон значений                |
|-------|----------------------------------|
| int8  | от -128 до 127                   |
| int16 | от -32768 до 32767               |
| int32 | от -2147483648 до 2147483647     |
| int64 | от -2^63 до 2^63-1               |

Обычно используется тип int, он автоматически выбирает оптимальный размер под твою платформу (32 или 64 бита).

Пример объявления переменной:

```go
var age int = 30
```

---

### 2. Числа с плавающей точкой (float32, float64)

Числа с плавающей точкой — это числа с дробной частью.
* float32 — используется для хранения небольших дробных чисел.
* float64 — используется чаще, это тип по умолчанию в Go.

```go
var height float64 = 1.83
```

---

### 3. Строки (string)

Строки в Go — это последовательности символов в двойных кавычках " ".

```go
var name string = "Иван"
var greeting string = "Hello, Kata Academy!"
```

Строки можно складывать (конкатенировать):

```go
var fullName string = "Иван" + " " + "Иванов"
```

---

### 4. Булевы значения (bool)

Булев тип может принимать всего два значения: true (истина) и false (ложь). Используется для условий и логических операций.

```go
var isActive bool = true
var isAdult bool = age >= 18
```

---

## Логические и арифметические операции с простыми типами данных

Типы данных позволяют нам выполнять с ними различные операции:

* Арифметические операции:

```go
var a int = 5
var b int = 2

var sum int = a + b // 7
var diff int = a - b // 3
var product int = a * b // 10
var quotient float64 = float64(a) / float64(b) // 2.5
```

* Логические операции:

```go
var isEqual bool = (a == b) // false
var isGreater bool = (a > b) // true
```

## Почему в Go строгая типизация?

Строгая типизация важна, так как она предотвращает ошибки, которые могли бы возникнуть из-за случайного совмещения несовместимых типов данных.

К примеру, ты не сможешь сложить строку и число напрямую:

```go
var num int = 10
var text string = " лет"

// ошибка: invalid operation: num + text (mismatched types int and string)
// var result = num + text
```

Но можно явно преобразовать типы:

```go
var result string = strconv.Itoa(num) + text // "10 лет"
```

## Как определить тип переменной?

Иногда ты можешь забыть или не знать, какого типа переменная. Go позволяет определить тип переменной следующим образом:

```go
package main

import "fmt"

func main() {
    var num = 10
    var text = "Hello"
    var isAdult = true

    fmt.Printf("Тип num: %T\n", num)
    fmt.Printf("Тип text: %T\n", text)
    fmt.Printf("Тип isAdult: %T", isAdult)
}
```

Вывод:

```
Тип num: int
Тип text: string
Тип isAdult: bool
```

## Выводы по теме лекции:
* Go имеет строгую статическую типизацию, что помогает избежать ошибок.
* Переменные определённого типа могут хранить только значения этого типа.
* Основные типы данных Go: целочисленные (int), числа с плавающей точкой (float64), строки (string), булевы (bool).
* Правильный выбор типа данных помогает сделать программу понятной и эффективной, а также облегчает её поддержку и расширение.
