<p style="margin: 0;">Освоить короткое объявление переменных и научиться &quot;видеть&quot; типы данных в Go &mdash; это поможет избежать множества ошибок в будущем.</p>

<p style="margin: 0;"><strong>Ваша задача:</strong> создать тестовые данные и вывести их так, чтобы было видно и значение, и его тип.</p>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>

<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Короткое объявление :=</h4>

<p>В Go есть умный оператор <code>:=</code>, который сам определяет тип по значению:</p>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code>age := <span style="color: #f78c6c;">30</span>      <span style="color: #546e7a;">// Go понимает: это int</span>
price := <span style="color: #f78c6c;">19.99</span>  <span style="color: #546e7a;">// Go понимает: это float64</span>
name := <span style="color: #c3e88d;">&quot;Eva&quot;</span>   <span style="color: #546e7a;">// Go понимает: это string</span></code></pre>
</div>

<div style="background: #e3f2fd; border: 1px solid #90caf9; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #1565c0; margin-top: 0;">🔍 Формат вывода для отладки</h4>

<p>Специальный паттерн <code>fmt.Printf(&quot;%v (%T)\n&quot;, x, x)</code> показывает:</p>

<ul style="margin: 10px 0;">
	<li><code>%v</code> &mdash; значение (value)</li>
	<li><code>%T</code> &mdash; тип (Type)</li>
</ul>

<p style="margin-bottom: 0;"><strong>Важно:</strong> переменная передаётся дважды &mdash; первый раз для значения, второй для типа!</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Результат программы</h2>

<p>После запуска ваша программа должна вывести:</p>

<pre style="background: #2c3e50; color: #ecf0f1; padding: 16px; border-radius: 8px; font-family: 'Courier New', monospace;">
<span style="color:#e74c3c;"><span style="font-family:Arial,Helvetica,sans-serif;">25 (int)
99.99 (float64)
Alice (string)
true (bool)</span></span></pre>

<div style="background: #fff3e0; border: 1px solid #ff9800; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #e65100; margin-top: 0;">🤔 Момент для размышления</h4>

<p>Почему Go выбрал именно <code>float64</code> для 99.99, а не <code>float32</code>? Почему для 25 выбран <code>int</code>, а не <code>int32</code> или <code>int64</code>?</p>

<p style="margin-bottom: 0;"><em>Подсказка: Go выбирает &quot;безопасные по умолчанию&quot; типы для вашей платформы.</em></p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📝 Структура программы</h2>

<pre style="background: #263238; color: #aed581; padding: 20px; border-radius: 8px; overflow-x: auto;">
<code><span style="color: #c792ea;">package</span> main

<span style="color: #c792ea;">import</span> <span style="color: #c3e88d;">&quot;fmt&quot;</span>

<span style="color: #c792ea;">func</span> <span style="color: #82aaff;">main</span>() {
    <span style="color: #546e7a;">// 1. Объявите все 4 переменные через :=</span>

    <span style="color: #546e7a;">// 2. Выведите каждую с её типом</span>

}</code></pre>

<p style="margin: 10px 0;">Все 4 объявления через <code>:=</code> и все 4 вызова <code>fmt.Printf</code> находятся <strong>внутри</strong> <code>func main()</code> &mdash; не в <code>init()</code> и не в helper-функциях.</p>

<h2 style="color: #34495e; margin-top: 28px;">✅ Чек-лист проверки</h2>

<div style="background: #f8f9fa; border-left: 4px solid #28a745; padding: 16px; margin: 16px 0;">
<ul style="margin-bottom: 0;">
	<li>☐ Программа начинается с <code>package main</code></li>
	<li>☐ Есть импорт <code>fmt</code> (и только его)</li>
	<li>☐ Функция <code>main()</code> с <code>{</code> на той же строке</li>
	<li>☐ Все 4 переменные объявлены через <code>:=</code></li>
	<li>☐ Используется <code>fmt.Printf</code>, НЕ <code>fmt.Println</code></li>
	<li>☐ Формат точно <code>&quot;%v (%T)\n&quot;</code></li>
	<li>☐ Порядок вывода: n &rarr; p &rarr; user &rarr; ok</li>
</ul>
</div>

<div style="background: #fff8e1; border-left: 4px solid #ffc107; padding: 16px; margin: 16px 0;">
<h4 style="color: #f57c00; margin-top: 0;">⚠️ Технические ограничения</h4>

<ul style="margin-bottom: 0;">
	<li>Запрещено <code>var</code> и <code>const</code></li>
	<li>Запрещены <code>println</code> и <code>fmt.Println</code></li>
	<li>Только указанные имена переменных</li>
	<li>Точные значения из таблицы</li>
	<li>Значение каждой переменной &mdash; простой литерал (например, <code>25</code>, <code>99.99</code>, <code>&quot;Alice&quot;</code>, <code>true</code>); приведение типов, арифметика или вызовы функций здесь не допускаются</li>
</ul>
</div>
