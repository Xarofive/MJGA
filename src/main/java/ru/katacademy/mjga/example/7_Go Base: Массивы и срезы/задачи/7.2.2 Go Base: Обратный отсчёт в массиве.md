<p style="margin-bottom: 10px;">Обратный отсчёт встречается везде: таймеры в приложениях, системы запуска ракет, игровые механики с ограничением времени, анимации обратного отсчёта в интерфейсах. <h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>

<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Классический цикл for с тремя компонентами</h4>

<p>Полная форма цикла for включает инициализацию, условие и пост-операцию:</p>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code class="language-go">// Структура классического for
for инициализация; условие; пост-шаг {
    // тело цикла
}

// Пример: обратный отсчёт
for i := 10; i &gt;= 0; i-- {
    fmt.Println(i)
}

// Пример: чётные числа
for n := 0; n &lt;= 20; n += 2 {
    fmt.Println(n)
}</code></pre>

<p style="margin-bottom: 0;">Инициализация выполняется один раз перед циклом, условие проверяется перед каждой итерацией, пост-шаг выполняется после тела цикла.</p>
</div>

<div style="background: #e3f2fd; border: 1px solid #90caf9; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #1565c0; margin-top: 0;">🎯 Динамические срезы и append</h4>

<p>Срезы в Go могут расти динамически с помощью функции append:</p>

<ul style="margin: 10px 0;">
	<li><strong>Создание пустого среза:</strong> <code>var numbers []int</code> или <code>numbers := []int{}</code></li>
	<li><strong>Добавление элементов:</strong> <code>numbers = append(numbers, value)</code></li>
	<li><strong>Важно:</strong> append возвращает новый срез, который нужно присвоить обратно</li>
	<li><strong>Порядок важен:</strong> элементы добавляются в конец среза в порядке вызовов append</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;"><strong>Создайте функцию для построения среза обратного отсчёта:</strong></p>

<h4 style="color: #e65100; margin-top: 16px;">🔧 Функция buildCountdown</h4>

<p><strong>Сигнатура:</strong></p>

<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>func buildCountdown(start int) []int</code></div>

<p><strong>Требования:</strong></p>

<ul style="margin: 10px 0;">
	<li>Принимает один параметр <code>start</code> &mdash; начальное значение отсчёта</li>
	<li>Возвращает срез целых чисел от <code>start</code> до 0 включительно</li>
	<li><strong>Граничный случай:</strong> если <code>start &lt; 0</code>, вернуть пустой срез <code>[]int{}</code></li>
	<li>Использовать классический цикл for с инициализацией, условием и декрементом</li>
	<li>Внутри цикла добавлять текущее значение счётчика в срез через <code>append</code></li>
	<li>Порядок элементов: от большего к меньшему (убывающая последовательность)</li>
</ul>

<h4 style="color: #e65100; margin-top: 16px;">📐 Алгоритмическая модель</h4>

<p>Результат: [start, start-1, start-2, ..., 2, 1, 0]</p>

<p>Длина результирующего среза: start + 1 (для start &ge; 0)</p>
</div>

<h4 style="color: #c62828; margin-top: 0;">⚠️ Частые ошибки и антипаттерны</h4>

<p><strong>Ошибка 1: Отсутствие защитной проверки</strong><br />
❌ Не проверять <code>start &lt; 0</code> перед циклом<br />
✅ Сначала проверить граничное условие и вернуть пустой срез</p>

<p><strong>Ошибка 2: Неправильный тип цикла</strong><br />
❌ <code>for start &gt;= 0 { ... }</code> &mdash; это while-стиль, не классический for<br />
✅ <code>for i := start; i &gt;= 0; i-- { ... }</code> &mdash; полная форма с тремя частями</p>

<p><strong>Ошибка 3: Неверное условие продолжения</strong><br />
❌ <code>i &gt; 0</code> &mdash; пропустит ноль<br />
✅ <code>i &gt;= 0</code> &mdash; включает ноль в последовательность</p>

<p><strong>Ошибка 4: Забыть присвоить результат append</strong><br />
❌ <code>append(slice, value)</code> &mdash; результат потерян<br />
✅ <code>slice = append(slice, value)</code> &mdash; сохраняем новый срез</p>

<p><strong>Ошибка 5: Инкремент вместо декремента</strong><br />
❌ <code>i++</code> &mdash; цикл будет бесконечным или не выполнится<br />
✅ <code>i--</code> &mdash; уменьшаем счётчик для обратного отсчёта</p>

<p style="margin-bottom: 0;"><strong>Ошибка 6: Отсутствие возврата для отрицательных значений</strong><br />
❌ Пропустить проверку <code>start &lt; 0</code> и уйти в бесконечный цикл<br />
✅ Вернуть пустой срез: <code>return []int{}</code> или <code>return nil</code></p>
