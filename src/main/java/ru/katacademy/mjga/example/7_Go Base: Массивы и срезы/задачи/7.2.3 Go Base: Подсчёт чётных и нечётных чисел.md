<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">📊 Подсчёт чётных и нечётных чисел</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>

<p style="margin: 0;">Научиться использовать цикл <code>for range</code> для обхода среза, применять оператор остатка от деления для определения чётности и организовывать раздельный подсчёт элементов по категориям.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">💼 Контекст и мотивация</h3>

<p style="margin-bottom: 10px;">Категоризация и подсчёт элементов &mdash; фундаментальная задача в программировании. В реальных системах это используется для статистики транзакций (успешные/неудачные), классификации пользователей (активные/неактивные), анализа данных (валидные/невалидные записи) и во множестве других сценариев обработки данных.</p>

<p style="margin: 0;"><strong>Задача:</strong> создать программу для подсчёта количества чётных и нечётных чисел в заданном срезе.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>

<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Цикл for range для срезов</h4>

<p>Цикл <code>for range</code> позволяет последовательно обходить все элементы среза:</p>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code class="language-go">nums := []int{10, 15, 20}
for index, value := range nums {
    fmt.Println(index, value)
}
// Вывод:
// 0 10
// 1 15
// 2 20

// Если индекс не нужен:
for _, value := range nums {
    fmt.Println(value)
}</code></pre>

<p style="margin-bottom: 0;">Переменная <code>value</code> получает копию каждого элемента среза по очереди.</p>
</div>

<div style="background: #e3f2fd; border: 1px solid #90caf9; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #1565c0; margin-top: 0;">🎯 Оператор остатка от деления (%)</h4>

<p>Оператор <code>%</code> возвращает остаток от деления:</p>

<ul style="margin: 10px 0;">
	<li><code>7 % 3 = 1</code> (7 = 3&times;2 + 1)</li>
	<li><code>8 % 2 = 0</code> (8 = 2&times;4 + 0)</li>
	<li><code>9 % 2 = 1</code> (9 = 2&times;4 + 1)</li>
</ul>

<p><strong>Проверка чётности:</strong> число чётное, если <code>число % 2 == 0</code></p>

<p style="margin-bottom: 0;"><strong>Инвариант:</strong> для любого целого числа <code>n</code>, выражение <code>n % 2</code> всегда равно либо 0 (чётное), либо 1 (нечётное).</p>
</div>

<div style="background: #fff3e0; border: 1px solid #ff9800; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #e65100; margin-top: 0;">📈 Счётчики для категорий</h4>

<p>При категоризации элементов используйте отдельные счётчики для каждой категории:</p>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code class="language-go">// Инициализация счётчиков до цикла
var countPositive, countNegative int

// Обновление внутри цикла
if value &gt; 0 {
    countPositive++
} else {
    countNegative++
}</code></pre>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;"><strong>Создайте программу для подсчёта чётных и нечётных чисел в срезе:</strong></p>

<h4 style="color: #e65100; margin-top: 16px;">📝 Структура программы</h4>

<ul style="margin: 10px 0;">
	<li>Начинается с <code>package main</code></li>
	<li>Импортирует только пакет <code>&quot;fmt&quot;</code> без псевдонимов</li>
	<li>Содержит только функцию <code>main()</code></li>
</ul>

<h4 style="color: #e65100; margin-top: 16px;">🔧 Требования к реализации</h4>

<p><strong>1. Объявление данных:</strong></p>

<ul style="margin: 10px 0;">
	<li>Создайте срез <code>nums</code> типа <code>[]int</code> &mdash; используйте <code>[]int{1, 2, 3, 4, 5, 6}</code> (точные значения)</li>
	<li>Объявите два счётчика для чётных и нечётных чисел (инициализированы нулями)</li>
	<li>Все объявления должны быть до цикла</li>
</ul>

<p><strong>2. Обход и подсчёт:</strong></p>

<ul style="margin: 10px 0;">
	<li>Используйте ровно один цикл <code>for range</code> для обхода <code>nums</code></li>
	<li>Формат цикла: <code>for _, value := range nums</code> (индекс не используется)</li>
	<li>Внутри цикла определите чётность через оператор <code>%</code></li>
	<li>Увеличивайте соответствующий счётчик на 1</li>
</ul>

<p><strong>3. Вывод результата:</strong></p>

<ul style="margin: 10px 0;">
	<li>После цикла используйте ровно один вызов <code>fmt.Println</code></li>
	<li>Передайте два аргумента: сначала счётчик чётных, затем нечётных</li>
	<li>Ожидаемый вывод: <code>3 3</code> (для среза {1, 2, 3, 4, 5, 6})</li>
</ul>

<h4 style="color: #e65100; margin-top: 16px;">⚠️ Ограничения</h4>

<ul style="margin: 10px 0;">
	<li>Не используйте <code>break</code> или <code>continue</code></li>
	<li>Не используйте дополнительные циклы <code>for</code></li>
	<li>Не создавайте дополнительные функции</li>
	<li>Счётчики должны увеличиваться только на 1</li>
</ul>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #c62828; margin-top: 0;">⚠️ Частые ошибки и антипаттерны</h4>

<p><strong>Ошибка 1: Неуниверсальная проверка чётности</strong><br />
❌ <code>if value % 2 == 1</code> &mdash; не работает для отрицательных чисел (в Go <code>-5 % 2 == -1</code>, а не <code>1</code>)<br />
✅ Для чётных: <code>value % 2 == 0</code>. Для нечётных: <code>value % 2 != 0</code> &mdash; универсальный способ, работающий и для отрицательных</p>

<p><strong>Ошибка 2: Объявление счётчиков внутри цикла</strong><br />
❌ Размещение <code>var evenCount int</code> внутри цикла<br />
✅ Объявите все счётчики до начала цикла</p>

<p><strong>Ошибка 3: Использование индекса вместо значения</strong><br />
❌ <code>for i, _ := range nums { if i % 2 == 0 ... }</code><br />
✅ Проверяйте чётность значения элемента, не индекса</p>

<p><strong>Ошибка 4: Множественные вызовы fmt.Println</strong><br />
❌ <code>fmt.Println(evenCount)</code> и <code>fmt.Println(oddCount)</code> отдельно<br />
✅ Один вызов: <code>fmt.Println(evenCount, oddCount)</code></p>

<p><strong>Ошибка 5: Неверный порядок счётчиков</strong><br />
❌ <code>fmt.Println(oddCount, evenCount)</code><br />
✅ Сначала чётные, потом нечётные</p>

<p style="margin-bottom: 0;"><strong>Ошибка 6: Изменение значений среза</strong><br />
❌ Использование других чисел вместо {1, 2, 3, 4, 5, 6}<br />
✅ Строго заданный срез согласно условию</p>
