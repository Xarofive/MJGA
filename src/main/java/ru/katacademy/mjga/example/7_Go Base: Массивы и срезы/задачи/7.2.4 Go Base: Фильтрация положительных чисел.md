<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🔍 Фильтрация положительных чисел</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>

<p style="margin: 0;">Научиться создавать новые срезы на основе существующих данных, используя динамическое добавление элементов через функцию <code>append</code> и условную фильтрацию.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">💼 Контекст и мотивация</h3>

<p style="margin-bottom: 10px;">Фильтрация данных &mdash; одна из самых частых операций в программировании. В системах аналитики отбирают положительные транзакции, в играх &mdash; активных игроков с положительным балансом, в датчиках &mdash; корректные показания выше нуля. Умение создавать новые коллекции без изменения исходных данных критично для безопасной работы с информацией.</p>

<p style="margin: 0;"><strong>Задача:</strong> из исходного среза чисел создать новый срез, содержащий только положительные значения.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>

<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Создание пустого среза для результата</h4>

<p>Для накопления отфильтрованных данных создают новый пустой срез:</p>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code class="language-go">// Способ 1: литерал пустого среза
result := []int{}

// Способ 2: объявление с типом
var result []int</code></pre>

<p style="margin-bottom: 0;">Оба способа создают срез нулевой длины, готовый к добавлению элементов.</p>
</div>

<div style="background: #e3f2fd; border: 1px solid #90caf9; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #1565c0; margin-top: 0;">🎯 Функция append и её особенности</h4>

<p>Функция <code>append</code> добавляет элементы в срез и возвращает обновлённый срез:</p>

<ul style="margin: 10px 0;">
	<li><strong>Важно:</strong> результат append нужно присвоить обратно: <code>slice = append(slice, value)</code></li>
	<li><strong>Первый аргумент:</strong> срез, куда добавляем</li>
	<li><strong>Остальные аргументы:</strong> добавляемые значения</li>
	<li><strong>Неправильно:</strong> <code>append(slice, value)</code> без присваивания &mdash; результат потеряется</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;"><strong>Создайте программу для фильтрации положительных чисел из заданного среза:</strong></p>

<h4 style="color: #e65100; margin-top: 16px;">📝 Обязательная структура</h4>

<ul style="margin: 10px 0;">
	<li>Начало с <code>package main</code></li>
	<li>Импорт только пакета <code>&quot;fmt&quot;</code></li>
	<li>Единственная функция <code>main()</code> без параметров</li>
</ul>

<h4 style="color: #e65100; margin-top: 16px;">🔧 Требования к реализации</h4>

<p><strong>1. Исходные данные:</strong></p>

<ul style="margin: 10px 0;">
	<li>Объявите срез <code>src</code> с точными значениями: <code>[]int{-2, -1, 0, 3, 5, -4}</code></li>
	<li>Срез объявляется один раз в начале функции</li>
	<li>Не изменяйте элементы исходного среза</li>
</ul>

<p><strong>2. Обработка данных:</strong></p>

<ul style="margin: 10px 0;">
	<li>Создайте новый пустой срез для результата (например, <code>res := []int{}</code>)</li>
	<li>Используйте цикл для обхода элементов исходного среза</li>
	<li>Для каждого элемента проверяйте условие <code>&gt; 0</code></li>
	<li>Положительные числа добавляйте в результат через <code>append</code></li>
	<li>Обязательно присваивайте результат append обратно: <code>res = append(res, ...)</code></li>
</ul>

<p><strong>3. Вывод результата:</strong></p>

<ul style="margin: 10px 0;">
	<li>Используйте одну инструкцию <code>fmt.Println</code> для вывода нового среза</li>
	<li>Выводите именно новый срез, а не исходный</li>
</ul>
</div>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #c62828; margin-top: 0;">⚠️ Частые ошибки и антипаттерны</h4>

<p><strong>Ошибка 1: Забыли присвоить результат append</strong><br />
❌ <code>append(res, value)</code> &mdash; результат теряется<br />
✅ <code>res = append(res, value)</code> &mdash; правильное использование</p>

<p><strong>Ошибка 2: Неверная проверка положительности</strong><br />
❌ <code>if value &gt;= 0</code> &mdash; включает ноль<br />
✅ <code>if value &gt; 0</code> &mdash; только положительные</p>

<p><strong>Ошибка 3: Изменение исходного среза</strong><br />
❌ <code>src[i] = 0</code> &mdash; модификация исходных данных<br />
✅ Создавайте новый срез, не трогая исходный</p>

<p><strong>Ошибка 4: Неправильные исходные данные</strong><br />
❌ <code>src := []int{1, 2, 3}</code> &mdash; другие значения<br />
✅ <code>src := []int{-2, -1, 0, 3, 5, -4}</code> &mdash; срез с положительными и отрицательными числами</p>

<p><strong>Ошибка 5: Множественный вывод</strong><br />
❌ Несколько <code>fmt.Println</code> в цикле<br />
✅ Один <code>fmt.Println</code> после формирования результата</p>

<p style="margin-bottom: 0;"><strong>Ошибка 6: Вывод исходного среза</strong><br />
❌ <code>fmt.Println(src)</code> &mdash; неверный срез<br />
✅ <code>fmt.Println(res)</code> &mdash; выводим новый срез с результатом</p>
</div>

<h4 style="color: #f57c00; margin-top: 0;">💡 Подсказка по подходу</h4>

<p>Представьте конвейер на фабрике: исходный срез &mdash; это лента с деталями разного качества. Ваша задача &mdash; создать новую коробку (пустой срез) и складывать туда только качественные детали (положительные числа). При этом исходная лента остаётся нетронутой. Функция <code>append</code> &mdash; это рука, которая берёт деталь и кладёт в коробку, но важно помнить: после добавления нужно обновить представление о том, что находится в коробке (присвоить результат обратно).</p>
