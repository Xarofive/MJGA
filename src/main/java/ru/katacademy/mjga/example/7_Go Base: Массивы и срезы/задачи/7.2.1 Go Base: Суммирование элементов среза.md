<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>

<p style="margin: 0;">Освоить работу со срезами и циклом <code>for range</code> для обхода коллекций, научиться накапливать значения с помощью оператора <code>+=</code>.</p>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">💼 Контекст и мотивация</h3>

<p style="margin-bottom: 10px;">Обработка коллекций данных &mdash; фундаментальная операция в программировании. От подсчёта общей стоимости товаров в корзине до анализа метрик в системах мониторинга &mdash; везде требуется умение эффективно обходить наборы данных и производить над ними вычисления. Цикл <code>for range</code> в Go предоставляет элегантный и безопасный способ итерации, исключающий ошибки с выходом за границы массива.</p>

<p style="margin: 0;"><strong>Задача:</strong> вычислить сумму всех элементов числового среза, используя современные идиомы Go.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>

<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Срезы и их инициализация</h4>

<p>Срез (slice) &mdash; это динамическая структура данных в Go, представляющая последовательность элементов одного типа:</p>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code class="language-go">// Создание среза с литералом
numbers := []int{1, 2, 3, 4, 5}

// Тип []int означает "срез целых чисел"
// Элементы перечисляются в фигурных скобках</code></pre>

<p style="margin-bottom: 0;">Срезы &mdash; это основной способ работы с коллекциями в Go, более гибкий чем массивы фиксированного размера.</p>
</div>

<div style="background: #e3f2fd; border: 1px solid #90caf9; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #1565c0; margin-top: 0;">🔄 Цикл for range</h4>

<p>Конструкция <code>for range</code> &mdash; идиоматичный способ итерации по элементам среза:</p>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code class="language-go">// Полная форма: получаем индекс и значение
for index, value := range slice {
    // используем index и value
}

// Только значение (индекс игнорируем через _)
for _, value := range slice {
    // используем только value
}

// Только индекс
for index := range slice {
    // используем только index
}</code></pre>

<p>Преимущества <code>for range</code>:</p>

<ul style="margin: 10px 0;">
	<li>Автоматический обход всех элементов</li>
	<li>Защита от выхода за границы</li>
	<li>Читаемость и лаконичность кода</li>
</ul>
</div>

<div style="background: #fce4ec; border: 1px solid #f48fb1; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #c2185b; margin-top: 0;">➕ Оператор накопления +=</h4>

<p>Оператор <code>+=</code> &mdash; краткая форма записи для накопления значений:</p>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code class="language-go">sum := 0        // Инициализация накопителя
sum += 5        // Эквивалентно: sum = sum + 5
sum += 10       // Теперь sum равно 15

// В контексте цикла:
total := 0
for _, num := range numbers {
    total += num  // Добавляем каждый элемент к общей сумме
}</code></pre>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;"><strong>Создайте программу для вычисления суммы элементов среза:</strong></p>

<h4 style="color: #e65100; margin-top: 16px;">📝 Структура программы</h4>

<ul style="margin: 10px 0;">
	<li>Начинается с <code>package main</code></li>
	<li>Импортирует только пакет <code>&quot;fmt&quot;</code></li>
	<li>Содержит только функцию <code>main</code> (без дополнительных функций)</li>
</ul>

<h4 style="color: #e65100; margin-top: 16px;">🔧 Требования к реализации</h4>

<p><strong>1. Объявление данных (до цикла):</strong></p>

<ul style="margin: 10px 0;">
	<li>Создайте срез <code>numbers</code> с точными значениями: <code>[]int{10, 20, 30, 40, 50}</code></li>
	<li>Объявите переменную для накопления суммы (инициализируйте нулём)</li>
	<li>Все объявления должны быть <strong>до</strong> цикла <code>for range</code></li>
</ul>

<p><strong>2. Обход среза:</strong></p>

<ul style="margin: 10px 0;">
	<li>Используйте <strong>только</strong> цикл <code>for range</code> (не классический <code>for</code> с индексом)</li>
	<li>Получайте значение элемента через <code>for range</code> (например: <code>for _, value := range numbers</code>)</li>
	<li>Не обращайтесь к элементам по индексу внутри цикла (не используйте <code>numbers[i]</code>)</li>
	<li>В программе должен быть ровно один цикл</li>
</ul>

<p><strong>3. Накопление суммы:</strong></p>

<ul style="margin: 10px 0;">
	<li>Внутри цикла накопите сумму, добавляя текущий элемент (рекомендуем <code>+=</code>)</li>
	<li>Не используйте инкремент (<code>++</code>) для накопления суммы</li>
	<li>Внутри тела цикла должен быть ровно один оператор накопления (<code>sum += value</code> или <code>sum = sum + value</code>). Посторонние переменные, инкременты или вызовы функций внутри цикла запрещены.</li>
</ul>

<p><strong>4. Вывод результата:</strong></p>

<ul style="margin: 10px 0;">
	<li>После завершения цикла выведите сумму через <code>fmt.Println</code></li>
	<li>Должен быть ровно один вызов <code>fmt.Println</code> &mdash; после цикла</li>
	<li>Не выводите промежуточные значения внутри цикла</li>
</ul>
</div>

<h4 style="color: #c62828; margin-top: 0;">⚠️ Частые ошибки и антипаттерны</h4>

<p><strong>Ошибка 1: Классический цикл вместо for range</strong><br />
❌ <code>for i := 0; i &lt; len(numbers); i++ { sum += numbers[i] }</code><br />
✅ <code>for _, value := range numbers { sum += value }</code></p>

<p><strong>Ошибка 2: Обращение по индексу внутри range</strong><br />
❌ <code>for i := range numbers { sum += numbers[i] }</code><br />
✅ <code>for _, value := range numbers { sum += value }</code></p>

<p><strong>Ошибка 3: Объявление среза внутри цикла</strong><br />
❌ Создание <code>numbers</code> после начала <code>for range</code><br />
✅ Все переменные объявляются до цикла</p>

<p><strong>Ошибка 4: Неправильные значения в срезе</strong><br />
❌ <code>[]int{1, 2, 3, 4, 5}</code> или любые другие числа<br />
✅ Точно <code>[]int{10, 20, 30, 40, 50}</code></p>

<p><strong>Ошибка 5: Вывод внутри цикла</strong><br />
❌ <code>fmt.Println(value)</code> на каждой итерации<br />
✅ Один <code>fmt.Println(sum)</code> после цикла</p>

<p style="margin-bottom: 0;"><strong>Ошибка 6: Использование инкремента для суммы</strong><br />
❌ <code>sum++</code> &mdash; прибавляет только 1, а не значение элемента<br />
✅ <code>sum += value</code> или <code>sum = sum + value</code></p>
</div>
