<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">✂️ Удаление диапазона из среза</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>

<p style="margin: 0;">Научиться удалять диапазон элементов из среза, используя идиому склейки через <code>append</code> и операцию срезания, оформив логику в виде переиспользуемой функции.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">💼 Контекст и мотивация</h3>

<p style="margin-bottom: 10px;">Удаление диапазона элементов из середины коллекции &mdash; частая операция при фильтрации данных, редактировании текста, управлении очередями задач. Идиома склейки срезов позволяет удалять целые подсегменты одной компактной строкой кода без циклов.</p>

<p style="margin: 0;"><strong>Задача:</strong> написать функцию <code>removeRange</code>, которая удаляет элементы с индекса <code>from</code> до <code>to</code> (не включая <code>to</code>) с сохранением порядка.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>

<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Идиома удаления диапазона через append и срезание</h4>

<p>Для удаления элементов из середины среза используется склейка: берём часть <strong>до</strong> удаляемого сегмента и часть <strong>после</strong> него, а затем соединяем их функцией <code>append</code>.</p>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code class="language-go">// Пример: исходный срез [A B C D E F G]
// Хотим убрать элементы с индекса 2 до 5 (C, D, E)

до   := s[:2]   // [A B]        — всё до удаляемого участка
после := s[5:]   // [F G]        — всё после удаляемого участка
result = append(до, после...)   // [A B F G]</code></pre>

<p style="margin-bottom: 0;">Подумайте, как обобщить этот приём для произвольных <code>from</code> и <code>to</code>.</p>
</div>

<div style="background: #e3f2fd; border: 1px solid #90caf9; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #1565c0; margin-top: 0;">🎯 Как работает склейка</h4>

<p>При выполнении <code>append</code> с двумя срезами:</p>

<ul style="margin: 10px 0;">
	<li><code>s[:N]</code> &mdash; берёт первые N элементов (до удаляемого участка)</li>
	<li><code>s[M:]</code> &mdash; берёт элементы начиная с индекса M до конца (после удаляемого участка)</li>
	<li>Троеточие <code>...</code> распаковывает второй срез для передачи в <code>append</code></li>
	<li>Элементы между N и M исключаются из результата</li>
</ul>
</div>

<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Обработка крайних случаев</h4>

<p>Перед выполнением склейки нужно проверить корректность границ:</p>

<ul style="margin: 10px 0;">
	<li><code>from &gt;= to</code> &mdash; ничего удалять не нужно</li>
	<li><code>from &lt; 0</code> &mdash; некорректный начальный индекс</li>
	<li><code>to &gt; len(s)</code> &mdash; выход за границы среза</li>
</ul>

<p style="margin-bottom: 0;">Во всех этих случаях функция должна вернуть копию исходного среза.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;"><strong>Реализуйте функцию удаления диапазона из среза:</strong></p>

<h4 style="color: #e65100; margin-top: 16px;">🔧 Функция removeRange</h4>

<p><strong>Сигнатура:</strong></p>

<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>func removeRange(s []int, from, to int) []int</code></div>

<p><strong>Параметры:</strong></p>

<ul style="margin: 10px 0;">
	<li><code>s []int</code> &mdash; исходный срез целых чисел</li>
	<li><code>from int</code> &mdash; начальный индекс удаляемого диапазона (включительно)</li>
	<li><code>to int</code> &mdash; конечный индекс (не включительно)</li>
</ul>

<p><strong>Возвращает:</strong> <code>[]int</code> &mdash; новый срез без элементов [from, to)</p>

<p><strong>Алгоритм:</strong></p>

<ul style="margin: 10px 0;">
	<li>Проверить корректность границ: если <code>from &gt;= to</code> или <code>from &lt; 0</code> или <code>to &gt; len(s)</code>, вернуть копию исходного среза</li>
	<li>Объединить часть среза до удаляемого сегмента с частью после него, используя <code>append</code> и операцию срезания</li>
</ul>

<p><strong>Ограничения:</strong></p>

<ul style="margin: 10px 0;">
	<li>Решение должно содержать только функцию <code>removeRange</code></li>
	<li>Используйте <code>append</code> для склейки срезов</li>
	<li>Используйте операцию срезания (<code>s[:from]</code>, <code>s[to:]</code>)</li>
	<li>Не импортируйте пакеты</li>
	<li>Не выводите данные на экран</li>
</ul>
</div>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #c62828; margin-top: 0;">⚠️ Частые ошибки и антипаттерны</h4>

<p><strong>Ошибка 1: Забыли троеточие</strong><br />
❌ <code>append(s[:from], s[to:])</code> &mdash; ошибка компиляции<br />
✅ <code>append(s[:from], s[to:]...)</code> &mdash; правильная распаковка</p>

<p><strong>Ошибка 2: Не проверили границы</strong><br />
❌ Сразу делать <code>append(s[:from], s[to:]...)</code> без проверки<br />
✅ Сначала проверить <code>from &gt;= to || from &lt; 0 || to &gt; len(s)</code></p>

<p><strong>Ошибка 3: Неправильная сигнатура</strong><br />
❌ <code>func removeRange(s []int, from int, to int)</code> &mdash; нет возврата<br />
✅ <code>func removeRange(s []int, from, to int) []int</code> &mdash; возвращает новый срез</p>

<p><strong>Ошибка 4: Использование циклов</strong><br />
❌ <code>for i := from; i &lt; to; i++ { ... }</code> &mdash; излишне сложно<br />
✅ <code>append(s[:from], s[to:]...)</code> &mdash; одна операция</p>

<p style="margin-bottom: 0;"><strong>Ошибка 5: Не используется срезание</strong><br />
❌ Ручное копирование элементов по одному<br />
✅ Используйте выражения срезов <code>s[:from]</code> и <code>s[to:]</code></p>
</div>

<h4 style="color: #f57c00; margin-top: 0;">💡 Подсказка</h4>

<p>Представьте срез как цепочку вагонов поезда. Чтобы удалить вагоны с позиции <code>from</code> до <code>to</code>, вы отцепляете их и соединяете напрямую последний вагон из <code>s[:from]</code> с первым вагоном из <code>s[to:]</code>. Операция <code>append</code> выполняет это соединение, а троеточие говорит &laquo;возьми все вагоны из второй части по одному&raquo;.</p>
