<h2 style="color: #34495e; margin-top: 30px;">1. Строки в Go: иммутабельность и внутреннее устройство</h2>

<p>Строка в Go &mdash; это неизменяемая последовательность байтов. Под капотом строка представлена структурой из указателя на данные и длины. Важнейшее свойство &mdash; иммутабельность: после создания строки её содержимое изменить нельзя. Это обеспечивает безопасность: любой код может читать строку, не опасаясь, что кто-то её изменит.</p>

<pre style="background: #f8f9fa; border: 1px solid #e9ecef; border-radius: 5px; padding: 15px; margin: 10px 0; overflow-x: auto;">
<code class="language-go">package main

import "fmt"

func main() {
    // Строка — это последовательность байтов
    s := "Hello, мир! 🌍"

    // len() возвращает количество БАЙТОВ, не символов
    fmt.Printf("Строка: %q\n", s)
    fmt.Printf("Длина в байтах: %d\n", len(s))

    // Попытка изменить строку приведёт к ошибке компиляции
    // s[0] = 'h'  // Ошибка: cannot assign to s[0]

    // Но можно создать новую строку
    s2 := "h" + s[1:]
    fmt.Printf("Новая строка: %q\n", s2)

    // Строки можно сравнивать напрямую
    s3 := "Hello, мир! 🌍"
    fmt.Printf("\ns == s3: %v\n", s == s3)

    // Пустая строка — это нулевое значение
    var empty string
    fmt.Printf("Пустая строка: %q, len=%d\n", empty, len(empty))
}</code></pre>

<h2 style="color: #34495e; margin-top: 30px;">2. Байты vs Символы: разница len() и количества рун</h2>

<p>Критически важно понимать: <code>len(s)</code> возвращает количество байтов в строке, а не количество видимых символов. В UTF-8 один символ может занимать от 1 до 4 байтов. ASCII символы занимают 1 байт, кириллица &mdash; 2 байта, большинство эмодзи &mdash; 4 байта.</p>

<pre style="background: #f8f9fa; border: 1px solid #e9ecef; border-radius: 5px; padding: 15px; margin: 10px 0; overflow-x: auto;">
<code class="language-go">package main

import (
    "fmt"
    "unicode/utf8"
)

func main() {
    // Сравним ASCII, кириллицу и эмодзи
    ascii := "Hello"
    cyrillic := "Привет"
    emoji := "👋🌍😀"
    mixed := "Go🚀"

    fmt.Println("=== Сравнение len() и RuneCountInString ===")

    fmt.Printf("ASCII:     %q\n", ascii)
    fmt.Printf("  Байтов: %d, Символов: %d\n",
        len(ascii), utf8.RuneCountInString(ascii))

    fmt.Printf("\nКириллица: %q\n", cyrillic)
    fmt.Printf("  Байтов: %d, Символов: %d\n",
        len(cyrillic), utf8.RuneCountInString(cyrillic))

    fmt.Printf("\nЭмодзи:    %q\n", emoji)
    fmt.Printf("  Байтов: %d, Символов: %d\n",
        len(emoji), utf8.RuneCountInString(emoji))

    fmt.Printf("\nСмешанный: %q\n", mixed)
    fmt.Printf("  Байтов: %d, Символов: %d\n",
        len(mixed), utf8.RuneCountInString(mixed))

    // Детальный разбор
    fmt.Println("\n=== Побайтовый разбор строки \"Go🚀\" ===")
    for i := 0; i &lt; len(mixed); i++ {
        fmt.Printf("Байт[%d]: 0x%02x (%c)\n", i, mixed[i], mixed[i])
    }
}</code></pre>

<h2 style="color: #34495e; margin-top: 30px;">3. Руны (rune): работа с Unicode символами</h2>

<p>Руна в Go &mdash; это алиас для типа int32, представляющий Unicode code point. Руны позволяют работать с отдельными символами независимо от их байтового представления. Тип rune критически важен для корректной обработки многоязычного текста.</p>

<pre style="background: #f8f9fa; border: 1px solid #e9ecef; border-radius: 5px; padding: 15px; margin: 10px 0; overflow-x: auto;">
<code class="language-go">package main

import "fmt"

func main() {
    s := "Hello, мир! 🌍"

    // Индексация даёт БАЙТ, а не символ
    fmt.Printf("s[0] = %d (байт), %c (как символ)\n", s[0], s[0])
    fmt.Printf("s[7] = %d (байт), %c (некорректно!)\n", s[7], s[7])

    // for range автоматически декодирует UTF-8 в руны
    fmt.Println("\nИтерация через for range:")
    for _, r := range s {
        fmt.Printf("%c ", r)
    }
    fmt.Println()

    // Руна — это int32
    var r rune = 'А'  // Кириллическая А
    fmt.Printf("\nРуна 'А': значение=%d, hex=U+%04X, символ=%c\n", r, r, r)

    // Литералы рун
    heart := '❤'
    fmt.Printf("Руна '❤': значение=%d, hex=U+%04X, символ=%c\n", heart, heart, heart)
}</code></pre>

<div style="background: #fff3cd; border: 1px solid #ffeaa7; border-radius: 8px; padding: 15px; margin: 20px 0;">
<h4 style="color: #856404; margin-top: 0;">⚠️ Руна &ne; видимый символ</h4>

<p>Работа с рунами покрывает большинство задач, но &laquo;видимый символ&raquo; может состоять из нескольких рун. Например, эмодзи семьи 👨&zwj;👩&zwj;👧 &mdash; это 5 рун (три символа + два невидимых соединителя ZWJ). Буквы с диакритическими знаками тоже могут состоять из нескольких рун.</p>

<p style="margin-bottom: 0;">Для базовых задач достаточно помнить: одна видимая &laquo;буква&raquo; не всегда равна одной руне. Для сложной обработки текста (работа с эмодзи, арабский, тайский) понадобятся специализированные библиотеки.</p>

<h2 style="color: #34495e; margin-top: 30px;">4. Преобразование между string и []rune</h2>

<p>Преобразование строки в срез рун позволяет работать с отдельными символами, но требует выделения памяти O(n) и времени O(n) для декодирования UTF-8. Это компромисс между удобством и производительностью.</p>

<pre style="background: #f8f9fa; border: 1px solid #e9ecef; border-radius: 5px; padding: 15px; margin: 10px 0; overflow-x: auto;">
<code class="language-go">package main

import "fmt"

func main() {
    original := "Привет, мир! 🎉"

    // Преобразование string -&gt; []rune
    runes := []rune(original)

    fmt.Printf("Исходная строка: %q\n", original)
    fmt.Printf("Байтов: %d, Рун: %d\n", len(original), len(runes))

    // Теперь можем безопасно работать с символами
    fmt.Println("\nРуны по индексам:")
    for i, r := range runes {
        fmt.Printf("  runes[%d] = %c (U+%04X)\n", i, r, r)
    }

    // Модификация отдельных символов
    runes[8] = '🌍'      // м → 🌍
    runes[12] = '!'      // пробел перед 🎉 → '!'
    runes = runes[:13]   // удаляем последнюю руну (🎉)

    // Преобразование []rune -&gt; string
    modified := string(runes)
    fmt.Printf("\nИзменённая строка: %q\n", modified)

    // Безопасное извлечение подстроки по символам
    runes2 := []rune(original)
    substring := string(runes2[0:6])
    fmt.Printf("Первые 6 символов: %q\n", substring)

    // Демонстрация стоимости преобразования
    longString := "A" + string(make([]byte, 1000)) + "B"
    fmt.Printf("\nДлинная строка: %d байт\n", len(longString))
    longRunes := []rune(longString)
    fmt.Printf("После преобразования: %d рун (занимает %d байт в памяти)\n",
        len(longRunes), len(longRunes)*4)
}</code></pre>

<h2 style="color: #34495e; margin-top: 30px;">5. Функции пакета unicode/utf8</h2>

<p>Пакет unicode/utf8 предоставляет функции для работы с UTF-8 без полного преобразования в руны. Это позволяет эффективно обрабатывать строки, минимизируя выделение памяти.</p>

<pre style="background: #f8f9fa; border: 1px solid #e9ecef; border-radius: 5px; padding: 15px; margin: 10px 0; overflow-x: auto;">
<code class="language-go">package main

import (
    "fmt"
    "unicode/utf8"
)

func main() {
    s := "Go языки 🚀"

    // RuneCountInString — подсчёт символов без выделения памяти
    count := utf8.RuneCountInString(s)
    fmt.Printf("Строка: %q\n", s)
    fmt.Printf("Символов: %d, Байтов: %d\n\n", count, len(s))

    // DecodeRuneInString — декодирование первой руны
    r, size := utf8.DecodeRuneInString(s)
    fmt.Printf("Первая руна: %c (U+%04X), занимает %d байт\n", r, r, size)

    // DecodeRuneInString для кириллицы
    r2, size2 := utf8.DecodeRuneInString(s[3:])  // Начиная с "языки"
    fmt.Printf("Руна с позиции 3: %c (U+%04X), занимает %d байт\n", r2, r2, size2)

    // ValidString — проверка корректности UTF-8
    valid := "Корректная строка"
    invalid := string([]byte{0xff, 0xfe, 0xfd})

    fmt.Printf("\n%q валидна: %v\n", valid, utf8.ValidString(valid))
    fmt.Printf("%q валидна: %v\n", invalid, utf8.ValidString(invalid))

    // RuneLen — размер руны в байтах
    fmt.Println("\nРазмеры разных рун:")
    fmt.Printf("  'A': %d байт\n", utf8.RuneLen('A'))
    fmt.Printf("  'Я': %d байт\n", utf8.RuneLen('Я'))
    fmt.Printf("  '🚀': %d байт\n", utf8.RuneLen('🚀'))
}</code></pre>

<h2 style="color: #34495e; margin-top: 30px;">6. Опасности срезания строк по байтам</h2>

<p>Срезание строки <code>s[a:b]</code> работает по байтовым индексам, что может привести к разрыву многобайтовых символов и получению невалидной UTF-8 последовательности. Это одна из самых распространённых ошибок при работе со строками.</p>

<p>Для безопасного извлечения подстроки по количеству символов напишем вспомогательную функцию:</p>

<pre style="background: #f8f9fa; border: 1px solid #e9ecef; border-radius: 5px; padding: 15px; margin: 10px 0; overflow-x: auto;">
<code class="language-go">package main

import (
    "fmt"
    "unicode/utf8"
)

// safeSubstring безопасно извлекает подстроку по количеству символов (рун)
func safeSubstring(s string, start, length int) string {
    runes := []rune(s)
    if start &gt;= len(runes) {
        return ""
    }
    end := start + length
    if end &gt; len(runes) {
        end = len(runes)
    }
    return string(runes[start:end])
}</code></pre>

    "fmt"
    "unicode"

func main() {

<code class="language-go">package main

import (
    "fmt"
    "unicode/utf8"

func truncate(s string, maxRunes int) string {
    if utf8.RuneCountInString(s) &lt;= maxRunes {
        return s
    }

    runes := []rune(s)
    if maxRunes &gt; 3 {
        return string(runes[:maxRunes-3]) + "..."
    }
    return "..."
}
