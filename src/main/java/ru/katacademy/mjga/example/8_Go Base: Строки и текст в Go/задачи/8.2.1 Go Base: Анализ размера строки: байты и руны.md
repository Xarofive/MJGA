<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🔤 Анализ размера строки: байты и руны</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>

<p style="margin: 0;">Научиться работать с многобайтовыми кодировками UTF-8, понимать разницу между физическим размером строки в байтах и количеством логических символов (рун), использовать специализированные функции для корректного подсчёта Unicode-символов.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">💼 Контекст и мотивация</h3>

<p style="margin-bottom: 10px;">В современных приложениях текст часто содержит символы различных языков, эмодзи и специальные знаки. Один визуальный символ может занимать от 1 до 4 байт в UTF-8. Это критично для баз данных с ограничениями на размер полей, систем обмена сообщениями с лимитами, обработки международного контента и оптимизации хранения данных.</p>

<p style="margin: 0;"><strong>Задача:</strong> создать функцию для одновременного подсчёта байтов и Unicode-символов в строке.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>

<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Байты vs Руны в Go</h4>

<p><strong>Байты:</strong> физический размер строки в памяти. В Go строки хранятся в UTF-8, где:</p>

<ul style="margin: 10px 0;">
	<li>ASCII-символы (латиница, цифры) &mdash; 1 байт</li>
	<li>Кириллица, арабский &mdash; обычно 2 байта</li>
	<li>Китайские иероглифы &mdash; 3 байта</li>
	<li>Эмодзи &mdash; 4 байта</li>
</ul>

<p><strong>Руны (rune):</strong> логические Unicode-символы. Одна руна = один визуальный символ, независимо от количества байт.</p>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code class="language-go">// Пример различий:
s := "Hi🚀"
len(s)                    // 6 байт (H=1, i=1, 🚀=4)
utf8.RuneCountInString(s) // 3 руны (H, i, 🚀)</code></pre>
</div>

<div style="background: #e3f2fd; border: 1px solid #90caf9; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #1565c0; margin-top: 0;">🎯 Функции для работы со строками</h4>

<ul style="margin: 10px 0;">
	<li><code>len(s)</code> &mdash; возвращает количество байт в строке</li>
	<li><code>utf8.RuneCountInString(s)</code> &mdash; возвращает количество рун (Unicode-символов)</li>
	<li>Пакет <code>unicode/utf8</code> предоставляет инструменты для работы с UTF-8</li>
</ul>

<p style="margin-bottom: 0;"><strong>Важно:</strong> простой перебор строки через индексы <code>s[i]</code> даёт байты, а не символы!</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;"><strong>Реализуйте функцию для анализа размерности строки:</strong></p>

<h4 style="color: #e65100; margin-top: 16px;">🔧 Функция ByteRuneCount</h4>

<p><strong>Сигнатура (обязательная):</strong></p>

<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>func ByteRuneCount(s string) (bytes int, runes int)</code></div>

<p><strong>Требования:</strong></p>

<ul style="margin: 10px 0;">
	<li>Принимает строку <code>s</code> любой длины и содержания</li>
	<li>Возвращает два именованных результата:
	<ul>
		<li><code>bytes</code> &mdash; количество байт в строке</li>
		<li><code>runes</code> &mdash; количество Unicode-символов (рун)</li>
	</ul>
	</li>
	<li>Обязательно использовать <code>len(s)</code> для подсчёта байт</li>
	<li>Обязательно использовать <code>utf8.RuneCountInString(s)</code> для подсчёта рун</li>
	<li>Пакет <code>unicode/utf8</code> будет подключён автоматически &mdash; вам не нужно писать строку import</li>
</ul>

<h4 style="color: #e65100; margin-top: 16px;">📝 Формат решения</h4>

<ul style="margin: 10px 0;">
	<li>Включите в ответ полную сигнатуру функции, например <code>func ByteRuneCount(s string) (bytes int, runes int) { ... }</code></li>
	<li>НЕ указывайте <code>package main</code></li>
	<li>НЕ добавляйте <code>import</code> директивы</li>
</ul>

<h4 style="color: #e65100; margin-top: 16px;">⚠️ Ограничения</h4>

<ul style="margin: 10px 0;">
	<li>Не используйте <code>fmt.Print*</code>, <code>print</code>, <code>println</code></li>
	<li>Функция должна возвращать значения через <code>return</code> без аварийной остановки программы</li>
	<li>Именованные возвращаемые значения обязательны (<code>bytes</code> и <code>runes</code>)</li>
	<li>Функция должна корректно обрабатывать строки до 100,000 символов</li>
</ul>
</div>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #c62828; margin-top: 0;">⚠️ Частые ошибки и антипаттерны</h4>

<p><strong>Ошибка 1: Ручной подсчёт рун через цикл</strong><br />
❌ Перебор через <code>for i := 0; i &lt; len(s); i++</code> считает байты, не руны<br />
✅ Используйте готовую функцию <code>utf8.RuneCountInString(s)</code></p>

<p><strong>Ошибка 2: Неправильная сигнатура</strong><br />
❌ <code>func ByteRuneCount(str string) (int, int)</code> &mdash; неименованные результаты<br />
✅ Точно: <code>(bytes int, runes int)</code></p>

<p style="margin-bottom: 0;"><strong>Ошибка 3: Добавление package или import</strong><br />
❌ Пакет и импорты подключаются автоматически<br />
✅ Включайте функцию целиком (сигнатуру и тело), без <code>package</code> и <code>import</code></p>
