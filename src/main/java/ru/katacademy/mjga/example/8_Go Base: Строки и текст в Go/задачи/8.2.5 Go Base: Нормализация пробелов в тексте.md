<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🧹 Нормализация пробелов в тексте</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>

<p style="margin: 0;">Научиться использовать функции пакета strings для элегантной нормализации пробельных символов в тексте.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">💼 Контекст и мотивация</h3>

<p style="margin-bottom: 10px;">Нормализация пробелов &mdash; базовая операция при обработке пользовательского ввода, парсинге документов и подготовке данных. Лишние пробелы, табуляции и переносы строк создают проблемы: некорректное отображение, ошибки парсинга, раздутые базы данных. В веб-формах, поисковых запросах, обработке CSV-файлов и логов критически важно приводить текст к единообразному формату.</p>

<p style="margin: 0;"><strong>Задача:</strong> реализовать функцию, которая заменяет любые последовательности пробельных символов одиночными пробелами.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>

<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Функция strings.Fields</h4>

<p>Разбивает строку на слова по любым пробельным символам:</p>

<ul style="margin: 10px 0;">
	<li>Пропускает все виды пробелов: обычные, табуляции (\t), переносы (\n, \r)</li>
	<li>Игнорирует множественные пробелы подряд</li>
	<li>Возвращает слайс строк без пустых элементов</li>
	<li>Для пустой строки или строки из одних пробелов возвращает пустой слайс</li>
</ul>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code class="language-go">words := strings.Fields("  hello   world\t\n")
// words = []string{"hello", "world"}</code></pre>
</div>

<div style="background: #e3f2fd; border: 1px solid #90caf9; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #1565c0; margin-top: 0;">🎯 Функция strings.Join</h4>

<p>Объединяет элементы слайса в строку с указанным разделителем:</p>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code class="language-go">result := strings.Join([]string{"Go", "is", "awesome"}, " ")
// result = "Go is awesome"

result = strings.Join([]string{"a", "b", "c"}, "-")
// result = "a-b-c"

result = strings.Join([]string{}, " ")
// result = "" (пустая строка для пустого слайса)</code></pre>

<p style="margin: 0;">Комбинация Fields и Join &mdash; идиоматичный способ нормализации в Go.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;"><strong>Реализуйте функцию нормализации пробелов:</strong></p>

<h4 style="color: #e65100; margin-top: 16px;">🔧 Функция NormalizeSpaces</h4>

<p><strong>Сигнатура:</strong></p>

<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>func NormalizeSpaces(s string) string</code></div>

<p><strong>Рекомендуемый подход:</strong></p>

<ul style="margin: 10px 0;">
	<li>Идиоматичное решение: <code>strings.Fields</code> + <code>strings.Join(..., &quot; &quot;)</code></li>
	<li>Допустимы и другие подходы, если результат корректен</li>
</ul>

<p><strong>Поведение:</strong></p>

<ul style="margin: 10px 0;">
	<li>Удаляет ведущие и замыкающие пробелы</li>
	<li>Заменяет любые последовательности пробельных символов одним пробелом</li>
	<li>Пробельные символы: пробел, \t, \n, \r и другие Unicode-пробелы</li>
	<li>Пустая строка или строка из одних пробелов &rarr; пустая строка</li>
</ul>

<p><strong>Ограничения:</strong></p>

<ul style="margin: 10px 0;">
	<li>Допустимые импорты: <code>strings</code> и <code>unicode</code></li>
	<li>Не использовать регулярные выражения</li>
	<li>Функция должна возвращать значение через <code>return</code> без аварийной остановки программы</li>
	<li>Не выводить данные на экран</li>
	<li>Должна быть единственной функцией в решении</li>
</ul>
</div>

<h4 style="color: #c62828; margin-top: 0;">⚠️ Частые ошибки и антипаттерны</h4>

<p><strong>Ошибка 1: Ручная обработка пробелов</strong><br />
❌ Цикл с проверкой каждого символа<br />
✅ Используйте готовые функции <code>strings.Fields</code> и <code>strings.Join</code></p>

<p><strong>Ошибка 2: Неправильный разделитель</strong><br />
❌ <code>strings.Join(words, &quot;&quot;)</code> &mdash; склеит слова без пробелов<br />
✅ <code>strings.Join(words, &quot; &quot;)</code> &mdash; ровно один пробел</p>

<p><strong>Ошибка 3: Забыли про специальные символы</strong><br />
❌ Обработка только обычных пробелов<br />
✅ <code>Fields</code> автоматически обрабатывает \t, \n, \r</p>

<p style="margin-bottom: 0;"><strong>Ошибка 4: Модификация исходной строки</strong><br />
❌ Попытка изменить параметр <code>s</code><br />
✅ Строки в Go неизменяемы, работайте с результатами функций</p>
