<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">👋 Персонализированное приветствие</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>

<p style="margin: 0;">Научиться разделять логику формирования данных и их вывода, создавая переиспользуемые функции для генерации текстовых сообщений.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">💼 Контекст и мотивация</h3>

<p style="margin-bottom: 10px;">В современных приложениях важно отделять бизнес-логику от представления данных. Функция, которая формирует приветствие, может использоваться в веб-интерфейсе, мобильном приложении, email-рассылке или чат-боте. Правильное разделение ответственности упрощает тестирование и повторное использование кода.</p>

<p style="margin: 0;"><strong>Задача:</strong> создать функцию для формирования персонализированного приветствия и использовать её в основной программе.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>

<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Функции в Go</h4>

<p>Функция &mdash; это именованный блок кода, который можно вызывать многократно:</p>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code class="language-go">func имя(параметры) возвращаемыйТип {
    // тело функции
    return значение
}
Пример:
func double(x int) int {
return x * 2
}
result := double(5)  // result = 10</code></pre>

<p style="margin-bottom: 0;">Функция может принимать параметры и возвращать значения.</p>
</div>

<div style="background: #e3f2fd; border: 1px solid #90caf9; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #1565c0; margin-top: 0;">🎯 Разделение ответственности</h4>

<p>Хорошая практика &mdash; разделять функции по назначению:</p>

<ul style="margin: 10px 0;">
	<li><strong>Функции обработки данных:</strong> формируют, вычисляют, преобразуют данные</li>
	<li><strong>Функции вывода:</strong> отображают результаты пользователю</li>
	<li><strong>Преимущества:</strong> легче тестировать, переиспользовать и модифицировать</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;"><strong>Создайте программу с двумя функциями для работы с приветствиями:</strong></p>

<h4 style="color: #e65100; margin-top: 16px;">📝 Структура программы</h4>

<ul style="margin: 10px 0;">
	<li>Начинается с <code>package main</code></li>
	<li>Импортирует только пакет <code>&quot;fmt&quot;</code></li>
	<li>Содержит ровно две функции: <code>main</code> и <code>buildGreeting</code></li>
</ul>

<h4 style="color: #e65100; margin-top: 16px;">🔧 Функция buildGreeting</h4>

<p><strong>Сигнатура:</strong></p>

<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>func buildGreeting(name string) string</code></div>

<p><strong>Требования:</strong></p>

<ul style="margin: 10px 0;">
	<li>Принимает один параметр <code>name</code> типа <code>string</code></li>
	<li>Возвращает строку-приветствие, включающую переданное имя</li>
	<li>НЕ выводит ничего на экран (без <code>fmt.Print</code>, <code>fmt.Println</code>, <code>fmt.Printf</code>)</li>
	<li>Должна содержать оператор <code>return</code></li>
</ul>

<div style="background: #e1f5fe; border: 1px solid #4fc3f7; border-radius: 8px; padding: 10px; margin: 10px 0;">
<p style="margin: 0;">💡 <strong>Обратите внимание:</strong> в отличие от следующей задачи (<code>personalGreeting</code>), здесь функция НЕ печатает сама, а <strong>возвращает строку</strong> через <code>return</code>. Вызывающий код (<code>main</code>) решает, что с ней делать.</p>
</div>

<h4 style="color: #e65100; margin-top: 16px;">⚡ Функция main</h4>

<p><strong>Сигнатура:</strong></p>

<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>func main()</code></div>

<p><strong>Требования:</strong></p>

<ul style="margin: 10px 0;">
	<li>Не принимает параметров и не возвращает значений</li>
	<li>Вызывает функцию <code>buildGreeting</code> с любым именем</li>
	<li>Выводит результат работы <code>buildGreeting</code> через <code>fmt.Println</code></li>
</ul>
</div>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #c62828; margin-top: 0;">⚠️ Частые ошибки и антипаттерны</h4>

<p><strong>Ошибка 1: Вывод внутри buildGreeting</strong><br />
❌ <code>func buildGreeting(name string) string { fmt.Println(&quot;Привет, &quot; + name) }</code><br />
✅ Функция должна только возвращать строку, не выводить её</p>

<p><strong>Ошибка 2: Отсутствие return</strong><br />
❌ <code>func buildGreeting(name string) string { greeting := &quot;Привет, &quot; + name }</code><br />
✅ Обязательно используйте <code>return greeting</code></p>

<p><strong>Ошибка 3: Игнорирование параметра name</strong><br />
❌ <code>return &quot;Привет!&quot;</code> (без использования name)<br />
✅ Возвращаемое значение должно включать переданное имя</p>

<p><strong>Ошибка 4: Лишние функции</strong><br />
❌ Создание дополнительных вспомогательных функций<br />
✅ Только две функции: main и buildGreeting</p>

<p><strong>Ошибка 5: Неправильная сигнатура</strong><br />
❌ <code>func buildGreeting(firstName, lastName string)</code><br />
✅ Ровно один параметр <code>name</code> типа <code>string</code></p>

<p style="margin-bottom: 0;"><strong>Ошибка 6: main не использует buildGreeting</strong><br />
❌ Вывод строки напрямую в main без вызова buildGreeting<br />
✅ main должна вызвать buildGreeting и вывести её результат</p>
</div>

	<li>Функция <code>buildGreeting</code> возвращает <code>string</code> и содержит <code>return</code>?</li>

<h4 style="color: #f57c00; margin-top: 0;">💡 Подсказка</h4>

<p>Представьте, что функция <code>buildGreeting</code> &mdash; это фабрика по производству приветствий. Она получает имя как сырьё и производит готовое приветствие как продукт. Функция <code>main</code> &mdash; это магазин, который заказывает приветствие на фабрике и показывает его покупателю. Фабрика не должна сама доставлять товар покупателю!</p>
