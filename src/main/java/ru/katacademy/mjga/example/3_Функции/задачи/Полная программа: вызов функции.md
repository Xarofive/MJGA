<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🎯 Полная программа: вызов функций</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>

<p style="margin: 0;">Освоить создание полноценной программы на Go: структура с package и import, создание вспомогательных функций и их вызов из main.</p>
</div>

<div style="background: #e1f5fe; border: 1px solid #4fc3f7; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #0277bd; margin-top: 0;">🔗 Связь с предыдущими модулями</h4>

<p style="margin: 0;">В модулях 1 и 2 вы познакомились со структурой Go-программы (<code>package main</code>, <code>import</code>, <code>func main()</code>) и научились работать с переменными и типами. Теперь мы добавляем к знакомой структуре новый элемент &mdash; <strong>вспомогательные функции</strong>. Программа по-прежнему начинается с <code>package main</code> и содержит <code>func main()</code>, но теперь рядом с ней появится ещё одна функция, которую <code>main</code> будет вызывать.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">💼 Контекст</h3>

<p style="margin-bottom: 10px;">Разделение логики на функции &mdash; основа чистого кода. Вспомогательные функции изолируют повторяющуюся логику, улучшают читаемость и тестируемость. Даже простой вывод баннера в отдельной функции демонстрирует принцип единственной ответственности.</p>

<p style="margin: 0;"><strong>Задача:</strong> создать программу с функцией вывода приветственного баннера.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>

<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Структура программы Go</h4>

<p>Минимальная исполняемая программа содержит:</p>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code class="language-go">package main      // объявление пакета

import "fmt"      // импорт библиотек

func main() {     // точка входа
    // код программы
}

func helper() {   // вспомогательные функции
    // логика функции
}</code></pre>

<p style="margin-bottom: 0;">Функция main &mdash; точка входа, откуда начинается выполнение.</p>
</div>

<div style="background: #e3f2fd; border: 1px solid #90caf9; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #1565c0; margin-top: 0;">🎯 Вызов функций</h4>

<p>Принципы организации кода:</p>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code class="language-go">// Вспомогательная функция
func showMessage() {
    fmt.Println("Сообщение")
}

// Главная функция
func main() {
    showMessage()  // вызов функции
}</code></pre>

<p style="margin-bottom: 0;">Функции вызываются по имени с круглыми скобками.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;"><strong>Создайте программу с функцией вывода баннера:</strong></p>

<h4 style="color: #e65100; margin-top: 16px;">📝 Структура программы</h4>

<ol style="margin: 10px 0;">
	<li><strong>Объявление пакета:</strong> <code>package main</code></li>
	<li><strong>Импорт:</strong> только пакет <code>fmt</code></li>
	<li><strong>Две функции:</strong> <code>main</code> и <code>showBanner</code></li>
</ol>

<h4 style="color: #e65100; margin-top: 16px;">🎯 Требования к функциям</h4>

<div style="background: #f5f5f5; border-left: 4px solid #2196f3; padding: 10px; margin: 10px 0;"><strong>1. Функция main:</strong>

<div style="background: #263238; color: #aed581; padding: 8px; border-radius: 4px; margin: 5px 0;"><code>func main() { // вызов showBanner() }</code></div>

<ul style="margin: 5px 0;">
	<li>Без параметров и возвращаемых значений</li>
	<li>Должна вызывать <code>showBanner()</code></li>
	<li>Больше никакого кода</li>
</ul>
</div>

<div style="background: #f5f5f5; border-left: 4px solid #4caf50; padding: 10px; margin: 10px 0;"><strong>2. Функция showBanner:</strong>

<div style="background: #263238; color: #aed581; padding: 8px; border-radius: 4px; margin: 5px 0;"><code>func showBanner() { // вывод с помощью fmt.Println }</code></div>

<ul style="margin: 5px 0;">
	<li>Без параметров и возвращаемых значений</li>
	<li>Использовать <code>fmt.Println</code> для вывода</li>
	<li>Можно вывести любой текст или число</li>
</ul>
</div>

<h4 style="color: #e65100; margin-top: 16px;">⚡ Важные ограничения</h4>

<ul style="margin: 10px 0;">
	<li>Ровно две функции: <code>main</code> и <code>showBanner</code></li>
	<li>Импортировать только <code>fmt</code>, не добавлять другие пакеты</li>
	<li>Использовать только <code>fmt.Println</code>, не <code>fmt.Print</code> или <code>fmt.Printf</code></li>
	<li>Функция <code>main</code> должна вызывать <code>showBanner</code></li>
	<li>Открывающая скобка <code>{</code> на той же строке, что и объявление функции</li>
</ul>
</div>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #c62828; margin-top: 0;">⚠️ Частые ошибки и ловушки</h4>

<p><strong>Ошибка 1: Забыли package main</strong><br />
Программа должна начинаться с <code>package main</code>.</p>

<p><strong>Ошибка 2: Неправильный импорт</strong><br />
Импортировать только <code>&quot;fmt&quot;</code>, не добавлять лишние пакеты.</p>

<p><strong>Ошибка 3: Лишние функции</strong><br />
Должно быть ровно две функции: <code>main</code> и <code>showBanner</code>.</p>

<p><strong>Ошибка 4: Забыли вызвать showBanner</strong><br />
В <code>main</code> обязательно должен быть вызов <code>showBanner()</code>.</p>

<p><strong>Ошибка 5: Неправильная функция вывода</strong><br />
Использовать только <code>fmt.Println</code>, не <code>fmt.Print</code> или <code>fmt.Printf</code>.</p>

<p style="margin-bottom: 0;"><strong>Ошибка 6: Параметры у функций</strong><br />
Обе функции должны быть без параметров: <code>func name()</code>.</p>
</div>

<h4 style="color: #f57c00; margin-top: 0;">💡 Подсказка по организации</h4>

<p>Представьте программу как спектакль: функция <code>main</code> &mdash; это режиссёр, который координирует действия. Функция <code>showBanner</code> &mdash; актёр с конкретной ролью (вывести приветствие). Режиссёр вызывает актёра на сцену в нужный момент. Даже для простого вывода разделение на функции улучшает структуру: если позже понадобится изменить баннер, достаточно отредактировать одну функцию.</p>
