<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">💰 Калькулятор скидок: функция с возвратом значения</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>

<p style="margin: 0;">Научиться создавать функции, которые принимают параметры, выполняют вычисления и возвращают результат для дальнейшего использования в программе.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">💼 Контекст</h3>

<p style="margin-bottom: 10px;">В бизнес-логике приложений часто требуются вычислительные функции: расчёт скидок, налогов, комиссий, процентов. Функции с возвращаемыми значениями позволяют инкапсулировать бизнес-логику и переиспользовать её в разных частях программы.</p>

<p style="margin: 0;"><strong>Задача:</strong> создать функцию для расчёта финальной цены товара с учётом скидки.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>

<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Функции с возвращаемым значением</h4>

<p>Функции могут не только выполнять действия, но и возвращать результат вычислений:</p>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code class="language-go">func имяФункции(параметры) типВозврата {
    // вычисления
    return результат
}
// Использование:
результат := имяФункции(аргументы)
fmt.Println(результат)</code></pre>

<p style="margin-bottom: 0;">Оператор <code>return</code> передаёт значение обратно в точку вызова функции.</p>
</div>

<div style="background: #e3f2fd; border: 1px solid #90caf9; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #1565c0; margin-top: 0;">🎯 Разделение ответственности</h4>

<p>Правильное проектирование функций:</p>

<ul style="margin: 10px 0;">
	<li><strong>Вычислительные функции:</strong> только расчёты, без побочных эффектов (вывода)</li>
	<li><strong>Чистые функции:</strong> при одинаковых входных данных всегда возвращают одинаковый результат</li>
	<li><strong>Тестируемость:</strong> функции без вывода легче тестировать</li>
	<li><strong>Композиция:</strong> результат одной функции можно передать в другую</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;"><strong>Создайте программу с функцией расчёта цены со скидкой:</strong></p>

<h4 style="color: #e65100; margin-top: 16px;">📝 Структура программы</h4>

<ul style="margin: 10px 0;">
	<li>Начинается с <code>package main</code></li>
	<li>Импортирует только пакет <code>&quot;fmt&quot;</code> (без псевдонима)</li>
	<li>Содержит ровно две функции: <code>main</code> и <code>discount</code></li>
	<li>Используйте только <code>fmt.Println</code> (не <code>fmt.Print</code> или <code>fmt.Printf</code>)</li>
</ul>

<h4 style="color: #e65100; margin-top: 16px;">🎯 Функция discount</h4>

<p><strong>Точная сигнатура (обязательно соблюдайте имена параметров):</strong></p>

<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>func discount(price int, percent int) int { // ваша реализация }</code></div>

<p><strong>Требования к реализации:</strong></p>

<ul style="margin: 10px 0;">
	<li>Параметр <code>price</code> &mdash; исходная цена товара (целое число)</li>
	<li>Параметр <code>percent</code> &mdash; процент скидки (целое число)</li>
	<li>Функция должна вычислить финальную цену после применения скидки</li>
	<li>Результат возвращается через <code>return</code></li>
	<li><strong>Важно:</strong> функция <code>discount</code> не должна ничего выводить на экран!</li>
	<li>Оба параметра должны использоваться в вычислениях</li>
</ul>

<h4 style="color: #e65100; margin-top: 16px;">⚡ Функция main</h4>

<ul style="margin: 10px 0;">
	<li>Должна вызвать функцию <code>discount</code> с любыми аргументами</li>
	<li>Должна вывести результат через <code>fmt.Println</code></li>
	<li>Вывод делается только в <code>main</code>, не в <code>discount</code></li>
</ul>
</div>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #c62828; margin-top: 0;">⚠️ Частые ошибки</h4>

<p><strong>Ошибка 1: Вывод внутри discount</strong><br />
Функция <code>discount</code> должна только вычислять и возвращать значение. Никаких выводов на экран &mdash; ни через <code>fmt.Print</code>/<code>fmt.Println</code>/<code>fmt.Printf</code>, ни через builtin <code>print</code>/<code>println</code>!</p>

<p><strong>Ошибка 2: Неверная сигнатура</strong><br />
Параметры должны называться именно <code>price</code> и <code>percent</code>, оба типа <code>int</code>.</p>

<p><strong>Ошибка 3: Забыли return</strong><br />
Функция должна явно возвращать результат через <code>return</code>.</p>

<p><strong>Ошибка 4: Не используются параметры</strong><br />
В вычислениях должны участвовать оба параметра: <code>price</code> и <code>percent</code>.</p>

<p><strong>Ошибка 5: Забыли вывести результат</strong><br />
В <code>main</code> обязательно нужно вызвать <code>discount</code> и вывести результат через <code>fmt.Println</code>.</p>

<p style="margin-bottom: 0;"><strong>Ошибка 6: Лишние функции</strong><br />
В программе должны быть только <code>main</code> и <code>discount</code>, никаких вспомогательных функций.</p>
</div>
