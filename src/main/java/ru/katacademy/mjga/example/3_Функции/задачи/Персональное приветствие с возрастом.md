<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🎂 Персональное приветствие с возрастом</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>

<p style="margin: 0;">Научиться создавать функции с несколькими параметрами разных типов и использовать их для форматированного вывода персонализированных сообщений.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">💼 Контекст и мотивация</h3>

<p style="margin-bottom: 10px;">В реальных приложениях часто требуется обрабатывать комбинации данных разных типов: текстовые (имена, адреса) и числовые (возраст, баллы, рейтинги). Умение работать с функциями, принимающими разнотипные параметры, критически важно для создания систем регистрации, CRM, социальных сетей и персонализированных сервисов.</p>

<p style="margin: 0;"><strong>Задача:</strong> создать функцию для вывода персонализированного приветствия с именем и возрастом пользователя.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>

<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Функции с несколькими параметрами</h4>

<p>Функции в Go могут принимать несколько параметров разных типов:</p>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code class="language-go">func имяФункции(параметр1 тип1, параметр2 тип2) {
    // тело функции
}
Примеры:
func showInfo(name string, age int) {
fmt.Println("Имя:", name, "Возраст:", age)
}
func calculate(price float64, quantity int) {
total := price * float64(quantity)
fmt.Println("Итого:", total)
}</code></pre>

<p style="margin-bottom: 0;">Каждый параметр должен иметь имя и тип. Порядок параметров важен при вызове функции.</p>
</div>

<div style="background: #e3f2fd; border: 1px solid #90caf9; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #1565c0; margin-top: 0;">🎯 Вывод с несколькими аргументами</h4>

<p>Функция <code>fmt.Println</code> может принимать множество аргументов:</p>

<ul style="margin: 10px 0;">
	<li><strong>Автоматические пробелы:</strong> между всеми аргументами добавляется пробел</li>
	<li><strong>Разные типы:</strong> можно комбинировать строки, числа и другие типы</li>
	<li><strong>Пример:</strong> <code>fmt.Println(&quot;Возраст:&quot;, 25, &quot;лет&quot;)</code> выведет <code>Возраст: 25 лет</code></li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;"><strong>Создайте программу с двумя функциями для персонализированного приветствия:</strong></p>

<h4 style="color: #e65100; margin-top: 16px;">📝 Структура программы</h4>

<ul style="margin: 10px 0;">
	<li>Начинается с <code>package main</code></li>
	<li>Импортирует только пакет <code>&quot;fmt&quot;</code></li>
	<li>Содержит ровно две функции: <code>main</code> и <code>personalGreeting</code></li>
</ul>

<h4 style="color: #e65100; margin-top: 16px;">🔧 Функция personalGreeting</h4>

<p><strong>Сигнатура:</strong></p>

<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>func personalGreeting(name string, age int)</code></div>

<p><strong>Требования:</strong></p>

<ul style="margin: 10px 0;">
	<li>Принимает два параметра: <code>name</code> (тип <code>string</code>) и <code>age</code> (тип <code>int</code>)</li>
	<li>Не возвращает значений</li>
	<li>Использует <code>fmt.Println</code> для вывода приветствия</li>
	<li>В выводе должны использоваться оба параметра: и имя, и возраст</li>
	<li>Используйте только <code>fmt.Println</code>, не <code>fmt.Print</code> или <code>fmt.Printf</code></li>
</ul>

<h4 style="color: #e65100; margin-top: 16px;">⚡ Функция main</h4>

<p><strong>Сигнатура:</strong></p>

<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>func main()</code></div>

<p><strong>Требования:</strong></p>

<ul style="margin: 10px 0;">
	<li>Не принимает параметров и не возвращает значений</li>
	<li>Вызывает функцию <code>personalGreeting</code> с двумя аргументами (любые имя и возраст)</li>
	<li>НЕ использует функции <code>fmt.Println</code>, <code>fmt.Print</code>, <code>fmt.Printf</code> напрямую</li>
</ul>
</div>

<h4 style="color: #c62828; margin-top: 0;">⚠️ Частые ошибки и антипаттерны</h4>

<p><strong>Ошибка 1: Неправильная сигнатура</strong><br />
❌ <code>func personalGreeting(n string, a int)</code> &mdash; неверные имена параметров<br />
✅ Точные имена: <code>name</code> и <code>age</code></p>

<p><strong>Ошибка 2: Неиспользованные параметры</strong><br />
❌ <code>fmt.Println(&quot;Привет!&quot;)</code> &mdash; не используются name и age<br />
✅ Оба параметра должны быть в выводе</p>

<p><strong>Ошибка 3: Возврат значения</strong><br />
❌ <code>func personalGreeting(name string, age int) string</code><br />
✅ Функция не должна ничего возвращать</p>

<p><strong>Ошибка 4: Неправильный вызов</strong><br />
❌ <code>personalGreeting(&quot;Анна&quot;)</code> &mdash; недостаточно аргументов<br />
✅ Всегда передавайте два аргумента</p>

<p><strong>Ошибка 5: Использование fmt.Print или fmt.Printf</strong><br />
❌ <code>fmt.Print(&quot;Привет, &quot;, name)</code><br />
✅ Используйте только <code>fmt.Println</code></p>

<p style="margin-bottom: 0;"><strong>Ошибка 6: Вывод в main</strong><br />
❌ Использование <code>fmt.Println</code> в функции main<br />
✅ Весь вывод только через <code>personalGreeting</code></p>
