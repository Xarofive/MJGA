<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">➕ Первая функция с параметрами: сумма двух чисел</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>

<p style="margin: 0;">Научиться создавать функции с параметрами и возвращаемым значением.</p>
</div>

<div style="background: #e1f5fe; border: 1px solid #4fc3f7; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #0277bd; margin-top: 0;">🔍 Важное уточнение</h4>

<p style="margin: 0;">Это первая задача, где функция принимает параметры. Ранее вы работали с функциями без аргументов &mdash; теперь добавляем входные данные.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">💼 Контекст</h3>

<p style="margin-bottom: 10px;">Параметры функций &mdash; один из ключевых инструментов программирования. Они позволяют функции работать с <strong>разными данными</strong> при каждом вызове, а не с фиксированными значениями внутри.</p>

<p style="margin: 0;"><strong>Задача:</strong> создать функцию, которая принимает два целых числа и возвращает их сумму. Это базовый шаблон, который вы будете использовать постоянно &mdash; функция получает входные данные, обрабатывает их и возвращает результат.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>

<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Параметры функций в Go</h4>

<p>Параметры указываются в скобках после имени функции. Если несколько параметров имеют один тип, его можно указать один раз:</p>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code class="language-go">// Сокращённая форма (оба параметра одного типа)
func Sum(a, b int) int {
    return a + b
}

// Полная форма (тип указан у каждого параметра)
func Sum(a int, b int) int {
    return a + b
}</code></pre>

<p style="margin-bottom: 0;">Оба варианта эквивалентны. Сокращённая форма предпочтительнее, когда параметры одного типа.</p>
</div>

<div style="background: #e3f2fd; border: 1px solid #90caf9; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #1565c0; margin-top: 0;">🎯 Как работают параметры</h4>

<p>При вызове функции значения аргументов подставляются в параметры:</p>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code class="language-go">result := Sum(2, 3)
// a получает значение 2
// b получает значение 3
// возвращается 2 + 3 = 5</code></pre>

<p style="margin-bottom: 0;">Каждый вызов может передавать разные значения &mdash; функция универсальна.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;"><strong>Создайте функцию, которая принимает два целых числа и возвращает их сумму:</strong></p>

<h4 style="color: #e65100; margin-top: 16px;">🎯 Сигнатура функции</h4>

<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>func Sum(a, b int) int</code></div>

<p>Функция принимает два параметра типа <code>int</code> и возвращает целое число &mdash; их сумму.</p>

<h4 style="color: #e65100; margin-top: 16px;">⚠️ Требования к решению</h4>

<ul style="margin: 10px 0;">
	<li>Функция принимает два параметра типа <code>int</code></li>
	<li>Функция возвращает сумму двух параметров</li>
	<li>Не используйте <code>package</code>, <code>import</code>, <code>fmt.Println</code></li>
	<li>Не используйте циклы (<code>for</code>) и <code>switch</code></li>
</ul>
</div>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #c62828; margin-top: 0;">⚠️ Частые ошибки</h4>

<p><strong>❌ Функция без параметров</strong><br />
<code>func Sum() int</code> &mdash; неправильно!<br />
<code>func Sum(a, b int) int</code> &mdash; правильно!</p>

<p><strong>❌ Забытый return</strong><br />
Функция должна <strong>возвращать</strong> результат через <code>return</code>, а не выводить его на экран.</p>

<p><strong>❌ Вывод на экран</strong><br />
<code>fmt.Println(a + b)</code> &mdash; не нужно!<br />
Используйте <code>return a + b</code>.</p>

<p style="margin-bottom: 0;"><strong>❌ Неверный тип возврата</strong><br />
Убедитесь, что функция возвращает <code>int</code>, а не <code>string</code> или другой тип.</p>
</div>

<div style="background: #e8f5e9; border: 1px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">

<ul style="margin-bottom: 0;">
	<li>Функция называется <code>Sum</code>?</li>
	<li>Сигнатура <code>func Sum(a, b int) int</code> (два параметра)?</li>
	<li>Функция возвращает сумму параметров?</li>
	<li>Работает с отрицательными числами и нулями?</li>
	<li>Нет <code>package</code> и <code>import</code>?</li>
	<li>Нет вывода на экран?</li>
</ul>
