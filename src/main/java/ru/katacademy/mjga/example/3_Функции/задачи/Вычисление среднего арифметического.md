<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">📊 Вычисление среднего арифметического</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>

<p style="margin: 0;">Научиться разбивать вычисления на промежуточные шаги с использованием переменных, что делает код более читаемым и отладочным.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">💼 Контекст и мотивация</h3>

<p style="margin-bottom: 10px;">В финансовой аналитике, научных вычислениях и обработке данных важно сохранять промежуточные результаты для отладки, логирования и повторного использования. Явное объявление переменных через <code>var</code> документирует намерения программиста и упрощает понимание алгоритма.</p>

<p style="margin: 0;"><strong>Задача:</strong> реализовать функцию для вычисления среднего арифметического двух чисел с явным сохранением промежуточных результатов.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>

<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Объявление переменных в Go</h4>

<p>В Go существует два основных способа объявления переменных:</p>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code class="language-go">// Явное объявление через var (рекомендуем для тренировки явных типов)
var total float64
var average float64 = 42.5
// Краткая форма тоже корректна
result := 3.14  // автоматически определяет тип
// Групповое объявление
var (
sum float64
count int
)</code></pre>

<p style="margin-bottom: 0;">Явное объявление делает типы очевидными и предотвращает ошибки.</p>
</div>

<div style="background: #e3f2fd; border: 1px solid #90caf9; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #1565c0; margin-top: 0;">🎯 Среднее арифметическое</h4>

<p>Среднее арифметическое двух чисел &mdash; это их сумма, делённая на количество:</p>

<ul style="margin: 10px 0;">
	<li><strong>Шаг 1:</strong> Найти сумму всех чисел</li>
	<li><strong>Шаг 2:</strong> Разделить сумму на количество чисел</li>
	<li><strong>Важно:</strong> Сохранение промежуточной суммы помогает при отладке и расширении функционала</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>

<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p style="margin-top: 0;"><strong>Создайте функцию для вычисления среднего арифметического:</strong></p>

<h4 style="color: #e65100; margin-top: 16px;">📝 Требования к функции</h4>

<ul style="margin: 10px 0;">
	<li>Имя функции: <code>AveragePair</code></li>
	<li>Не указывайте <code>package</code> и <code>import</code> &mdash; только код функции</li>
	<li>Функция должна быть единственной в решении</li>
</ul>

<h4 style="color: #e65100; margin-top: 16px;">🔧 Сигнатура функции</h4>

<div style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>func AveragePair(first, second float64) float64</code></div>

<p><strong>Параметры:</strong></p>

<ul style="margin: 10px 0;">
	<li><code>first</code> &mdash; первое число типа <code>float64</code></li>
	<li><code>second</code> &mdash; второе число типа <code>float64</code></li>
	<li>Возвращает среднее арифметическое типа <code>float64</code></li>
</ul>

<h4 style="color: #e65100; margin-top: 16px;">⚡ Рекомендуемые переменные</h4>

<p>Для читаемости рекомендуем объявить внутри функции две переменные:</p>

<h4 style="color: #e65100; margin-top: 16px;">📌 Рекомендуемый алгоритм для читаемости</h4>

<ol style="margin: 10px 0;">
	<li>Объявите переменные <code>total</code> и <code>average</code> типа <code>float64</code></li>
	<li>Вычислите сумму параметров и сохраните в <code>total</code></li>
	<li>Разделите <code>total</code> на 2 и сохраните в <code>average</code></li>
	<li>Верните <code>average</code> через <code>return</code></li>

<p style="margin: 8px 0; color: #555;"><em>Эквивалентный однострочный вариант: <code>return (first + second) / 2</code> &mdash; оба корректны.</em></p>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #c62828; margin-top: 0;">⚠️ Частые ошибки и антипаттерны</h4>

<p><strong>Ошибка 1: Потеря читаемости вычислений</strong><br />
Можно использовать и <code>:=</code>, и <code>var</code>, но важно, чтобы шаги вычисления были понятны.</p>

<p><strong>Ошибка 2: Неверное вычисление среднего</strong><br />
Проверяйте формулу: <code>(first + second) / 2</code> или эквивалентное выражение.</p>

<p><strong>Ошибка 3: Непонятные имена переменных</strong><br />
Используйте имена, которые отражают смысл вычислений (например <code>total</code>, <code>average</code>).</p>

<p><strong>Ошибка 4: Деление на 2.0 вместо 2</strong><br />
Оба варианта корректны: <code>total / 2</code> и <code>total / 2.0</code></p>

<p><strong>Ошибка 5: Вывод внутри функции</strong><br />
❌ <code>fmt.Println(average)</code><br />
✅ Возвращайте значение через <code>return</code> (например <code>return average</code>)</p>

<p style="margin-bottom: 0;"><strong>Ошибка 6: Возврат не среднего значения</strong><br />
Проверьте, что <code>return</code> отдаёт именно среднее арифметическое.</p>

<h4 style="color: #f57c00; margin-top: 0;">💡 Подсказка</h4>

<p>Представьте, что вы готовите отчёт для начальника. Вам нужно не только показать итоговое среднее, но и продемонстрировать промежуточные вычисления для прозрачности. Переменная <code>total</code> &mdash; это промежуточный результат, который можно проверить, а <code>average</code> &mdash; финальный ответ. Такой подход облегчает отладку: если результат неверный, можно проверить каждый шаг отдельно.</p>
