# 7.4 Поддержка римских чисел

В этой лекции мы научимся добавлять поддержку римских чисел в наш калькулятор. Это позволит калькулятору принимать на вход как арабские числа (1, 2, 3...), так и римские (I, II, III...), а также возвращать результат вычислений в соответствующей системе счисления.

## Зачем нужны римские числа?

Римская система счисления — это исторический способ записи чисел, основанный на буквах латинского алфавита:

| Римская цифра | Арабское число |
|---------------|-----------------|
| I             | 1               |
| II            | 2               |
| III           | 3               |
| IV            | 4               |
| V             | 5               |
| VI            | 6               |
| VII           | 7               |
| VIII          | 8               |
| IX            | 9               |
| X             | 10              |

Использование римских чисел сделает наш калькулятор более универсальным и поможет лучше понять работу с различными системами счисления.

## Шаги реализации

Реализация поддержки римских чисел требует следующих шагов:

1. Определить соответствие римских и арабских чисел.
2. Реализовать функции для конвертации из римских чисел в арабские и обратно.
3. Добавить проверку на тип используемых чисел.

## Примеры реализации функций

### Функция для преобразования римских чисел в арабские:

```go
func romanToArabic(roman string) int {
	romanMap := map[byte]int{
		'I': 1, 'V': 5, 'X': 10,
	}
	result := 0
	for i := 0; i < len(roman); i++ {
		if i < len(roman)-1 && romanMap[roman[i]] < romanMap[roman[i+1]] {
			result -= romanMap[roman[i]]
		} else {
			result += romanMap[roman[i]]
		}
	}
	return result
}
```

4. Реализовать проверку входных данных на соответствие одной системе счисления.

Пример функции, проверяющей систему счисления:

```go
func isRoman(s string) bool {
	romanNumerals := "IVXLCDM"
	for _, char := range s {
		if !strings.ContainsRune(romanNumerals, char) {
			return false
		}
	}
	return true
}
```

## Примеры использования калькулятора

- Ввод римских чисел:
```
Input: IV + II
Output: VI
```

- Программа преобразует `IV` в `4` и `II` в `2`.
- Выполняет операцию: `4 + 2 = 6`.
- Возвращает результат обратно в виде римского числа `VI`.

- В случае, если входные данные смешанные (`I + 1`), программа должна завершиться паникой:

```
panic: используются одновременно разные системы счисления
```

Таким образом, калькулятор будет полноценно поддерживать арифметику с римскими числами, проверять корректность введённых данных и обеспечивать правильный вывод результата.
