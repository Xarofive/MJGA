# Работа с вводом данных в Go

На этом этапе мы рассмотрим, как получать данные от пользователя в программе на Go.

---

## 1. Использование fmt.Scan и fmt.Scanln

Для чтения простых входных данных в Go удобно использовать функции `fmt.Scan` и `fmt.Scanln` из стандартного пакета fmt.

### Пример использования `fmt.Scan`

Функция `fmt.Scan` позволяет считывать данные, разделенные пробелами:

```go
package main

import (
    "fmt"
)

func main() {
    var a, b int

    fmt.Println("Введите два числа через пробел:")
    _, err := fmt.Scan(&a, &b)
    if err != nil {
        fmt.Println("Ошибка ввода:", err)
        return
    }

    fmt.Printf("Вы ввели: %d и %d\n", a, b)
}
```

### Пример использования `fmt.Scanln`

Функция `fmt.Scanln` считывает данные до первого переноса строки:

```go
package main

import (
    "fmt"
)

func main() {
    var input string

    fmt.Println("Введите строку данных:")
    _, err := fmt.Scanln(&input)
    if err != nil {
        fmt.Println("Ошибка ввода:", err)
        return
    }

    fmt.Println("Вы ввели:", input)
}
```

## 2. Использование bufio.Scanner

Для более гибкой обработки пользовательского ввода лучше использовать пакет `bufio`. Он позволяет работать со строками более удобно и надежно.

### Пример использования `bufio.NewScanner`

```go
package main

import (
    "bufio"
    "fmt"
    "os"
)

func main() {
    scanner := bufio.NewScanner(os.Stdin)

    fmt.Println("Введите данные:")
    if scanner.Scan() {
        input := scanner.Text()
        fmt.Println("Вы ввели:", input)
    }

    if err := scanner.Err(); err != nil {
        fmt.Println("Ошибка ввода:", err)
    }
}
```
