# 7.1 Описание проекта и требования

## Цель проекта
Создать консольный калькулятор, способный выполнять простые арифметические операции с двумя числами. Программа будет поддерживать работу как с арабскими, так и с римскими числами.

## Основные требования
- Калькулятор выполняет четыре базовые операции: сложение (+), вычитание (-), умножение (*) и деление (/).
- Данные вводятся пользователем в одну строку формата: `число оператор число`. Например: `5 + 3` или `V * II`.
- Калькулятор поддерживает числа от 1 до 10 включительно (I, II, III, IV, V, VI, VII, VIII, IX, X) для обеих систем счисления.
- При вводе арабских чисел результатом может быть любое целое число, включая отрицательные числа и ноль.
- При вводе римских чисел результат должен быть строго положительным. Если результат меньше единицы, программа должна выдать ошибку и завершить работу.
- Калькулятор должен работать только с одной системой счисления одновременно. Например, ввод `1 + V` недопустим и должен вызывать ошибку.
- Деление выполняется нацело, дробная часть отбрасывается.

## Примеры работы программы

### Корректные примеры:
```
Input: 3 + 7
Output: 10

Input: X - II
Output: VIII

Input: VI / III
Output: II

Input: IX * X
Output: XC
```

### Некорректные примеры (должны вызвать ошибку):
```
Input: 11 + 2
Ошибка: числа должны быть в диапазоне от 1 до 10

Input: I - II
Ошибка: в римской системе нет отрицательных чисел и нуля

Input: V + 1
Ошибка: используются разные системы счисления

Input: 1 + 2 + 3
Ошибка: неверный формат ввода, требуется два операнда и один оператор
```
