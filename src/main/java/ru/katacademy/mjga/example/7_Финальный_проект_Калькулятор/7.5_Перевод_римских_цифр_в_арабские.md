# Перевод римских чисел в арабские в Go

В этой лекции мы рассмотрим, как переводить римские числа в обычные арабские числа в языке программирования Go. Это необходимо для реализации нашего калькулятора, способного работать с двумя системами счисления.

## Введение в римские числа

Римская система счисления использует буквы латинского алфавита для обозначения чисел:

- I = 1
- V = 5
- X = 10
- L = 50
- C = 100
- D = 500
- M = 1000

Правила образования чисел основаны на сложении и вычитании значений букв. Например:

- II = 2 (1 + 1)
- IV = 4 (5 - 1)
- IX = 9 (10 - 1)

## Простое преобразование с помощью map

Для небольшого набора чисел (до V) можно использовать простую карту:

```go
package main

import (
	"fmt"
)

func simpleRomanToInt(s string) int {
	romanMap := map[string]int{
		"I":   1,
		"II":  2,
		"III": 3,
		"IV":  4,
		"V":   5,
	}

	return romanMap[s]
}

func main() {
	fmt.Println("I =", simpleRomanToInt("I"))     // 1
	fmt.Println("III =", simpleRomanToInt("III")) // 3
	fmt.Println("IV =", simpleRomanToInt("IV"))   // 4
}
```

## Продвинутый перевод римских чисел в арабские

Продвинутый перевод римских чисел в арабские числа можно выполнить, используя мапу в Go и правила сложения и вычитания. Основная идея заключается в том, чтобы пройтись по строке с римским числом и складывать значения символов, учитывая правила вычитания:

- Если текущий символ меньше следующего, значение текущего символа вычитается из результата.
- В противном случае, значение текущего символа прибавляется к результату.

Пример реализации:

```go
package main

import (
	"fmt"
)

func romanToInt(s string) int {
	romanMap := map[byte]int{
		'I': 1,
		'V': 5,
		'X': 10,
		'L': 50,
		'C': 100,
		'D': 500,
		'M': 1000,
	}

	result := 0
	for i := 0; i < len(s); i++ {
		if i < len(s)-1 && romanMap[s[i]] < romanMap[s[i+1]] {
			result -= romanMap[s[i]]
		} else {
			result += romanMap[s[i]]
		}
	}

	return result
}

func main() {
	fmt.Println("III =", romanToInt("III"))   // 3
	fmt.Println("IV =", romanToInt("IV"))     // 4
	fmt.Println("IX =", romanToInt("IX"))     // 9
	fmt.Println("XIII =", romanToInt("XIII")) // 13
	fmt.Println("XIX =", romanToInt("XIX"))   // 19
}
```

### Объяснение кода:

- Используется карта (`romanMap`) для хранения значений римских цифр.
- Цикл проходит по каждому символу строки.
- Если текущий символ меньше следующего, его значение вычитается, иначе прибавляется.
- В результате получаем корректное арабское число.

Таким образом, мы получаем эффективный и универсальный способ перевода римских чисел в арабские, подходящий для использования в нашем калькуляторе.
