<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🔁 Continue в разных формах цикла</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Понять, как <code>continue</code> ведёт себя в классическом <code>for</code> и в <code>while</code>, где счётчик нужно контролировать вручную.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Сценарий</h2>
<p>Вы готовите два одинаковых прохода по числам от 0 до 9, но значение 5 должно быть пропущено. Первый проход написан через <code>for</code>, второй через <code>while</code>. На выходе программа должна показать, что оба цикла дают одинаковый результат, если правильно обновлять счётчик.</p>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<ul>
	<li><code>continue</code> завершает текущую итерацию и переходит к следующей.</li>
	<li>В классическом <code>for</code> выражение <code>i++</code> всё равно выполнится после <code>continue</code>.</li>
	<li>В <code>while</code> счётчик не изменится сам: если пропустить инкремент, цикл может стать бесконечным.</li>
	<li>Одинаковая идея в разных циклах может требовать разного расположения кода.</li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p><strong>Создайте программу с двумя циклами.</strong></p>
<ul style="margin: 10px 0;">
	<li>создайте класс <code>Main</code> и метод <code>public static void main(String[] args)</code>;</li>
	<li>первый цикл должен быть классическим: <code>for (int i = 0; i &lt; 10; i++)</code>;</li>
	<li>если <code>i == 5</code>, выполните <code>continue</code>;</li>
	<li>выведите числа от 0 до 9 без 5;</li>
	<li>второй цикл должен быть <code>while</code> с переменной, объявленной до цикла;</li>
	<li>если значение равно 5, сначала увеличьте счётчик, потом выполните <code>continue</code>;</li>
	<li>после обычного вывода тоже увеличивайте счётчик;</li>
	<li>вывод должен содержать две одинаковые последовательности без разделителей и лишнего текста.</li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">🤔 Момент для размышления</h2>
<p>Почему в <code>for</code> можно написать <code>continue</code> до вывода и не думать об инкременте, а в <code>while</code> перед <code>continue</code> обязательно нужно увеличить счётчик?</p>

<h2 style="color: #34495e; margin-top: 28px;">📝 Структура программы</h2>
<ol>
	<li>Создайте первый цикл <code>for</code> от 0 до 9.</li>
	<li>Внутри него пропустите 5 через <code>continue</code>.</li>
	<li>Выведите остальные значения.</li>
	<li>Создайте переменную для второго цикла.</li>
	<li>В <code>while</code> отдельно обработайте значение 5: увеличьте счётчик и выполните <code>continue</code>.</li>
	<li>Для остальных значений выведите число и увеличьте счётчик после вывода.</li>
</ol>

<h2 style="color: #34495e; margin-top: 28px;">⚠️ Технические ограничения</h2>
<ul>
	<li>Используйте именно два цикла: один <code>for</code> и один <code>while</code>.</li>
	<li>Не заменяйте <code>continue</code> условием <code>if (i != 5)</code> вокруг вывода.</li>
	<li>Не используйте готовые структуры данных, накопление текста и дополнительные методы.</li>
	<li>Откажитесь от ручного вывода готовых чисел: задача проверяет работу циклов.</li>
	<li>Не добавляйте поясняющий текст в вывод.</li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">🐛 Отладка: если что-то не работает</h2>
<ul>
	<li>Если программа зависла, проверьте, увеличивается ли счётчик перед <code>continue</code> во втором цикле.</li>
	<li>Если выводится 5, проверьте условие <code>== 5</code> и расположение <code>continue</code>.</li>
	<li>Если строк меньше или больше 18, пересчитайте выводы в обоих циклах.</li>
	<li>Если последовательности отличаются, сравните границы циклов: обе должны проходить значения от 0 до 9.</li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">💪 Дополнительный вызов (после сдачи задачи)</h2>
<p>Добавьте третий цикл <code>do-while</code>, который выводит такую же последовательность без 5. Сравните, где в нём удобнее увеличивать счётчик.</p>

<h2 style="color: #34495e; margin-top: 28px;">🎨 Где это пригодится</h2>
<p><code>continue</code> часто используют при обработке списков данных: пропустить заблокированного пользователя, пустую строку, некорректную заявку или уже обработанную запись. Разница между <code>for</code> и <code>while</code> важна в задачах, где цикл зависит от ручного изменения состояния: чтение файла, ожидание события, обработка очереди.</p>
