<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">Задача 2.1: Объявление переменных</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Научиться объявлять переменные с явным указанием типа и выводить их значения в консоль.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">🎯 Сценарий</h3>
<p style="margin-bottom: 10px;">Вы создаёте простой профиль пользователя для учебной программы.</p>
<p style="margin: 0;"><strong>Ваша задача:</strong> сохранить данные в переменные разных типов и вывести каждое значение с новой строки.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Явное объявление переменной</h4>
<p>В Java при объявлении переменной сначала указывают тип, затем имя, затем значение.</p>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>int age = 25;
System.out.println(age);</code></pre>
<p style="margin-bottom: 0;">Тип должен подходить к значению: число без дробной части — <code>int</code>, дробное число — <code>double</code>, текст — <code>String</code>, логическое значение — <code>boolean</code>.</p>
</div>

Alice

true</code></pre>

<p style="margin-bottom: 0;">Почему значение <code>"Alice"</code> нужно записывать в кавычках, а <code>25</code> и <code>true</code> — без кавычек?</p>
</div>
