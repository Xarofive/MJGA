<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">Задача 2.5: Расчёт с налогом</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Закрепить вычисления с <code>int</code>, <code>double</code> и <code>String</code> на примере итоговой стоимости покупки.</p>
</div>

<div style="background: linear-gradient(to right, #ffeaa7, #fdcb6e); border-radius: 8px; padding: 16px; margin: 20px 0;">
<h3 style="color: #2d3436; margin-top: 0;">🎯 Сценарий</h3>
<p style="margin-bottom: 10px;">Вы делаете простой расчёт для чека: цена товара, количество, налог и итоговая сумма в валюте.</p>
<p style="margin: 0;"><strong>Ваша задача:</strong> объявить исходные переменные, выполнить расчёты и вывести итог в заданном формате.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<h4 style="color: #455a64; margin-top: 0;">Расчёт по шагам</h4>
<p>Сложную формулу проще читать, если разбить её на промежуточные переменные.</p>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;"><code>double total = price * qty;
double tax = total * taxPercent / 100;
double finalPrice = total + tax;</code></pre>
<p style="margin-bottom: 0;">Валюта хранится как <code>String</code>, потому что это текстовая подпись, а не число для вычислений.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<p><strong>Объявите исходные переменные:</strong></p>
<ul>
	<li><code>int taxPercent = 13;</code></li>
	<li><code>String currency = "RUB";</code></li>
	<li><code>double price = 1500.00;</code></li>
	<li><code>int qty = 2;</code></li>
</ul>
<p><strong>Затем выполните расчёты:</strong></p>
<ul>
	<li><code>double total = price * qty;</code></li>
	<li><code>double tax = total * taxPercent / 100;</code></li>
	<li><code>double finalPrice = total + tax;</code></li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Результат программы</h2>
<pre style="background: #2c3e50; color: #ecf0f1; padding: 16px; border-radius: 8px; overflow-x: auto;"><code>Price: 1500.0
Qty: 2
Tax: 390.0
Final: 3390.0 RUB</code></pre>

<div style="background: #fff3e0; border: 1px solid #ff9800; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #e65100; margin-top: 0;">🤔 Момент для размышления</h4>
<p style="margin-bottom: 0;">Почему налог считается от <code>total</code>, а не от <code>price</code>? Что изменится, если количество товара будет больше?</p>
</div>

    }
}</code></pre>

<ul style="margin-bottom: 0;">
