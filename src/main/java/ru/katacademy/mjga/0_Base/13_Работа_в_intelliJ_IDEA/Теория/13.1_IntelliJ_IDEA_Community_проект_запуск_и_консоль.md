<h2 style="color: #34495e; margin-top: 28px;">1. Community Edition подходит для курса</h2>

<div style="background: #e3f2fd; border: 1px solid #90caf9; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #1565c0; margin-top: 0;">💡 Что важно знать</h4>

<p style="margin-bottom: 10px;">У IntelliJ IDEA есть две основные версии: Community и Ultimate. Для этого курса достаточно Community Edition.</p>

<p style="margin: 0;">Community Edition умеет работать с обычными Java-программами, Gradle/Maven-проектами, запуском классов, тестами и Debugger. Ultimate нужна в основном для расширенной работы с enterprise-фреймворками, базами данных и веб-инструментами, но для Java Core это не требуется.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">2. Как открыть проект</h2>

<div style="background: #fff8e1; border: 1px solid #ffc107; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #f57c00; margin-top: 0;">📚 Порядок действий</h4>

<ol style="margin: 10px 0;">
	<li>Откройте IntelliJ IDEA.</li>
	<li>Нажмите <strong>Open</strong>.</li>
	<li>Выберите папку проекта, а не отдельный файл.</li>
	<li>Дождитесь, пока IDEA проиндексирует проект и загрузит Gradle/Maven, если они есть.</li>
	<li>Откройте дерево файлов слева в окне <strong>Project</strong>.</li>
</ol>

<p style="margin-bottom: 0;">Если проект открылся правильно, слева видны папки, Java-файлы и структура проекта. Если открыт только один файл, запуск и подсказки могут работать неполно.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">3. Где находится код</h2>

<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code>project
└── src
    └── main
        └── java
            └── Main.java</code></pre>
</div>

<p>В учебных Java-проектах код часто лежит в папке <code>src/main/java</code>. Внутри могут быть пакеты, а в пакетах &mdash; классы.</p>

<div style="background: #e8eaf6; border: 1px solid #3f51b5; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #283593; margin-top: 0;">🔍 Что искать в классе</h4>

<p style="margin-bottom: 10px;">Для запуска обычной программы нужен метод <code>main</code>:</p>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code class="language-java">public class Main {
    public static void main(String[] args) {
        System.out.println("Hello, Java!");
    }
}</code></pre>

<p style="margin: 0;">Рядом с классом или методом IDEA показывает зелёную кнопку запуска. Если её нет, проверьте сигнатуру метода <code>main</code>.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">4. Как запустить программу</h2>

<div style="background: #e8f5e9; border: 1px solid #4caf50; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #2e7d32; margin-top: 0;">✅ Базовый запуск</h4>

<ol style="margin: 10px 0;">
	<li>Откройте класс с методом <code>main</code>.</li>
	<li>Нажмите зелёный треугольник слева от метода или класса.</li>
	<li>Выберите <strong>Run 'Main.main()'</strong>.</li>
	<li>Посмотрите результат в нижнем окне <strong>Run</strong>.</li>
</ol>
</div>

<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code class="language-java">public class Main {
    public static void main(String[] args) {
        int price = 100;
        int discount = 15;
        int finalPrice = price - discount;

        System.out.println(finalPrice);
    }
}</code></pre>
</div>

<p>После запуска в консоли появится:</p>

<div style="background: #f0f4f8; border-radius: 8px; padding: 16px; margin: 16px 0;">
<pre style="background: white; border: 1px solid #e9ecef; border-radius: 5px; padding: 12px;"><code>85</code></pre>
</div>

<h2 style="color: #34495e; margin-top: 28px;">5. Run tool window: консоль и ввод</h2>

<div style="background: #fff3e0; border: 1px solid #ff9800; border-radius: 8px; padding: 14px; margin: 16px 0;">
<h4 style="color: #e65100; margin-top: 0;">⌨️ Если программа ждёт ввод</h4>

<p>Код со <code>Scanner</code> может остановиться и ждать, пока вы введёте данные в консоль:</p>

<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px; margin: 10px 0;">
<code class="language-java">import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        int age = scanner.nextInt();
        System.out.println("Age: " + age);
    }
}</code></pre>

<p style="margin-bottom: 0;">После запуска кликните в нижнюю консоль, введите число и нажмите Enter.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">6. Частые проблемы при запуске</h2>

<div style="background: #ffebee; border: 1px solid #ef5350; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h4 style="color: #c62828; margin-top: 0;">⚠️ Ошибки и что они означают</h4>

<p><strong>main method not found</strong><br />
IDEA не нашла метод запуска. Проверьте, что метод написан как <code>public static void main(String[] args)</code>.</p>

<p><strong>Cannot resolve symbol</strong><br />
IDEA не знает такое имя переменной, класса или метода. Часто причина &mdash; опечатка или переменная объявлена в другой области видимости.</p>

<p><strong>Class is never used</strong><br />
Это не ошибка запуска, а предупреждение. Оно означает, что класс пока нигде не используется.</p>

<p style="margin-bottom: 0;"><strong>Process finished with exit code 0</strong><br />
Программа завершилась успешно. Это нормальное сообщение, а не ошибка.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">7. Мини-чек-лист перед дебагом</h2>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 15px; margin: 20px 0;">
<ul style="margin: 10px 0;">
	<li>Проект открыт как папка проекта, а не как отдельный файл</li>
	<li>Класс содержит корректный метод <code>main</code></li>
	<li>Вы запускаете именно тот класс, который хотите проверить</li>
	<li>Консоль находится в нижнем окне <strong>Run</strong></li>
	<li>Если программа ждёт ввод, курсор стоит в консоли</li>
	<li>Если программа не останавливается, используйте кнопку <strong>Stop</strong></li>
</ul>
</div>
