<h1 style="text-align: center; color: #2c3e50; margin-top: 0;">🧹 Нормализация пробелов в тексте</h1>

<div style="background: #e8f5e9; border: 2px solid #4caf50; border-radius: 8px; padding: 14px; margin: 20px 0;">
<h3 style="color: #2e7d32; margin-top: 0;">📌 Цель задачи</h3>
<p style="margin: 0;">Удалить лишние пробелы по краям и заменить любые группы пробельных символов одним обычным пробелом.</p>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Сценарий</h2>
<p>Пользователь вводит текст с лишними пробелами, табами и переводами строк. Перед сохранением или сравнением текст нужно привести к аккуратному виду.</p>

<h2 style="color: #34495e; margin-top: 28px;">💡 Что нужно знать</h2>
<ul>
	<li><code>trim()</code> убирает пробелы по краям строки.</li>
	<li><code>split("\\s+")</code> разбивает строку по группам whitespace.</li>
	<li><code>String.join(" ", words)</code> собирает слова с одним пробелом между ними.</li>
	<li>Строка только из пробелов должна стать пустой строкой.</li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">📋 Техническое задание</h2>
<div style="background: #fff3e0; border: 2px solid #ff9800; border-radius: 8px; padding: 16px; margin: 16px 0;">
<p><strong>Реализуйте метод:</strong></p>
<pre style="background: #263238; color: #aed581; padding: 12px; border-radius: 4px;"><code class="language-java">static String normalizeSpaces(String s)</code></pre>
<ul>
	<li>Пробелы в начале и конце нужно убрать.</li>
	<li>Любую группу whitespace внутри строки заменить одним пробелом.</li>
	<li>Для строки только из пробелов вернуть пустую строку.</li>
	<li>Используйте <code>trim()</code>, <code>split("\\s+")</code> и <code>String.join(" ", words)</code>.</li>
	<li>Решение должно содержать только метод без вывода в консоль.</li>
</ul>
</div>

<h2 style="color: #34495e; margin-top: 28px;">🎯 Результат программы</h2>
<ul>
	<li><code>normalizeSpaces("  hello   world  ")</code> → <code>"hello world"</code></li>
	<li><code>normalizeSpaces("a\tb\nc")</code> → <code>"a b c"</code></li>
	<li><code>normalizeSpaces("     ")</code> → <code>""</code></li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">🤔 Момент для размышления</h2>
<p>Почему перед <code>split("\\s+")</code> удобно сначала применить <code>trim()</code>?</p>

<h2 style="color: #34495e; margin-top: 28px;">📝 Структура программы</h2>
<ol>
	<li>Уберите пробелы по краям через <code>trim()</code>.</li>
	<li>Если строка стала пустой, верните <code>""</code>.</li>
	<li>Разбейте строку по группам whitespace.</li>
	<li>Соберите слова через один обычный пробел.</li>
	<li>Верните результат.</li>
</ol>

<h2 style="color: #34495e; margin-top: 28px;">⚠️ Технические ограничения</h2>
<ul>
	<li>Используйте именно инструменты из задания: <code>trim</code>, <code>split</code>, <code>String.join</code>.</li>
	<li>Не оставляйте пробелы по краям результата.</li>
	<li>Не используйте вывод в консоль.</li>
	<li>Откажитесь от ручной замены только двойных пробелов: whitespace бывает разным.</li>
</ul>

<h2 style="color: #34495e; margin-top: 28px;">🐛 Отладка: если что-то не работает</h2>
<ul>
	<li>Если у результата есть пробел в начале, проверьте <code>trim()</code>.</li>
	<li>Если табы не заменяются, проверьте регулярное выражение <code>\\s+</code>.</li>
	<li>Если строка из пробелов даёт ошибку или пробел, отдельно обработайте пустой результат после <code>trim()</code>.</li>
</ul>
